> **编译日期**: 2026-06-02
> **规则条数**: 22
> **置信度说明**: EXTRACTED=ACSM明确陈述 | INFERRED=综合推导

## 1. FITT-VP框架 (ACSM核心)

### RULE-PRE-001: FITT-VP — 运动处方的6要素
- **类型**: book-thesis
- **来源**: ACSM Guidelines 11th Ed, Ch.5
- **置信度**: EXTRACTED

任何运动处方必须定义以下6个要素：

| 要素 | 英文 | 含义 |
|------|------|------|
| F - Frequency | 频率 | 每周几天？ |
| I - Intensity | 强度 | 多努力？（%VO₂max、%HR、%1RM、RPE） |
| T - Time | 时间 | 每次多久？ |
| T - Type | 类型 | 什么运动？ |
| V - Volume | 总量 | 每周总运动量（MET-min/wk或kcal/wk） |
| P - Progression | 进阶 | 如何逐步增加？ |

## 2. 有氧运动处方 (Aerobic Exercise Prescription)

### RULE-PRE-002: 健康成年人的有氧运动处方标准
- **类型**: default
- **来源**: ACSM Ch.5 + 2018 U.S. Physical Activity Guidelines
- **置信度**: EXTRACTED

| FITT | 推荐 |
|------|------|
| **频率** | ≥3-5天/周（中等强度）；≥3天/周（较高强度） |
| **强度** | 中等(40-59% HRR) 和/或 较高(60-89% HRR) |
| **时间** | 30-60 min/天(中等)；20-60 min/天(较高) |
| **类型** | 大肌群参与的持续性有节律运动(步行/跑步/骑车/游泳) |
| **总量** | ≥500-1,000 MET-min/wk 或 ≥150 min/wk中等或≥75 min/wk较高强度 |
| **进阶** | "低起点，缓慢加" → 先延长时间(每1-2周+5-10min)，再加强度 |

**最低有效量**: 150 min/wk中等强度或75 min/wk较高强度
**最佳效益量**: 300 min/wk中等或150 min/wk较高强度

### RULE-PRE-003: 有氧运动强度量化方法
- **类型**: default
- **来源**: ACSM Ch.5
- **置信度**: EXTRACTED

| 强度级别 | %HRR/VO₂R | %HRmax | RPE (1-10) | RPE (6-20) | METs |
|---------|-----------|--------|------------|------------|------|
| 极轻 | <30 | <57 | <2 | <9 | <3 |
| 轻 | 30-39 | 57-63 | 2-3 | 9-11 | 3-4 |
| **中等** | **40-59** | **64-76** | **4-5** | **12-13** | **3-5.9** |
| 较高 | 60-89 | 77-95 | 6-8 | 14-17 | ≥6 |
| 最大 | ≥90 | ≥96 | ≥9 | ≥18 | — |

**公式选择**:
- %HRR (Karvonen公式): THR = (HRmax - HRrest) × %Intensity + HRrest
- %HRmax: THR = HRmax × %Intensity
- HRmax估算: 208 - 0.7×age (Tanaka) 优于 220-age
- %HRR与%VO₂R高度对应 → 是最推荐的处方方法

### RULE-PRE-004: 运动总量 — 剂量-反应关系
- **类型**: trigger
- **来源**: ACSM Ch.5
- **置信度**: EXTRACTED

| 周运动量 | 健康效益 |
|---------|---------|
| <500 MET-min/wk | 极低量→久坐不动的部分改善 |
| **500-1,000 MET-min/wk** | **显著降低CVD发病率和死亡率** |
| >1,000 MET-min/wk | 额外效益递减但持续增加 |

**换算实例**:
- 7 METs × 30min × 3天/wk = 630 MET-min/wk (满足最低推荐)
- 达标方案: 每周150min中等强度(≈500-1,000 MET-min) 或 每周1,000 kcal消耗
- 每日步数: 最低7,000-8,000步，其中至少3,000步快走(≥100步/min)

## 3. 抗阻训练处方 (Resistance Exercise Prescription)

### RULE-PRE-005: 健康成人抗阻训练处方 (ACSM标准)
- **类型**: default
- **来源**: ACSM Ch.5, Table 5.5
- **置信度**: EXTRACTED

| FITT | ACSM推荐 |
|------|---------|
| **频率** | 每肌群≥2天/周（非连续日）；全身训练2-3x/wk |
| **强度** | 初学者60-70% 1RM (8-12 RM)；有经验者依目标而定 |
| **组数** | 初学者每肌群1-3组；有经验者≥3组 |
| **方式** | 多关节+单关节+核心；包含主动肌和拮抗肌 |
| **休息** | 取决于目标（力量2-5min，增肌60s-3min） |
| **进阶** | 先加次数→再加负荷→再加组数→再加频率 |

**最低有效量**: 每肌群≥1组，8-12次，每周2天
**周训练量-反应关系**:
- 低量: 每肌群<5组/周 → 维持/初学改善
- 中量: 每肌群5-9组/周 → 显著改善
- 高量: 每肌群>10组/周 → 最大化肥大和力量
- 上限未知（过量→停滞/退步）

## 4. 柔韧性训练处方 (Flexibility Prescription)

### RULE-PRE-006: 柔韧性训练FITT (ACSM标准)
- **类型**: default
- **来源**: ACSM Ch.5, Table 5.6
- **置信度**: EXTRACTED

| FITT | 推荐 |
|------|------|
| **频率** | ≥2-3天/周（每天效果最佳） |
| **强度** | 拉伸至紧绷或轻微不适（非疼痛） |
| **时间** | 静力性10-30s/次（老年人30-60s）；PNF: 3-6s收缩+10-30s辅助拉伸 |
| **重复** | 每动作2-4次，累计拉伸≥60-90s |
| **方式** | 静力性/动力性/弹振/PNF均有效 |

**关键使用规则**:
- 运动前: **动力性拉伸**（≤30s）作为热身 → 不影响后续表现
- 运动后或单独: **静力性拉伸**（10-30s） → 提高ROM
- ⚠️ 运动前长时间静力性拉伸(>60s) → 降低爆发力/冲刺表现
- 柔韧性提高需要3-4周规律训练

## 5. 风险分层与运动前筛查 (Risk Stratification)

### RULE-PRE-007: 运动前筛查 — 何时需要医学许可
- **类型**: safety-gate
- **来源**: ACSM Ch.2
- **置信度**: EXTRACTED

使用 **PAR-Q+** 或 **ACSM筛查流程**:

**绿灯 — 可直接开始中等强度运动**:
- 无心血管/代谢/肾脏疾病症状或诊断
- 无症状（胸痛、头晕、呼吸困难、心悸）

**黄灯 — 建议中等强度，较高强度需医学许可**:
- 已知但已控制的CVD危险因素（高血压服药后控制良好）
- 糖尿病前期、血脂边缘升高

**红灯 — 任何强度都需医学许可**:
- 已知CVD、糖尿病（有并发症）、肾脏疾病
- 运动中胸痛/头晕/晕厥史
- SBP≥160或DBP≥100 mmHg

### RULE-PRE-008: ACSM心血管风险因素
- **类型**: safety-gate
- **来源**: ACSM Ch.2
- **置信度**: EXTRACTED

| 风险因素 | 阳性阈值 |
|---------|---------|
| 年龄 | 男≥45岁，女≥55岁 |
| 家族史 | 一级亲属 MI/CABG/猝死: 男<55岁，女<65岁 |
| 吸烟 | 当前吸烟或戒烟<6个月 |
| 久坐 | 不满足最低体力活动推荐(≥30min×3天/wk) |
| 肥胖 | BMI≥30kg/m² 或 腰围男>102cm/女>88cm |
| 高血压 | SBP≥130 或 DBP≥80 mmHg 或 服降压药 |
| 高血脂 | LDL≥130mg/dL 或 HDL<40mg/dL 或 服降脂药 |
| 高血糖 | FPG≥100mg/dL 或 HbA1c≥5.7% |

**行动**: ≥2个风险因素 = 较高强度运动前建议医学评估

## 6. 特殊人群运动处方 (Special Populations)

### RULE-PRE-009: 高血压患者运动处方
- **类型**: safety-gate
- **来源**: ACSM Ch.9
- **置信度**: EXTRACTED

| 要素 | 建议 |
|------|------|
| **频率** | 有氧大多数天(≥5天/wk)；抗阻2-3天/wk(非连续) |
| **强度** | 有氧中等(40-59% HRR)；抗阻60-70% 1RM |
| **时间** | 有氧≥30min/天；抗阻8-12次/组 |
| **类型** | 有氧(步行/骑车/游泳)；抗阻(大肌群多关节) |

**安全红线**:
- 🛑 运动前SBP≥160 或 DBP≥100 → 先控血压再运动
- ⚠️ 避免Valsalva（憋气）→ 全程自然呼吸
- ⚠️ β-blocker使用者：HR反应被药物限制，用RPE代替HR评估
- ✅ 等长收缩不过度（握力/平板支撑≤10s收缩）
- 一次有氧运动后降压效应持续22h(Post-Exercise Hypotension)

### RULE-PRE-010: 2型糖尿病患者运动处方
- **类型**: safety-gate
- **来源**: ACSM Ch.9
- **置信度**: EXTRACTED

| 要素 | 建议 |
|------|------|
| **频率** | 有氧≥3-5天/wk（不超过连续2天不运动）；抗阻2-3天/wk |
| **强度** | 有氧中等到较高(40-75% HRR)；抗阻60-85% 1RM |
| **时间** | 有氧每周≥150min中等或75min较高 |
| **类型** | 有氧(大肌群持续性)；抗阻(大肌群)；柔韧性 |

**安全要求**:
- 🔴 运动前测血糖:
  - <100 mg/dL (5.6 mmol/L): 吃15-30g碳水再运动
  - >250 mg/dL (13.9 mmol/L) + 酮体阳性: 推迟运动
  - 100-250 mg/dL: 安全运动
- ⚠️ 注射胰岛素者：避免在胰岛素峰值时间运动，或减少胰岛素剂量
- ⚠️ 运动后12h内低血糖风险(迟发性低血糖)
- ✅ 每周至少3天运动，不过连续2天不运动（胰岛素敏感性窗口约48h）
- 有神经病变者：改为非负重运动(游泳/骑车) + 每天足部检查

### RULE-PRE-011: 肥胖人群运动处方
- **类型**: decision-changing
- **来源**: ACSM Ch.10
- **置信度**: EXTRACTED

| 目标 | 运动量 |
|------|--------|
| **维持健康** | ≥150 min/wk 中等强度 |
| **减重(临床意义)** | 200-300 min/wk 中等至较高强度 |
| **减重后维持** | 250-300 min/wk |

- 先追求时长（频率和每次持续时间），再追求强度
- 关节保护: 优先非负重有氧（游泳/骑车→步行→慢跑）
- NEAT（非运动性产热）增加: 每日多站、多走、减少坐
- 减重>5%需饮食+运动结合，单独运动减重有限

### RULE-PRE-012: 骨质疏松症运动处方
- **类型**: safety-gate
- **来源**: ACSM Ch.10
- **置信度**: EXTRACTED

| 要素 | 建议 |
|------|------|
| **有氧** | 负重活动（步行/慢跑/跳舞/爬楼梯）≥3-5天/wk, 中等强度, 30-60min |
| **抗阻** | 2-3天/wk, 中到高强度(70-85% 1RM), 8-12次, 重点: 脊柱伸肌/髋伸肌 |
| **柔韧性** | ≥2-3天/wk, 避免脊柱极限屈曲/旋转 |

**禁止**:
- ❌ 爆发性/高冲击动作（跳深等）
- ❌ 脊柱屈曲+旋转组合（如卷腹+扭转）
- ❌ 腰椎极限屈曲动作（仰卧起坐、坐姿体前屈）

### RULE-PRE-013: 慢性腰痛运动处方
- **类型**: trigger
- **来源**: ACSM Ch.10
- **置信度**: EXTRACTED

| 疼痛类型 | 运动方案 |
|---------|---------|
| 全身性疼痛(多部位) | 渐进低强度有氧运动(步行/水中运动) |
| 非全身性疼痛 | 中-高强度有氧运动 |
| 特异性下背痛(盘源性/小关节) | 避免脊柱极限屈伸+旋转；McGill Big 3（鸟狗/侧桥/卷腹改良） |

- ✅ 绝对卧床休息 > 2天反而延迟恢复 → 保持轻度活动
- ✅ 核心稳定训练（死虫式ex-030、鸟狗式ex-029、平板支撑ex-028）
- 慢性疼痛与恐惧-回避(Fear-Avoidance)心理模式密切相关 → 教育+渐进暴露

### RULE-PRE-014: 孕期与产后运动
- **类型**: safety-gate
- **来源**: ACSM Ch.6
- **置信度**: EXTRACTED

**孕期（无禁忌症者）**:
- 频率: ≥3天/wk（最好每天30min）
- 强度: 中等(RPE 12-14)，"谈话测试"可通过
- 类型: 步行/游泳/固定自行车/孕期瑜伽
- 🛑 禁止: 仰卧运动(>16周→子宫压迫下腔静脉)、接触性运动、热瑜伽、潜水
- 🛑 终止信号: 阴道出血、腹痛、羊水漏、胸痛、头晕

**产后**:
- 顺产: 无并发症者产后数天可逐步恢复轻活动
- 剖腹产: 等术后6-8周伤口愈合
- 逐步恢复抗阻训练，关注盆底肌康复(Kegel)

## 7. 运动处方进阶策略 (Progression)

### RULE-PRE-015: "Start Low, Go Slow" 原则
- **类型**: default
- **来源**: ACSM Ch.5
- **置信度**: EXTRACTED

**三阶段进阶模型**:

| 阶段 | 时长 | 策略 |
|------|------|------|
| **初始适应**(Initial) | 1-4周 | 低→中等强度，追求规律性而非强度 |
| **改善**(Improvement) | 4-8月 | 每1-2周增加一个FITT变量（时间→强度→频率） |
| **维持**(Maintenance) | 持续 | 一旦达到目标，保持至少150min/wk中等强度 |

**进阶顺序（优先）**:
1. 先延长每次运动时间（每1-2周+5-10min/次）
2. 再增加频率（每周+1天）
3. 再提高强度
4. 每次只改变**一个**FITT变量 → 避免过度损伤

## 8. 运动与心理健康

### RULE-PRE-016: 运动与抑郁/焦虑
- **类型**: trigger
- **来源**: ACSM Ch.5, 研究综述
- **置信度**: INFERRED

- ≥150 min/wk 中等强度有氧 → 抑郁症状减少20-30%
- 抗阻训练同样有效（≥2x/wk, 8-12RM）
- 单次运动即可产生急性抗焦虑效应（持续2-4h）
- 最佳: 有氧+抗阻结合，≥3x/wk，每次≥30min
- 户外运动对心理健康额外加分（"绿色运动"效应）
