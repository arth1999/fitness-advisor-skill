# 安全边界与就医指引

> ⚠️ 待填充内容需来自运动医学教材和临床实践指南，以下为框架骨架。

## 必须建议立即就医的危险信号
- [ ] 胸痛/胸闷（运动诱发或加重）
- [ ] 眩晕/晕厥（运动中出现）
- [ ] 关节急性肿胀/变形/无法承重
- [ ] 运动后持续加重的疼痛（非酸痛）
- [ ] 呼吸困难（与运动强度不匹配）
- [ ] 心悸/心律不齐（运动中或运动后持续）
- [ ] 单侧肢体麻木/无力

## 建议暂停训练并观察的情形
- [ ] 轻度肌肉拉伤（局部压痛、无肿胀）
- [ ] 关节轻微不适（非急性）
- [ ] 持续超过72小时的 DOMS
- [ ] 训练意愿持续下降 + 静息心率升高
- [ ] 连续两周力量不增长反而下降
