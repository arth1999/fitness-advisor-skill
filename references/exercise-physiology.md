## 1. 能量系统规则 (Energy Systems)

### RULE-PHY-001: 运动持续时间决定主导供能系统
- **类型**: decision-changing
- **来源**: 运动生理学 第2章 p.44-59 + 表2-6
- **置信度**: EXTRACTED
- **关联**: [[sports-nutrition#fuel-timing]]

**条件**: IF 已知运动持续时间
**建议**: THEN 根据下表确定主导供能系统及训练重点

| 运动持续时间 | 无氧供能占比 | 有氧供能占比 | 主导系统 | 典型运动 |
|-------------|-------------|-------------|---------|---------|
| 0-10s | 94% | 6% | ATP-CP（磷酸原） | 短跑、跳跃、举重 |
| 10-30s | 88-73% | 12-27% | ATP-CP + 快速糖酵解 | 200m、50m游泳 |
| 30-90s | 55-44% | 45-56% | 糖酵解 + 有氧过渡 | 400m、100m游泳 |
| 90s-3min | 37-27% | 63-73% | 有氧 + 糖酵解 | 800m、1500m |
| 3min+ | 21%→ | 79%→ | 有氧氧化 | 长跑、自行车、游泳 |

**原理**: 各能量系统输出功率不同（磷酸原 $56 J·kg^{-1}·s^{-1}$ > 糖酵解 $29.3 J·kg^{-1}·s^{-1}$ > 有氧 $15 J·kg^{-1}·s^{-1}$），运动强度随持续时间递减，供能系统按功率从高到低依次调用。

### RULE-PHY-002: 磷酸原系统容量与恢复
- **类型**: trigger
- **来源**: 运动生理学 第2章 p.44-46
- **置信度**: EXTRACTED
- **关联**: [[strength-training#rest-periods]]

**条件**: IF 训练目标 = 爆发力/最大力量（1-5RM，<10s/组）
**建议**: THEN
- 组间休息 ≥ 3min（磷酸原恢复半衰期约20-30s，完全恢复需3-5min）
- ATP-CP系统最大容量 ≈ 420 J/kg体重，仅支持约7.5s全力运动
- 运动后CP再合成依赖有氧代谢，保持低强度活动（非完全静坐）促进恢复

**原理**: 安静时肌肉CP含量约28 mmol/kg（约为ATP的3-5倍），全力运动时CP在10s内下降75-85%。CP再合成需要ATP提供能量，而ATP来自有氧代谢。

---

### RULE-PHY-003: 糖酵解系统限制因素
- **类型**: decision-changing
- **来源**: 运动生理学 第2章 p.47-48
- **置信度**: EXTRACTED

**条件**: IF 运动强度 > 无氧阈 AND 持续时间 10-90s
**建议**: THEN
- 糖酵解供能约33s达到极限（最大容量962 J/kg ÷ 最大功率29.3 J/(kg·s) = 33s）
- 乳酸累积是主要限制因素——pH下降抑制糖酵解关键酶（PFK）
- 血乳酸正常安静值 < 2 mmol/L，全力400m跑后可达20-25 mmol/L
- 缓冲能力可训练：通过重复冲刺训练(10-30s)提高肌肉缓冲容量

**原理**: 1mol葡萄糖经糖酵解净产2mol ATP（仅为有氧氧化的1/18-1/19），但输出功率是有氧的2倍。乳酸生成速率超过清除速率时→pH↓→PFK酶活性↓→ATP合成↓→疲劳。

---

### RULE-PHY-004: 有氧氧化系统——底物利用顺序
- **类型**: trigger
- **来源**: 运动生理学 第2章 p.49-50
- **置信度**: EXTRACTED
- **关联**: [[sports-nutrition#carb-fat-utilization]]

**条件**: IF 运动持续 > 2min
**建议**: THEN 底物利用按以下顺序：
1. 首先动用肌糖原（直接可用）
2. 糖原储备下降后→肝糖原分解→血糖
3. 血糖下降→脂肪动员（β-氧化）
4. 仅长时间饥饿/超大运动量时→蛋白质分解供能

**原理**: 糖（4 kcal/g）→脂肪（9 kcal/g）→蛋白质（4 kcal/g）。脂肪能量密度最高但氧化速率最慢（仅0.24 mmol ATP/kg干肌/s，约糖的1/2），因此"脂肪在糖的火焰中燃烧"——糖耗竭后即使脂肪充足也会疲劳。

---

### RULE-PHY-005: 运动节省化（Exercise Economy）
- **类型**: decision-changing
- **来源**: 运动生理学 第2章 p.58
- **置信度**: EXTRACTED
- **关联**: [[exercise-prescription#aerobic-training]]

**条件**: IF 训练目标 = 提升有氧耐力（尤其高阶运动员）
**建议**: THEN
- VO₂max达到平台后，成绩提升更多依赖运动节省化
- 同等速度下摄氧量越低 = 节省化越好 = 10km成绩可相差约1min
- 训练方法：技术优化 + 长期大运动量低强度（LSD）+ 力量训练（提高跑步经济性）
- 评估：同等亚极量负荷下的摄氧量(VO₂)比较

**原理**: 两名世界级10km运动员VO₂max相同(70 mL/kg/min)，节省化差异导致成绩差1分钟。节省化反映神经肌肉协调+能量代谢效率+生物力学优化。

## 2. 肌纤维类型规则 (Muscle Fiber Types)

### RULE-PHY-006: 肌纤维类型特征对比
- **类型**: book-thesis
- **来源**: 运动生理学 第1章 p.28-38 + 表1-1,1-2,1-3
- **置信度**: EXTRACTED

| 特征 | I型（慢肌） | IIa型（快肌-氧化） | IIb型（快肌-酵解） |
|------|-----------|------------------|-----------------|
| 收缩速度 | 慢（~100ms） | 快（~50ms） | 最快（~40ms） |
| 力量输出 | 低 | 中-高 | 最高 |
| 抗疲劳性 | 最高 | 中 | 最低 |
| 线粒体密度 | 高 | 中 | 低 |
| 毛细血管化 | 丰富 | 中等 | 少 |
| 糖原含量 | ≈80 mmol/kg | ≈83 mmol/kg | ≈89 mmol/kg |
| 甘油三酯 | 高（7.1） | 中（4.2） | 低 |
| 氧化酶活性(SDH) | 7.1 | 4.8 | 2.5 |
| 糖酵解酶活性(LDH) | 59 | 221 | 293 |
| 肌浆网发育 | + | ++ | +++ |
| 主要供能 | 有氧氧化 | 有氧+糖酵解 | 糖酵解 |

**训练含义**: 
- 耐力训练→I型纤维优先募集 + IIb→IIa转化（快→慢方向，不可逆）
- 力量/爆发训练→II型纤维优先肥大（II型是I型面积约2-3倍）

### RULE-PHY-007: 肌纤维类型的遗传性与可训练性
- **类型**: book-thesis
- **来源**: 运动生理学 第1章 p.34-38
- **置信度**: EXTRACTED
- **关联**: [[strength-training#individualization]]

**条件**: IF 用户询问"我天生适合什么运动"
**建议**: THEN
- 肌纤维类型百分构成由遗传决定（单卵双生子高度一致）
- 人上肢肌I型纤维比例40-67%，下肢35-82%（比目鱼肌I型可达82%）
- I型纤维比例高（>60%）→耐力型，对耐力训练反应好
- II型纤维比例高（>60%）→力量/速度型，对力量训练反应好
- 通过训练，IIb→IIa转化，但I↔II不可互转
- **东亚人群数据不足，肌纤维分布特征研究多基于西方人群**

**原理**: IIb→IIa转化是运动神经元放电模式改变引起的表型转换。大α运动神经元（支配II型）在耐力训练下可能转向持续低频放电模式，诱导肌纤维代谢酶谱向氧化型转变。

---

### RULE-PHY-008: 运动单位募集原则 — Henneman大小原则
- **类型**: trigger
- **来源**: 运动生理学 第1章 p.34-35, 第3章 p.83-84
- **置信度**: EXTRACTED
- **关联**: [[strength-training#load-selection]]

**条件**: IF 训练需要选择负荷强度
**建议**: THEN
- 低强度（<40% 1RM）→仅募集小α运动神经元（I型纤维）
- 中等强度（40-60% 1RM）→募集I型 + 部分IIa型
- 高强度（60-85% 1RM）→募集I型 + IIa + 部分IIb型
- 极强/爆发（>85% 1RM）→募集所有类型（包括最大阈值的IIb型）
- **II型纤维只能通过高强度负荷或疲劳状态下募集**

**原理**: 大α运动神经元（支配II型纤维）兴奋阈值高，必须有足够强的神经驱动才能激活。这是力量训练需要大负荷的生理学基础。

---

### RULE-PHY-009: 年龄对肌纤维类型的影响
- **类型**: trigger
- **来源**: 运动生理学 第1章 p.33
- **置信度**: EXTRACTED
- **关联**: [[special-populations#elderly]]

**条件**: IF 用户年龄 > 50岁
**建议**: THEN
- I型纤维比例随年龄增加：20-29岁I型41% → 60-65岁I型55%
- II型纤维（尤其IIb）随年龄减少：IIa 34%→20%，IIb 23.4%→12.9%
- 这是肌肉减少症（Sarcopenia）的生理基础
- **必须进行抗阻训练维持II型纤维**——不做力量训练的老年人II型纤维退化最快
- 阻力训练可逆转或延缓此过程（即使在70-80岁）

**原理**: II型纤维减少的可能机制：①运动神经元改变（位相型→紧张型）诱导II→I转变；②废用性萎缩先影响II型纤维。抗阻训练增加运动神经元放电频率，维持II型纤维。

## 3. 心血管反应规则 (Cardiovascular)

### RULE-PHY-010: 心率对运动的急性反应 — 心搏峰与最佳心率范围
- **类型**: decision-changing
- **来源**: 运动生理学 第8章 p.173-177
- **置信度**: EXTRACTED
- **关联**: [[exercise-prescription#intensity-zones]]

**条件**: IF 设计有氧训练计划
**建议**: THEN 了解两个关键心率节点：

| 概念 | 心率值 | 意义 |
|------|--------|------|
| 心搏峰 (SV Peak) | 110-130 bpm | 每搏输出量达峰值的HR，此后SV不增，CO增加依赖HR |
| 最佳心率范围 | 110-180 bpm | 心输出量可持续增加，SV已最大，训练心脏泵血效率 |
| 心输出量上限 | >200 bpm | SV开始减少（舒张期过短→心室充盈不足） |

**训练应用**:
- 心搏峰区间（110-130 bpm）训练：最大化SV，锻炼心肌泵血功能
- 最佳心率范围上限（160-180 bpm）：训练最大心输出量，提高有氧能力上限

**原理**: CO = SV × HR。运动初期SV随HR增加而增加，至110-130 bpm达峰值(SV约110-130mL)，之后CO增加全赖HR。HR>200时舒张期过短→充盈不足→SV反而下降。

---

### RULE-PHY-011: 安静心率与心脏功能
- **类型**: default
- **来源**: 运动生理学 第8章 p.170
- **置信度**: EXTRACTED

| 状态 | 安静心率 | 含义 |
|------|---------|------|
| 优秀耐力运动员 | 40-50 bpm | 心肌肥大，每搏量大，心率低即可满足CO需求 |
| 运动人群 | 50-65 bpm | 心脏功能良好 |
| 一般成人 | 65-80 bpm | 正常范围 |
| >80 bpm（静息） | — | 可能与低心肺适能有关，建议提升有氧运动 |

**安全阀**:
- 安静心率突然升高 > 5-10 bpm → 可能过度训练/感染，建议减量或休息
- 早晨静息心率持续升高超过3天 → 潜在的过度训练信号

---

### RULE-PHY-012: 运动时血液重新分配
- **类型**: book-thesis
- **来源**: 运动生理学 第8章 p.178-180
- **置信度**: EXTRACTED

**条件**: IF 用户有餐后立即运动的习惯
**建议**: THEN
- 运动中骨骼肌血流量从安静时的15-20%增加到80-85%
- 内脏（胃肠、肾）血流量显著减少（从25%降至3-5%）
- **餐后至少等1.5-2h再运动**（大餐等2-3h），否则消化不良+运动时胃肠缺血→腹痛
- 运动后不要立即进食大餐——内脏血流恢复需要15-30min

## 4. VO₂max与无氧阈规则 (Aerobic Capacity)

### RULE-PHY-013: VO₂max参考值与分级
- **类型**: default
- **来源**: 运动生理学 第10章 p.259-265 + ACSM
- **置信度**: EXTRACTED
- **关联**: [[exercise-prescription#aerobic-assessment]]

| 分级 | 男性 (mL/kg/min) | 女性 (mL/kg/min) |
|------|-----------------|-----------------|
| 优秀 | >50 | >42 |
| 良好 | 42-50 | 35-42 |
| 一般 | 35-41 | 28-34 |
| 差 | <35 | <28 |

**补充**:
- 精英越野滑雪男最高记录 ≈ 94 mL/kg/min
- 精英女运动员最高 ≈ 85 mL/kg/min
- VO₂max遗传度70-95%，但训练可提高5-25%
- 训练初期提升来自心肺（中央机制），后期来自肌肉摄氧能力（外周机制）
- 男性12-13岁后开始显著高于女性（睾酮促进红细胞生成+心脏容积增大）

---

### RULE-PHY-014: 无氧阈 (Anaerobic Threshold) — 训练强度的关键标志
- **类型**: decision-changing
- **来源**: 运动生理学 第10章 p.267-270
- **置信度**: EXTRACTED
- **关联**: [[exercise-prescription#intensity-zones]]

**条件**: IF 训练目标 = 提升有氧耐力
**建议**: THEN 以无氧阈(AT)为关键训练强度参考：

| 人群 | AT (% VO₂max) | 训练含义 |
|------|--------------|---------|
| 健康青年 | 50-60% | AT以下 = 纯有氧训练 |
| 训练者 | 60-70% | AT附近 = 最佳有氧刺激 |
| 优秀耐力运动员 | 70-80% | 高AT是竞技能力的标志 |

**关键原则**:
- AT比VO₂max更有可训练性（遗传度更低）
- AT强度训练（60-80% VO₂max）是发展有氧耐力最有效的强度
- 血乳酸4.0 mmol/L为通用乳酸阈参考值，但个体差异大（1.7-7.5 mmol/L）
- 通气阈(VT) ≈ 乳酸阈(LT)：呼吸突然加快加深的转折点

---

### RULE-PHY-015: EPOC (运动后过量氧耗) 与恢复/减重
- **类型**: trigger
- **来源**: 运动生理学 第10章 p.266-267
- **置信度**: EXTRACTED
- **关联**: [[exercise-prescription#weight-loss-programming]]

**条件**: IF 训练目标 = 减重/燃脂
**建议**: THEN
- HIIT后EPOC持续更长（可达24-48h），额外消耗50-150 kcal/day
- 中等强度稳态有氧EPOC约15min-2h
- EPOC幅度主要由运动强度（非时长）决定
- 运动中的氧亏越大→EPOC越多
- EPOC机制：磷酸原恢复+乳酸清除+儿茶酚胺+体温+心率恢复

**减重应用**: HIIT + 力量训练的组合产生最大EPOC效应，优于纯稳态有氧。

## 5. 激素反应规则 (Hormonal Responses)

### RULE-PHY-016: 生长激素(GH)释放的运动强度阈值
- **类型**: trigger
- **来源**: 运动生理学 第4章 p.113
- **置信度**: EXTRACTED
- **关联**: [[strength-training#hypertrophy]]

**条件**: IF 训练目标 = 增肌/减脂（希望最大化GH分泌）
**建议**: THEN
- GH释放的临界阈值 = 50-66% VO₂max强度
- <50% VO₂max强度→GH无显著变化
- >66% VO₂max强度→GH显著升高（可达安静时的35倍）
- 高强度间歇（>80% VO₂max）> 中等强度持续
- 大肌群多关节动作（深蹲/硬拉）GH反应最大
- 组间休息60-90s比长休息(>3min)更能刺激GH

**原理**: 乳酸堆积和高强度运动刺激下丘脑GHRH分泌→垂体GH分泌↑。GH促进脂肪分解+蛋白质合成。GH峰值出现在运动后30-60min。

---

### RULE-PHY-017: 睾酮与皮质醇比值 (T/C Ratio) — 训练负荷监控
- **类型**: decision-changing
- **来源**: 运动生理学 第4章 p.123-125
- **置信度**: INFERRED
- **关联**: [[strength-training#load-monitoring]]

**条件**: IF 训练计划需要监控是否过度训练
**建议**: THEN
- T/C比值下降 > 30% → 提示过度训练/恢复不足风险
- 睾酮主导合成代谢（肌肉构建），皮质醇主导分解代谢
- 高强度+大训练量+热量不足→皮质醇↑ + 睾酮↓
- 恢复良好→T/C比值回升
- **注意**: 这是竞技体育监测指标，普通健身者通常不需要血检监控

**训练减载信号**:
- 安静心率↑ + 睡眠质量↓ + T/C比值↓ → 建议减载1-2周

## 6. 训练适应规则 (Training Adaptations)

### RULE-PHY-018: 力量增长的神经适应 vs. 肌肉适应
- **类型**: decision-changing
- **来源**: 运动生理学 第1章 p.11-15, 第11章
- **置信度**: INFERRED
- **关联**: [[strength-training#progression-model]]

**条件**: IF 用户"练了几个月力量不增长了"
**建议**: THEN 区分两个阶段：

| 阶段 | 时间 | 主要机制 | 表现 |
|------|------|---------|------|
| 初期力量增长 | 前4-8周 | 神经适应（运动单位募集↑ + 同步化↑ + 拮抗肌共收缩↓） | 力量↑但肌肉体积无明显变化 |
| 后期力量增长 | 8周后 | 肌肉肥大（肌原纤维增生→横截面积↑） | 肌肉体积增加 + 力量持续上升 |
| 平台期 | 6-12个月 | 适应已达极限 | 需改变训练变量（负荷/组数/频率/练习选择） |

**行动计划**: 如果力量停滞——①增加训练量（组数×次数）；②改变训练频率；③增加动作种类；④周期性安排（非线性周期化）。

---

### RULE-PHY-019: 有氧训练的中枢vs外周适应
- **类型**: book-thesis
- **来源**: 运动生理学 第10章 p.270-278
- **置信度**: EXTRACTED

| 适应类型 | 具体变化 | 出现时间 |
|---------|---------|---------|
| **中枢适应** | 左心室容积↑、每搏量↑、心输出量↑、血容量↑ | 4-8周 |
| **外周适应** | 线粒体数量↑、氧化酶活性↑、毛细血管密度↑、肌红蛋白↑ | 8-16周 |
| **代谢适应** | 脂肪氧化能力↑、糖原储备↑、乳酸阈值右移 | 持续（12-24周后明显） |

---

### RULE-PHY-020: 停训/失训 (Detraining) 的时间线
- **类型**: trigger
- **来源**: 运动生理学 第10-11章
- **置信度**: EXTRACTED
- **关联**: [[exercise-prescription#maintenance-dose]]

| 停训时长 | 丧失内容 | 恢复难度 |
|---------|---------|---------|
| < 2周 | 无明显下降（VO₂max略降3-5%，实为血浆容量减少） | 1-2次训练恢复 |
| 2-4周 | VO₂max↓6-12%，肌肉糖原↓，肌力开始下降 | 2-4周恢复训练 |
| 4-8周 | VO₂max↓12-20%，肌纤维面积↓（尤其II型） | 8-12周重新建立 |
| >8周 | 接近久坐水平，但神经肌肉效率仍优于从未训练者 | 需重新周期化 |

**关键发现**: 减量50%（而非停训）可维持VO₂max和肌力长达4-8周。受伤/繁忙时优先保持1-2次/周强度训练而非完全停止。

## 7. 恢复与过度训练规则 (Recovery & Overtraining)

### RULE-PHY-021: 超量恢复 (Supercompensation) 原理
- **类型**: book-thesis
- **来源**: 运动生理学 第12章 p.274-280
- **置信度**: EXTRACTED
- **关联**: [[exercise-prescription#periodization]]

**条件**: IF 设计训练频率
**建议**: THEN 根据训练强度确定恢复时间：

| 训练类型 | 恢复时间 | 适宜频率 |
|---------|---------|---------|
| 轻-中等有氧 | 4-12h | 每天 |
| 高强度有氧 | 24-48h | 隔天 |
| 抗阻训练（同肌群） | 48-72h | 2-3次/周 |
| 爆发力/速度 | 48-72h | 2-3次/周 |
| 极量离心训练 | 72-96h | 1-2次/周 |

**原理**: 训练→消耗→恢复→超量恢复→适应。下一次训练应在超量恢复峰值附近（既非完全未恢复也非超量恢复消退期）。

---

### RULE-PHY-022: 过度训练（OTS）识别与干预
- **类型**: safety-gate
- **来源**: 运动生理学 第12章 p.280-286
- **置信度**: INFERRED
- **关联**: [[sports-medicine#overtraining-diagnosis]]

**条件**: IF 用户报告：持续疲劳 + 运动表现下降 + 安静HR↑ + 睡眠障碍
**建议**: THEN 进行OTS（过度训练综合征）筛查：

**交感型OTS**（早期）:
- HR↑ 5-10 bpm（静息）
- 烦躁/易怒/失眠
- 训练时RPE升高但表现未提升
- 继续训练会恶化

**副交感型OTS**（晚期，更严重）:
- HR↓（反常）
- 低血糖倾向
- 极度疲劳/抑郁
- 恢复期以月计算（3-6月+）

**行动计划**: 
1. 立即停止高强度训练
2. 仅允许低强度活动（<60% HRmax，每次<30min）
3. 保证每天8+h睡眠
4. 营养支持：热量不亏空，蛋白质1.6-2.0g/kg
5. 症状持续>2周→转诊运动医学专科

---

### RULE-PHY-023: 主动恢复 vs 被动恢复
- **类型**: micro-decision
- **来源**: 运动生理学 第12章 p.276-279
- **置信度**: INFERRED

**条件**: IF 高强度训练后需要促进恢复
**建议**: THEN 
- 高强度训练后24h内：主动恢复（低强度有氧，<50% HRmax，10-20min）+ 动态拉伸
- 赛后/大负荷后48h内：轻活动（游泳/散步/骑行）+ 泡沫轴 + 充足睡眠
- 仅完全被动休息（静卧）不利于乳酸清除——保持轻度活动可加速代谢废物清除50%以上
- 冷水浸泡(10-15°C, 10-15min)对DOMS有效但可能减弱肌肉肥大适应（力量期慎用）

## 8. 环境生理规则 (Environmental)

### RULE-PHY-024: 热环境运动安全
- **类型**: safety-gate
- **来源**: 运动生理学 第15章
- **置信度**: EXTRACTED

**条件**: IF 环境温度 > 30°C 或湿度 > 70%
**建议**: THEN
- 运动前补液：5-7 mL/kg体重，运动前2-4h
- 运动中补液：每15-20min饮用150-300mL
- 每脱水1%体重→核心温度↑0.1-0.2°C + HR↑5-10 bpm
- 脱水>2%体重→有氧能力显著下降
- 脱水>4%体重→恶心/呕吐/热射病风险

**热适应** (10-14天):
- 每天在热环境中运动30-100min
- 适应后：出汗率↑、汗液钠浓度↓、核心温↓、HR↓

## 9. 运动代谢规则 (Exercise Metabolism — from Advanced Nutrition)

> **来源**: Advanced Nutrition and Human Metabolism, 8th Edition, Chapter 7
> **编译日期**: 2026-06-02

### RULE-PHY-025: 运动强度决定燃料利用比例
- **类型**: decision-changing
- **来源**: Advanced Nutrition Ch.7, Coyle EF. Am J Clin Nutr. 1995
- **置信度**: EXTRACTED
- **关联**: [[sports-nutrition#fuel-timing]] [[exercise-prescription#intensity-zones]]

**条件**: IF 需要根据运动目标调整营养策略
**建议**: THEN 根据下表了解不同强度下燃料利用比例（运动30min后）：

| 强度 (% VO₂max) | 肌糖原 | 肌内甘油三酯 | 血浆游离脂肪酸 | 血浆葡萄糖 | 主导燃料 |
|-----------------|--------|-------------|--------------|-----------|---------|
| 25% (低) | 极少 | ~10% | ~70% | ~10% | 血浆脂肪酸 (80-90%) |
| 65% (中高) | **主力** | ~25% | ~20% | ~10% | 肌糖原 + 脂肪酸各半 |
| 85% (极高) | **绝对主力** | ~15% | ~15% | ~15% | 肌糖原 (60%+) |

**训练/营养含义**:
- 低强度(<50% VO₂max)：燃脂为主，适合减脂和恢复日
- 中等(60-75% VO₂max)：脂肪酸氧化达最大速率，葡萄糖开始重要
- 高强度(>80% VO₂max)：肌糖原绝对主导 → 必须保证糖原储备充足
- **减脂策略**: 中低强度长时间 + 空腹/低胰岛素状态（脂肪氧化最高）
- **高强度训练日**: 训练前3-4h补碳水以保证糖原

---

### RULE-PHY-026: 长时间运动中燃料来源的动态变化
- **类型**: trigger
- **来源**: Advanced Nutrition Ch.7, Figure 7.14 + Coyle 1995
- **置信度**: EXTRACTED
- **关联**: [[sports-nutrition#endurance-fueling]]

**条件**: IF 运动持续 > 1h 且强度 65-75% VO₂max
**建议**: THEN

| 运动时间 | 肌糖原占比 | 肌内脂肪 | 血浆脂肪酸 | 血浆葡萄糖 | 关键行动 |
|---------|-----------|---------|-----------|-----------|---------|
| 0-30min | ~65% | ~25% | ~15% | ~5% | — |
| 1h | ~60% | ~20% | ~20% | ~15% | 开始补水 |
| 2h | ~55% | ~15% | ~25% | ~20% | **开始补碳水(30-60g/h)** |
| 3h | ~50% | ~10% | ~30% | ~25% | 碳水+电解质 |
| 4h | ~45%↓ | ~5%↓ | ~35%↑ | ~30%↑ | 碳水60-80g/h |

**关键原则**:
- 运动2h后肌糖原接近耗竭→必须外源补碳水维持血糖
- 肌内甘油三酯也在消耗（4h后只剩一半）
- 血浆游离脂肪酸随时间占比上升——但脂肪酸氧化速率有上限（受CAT I酶限制）
- **马拉松等长时运动不补碳水→"撞墙"（血糖↓+肌糖原↓→被迫降速）**

---

### RULE-PHY-027: AMPK — 细胞能量代谢的"主开关"
- **类型**: book-thesis
- **来源**: Advanced Nutrition Ch.7, Hardie DG. Nat Rev Mol Cell Biol. 2012
- **置信度**: EXTRACTED
- **关联**: [[exercise-physiology#PHY-004]]

**条件**: IF 需要理解"为什么运动改善代谢健康"
**建议**: THEN AMPK是关键机制：
- AMPK被AMP/ATP比值升高激活（提示细胞能量低）
- 运动→AMP↑→AMPK激活→：
  - GLUT4转位至细胞膜→独立于胰岛素的葡萄糖摄取↑（运动降糖的核心机制）
  - 脂肪酸氧化↑（抑制ACC→减少malonyl-CoA→解除CAT I抑制）
  - 糖原合成↓、糖异生↓（将能量导向ATP生产）
  - mTOR活性↓→蛋白质合成暂时下降
- 二甲双胍的降糖作用部分通过激活AMPK

**训练含义**:
- 运动在低糖原状态下进行（"train low"）→AMPK激活更强→线粒体生物合成↑→代谢适应更强
- 但"train low"会降低训练强度→仅在基础期/低强度训练日使用
- 关键比赛前必须"train high"（糖原充足）以保证强度

---

### RULE-PHY-028: 碳水化合物负荷（Carb Loading）— 耐力赛前策略
- **类型**: decision-changing
- **来源**: Advanced Nutrition Ch.7, Bergstrom & Hultman 1967 + Ivy 2001
- **置信度**: EXTRACTED
- **关联**: [[sports-nutrition#carb-loading]]

**条件**: IF 准备 > 90min持续性耐力赛事（马拉松/铁三/越野跑）
**建议**: THEN

**改良方案（推荐）**:
| 天 | 运动 | 碳水摄入 | 目标 |
|----|------|---------|------|
| Day 1-3 | 正常训练量↓ | 50% 总热量来自碳水 | 维持 |
| Day 4-6 | 递减→休息 | 70% 总热量来自碳水 | 超量储备 |
| Day 7 | 休息 | 赛前餐(高碳水) | 比赛日 |

- 改良方案可使肌糖原达220 mmol/kg（比正常高20-40%）
- 经典方案（先耗竭再补充）副作用大（头晕、易怒、训练质量差）→**不推荐**

**不适用**:
- < 60min 运动 → 正常饮食糖原已够
- 高强度间歇运动 → 依赖ATP-CP/糖酵解，不依赖糖原总量
- 减重阶段 → 碳水负荷与减重目标冲突

### RULE-PHY-029: "Train Low, Compete High" — 代谢适应策略
- **类型**: decision-changing
- **来源**: Advanced Nutrition Ch.7, Bartlett JD. Eur J Sport Sci. 2015
- **置信度**: AMBIGUOUS
- **关联**: [[exercise-prescription#advanced-strategies]]

**条件**: IF 高水平耐力运动员寻求额外代谢适应
**建议**: THEN
- **Train Low**: 在低碳水可用状态下进行部分训练（如晨起空腹训练或一天两次训练中第二次低碳水）
  - 增强AMPK信号→促进线粒体生物合成 + 脂肪酸氧化能力↑
  - 提高代谢灵活性（肌肉在低碳水下仍能维持输出）
- **Compete High**: 关键训练课（高强度/长距离）和比赛日→保证糖原充足 + 赛中补碳水
- ⚠️ **证据等级**: 初步研究支持，但最佳方案未确立（AMBIGUOUS标记）
- ⚠️ **风险**: 空腹训练免疫抑制风险↑，女性慎用（RED-S风险）
- **不推荐新手使用** — 仅在基础扎实的运动员中谨慎尝试

---

### RULE-PHY-030: 训练后营养时机 — 蛋白质+碳水协同
- **类型**: trigger
- **来源**: Advanced Nutrition Ch.7, Tipton & Witard 2007
- **置信度**: EXTRACTED
- **关联**: [[sports-nutrition#post-exercise]] [[strength-training#nutrition-timing]]

**条件**: IF 抗阻训练或高强度耐力训练后
**建议**: THEN
- 训练后立即补充**蛋白质20-25g + 碳水0.8-1.2g/kg体重**
- 碳水→胰岛素↑→：
  - 蛋白质合成↑（mTOR通路）
  - 蛋白质分解↓（抑制泛素-蛋白酶体通路）
  - 糖原再合成↑
- 耐力训练后单独补碳水可恢复糖原
- 抗阻训练后蛋白质+碳水优于单独蛋白质
- **蛋白质来源**: 完整食物蛋白质与蛋白补剂效果相同 — 食物优先原则
- 运动后窗口期（0-2h）胰岛素敏感性最高

### RULE-PHY-031: 三大宏观营养素互转规律
- **类型**: book-thesis
- **来源**: Advanced Nutrition Ch.7, Section "Interconversion of Fuel Molecules"
- **置信度**: EXTRACTED

**关键生化事实**（常被误解）:
- ✅ 碳水 → 脂肪（通过乙酰CoA → 丙二酰CoA → 脂肪酸合成）
- ✅ 碳水 → 非必需氨基酸（通过丙酮酸 → TCA中间体）
- ✅ 蛋白质 → 葡萄糖（生糖氨基酸通过TCA → 草酰乙酸 → 糖异生）
- ✅ 蛋白质 → 脂肪（生酮氨基酸 → 乙酰CoA → 脂肪酸）
- ❌ **脂肪 → 葡萄糖（不可行！）** — 丙酮酸脱氢酶反应不可逆，乙酰CoA不能净转化为丙酮酸
- ❌ **脂肪 → 氨基酸（不可行！）** — 同样原因

**训练/减重含义**:
- 低碳水饮食下，即使脂肪充足，大脑和红细胞仍需葡萄糖（每天至少130g）
  → 只能来自糖异生（氨基酸/甘油）或酮体替代
- 脂肪不能变成糖 → 长时间低碳水+高强度训练→糖异生↑→肌肉分解↑（皮质醇↑）
- 减重时需保证足够碳水或蛋白质以保护肌肉
