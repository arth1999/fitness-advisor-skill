# 第1章 腰背疼痛

## 章节概述

腰背疼痛是力量型运动员最常见的损伤之一，超过80%的成年人一生中都会经历腰痛。本章基于斯图尔特·麦吉尔博士数十年的脊柱研究成果，建立了从筛查到康复再到重返运动的完整框架。

---

## 1. MRI的局限性

- 在无腰背疼痛的健康人群中，20岁群约1/3、40岁群约1/2存在椎间盘膨出
- 在一项研究中，84%出现腰背痛的患者MRI结果与无症状期扫描结果无差异
- MRI无法区分新伤与旧伤、不显示功能状态
- 仅凭MRI影像诊断疼痛源如同"只凭赛车照片就判断其在三挡换四挡时有异响"

## 2. 腰背损伤解剖学基础

### 损伤机制公式

```
功率 = 力量 × 速度
```

- 移动脊柱时应在最小负荷下进行（舞者、高尔夫球手安全）
- 举起重物时应锁定脊柱不动，通过髋关节运动
- 脊柱同时承重+移动=损伤风险大幅增加

### 四种主要损伤类型

| 损伤 | 机制 | 关键特征 |
|------|------|----------|
| **椎间盘膨出/突出** | 负重下脊柱屈曲→纤维环分离→髓核渗出 | 脊柱承重+移动同时发生；屁股眨眼是常见诱因 |
| **椎骨终板骨折** | 反复超负荷无充分恢复→终板压垮 | 微骨折→恢复不足→累积至大问题 |
| **小面关节损伤** | 脊柱伸展过限或伸展+旋转；椎间盘损伤后转移负荷 | 占慢性腰背痛15%-40% |
| **椎骨滑脱** | 椎弓峡部应力性骨折（反复弓腰）；遗传因素显著 | 可进展为脊柱滑脱症（L5前滑） |

### 神经疼痛

- **坐骨神经痛**：沿大腿后侧放射——椎间盘膨出、椎管狭窄或梨状肌压迫
- **股神经疼痛**：沿大腿前侧放射

### 关于"肌肉疼痛"

大多数腰背痛不是肌肉拉伤——肌肉痉挛是身体为稳定深层次损伤（椎间盘、小面关节等）的继发性代偿反应。

## 3. 疼痛筛查流程

### 步骤1：症状自评

找出：
- 哪些动作/姿势/负荷引发疼痛？
- 哪些动作/姿势/负荷不引发疼痛？
- 训练房内外的疼痛诱因

**将疼痛按激惹类型初步归类**：屈曲激惹、伸展激惹、旋转伸展激惹、负荷激惹。

### 步骤2：筛查测试

#### 姿势（体位）评估
- 拍摄站立侧面/前面/后面照片
- 观察：特定体位是否与症状相关（不评判"好/坏"）
- 如站姿伸展引发疼痛，尝试"挺身站立"调整

#### 坐姿上拉测试
- 中立位、屈曲位、伸展位分别施加压力
- 中立位也痛→严重负荷激惹（需暂时停训）
- 屈曲位痛→屈曲激惹；伸展位痛→伸展激惹

#### 俯卧测试
- 趴1-2分钟→疼痛缓解=屈曲激惹（建议每天趴几次）
- 趴着更痛→伸展激惹

#### 俯卧髋伸展测试（单侧）
- 抬腿产生疼痛+腹部垫枕头后改善→旋转伸展激惹

#### 动作评估
- 自重深蹲→观察屁股眨眼（骨盆后倾+腰椎屈曲）
- 负重杠铃站立弓腰→是否引发疼痛
- 杠铃RDL→前倾时痛=屈曲激惹；起身时痛=伸展激惹
- 单腿蹲→双侧对比、控制质量
- 5英寸触墙测试→踝关节背屈灵活性

#### 负荷测试
- 负重前平举：收紧核心后仍痛→严重负荷激惹（停训）
- 脚跟落地测试：收紧核心后仍痛→可能终板骨折（停训）

### 步骤3：髋关节伸展协调性（单腿臀桥）

- 单腿臀桥10秒保持→感受臀肌是否主要发力
- 非臀肌主导=臀肌失忆症→转化至双腿臀桥训练

### 步骤4：髋关节灵活性评估

- 仰卧髋旋转（双侧差异>几英寸=问题）
- Thomas测试、FABER测试

### 步骤5：胸椎灵活性评估

- 坐姿旋转测试（应达45°）
- PVC管墙壁测试（过顶活动度）

## 4. 腰背疼痛分类与短期管理

### 屈曲激惹

| 情境 | 策略 |
|------|------|
| 起床 | 侧身→手臂撑起（不直接坐起） |
| 坐立 | 腰部垫毛巾卷；挺直不驼背 |
| 站立 | 挺直身体 |
| 捡/搬 | 单膝跪/蹲下；髋关节铰链运动 |
| 康复练习 | 每天趴2-3次 |

### 伸展激惹

| 情境 | 策略 |
|------|------|
| 睡觉 | 俯睡→腹部垫枕；仰睡→膝下垫枕 |
| 坐立 | 靠椅背，不坐椅子边缘 |
| 站立 | 确保不反弓 |
| 捡/搬 | 髋关节启动→髋伸展；蹲下捡物 |

### 旋转伸展激惹

| 情境 | 策略 |
|------|------|
| 侧睡 | 膝间夹枕 |
| 起床 | 双腿并拢+躯干一起翻转 |
| 坐立 | 不跷二郎腿；不偏坐一侧 |
| 捡/搬 | 限制躯干旋转 |

### 负荷激惹

- 避免跑步和冲击性活动
- 仅在无痛范围内举重
- 严重时暂时完全停止负重训练

## 5. 重建过程

### 核心稳定性理论基础

- 核心肌群：腹部肌肉+背部肌肉（竖脊肌）+背阔肌+腰大肌+臀肌
- 关键是**等长耐力**（肌肉刚度/硬度），非动态力量
- 需要所有核心肌肉协同收缩，不存在可单独训练的核心肌肉
- 麦吉尔三大练习：**麦吉尔卷腹 + 侧平板支撑 + 鸟狗式**

### 灵活性优先

在三大练习前先处理髋/胸椎灵活性受限，否则代偿会削弱训练效果。

### 猫驼式热身

5-6次脊柱屈曲+伸展循环（无痛范围）。

### 麦吉尔卷腹

- 仰卧，一侧屈膝一侧伸直
- 头抬离地面仅1英寸→保持10秒
- **不要**抬太高（避免腰椎屈曲）
- 倒金字塔式方案：5-3-1升到8-6-4
- 呼吸：小口吸气呼气（如通过吸管呼吸）

### 侧平板支撑

- 膝支撑→全侧平板进阶
- 臀部前后移动（蹲起/下蹲提示）
- 每侧相同倒金字塔方案

### 鸟狗式

- 四肢着地→对侧手臂+腿伸展→保持10秒
- 手膝相触（交替间）
- 专注腰部不移动

### 臀肌激活

- 双腿臀桥：2组×20次，每次保持5秒
- 进阶到单腿臀桥
- 腘绳肌痉挛时：脚跟更靠近臀部+脚趾下压

### 早期康复计划

- 每天3次步行，每次10-15分钟
- 每天1次三大练习（几周后每天2次）
- 早晨刚起床后不进行核心训练（椎间盘最易受伤时）

## 6. 第三阶段：重建运动表现

### 四类训练

| 类别 | 练习示例 | 平面 |
|------|----------|------|
| 推 | 深蹲、推雪橇 | 矢状面 |
| 拉 | 硬拉、反向划船 | 矢状面 |
| 单侧负重 | 单侧农夫行走 | 额状面 |
| 抗旋转 | 站姿核心抗旋前推、单臂划船 | 水平面 |

**加载顺序**：矢状面→额状面→水平面（扭转力对脊柱压力最大，约4倍）

### 深蹲回归路径

```
高脚杯深蹲（胸前持壶铃）
    ↓
前深蹲（杠铃在锁骨前）
    ↓
后深蹲（杠铃在背上）
```

### 硬拉回归路径

```
垫木硬拉（提铃位置抬高至膝关节以上）
    ↓
逐步降低垫木高度
    ↓
地面硬拉
```

- 相扑式硬拉对腰部压力更小（躯干更竖直、力臂更短）
- 拉弯杠铃杆：募集背阔肌增加核心刚度

### 变式训练

- **僵尸式前蹲**：双臂伸直保持杠铃在肩上，迫使躯干直立
- **铁链深蹲**：利用力量-速度曲线，摆动铁链训练本体感觉
- **壶铃甩摆**：后链爆发力+全身协调性
- **等长硬拉**：不同位置等长保持2-5秒

### 关于举重腰带

- 仅在大重量（≥85%1RM）的少数训练中使用
- 正确使用：在腰带内扩张腹部，非仅仅系紧
- 训练前几年不建议使用——建立天然举重腰带
- 不以缓解疼痛为目的使用腰带

---

## 7. 警示练习

- **器械俯卧直腿后摆**：应通过肘关节支撑上半身+限制腰部运动，专注于髋关节而非腰椎
- **罗马椅/GHD腰背伸展**：应使骨盆顶部略超支撑垫，保持腰背中立位（等长），通过腘绳肌和臀肌完成

## 8. 拉伸腘绳肌能解决腰背痛吗？

**答案：不能。**

- 腘绳肌"紧张"在腰背痛患者中是神经系统紊乱（拉伸耐受性改变），而非肌肉长度问题
- 腰背痛的解决步骤：
  1. 降低疼痛敏感性（避免诱因）
  2. 提高核心稳定性+相邻关节灵活性
  3. 提高动作质量（髋关节铰链+臀肌激活）
- 有些运动员腘绳肌相对僵硬是优势（如弹簧般的弹性）
- 训练前拉伸超过30秒会降低力量/爆发力/速度长达30分钟
- 动态热身（步行、弓步、自重深蹲等）比静态拉伸更好地改善灵活性
