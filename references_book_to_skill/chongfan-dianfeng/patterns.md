# 康复诊疗通用规律 — 重返巅峰

## 模式 1：疼痛诱因分类系统

**跨部位通用模型**：所有关节/部位的疼痛均可按诱发机制归类为四种激惹类型。

| 激惹类型 | 诱发条件 | 常见损伤 | 核心策略 |
|----------|----------|----------|----------|
| 屈曲激惹 | 关节/脊柱屈曲时疼痛 | 椎间盘膨出、坐姿疼痛 | 髋关节铰链运动、避免弯腰 |
| 伸展激惹 | 关节/脊柱过度伸展时疼痛 | 小面关节刺激、腰椎滑脱 | 腹部垫枕、限制过伸 |
| 旋转伸展激惹 | 伸展+旋转时疼痛 | 小面关节+旋转刺激 | 限制躯干旋转、双腿并拢移动 |
| 负荷激惹 | 负重或快速加载时疼痛 | 终板骨折、核心不稳 | 暂时停训、逐步重建负荷 |

**规律**：确定激惹类型是选择康复策略的首要步骤。不分类直接治疗如同"在不了解病因的情况下开药"。

---

## 模式 2：三联筛查流程

**跨部位适用**：全书对所有关节疼痛筛查遵循统一的三步法。

```
步骤1: 症状自评
└── 列出诱发/不诱发疼痛的动作、姿势、负荷

步骤2: 动作筛查
├── 姿势评估 ── 站立/坐姿下脊柱体位与症状关联
├── 动作评估 ── 自重蹲、单腿蹲、RDL、髋伸展
├── 负荷测试 ── 负重前平举、脚跟落地、抱膝跳
└── 灵活性筛查 ── 髋旋转、踝背屈、胸椎旋转、FABER

步骤3: 分类判定
└── 确定激惹类型 → 进入对应康复路径
```

**规律**：筛查不做减法——每个部位都需检查其上、下方相邻关节的灵活性（腰背痛查髋和胸椎，膝痛查髋和踝，肩痛查胸椎）。

---

## 模式 3：三阶段康复框架

**全书统一的康复路径**：所有部位损伤康复遵循相同的三阶段结构。

### 第一阶段：消除症状 + 创造愈合环境

- 停止所有诱发疼痛的动作/负荷
- 调整日常姿势（坐、站、睡、捡/搬东西）
- 每日步行计划（3次/天，每次10-15分钟）
- 无痛范围内的主动活动

**关键原则**："在康复期间，你不做的事情通常与你做的事情同样重要。" —— 麦吉尔

### 第二阶段：解决根本问题

```
┌─────────────────────────────────────────┐
│  灵活性优先（关节受限）                    │
│  ├── 髋关节 / 踝关节 / 胸椎灵活性练习      │
│  └── 软组织放松（泡沫轴滚压）              │
│              ↓                           │
│  核心稳定性训练（等长练习）                 │
│  ├── 三大练习（麦吉尔卷腹+侧平板+鸟狗式）     │
│  └── 臀肌激活（臀桥+单腿臀桥）             │
│              ↓                           │
│  运动模式纠正                              │
│  ├── 髋关节铰链运动                        │
│  ├── 肩胛骨控制                            │
│  └── 落地机制纠正                          │
└─────────────────────────────────────────┘
```

**规律**：灵活性始终优先于稳定性——不解决相邻关节灵活性受限，核心稳定性训练效果大打折扣。

### 第三阶段：重建运动表现

- 多平面对抗训练（矢状面→额状面→水平面）
- 推/拉/单侧负重/抗旋转四类练习
- 渐进负荷：高脚杯深蹲→前深蹲→后深蹲
- 垫木硬拉→逐步降低垫木高度→地面硬拉
- 加入铁链/弹力带等不稳定负荷训练

## 模式 4：灵活性优先 + 重复测试验证

**全书一致的方法**：

```
前提：关节灵活性受限 → 相邻关节代偿 → 损伤

方法：
1. 基线测试（记录筛查结果）
2. 针对性灵活性练习（1组10-20次）
3. 立即重复测试（观察变化）
4. 有效 → 纳入每日训练，无效 → 换练习重试
```

**规律**：不存在万能灵活性练习——通过"测试→练习→复测"循环找到适合自己的练习是康复的核心机制。

---

## 模式 5：核心稳定性 = 肌肉等长收缩耐力（非力量）

**打破传统误区**：

| 传统观念 | 本书观点 |
|----------|----------|
| 核心训练=腹肌力量 | 核心训练=稳定性耐力 |
| 动态卷腹/转体 | 等长保持（麦吉尔三大练习） |
| 力量=保护 | 肌肉刚度/硬度=脊柱保护 |
| 单独训练某块肌肉 | 所有核心肌肉协同训练 |
| 呼气发力 | 屏住呼吸收紧核心（大重量时） |

**规律**：许多腰背疼痛的人背部很强壮，但缺乏核心稳定性耐力。等长练习（时间耐力）优于动态练习（力量训练）用于损伤康复。

---

## 模式 6：肌腱损伤 = 负荷管理问题（非炎症）

**肌腱病理连续体模型**（跨膝/踝/髋/肘通用）：

```
正常肌腱
    │ 急性超负荷 / 休息后快速恢复训练
    ↓
反应性肌腱病（可逆：几周休息+正确负荷）
    │ 持续超负荷 + 忽略疼痛
    ↓
肌腱损伤（胶原蛋白破坏）
    │ 持续超负荷
    ↓
退行性肌腱病（胶原蛋白消亡、无疼痛但功能丧失）
```

**规律**：
- 肌腱疼痛 ≠ 炎症，"肌腱炎"术语不正确
- 康复核心：在无痛范围内施加可控的渐进负荷（非完全休息）
- 分为等长→离心→向心→能量存储负荷的渐进序列

## 模式 7：代偿模式——远端问题近端体现

**全书揭示的因果链**：

```
踝关节背屈受限
    │
    ↓
深蹲时膝关节无法充分前移
    │
    ↓
骨盆后倾/屁股眨眼（腰椎屈曲代偿）
    │
    ↓
腰背痛（L4/L5 或 L5/S1 椎间盘膨出风险）

类似链：
胸椎旋转受限 → 腰椎代偿旋转 → 小面关节刺激
髋关节内旋不足 → 膝关节外翻代偿 → 髌股关节疼痛
肩胛骨前锯肌无力 → 肩峰下空间减少 → 肩关节撞击
```

**规律**：疼痛部位往往不是问题根源。治疗必须检查疼痛部位上方和下方所有相邻关节。

---

## 模式 8：臀肌失忆症——跨部位损伤的共同表现

**现象**：疼痛导致大脑减少对臀肌的神经驱动。

**影响链**：
```
臀肌失忆 → 髋伸展依赖竖脊肌/腘绳肌
    │
    ├─→ 脊柱压力过大 → 腰背痛
    ├─→ 骨盆不稳 → 膝关节外翻 → 膝痛
    └─→ 髋关节控制差 → 髋关节撞击/外侧痛
```

**康复方案**：双腿臀桥→单腿臀桥→等长深蹲（保持）→壶铃甩摆

---

## 模式 9：闭链优于开链（负荷下康复）

**规律**：对于大多数关节疼痛，闭链运动（脚/手固定接触地面/器械）优于开链运动（肢体自由移动）。

| 优势 | 原理 |
|------|------|
| 压力分散 | 关节接触面更大，单位面积压力更小 |
| 运动轨迹更优 | 多关节协同控制，减少关节偏移 |
| 更接近功能需求 | 模拟日常和运动中的动作模式 |

---

## 模式 10：无冰原则——积极恢复替代被动治疗

**核心论点**：

```
损伤后愈合的三阶段：炎症 → 修复 → 重建
                       ↑
              冰敷在这里设路障阻止白细胞进入
    
正确做法：
├── 急性期：无痛范围内主动活动（等长收缩）
├── 消肿：肌肉收缩→驱动淋巴系统
├── 防萎缩：NMES设备辅助（无法自主收缩时）
└── 训练后恢复：泡沫轴、步行、NMES（非冰水浴）
```

**规律**：让好的东西（白细胞）进来，让坏的东西（肿胀/代谢废物）排出。冰敷两个目标都无法达成。

---

## 模式 11：姿势定义——体位与症状关联（非好与坏）

**规律**：姿势评估的目标不是判断"好姿势"或"坏姿势"，而是观察特定脊柱体位是否与症状相关联。

- 如果站立时过度前凸导致疼痛，那么该体位就是康复需要调整的方向
- 如果趴着减轻疼痛（伸展位缓解），则说明属于屈曲激惹
- 姿势调整是短期策略，不是最终目标

---

## 模式 12：正确的举重呼吸——瓦尔萨尔瓦策略

**规律**：大重量举重时（≥70%1RM），应深吸气→收紧核心→屏住呼吸→通过最小阻力呼气（"嘶"声）过黏滞点。

- "向下吸气，向上呼气"的建议对大重量不适用
- 核心肌群+腹内压=天然举重腰带
- 举重腰带是辅助工具，非腰背痛解决方案
- 正确使用腰带：在腰带内扩张腹部，而非仅仅系紧

---

## 模式 13：筛查-分类-康复的决策树

```
运动员出现疼痛
    │
    ├→ 何时看医生？
    │   ├── 体重无故减轻
    │   ├── 腹/骨盆底松弛、麻木
    │   ├── 膝关节绞索/明显肿胀
    │   ├── 完全断裂（挤压测试失败）
    │   └── 持续恶化（数周无改善）
    │
    ├→ 可以自我处理：
    │   │
    │   ├→ 步骤1: 确定激惹类型（屈曲/伸展/旋转+伸展/负荷）
    │   │
    │   ├→ 步骤2: 调整姿势与日常活动（避免诱因）
    │   │
    │   ├→ 步骤3: 灵活性筛查+针对性练习
    │   │          （髋/踝/胸椎灵活性+重复测试验证）
    │   │
    │   ├→ 步骤4: 核心稳定性训练（三大练习）
    │   │          （倒金字塔式方案：6-4-2→8-6-4→10-8-6）
    │   │
    │   ├→ 步骤5: 运动模式纠正（髋关节铰链/肩胛骨控制等）
    │   │
    │   └→ 步骤6: 渐进负荷重返运动
    │              （自重→轻负荷→中负荷→大负荷→专项运动）
    │
    └→ 周/月后仍无改善 → 寻求专业物理治疗师
```
