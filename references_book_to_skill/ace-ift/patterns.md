# ACE Personal Trainer Manual -- Reusable Patterns

## Pattern 1: Client Intake and Screening Workflow
```
New Client Inquiry
    -> Schedule initial consultation (free, not a full workout)
        -> Conduct health-history interview (HHQ, PAR-Q, exercise history, attitude questionnaire)
        -> CAD risk factor stratification (low/moderate/high)
        -> If high risk or signs/symptoms present -> Medical referral before exercise
        -> If low/moderate risk -> Obtain informed consent
            -> Determine behavior-change stage (TTM)
            -> Set initial goals collaboratively
            -> Begin Phase 1 exercise
```

## Pattern 2: ACE IFT Assessment Sequence
```
1. Health history review (always first)
2. Static postural assessment (anterior, lateral, posterior views)
3. Movement screens (bend-and-lift, hurdle step, shoulder push/pull)
4. Flexibility and joint mobility (Thomas test, Apley's scratch, ankle mobility)
5. Core function (McGill endurance test battery)
6. Balance (static -> dynamic)
7. Physiological: RHR, BP, body composition
8. Cardiorespiratory: Submaximal test (cycle ergometer, Rockport walk, step test) or VT1/VT2 field tests
```
Note: Assessments should match the client's current training phase. Not all assessments are performed at intake.

## Pattern 3: ACE IFT Dual-Module Phase Assignment
For each client, assign to a phase in BOTH modules independently:
```
Client: [Name]
  Functional Movement & Resistance: [Phase 1/2/3/4]
    Phase-specific assessments: [...]

  Cardiorespiratory Training: [Phase 1/2/3/4]
    Phase-specific assessments: [...]
    Key training methods: [...]
  Behavior-Change Stage: [Precontemplation/Contemplation/Preparation/Action/Maintenance]
  Next reassessment date: [4-8 weeks]
```

## Pattern 4: Exercise Program Design Template (FITT-VP)
```
Client Goal: [endurance/hypertrophy/strength/power/weight loss/general health]
ACE IFT Phase(s): [resistance phase] / [cardio phase]

Warm-up: [5-10 min dynamic movement, movement prep]

Resistance Training:
  Frequency: [days/week per muscle group]
  Exercises: [list in order: multi-joint -> single-joint]
    Exercise 1: [sets] x [reps] @ [load or %1-RM], tempo [ecc/iso/con/iso], rest [sec]
    Exercise 2: ...
  Progression plan: [increase reps -> increase load -> increase sets -> decrease rest]

Cardiorespiratory Training:
  Frequency: [days/week]
  Intensity: [RPE, HR zone, or talk test reference to VT1/VT2]
  Duration: [minutes]
  Type: [mode(s)]
  Method: [steady-state / intervals / mixed]

Cool-down: [5-10 min low-intensity cardio + static stretching 15-60 sec per major muscle]

Behavioral Focus: [key adherence strategy for this client/phase]
```

## Pattern 5: Program Progression Decision Tree
```
Has client achieved Phase exit criteria?
  YES -> Assess readiness for next phase. Introduce Phase N+1 elements gradually.
  NO -> Continue current phase. Modify ONE variable:
    -> Is client tolerating current volume?
      YES -> Increase intensity slightly (load, speed, or ROM)
      NO -> Has recovery been adequate?
        YES -> Reduce intensity or volume temporarily; reassess technique
        NO -> Address sleep, nutrition, stress; reduce training load
```

## Pattern 6: Exercise Regression and Progression Ladder (Any Movement)
```
Regression <----------------------------------------------> Progression
Reduce load      Reduce ROM     Reduce stability     Base     Add load      Add ROM    Add instability    Add speed
(body weight)    (partial)      (bilateral, seated)          (bands, DB)   (full)     (single leg, ball)  (power)

Example: Squat
  Seated sit-to-stand -> BW squat partial -> BW squat full -> Goblet squat -> Front squat -> Back squat -> Overhead squat -> Jump squat
```

## Pattern 7: Medical Referral Decision Tree
```
Client reports or assessment reveals:
  -> Known cardiovascular, pulmonary, or metabolic disease -> Medical clearance required
  -> Signs/symptoms of disease (chest pain, dyspnea, dizziness, etc.) -> Medical referral required
  -> >= 2 CAD risk factors without symptoms -> Medical exam recommended before vigorous exercise
  -> Persistent musculoskeletal pain (> 1-2 weeks) -> Medical referral
  -> Seeking specific nutritional guidance/meal plan -> Refer to Registered Dietitian
  -> Psychological/emotional issues beyond exercise -> Refer to mental health professional
```

## Pattern 8: Behavioral Strategy Matching
```
Client Stage (TTM) -> Primary Strategy
  Precontemplation -> Education, awareness building, empathy. Do NOT prescribe exercise yet.
  Contemplation -> Explore pros/cons, address barriers, build confidence through small wins.
  Preparation -> Concrete action plan, realistic goal setting, identify social support.
  Action -> Relapse prevention, positive reinforcement, self-monitoring tools, build autonomy.
  Maintenance -> Variety in programming, new challenges, develop independence, plan for lapses.
```

## Pattern 9: Cardiorespiratory Training Zone Prescription
```
Zone 1 (below VT1): 
  Talk test = comfortable conversation
  RPE = 3-4/10 (0-10 scale) or 12-13 (6-20 scale)
  HR = < 70% HRR
  Use: Phase 1 aerobic base, recovery sessions, warm-up/cool-down

Zone 2 (VT1 to below VT2):
  Talk test = difficult to speak continuously (a few sentences at a time)
  RPE = 5-6/10 or 14-15
  HR = 70-80% HRR
  Use: Phase 2 aerobic efficiency intervals

Zone 3 (at and above VT2):
  Talk test = cannot speak more than 1-2 words
  RPE = 7-8/10 or 16-17
  HR = 80-90% HRR
  Use: Phase 3 anaerobic endurance intervals

Zone 4 (maximal):
  Talk test = cannot speak at all
  RPE = 9-10/10 or 18-20
  Use: Phase 4 anaerobic power (performance athletes only)
```

## Pattern 10: Special Population Training Modifications
```
Population        | Key Modifications
Cardiac           | Avoid Valsalva, isometrics; monitor RPE; gradual progression
Hypertension      | Avoid heavy isometrics; monitor BP; moderate intensity
Diabetes          | Monitor glucose; have carbs available; foot care essential
Obesity           | Low impact; start with short bouts; emphasize consistency
Older adults      | Limit 2 hard sessions/week; protect muscle; avoid risk
Youth             | Skill development; no max lifts before maturity; variety
Pregnancy         | No supine after 1st tri; no Valsalva; moderate intensity
Low-back pain     | Core stabilization; neutral spine; avoid spinal flexion load
Arthritis         | Exercise in pain-free range; pool ideal; 10+ min warm-up
Osteoporosis      | Weight-bearing cardio; spinal extension; balance; no spinal flexion
```
