# ACE Integrated Fitness Training (IFT) -- Agent Skill

## Overview

This skill encodes the ACE Personal Trainer Manual (5th Edition), the primary textbook for the ACE Certified Personal Trainer credential. At its core is the **ACE Integrated Fitness Training (ACE IFT) Model** -- a comprehensive, evidence-based system for designing personalized exercise programs that progress clients along a **Function-Health-Fitness-Performance continuum**.

The ACE IFT Model organizes all training variables across two parallel modules:
1. **Functional Movement and Resistance Training** (4 phases: Stability/Mobility -> Movement -> Load -> Performance)
2. **Cardiorespiratory Training** (4 phases: Aerobic Base -> Aerobic Efficiency -> Anaerobic Endurance -> Anaerobic Power)

All training is built on a foundation of **rapport** and **behavior-change strategies**.

## Guiding Principles

When applying this skill, follow these fundamental principles:

1. **Rapport first, training second**: All successful programs are built on trust and positive exercise experiences. Behavior-change strategy precedes exercise prescription.

2. **Assess before you prescribe**: Health-history screening -> postural assessment -> movement screening -> physiological testing. Each assessment informs the next. Never skip directly to loaded exercise.

3. **Stability and mobility before movement, movement before load**: Correct imbalances, establish motor control, then teach movement patterns with body weight before adding external resistance.

4. **Aerobic base before anaerobic work**: Build the cardiorespiratory foundation with steady-state exercise below VT1 before introducing intervals or high-intensity training.

5. **One phase may not fit all modules**: A client may be in Phase 1 of functional movement and Phase 2 of cardiorespiratory training simultaneously. Assess and program each module independently.

6. **Individualize progression continuously**: No two clients follow identical timelines. Progression speed depends on adaptation rate, recovery capacity, and consistency.

7. **Work within your scope of practice**: Never diagnose, never prescribe diets, never treat injuries, never provide psychological counseling. When in doubt, refer to the appropriate healthcare professional.

8. **Use evidence-based protocols**: All program design should reference ACSM guidelines, published position stands, and the ACE IFT model phases and exit criteria.

## Integration Notes

- This skill covers program design for apparently healthy individuals and those with medical clearance. It does NOT replace medical advice.
- The ACE IFT model is particularly strong at bridging general fitness with clinical exercise: Phase 1 addresses functional deficits and health improvement for sedentary, deconditioned, and older adult populations.
- Ventilatory threshold-based training (VT1/VT2) plus the talk test provides a practical, equipment-free method for prescribing cardiorespiratory intensity without needing laboratory testing.
- The dual-module design (functional movement/resistance + cardiorespiratory) ensures trainers address all components of fitness without the common error of emphasizing one domain while neglecting another.
