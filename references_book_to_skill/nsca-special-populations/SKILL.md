# NSCA's Essentials of Training Special Populations -- Structured Skill

## Knowledge Base Overview

This skill converts NSCA's Essentials of Training Special Populations into a structured knowledge base covering exercise programming for individuals with chronic diseases and special conditions. The core framework is built around condition-specific FITT-VP parameters, contraindications, safety guidelines, and medication interactions.

## Core Clinical Decision Chain

```
Health Appraisal & Medical Clearance (ch02)
  ├── PAR-Q+ (7 core + follow-up questions)
  ├── Medical History Questionnaire
  ├── Informed Consent
  ├── Medical Clearance Form
  └── Determine: Supervised vs Medically Supervised Program
        │
        ▼
Fitness Assessment (ch02)
  ├── Non-fatiguing tests (HR, BP, height, weight, flexibility, body comp)
  ├── Agility tests
  ├── Maximal strength tests (1RM)

  ├── Condition-specific FITT-VP
  ├── Exercise Modifications & Precautions

  └── Case Study Application
```

## Key Numbers (Must-Know)

| Parameter | Value |
|-----------|-------|
| General aerobic minimum | >=150 min/wk moderate or >=75 min/wk vigorous |
| General resistance training | >=2 days/wk, 8-12 exercises, 60-80% 1RM |
| Weight loss target | >250 min/wk exercise |
| Weight regain prevention | 300+ min/wk, 60-90 min/day |
| Moderate intensity | 40-<60% HRR/VO2R, RPE 12-13 |
| Vigorous intensity | >=60-90% HRR/VO2R, RPE >=14 |
| Hypertension exercise BP limit | SBP <220, DBP <105 mmHg |
| Angina: below ischemic threshold | 10-15 bpm below |
| Diabetes glucose check threshold | <100 mg/dl pre-exercise: consume 15-30g CHO |
| Diabetes ketoacidosis threshold | >250 mg/dl + ketones: avoid exercise |
| Pulmonary SpO2 limit | >85-90% during exercise |
| Older adult resistance start | 40-50% 1RM progressing to 60-80% |
| Pregnancy exercise | >=3 days/wk, moderate, >=150 min/wk |
| Youth physical activity | >=60 min/day moderate to vigorous |
