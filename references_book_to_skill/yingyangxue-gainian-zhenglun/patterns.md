# 营养学：概念与争论 — 思维框架与决策模式

## 1. 全书的组织逻辑

```
个体营养素 → 食物 → 膳食模式 → 生命阶段 → 公共健康 → 全球问题
```

本书从微观到宏观展开，引导读者从单个营养素理解上升到整体膳食和全球视角。

## 2. 营养科学探究方法（科学方法）

```
观察 → 提出假说 → 设计研究 → 收集数据 → 
分析结果 → 得出结论 → 发表 → 同行评议 → 重复验证
```

本书的核心信息之一是：**一个研究不足以建立真理**。营养学知识来源于多个研究的一致性结论，而非某个"突破性发现"。

## 3. 区分营养事实与伪科学的判断框架

是否可靠的营养信息？问以下问题：
1. 是否有在同行评审期刊上发表的科学研究支持？
2. 信息提供者是否有适当的资质？
3. 声称是否"太神奇以至于不像真的"？
4. 是否基于一小部分或选择性的研究？
5. 是否试图销售产品？——经济利益冲突是一个重要的危险信号
6. 信息来源是否承认任何局限性和反证据？
7. 信息来源是否为声誉良好的专业或政府机构？

## 4. 健康膳食计划的核心框架

```
充足性(Adequacy) + 平衡(Balance) + 能量控制(kCalorie Control) + 
营养密度(Nutrient Density) + 节制(Moderation) + 多样性(Variety)
```

在这六个原则指导下，通过DRIs、膳食指南和MyPlate等工具实现。

## 5. 慢性病-营养关系谱

```
纯遗传疾病 → 遗传-生活方式交互疾病 → 纯生活方式疾病
（营养影响最小）                          （营养影响最大）
```

本书第1章建立的概念——确定某疾病的营养可预防性和营养治疗的潜力。

## 6. 行为改变六阶段

```
前意向期 → 意向期 → 准备期 → 行动期 → 维持期 → 终止/适应期
```

为膳食咨询和行为改变干预提供心理框架。了解个体处于哪一阶段对于给予适当的建议和支持至关重要。

## 7. 能量平衡的管理逻辑

```
能量摄入(食物+饮料)
- 能量消耗(BMR + TEF + 身体活动)
= 能量平衡(正值=增重 / 负值=减重)
```

饮食方面：关注能量密度和营养素密度
活动方面：包括有目的运动 + NEAT（非运动性活动）

## 8. 全生命周期营养的关键节点

```
孕前 → 孕期 → 哺乳期 → 婴儿期 → 儿童期 → 
青少年期 → 成年期 → 老年期
```

每个阶段都有其独特的营养素优先级：
- **孕期**：叶酸（神经管缺陷预防）+ 铁 + 碘 + DHA
- **婴幼儿**：母乳或铁强化配方食品
- **青少年**：钙（峰值骨量）+ 铁（女性）+ 蛋白质
- **老年**：蛋白质（抗肌肉衰减）+ 钙/维生素D（骨健康）+ B12（补充剂）

## 9. 运动营养四要素

```
CHO供给(赛前+赛中+赛后) + 蛋白质时机(刺激MPS) + 
补液(防止脱水和低钠血症) + 能量可利用性(支持所有生理功能)
```

## 10. 食品安全控制链

```
农田/养殖场 → 收获/运输 → 加工 → 存储 → 运输 → 
零售/门店 → 消费者采购 → 家庭储存 → 制备 → 消费前存储
```

每个环节都有特定的安全风险和控制措施。消费者的四大原则：
**Clean + Separate + Cook + Chill**

## 11. 全球粮食系统的可持续性框架

```
人口增长 → 粮食需求↑ → 生产压力↑ → 资源耗竭 + 环境影响↑
← 减少食物浪费 + 可持续膳食转变 + 技术创新 ←
```

解决方案需要在生产端（技术改良、效率提升）和消费端（膳食模式转变、减少浪费）同时采取行动。
