# 第7章 维生素

## 章节概述

维生素是有机微量营养素，不提供能量但对生命至关重要。本章全面介绍13种维生素的分类、功能、缺乏症、食物来源和中毒风险。将维生素分为脂溶性（A、D、E、K）和水溶性（B族和C）两大类。

## 核心知识点

### 1. 脂溶性维生素（Fat-Soluble Vitamins）

**维生素A（视黄醇，Retinol）**：
- 功能：视觉（视紫红质组成部分）、基因表达调控、免疫、上皮组织维护
- 动物来源：肝脏、鱼肝油、蛋黄、强化乳制品
- 植物来源：β-胡萝卜素（维生素A原）——深色蔬菜和水果（胡萝卜、甜薯、菠菜）
- 缺乏：夜盲症→干眼症→角化症、感染抵抗力下降
- 过量：肝损伤、骨损伤、致畸作用（孕期需警惕）

**维生素D**：
- 功能：钙和磷的吸收与骨骼矿化、免疫调节、肌肉功能
- 独特之处：皮肤在UVB光照下可合成（"阳光维生素"），食物来源有限
- 缺乏：儿童佝偻病（Rickets）、成人骨软化症（Osteomalacia）、增加多种慢性病风险
- 过量：高钙血症→肾结石、血管和组织钙化。UL = 100 μg（4000 IU）/d

**维生素E（生育酚，Tocopherol）**：
- 功能：主要脂溶性抗氧化剂，保护细胞膜免受氧化损伤
- 食物来源：植物油、坚果和种子、全谷物。人体储存丰富，缺乏罕见
- 过量补充可能干扰维生素K活性（出血风险）

**维生素K**：
- 功能：血液凝固（凝血因子II、VII、IX、X的γ-羧化）、骨蛋白（骨钙素）的活化
- 来源：绿叶蔬菜（K1，叶绿醌）、肠道菌群合成（K2，甲萘醌）
- 缺乏：新生儿（肠道菌群未建立、初乳K含量低）易出血——出生时常规注射维生素K

### 2. 水溶性维生素（Water-Soluble Vitamins）

**B族维生素**（B1、B2、B3、泛酸、生物素、B6、叶酸、B12）：
- 共同特征：在能量代谢中作为辅酶
- B1（硫胺素）：TTP/TPP辅酶，参与CHO代谢。缺乏=脚气病（Beriberi）（干型=外周神经病变；湿型=心血管症状）
- B2（核黄素）：FAD和FMN辅酶，参与氧化还原反应。缺乏=口角炎、舌炎
- B3（烟酸）：NAD和NADP辅酶，参与能量代谢。缺乏=糙皮病（皮炎、腹泻、痴呆）
- B6（吡哆醇）：氨基酸代谢辅酶。缺乏=贫血、皮炎、神经病变
- 叶酸（B9）：DNA合成和一碳代谢。缺乏=巨幼细胞性贫血、神经管缺陷（NTDs）
- B12（钴胺素）：唯一含矿物质的维生素。缺乏=巨幼细胞性贫血+不可逆神经损害。需内因子辅助吸收
- 泛酸和生物素：广泛分布，缺乏罕见

**维生素C（Ascorbic Acid/抗坏血酸）**：
- 功能：胶原蛋白合成中的脯氨酸羟化→维持血管壁和结缔组织完整性；强抗氧化剂；促进非血红素铁吸收
- 缺乏：坏血病（牙龈出血、皮下出血、伤口不愈合）
- 食物来源：新鲜水果和蔬菜（柑橘类、猕猴桃、辣椒、西兰花）。维生素C对热敏感，烹饪可破坏

### 3. 水溶性和脂溶性维生素的关键区别

| 特征 | 水溶性 | 脂溶性 |
|------|--------|--------|
| 吸收 | 直接进入血液 | 首先进入淋巴 |
| 运输 | 自由运行 | 需要蛋白载体 |
| 储存 | 体内储存少 | 储存在脂肪组织 |
| 排泄 | 多余量由尿液排出 | 不易排泄，倾向于储存 |
| 中毒风险 | 一般较低 | 高剂量有风险（尤其A和D） |
| 需摄入频率 | 经常（1-3天） | 不需要那么频繁 |

### 争论7：维生素补充剂——利与弊？

探讨维生素补充剂在预防慢性病中是否有效。大型随机试验（如 Physicians' Health Study II, SELECT等）总体发现，常规服用多种维生素和矿物质补充剂对一般健康人群的死亡率、心血管疾病或癌症发生率无显著降低。某些特定补充场景有获益证据（如叶酸预防NTDs、维生素D/钙预防老年人骨折）。补充剂不是均衡膳食的替代品，食物中的多种生物活性物质的协同效应无法通过补充剂复制。
