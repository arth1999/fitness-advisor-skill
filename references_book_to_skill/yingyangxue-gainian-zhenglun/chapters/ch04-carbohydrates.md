# 第4章 碳水化合物：糖、淀粉、糖原和纤维

## 章节概述

碳水化合物（CHO）是人体首选的能量来源，尤其在脑和神经系统功能中至关重要。本章系统介绍CHO的化学分类、消化吸收、血糖调节、糖与健康以及膳食纤维的多种益处。

## 核心知识点

### 1. CHO的化学分类

**单糖**（最简单的糖单位）：
- 葡萄糖（血糖和基本细胞燃料）
- 果糖（水果甜味）
- 半乳糖（乳制品组成）

**双糖**（两个单糖结合）：
- 蔗糖 = 葡萄糖 + 果糖（食糖）
- 乳糖 = 葡萄糖 + 半乳糖（乳制品）
- 麦芽糖 = 葡萄糖 + 葡萄糖（淀粉消化产物）

**多糖**（多个葡萄糖单位组成的聚合物）：
- 淀粉：植物储存形式（谷类、土豆、豆类——膳食中的主要CHO）
- 糖原：动物和人体储存形式（肝脏和肌肉中）
- 纤维素：植物结构成分，人类无法消化（膳食纤维的主要组成）

### 2. 血糖调节

血糖浓度受精密激素调控：
- **胰岛素**（胰岛β细胞分泌）：降低血糖——促进细胞摄取葡萄糖和糖原、脂肪合成
- **胰高血糖素**（胰岛α细胞分泌）：升高血糖——促进肝糖原分解和糖异生
- 1型糖尿病：自身免疫破坏β细胞→胰岛素绝对缺乏
- 2型糖尿病：胰岛素抵抗→相对胰岛素缺乏（占糖尿病约90%-95%）
- 低血糖（Hypoglycemia）：血糖过低，可能导致头晕、乏力、意识障碍

### 3. 血糖指数（GI）与血糖负荷（GL）

- GI：含50 g CHO的食物餐后血糖应答与50 g参考食物（葡萄糖/白面包）的比值
- 影响GI的因素：CHO类型、纤维含量、脂肪/蛋白质含量、加工方式、成熟度
- GL = GI × 一份食物中可用CHO克数 / 100。既考虑CHO"质"也考虑"量"
- 低GI/GL膳食与降低2型糖尿病和心血管疾病风险关联

### 4. 添加糖与健康

过量添加糖（Added Sugars）的主要健康关切：
- 增加能量密度而无相应营养素供给（"空卡路里"）
- 与超重/肥胖、2型糖尿病和龋齿关联
- 含糖饮料是最大的单一添加糖来源
- 美国膳食指南推荐添加糖<总能量10%
- WHO推荐进一步限制到<5%总能量（获得更多健康益处时）

### 5. 膳食纤维

分类：可溶性纤维（如果胶、β-葡聚糖）和不溶性纤维（如纤维素、木质素）

健康益处：
- 促进规律肠道功能，预防便秘
- 可溶性纤维降低血胆固醇（结合胆汁酸）
- 延缓CHO消化吸收，改善血糖反应
- 增加饱腹感（有助于体重管理）
- 维持健康肠道微生物群（提供益生元）

推荐摄入量：女性25 g/d、男性38 g/d（实际平均摄入仅约15 g/d）

### 争论4：糖与添加糖的争议

探讨添加糖是否真的是导致肥胖和慢性病的"元凶"：
- 添加糖摄入与健康风险之间的关联证据存在（但主要是流行病学证据）
- 能量过剩（无论来源）是肥胖的根本原因
- 低糖产品的营销可能误导消费者（"低糖"不一定等于"健康"）
- 食品工业的利益关系和科学研究资助中的偏见
- 添加糖税作为公共政策选项的讨论

## 关键术语

- 单糖/双糖/多糖
- 胰岛素/胰高血糖素
- 血糖生成指数（GI）/血糖负荷（GL）
- 添加糖（Added Sugars）
- 膳食纤维（Dietary Fiber）
- 糖异生（Gluconeogenesis）
- 酮体（Ketone Bodies）
- 乳糖不耐受（Lactose Intolerance）
