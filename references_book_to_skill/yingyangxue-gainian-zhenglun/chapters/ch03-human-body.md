# 第3章 奇妙的人体

## 章节概述

本章精练地介绍人体功能，从细胞到器官系统，着重介绍消化系统及其微生物组。理解消化生理是理解营养素如何被利用的前提。本章还深入讨论酒精的代谢与健康效应。

## 核心知识点

### 1. 人体的层次组织

细胞 → 组织 → 器官 → 器官系统 → 完整人体

细胞是生命的基本单位。人体细胞含有全套基因（基因组），但不同类型的细胞表达不同的基因组合以执行特殊功能。

### 2. 消化系统概述

消化道是一条从口腔到肛门的连续管道，长约9米（30英尺）。依次经过：
- **口腔**：机械消化（咀嚼）和化学消化（唾液淀粉酶启动淀粉分解）
- **食管**：蠕动将食团推送至胃
- **胃**：胃酸（HCl）激活胃蛋白酶原、杀灭细菌；胃蛋白酶启动蛋白质消化；胃的搅拌作用形成食糜
- **小肠**（约7米）：十二指肠、空肠、回肠——消化和吸收的主要场所
- **大肠/结肠**：水分和电解质的重吸收，微生物发酵
- **直肠和肛门**：储存和排出粪便

### 3. 消化系统的附属器官

- **肝脏**：产生胆汁（脂肪乳化）、营养素代谢中心、储存糖原和多种维生素、解毒
- **胆囊**：储存和浓缩胆汁
- **胰腺**：分泌消化酶（胰蛋白酶、胰脂肪酶、胰淀粉酶等）和碳酸氢盐（中和胃酸）进入小肠；同时分泌胰岛素和胰高血糖素调控血糖

### 4. 吸收过程

营养素主要通过小肠绒毛（villi）和微绒毛（microvilli）上的肠细胞吸收，大大增加了吸收表面积。

- **碳水化合物**：以单糖形式吸收（葡萄糖、果糖、半乳糖）进入毛细血管→肝门静脉→肝脏
- **蛋白质**：以氨基酸、二肽和三肽形式吸收→肝门静脉→肝脏
- **脂肪**：以甘油一酯和脂肪酸形式吸收→肠细胞内再合成甘油三酯→乳糜微粒→淋巴系统→血液
- **水溶性维生素**：大部分直接吸收进入血液
- **脂溶性维生素**：随脂肪吸收路径
- **大多数矿物质**：按需要被调控吸收
- **水**：被动渗透方式吸收

### 5. 微生物组（Microbiome）

人体肠道内寄居着约100万亿个微生物（细菌、古菌、病毒、真菌），包含的基因数量是人类基因组的100倍以上。肠道微生物组的功能：
- 发酵未被消化的CHO和纤维→短链脂肪酸（SCFAs）
- 合成维生素K和部分B族维生素
- 调节免疫系统
- 影响体重和代谢（某些菌群组成与肥胖相关）
- 肠-脑轴的信号通路

膳食对微生物组有深远影响：高纤维膳食促进有益菌的增长和多样性。

### 6. 常见消化问题

**便秘（Constipation）**：膳食纤维摄入不足、液体摄入不足、缺乏运动是主要风险因素
**腹泻（Diarrhea）**：可能由感染、食物不耐受、肠易激综合征等引起——可能导致严重的液体和电解质丢失
**胃食管反流（GERD）**：胃酸反流入食管，可能由食管下括约肌功能减弱引起
**消化性溃疡**：幽门螺杆菌（H. pylori）感染和NSAIDs滥用是主要原因

### 争论3：酒精

核心问题：酒精对健康有益还是有害？
- 适度饮酒（女性≤1杯/天，男性≤2杯/天）可能与心血管保护相关（仍存争议，可能被混淆因素解释）
- 过量饮酒的健康风险明确：肝病（脂肪肝→肝硬化）、某些癌症（口咽、食管、肝脏、乳腺、结直肠）、高血压、脑损伤、营养不良
- 特定人群应完全禁酒：孕妇、肝病患者、某些药物使用者、有酒精依赖史者
- 酒精提供能量（7 kcal/g）但基本不含营养素（"空卡路里"）
