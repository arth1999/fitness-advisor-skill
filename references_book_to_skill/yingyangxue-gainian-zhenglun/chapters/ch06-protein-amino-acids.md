# 第6章 蛋白质与氨基酸

## 章节概述

蛋白质是生命的结构基础和工作分子。本章介绍蛋白质的氨基酸组成、蛋白质的合成与功能、膳食蛋白质质量的评价方法、蛋白质缺乏与过剩的后果以及素食膳食的蛋白质充分性问题。

## 核心知识点

### 1. 氨基酸与蛋白质结构

蛋白质由20种氨基酸组成，氨基酸通过肽键连接成多肽链，折叠成特定三维结构决定其功能。其中9种为**必需氨基酸**（人体不能合成）：组氨酸、异亮氨酸、亮氨酸、赖氨酸、蛋氨酸、苯丙氨酸、苏氨酸、色氨酸、缬氨酸。**条件必需氨基酸**在某些生理状态下可能需要从膳食获取（如疾病或应激时）。

### 2. 蛋白质的功能

- **结构组分**：胶原蛋白、角蛋白、肌肉收缩蛋白（肌动蛋白和肌凝蛋白）
- **酶**：催化所有生化反应
- **激素**：某些激素是蛋白质（如胰岛素、生长激素）
- **免疫**：抗体是蛋白质
- **运输**：血红蛋白运输氧气，脂蛋白运输脂类
- **液体平衡**：血浆蛋白（白蛋白）维持渗透压
- **酸碱平衡**：蛋白质作为缓冲剂
- **能量来源**：在CHO和脂肪不足时提供能量（4 kcal/g）

### 3. 蛋白质的消化与代谢

- 胃中HCl使蛋白质变性，胃蛋白酶启动分解
- 小肠中胰腺和小肠产生的蛋白酶继续分解为氨基酸、二肽和三肽
- 肠细胞吸收后进入肝门静脉→肝脏
- 氨基酸池（Amino Acid Pool）提供随时可用的氨基酸用于蛋白质合成
- 蛋白质转换（Protein Turnover）：体内持续进行蛋白质合成和分解的动态平衡
- 氮平衡：氮摄入量 = 氮排出量。正值=生长或恢复；负值=分解代谢状态

### 4. 膳食蛋白质质量

蛋白质质量取决于：
- **消化率**：动物蛋白通常比植物蛋白更易消化
- **氨基酸组成**：必需氨基酸的种类和比例与人体需求的接近程度
- **完全蛋白（Complete Protein）**：含所有必需氨基酸且比例适当——动物蛋白（肉、鱼、蛋、乳）和大豆蛋白
- **不完全蛋白（Incomplete Protein）**：缺乏一种或多种必需氨基酸——大多数植物蛋白
- **互补蛋白（Complementary Proteins）**：两种或多种不完全蛋白组合可提供所有必需氨基酸——例如谷物+豆类（经典搭配：大米+豆子）

蛋白质质量评价方法：PDCAAS和DIAAS

### 5. 蛋白质推荐摄入量

- 成人RDA：0.8 g/(kg·d)（基于氮平衡研究）
- AMDR：总能量的10%-35%
- 运动员：1.2-2.0 g/(kg·d)
- 美国成人实际平均摄入量远超RDA（男性约100g/d，女性约70g/d）

### 6. 蛋白质缺乏与过剩

**缺乏**（蛋白质-能量营养不良，PEM）：
- 消瘦症（Marasmus）：长期总能量+蛋白质严重缺乏→全身消瘦
- 夸希奥科病（Kwashiorkor）：能量摄入相对充足但蛋白质严重缺乏→水肿（低白蛋白血症）+脂肪肝+皮肤损伤

**过剩的潜在后果**（存在争议）：
- 高蛋白膳食对健康肾脏人群通常安全
- 极为高蛋白（>2.5-3 g/kg/d）+已有肾病→加速肾功能衰退
- 动物蛋白的高硫氨基酸含量可能促进尿钙排出
- 高红肉和加工肉类摄入与结直肠癌和心血管疾病风险增加相关

### 争论6：素食与健康

素食膳食模式与较低的BMI、心血管疾病、高血压、2型糖尿病和某些癌症风险相关。关键是是否能通过多种植物食物满足所有必需营养素需求：
- 纯素食者需补充维生素B12（绝对必需）
- 铁和锌的生物利用度需要关注
- 钙摄入在去掉乳制品时需要规划
- 蛋白质通过多样化植物来源+互补效应可以满足
