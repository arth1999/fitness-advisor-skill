# 第2章 营养学工具——标准与指南

## 章节概述

本章介绍营养学中用于评价和计划膳食的核心工具体系，包括膳食营养素参考摄入量（DRIs）、膳食指南、食品标签以及膳食计划方法。这些工具将营养科学转化为可操作的膳食建议。

## 核心知识点

### 1. 膳食营养素参考摄入量（DRIs）

DRIs是一组用于计划和评价健康人群膳食的营养素摄入量参考值：
- **EAR（平均需要量）**：满足50%个体需要的量
- **RDA（推荐膳食供给量）**：满足97%-98%个体需要的量 = EAR + 2SD
- **AI（适宜摄入量）**：证据不足以确定EAR/RDA时的替代值
- **UL（可耐受最高摄入量）**：不产生不良效应的最高日摄入量
- **AMDR（宏量营养素可接受范围）**：蛋白质10%-35%、脂肪20%-35%、CHO 45%-65%
- **EER（估计能量需要量）**：维持能量平衡的膳食能量摄入预测值

### 2. 膳食指南

**《美国居民膳食指南（2015-2020）》**五大核心指南：
1. 一生遵循健康膳食模式
2. 关注多样化、营养素密度和摄入量
3. 限制来自添加糖和饱和脂肪的能量，减少钠摄入
4. 转向更健康的食物和饮料选择
5. 支持所有人选择的健康膳食模式

**遵循健康膳食模式的关键建议**：
- 在各种食物组中选择多样化的高营养素密度食物
- 蔬菜（深绿色、红色和橙色、豆类、淀粉类等）
- 水果（尤其完整水果）
- 谷物（至少一半为全谷物）
- 脱脂或低脂乳制品（或强化大豆饮料）
- 多样化的蛋白质食物
- 油（以不饱和脂肪替代饱和脂肪）

### 3. 食品标签

**营养成分标签**的核心要素：
- 份量（Serving Size）：基于实际消费量调整的标准化份量
- 能量（Calories）：每份提供的千卡数
- 营养素含量：需限制的（饱和脂肪、反式脂肪、钠、添加糖）和需充足摄入的（膳食纤维、维生素D、钙、铁、钾）
- %每日值（%DV）：基于每日2000 kcal膳食的参考值

### 4. 膳食计划方法

**MyPlate（我的餐盘）**：将餐盘分为五组
- 水果：约20%
- 蔬菜：约30%
- 谷物：约30%
- 蛋白质：约20%
- 乳制品：在旁边（如一杯牛奶）

**食物交换份系统**：将食物按CHO、蛋白质和脂肪含量分类，便于糖尿病患者和体重管理者进行精确的食物交换。

### 5. 植物化合物

植物中非必需但有生物活性的化学成分（如类黄酮、类胡萝卜素、植物固醇等）。许多植物化合物具有抗氧化、抗炎或其他潜在健康保护功能。食物优于补充剂——天然食物提供多种植物化合物的协同效应。

### 争论2：膳食补充剂——利与弊

核心议题：是否需要膳食补充剂？
- 理想情况下，均衡膳食可满足大多数健康人的营养需求
- 特定人群可能需要补充：育龄妇女（叶酸）、孕妇（铁、叶酸）、老年人（B12、维生素D）、纯素食者（B12）、日照不足者（维生素D）
- 大多数补充剂的健康声称缺乏足够科学证据
- 过量补充的风险：某些维生素和矿物质的安全范围较窄（脂溶性维生素A和D、铁等的毒性）
- 补充剂不受与药品同样严格的FDA审批监管

## 关键术语

- AI（Adequate Intake）

- AMDR（Acceptable Macronutrient Distribution Range）
- %每日值（%Daily Value, %DV）
- 营养素密度（Nutrient Density）
