# 第11章 营养与慢性疾病

## 章节概述

本章将前十章所学的营养学原理综合应用于慢性病的预防和管理，聚焦心血管疾病（CVD）、糖尿病和癌症这三大主要营养相关慢性病。强调营养在疾病预防中的核心地位——无论是作为独立危险因素还是与其他风险因素协同作用。

## 核心知识点

### 1. 心血管疾病（CVD）的营养预防

CVD是美国和中国的首要死因。动脉粥样硬化是CVD的病理基础。

**主要的膳食相关CVD风险因素**：
- 高LDL胆固醇：SFA和反式脂肪是主要膳食升高因素
- 高血压：高钠摄入是最重要的膳食因素；低钾、低钙、低镁也可能参与
- 肥胖/代谢综合征
- 糖尿病（显著增加CVD风险）
- 炎症（hs-CRP等标志物）

**保护性膳食策略**：
- 以不饱和脂肪（尤其MUFA和ω-3 PUFA）替代SFA
- 增加可溶性纤维（≥5-10 g/d可额外降低LDL 3%-5%）
- DASH膳食模式：高水果蔬菜、低脂乳制品、全谷物、低钠高钾
- 地中海膳食模式：丰富的植物性食物、橄榄油为主要脂肪来源、适量鱼和红酒
- 限制加工肉类和含糖饮料
- 保持健康体重 + 规律身体活动

**血压分类**（美国心脏协会标准）：正常<120/<80；升高120-129/<80；高血压1期130-139/80-89；高血压2期≥140/≥90 mmHg

### 2. 糖尿病

**2型糖尿病的病理**：胰岛素抵抗（肌肉、肝脏和脂肪组织对胰岛素反应减弱） + 逐渐的β细胞功能衰竭。

**营养在2型糖尿病中的作用**：
- 减重是核心治疗策略（5%-10%体重减轻可产生临床意义改善）
- CHO"质"比"量"更重要——全谷物和膳食纤维优于精制CHO
- 血糖指数和血糖负荷的概念在膳食管理中有实用价值
- 不限任何单一营养素的均衡中度CHO膳食可接受

**1型糖尿病**：自身免疫破坏胰岛β细胞。营养治疗重点是使外源性胰岛素与CHO摄入和身体活动同步匹配——**碳水化合物计数**是核心技能。

**糖尿病的常见误区**：吃太多糖"导致"糖尿病（不准确——2型糖尿病是多因素疾病）；糖尿病患者不能吃任何CHO（错误——需要的是管理而非消除）；"糖尿病专用"食品是必需的（错误——常规健康膳食即足够）

### 3. 癌症

癌症是细胞生长失控的一组疾病。致癌过程是多阶段的：启动（Initiation）→ 促进（Promotion）→ 进展（Progression）。营养因素主要影响促进和进展阶段。

**WCRF/AICR持续更新的癌症预防膳食建议**：
- 保持健康的体重（在所有建议中优先级最高——肥胖与至少13种癌症相关）
- 规律身体活动
- 食用富含全谷物、蔬菜、水果和豆类的膳食
- 限制快餐和高度加工食品
- 限制红肉和加工肉类
- 限制含糖饮料
- 限制酒精
- 不要使用补充剂来预防癌症（食物而非补充剂）
- 对于母亲：尽可能母乳喂养

**确立的膳食-癌症风险关联**（国际癌症研究机构IARC分类）：
- 加工肉类→结直肠癌（1类——确立因果）
- 红肉→结直肠癌（2A类——很可能致癌）
- 酒精→口咽、食管、肝、乳腺、结直肠等多种癌症（1类）
- 黄曲霉毒素→肝癌（1类）

### 4. 精准医学（Precision Medicine）

精准医学考虑个体在基因、环境和生活方式上的差异来定制个性化的预防和治疗方法：
- 营养基因组学（Nutrigenomics）：研究营养素如何影响基因表达
- 营养遗传学（Nutrigenetics）：研究基因变异如何影响对营养素和膳食的反应
- 虽然DNA检测的商业化越来越普及，但基于基因信息的个体化膳食建议目前仍然是非常有限且需谨慎对待的领域

### 争论11：营养基因组学

探讨DNA检测和个性化营养是否已准备好进入主流临床实践。当前证据表明，在许多情况下，针对特定基因型的膳食建议并未产生比"标准"健康膳食建议更好的结局。这项技术有巨大潜力但还在发展中。
