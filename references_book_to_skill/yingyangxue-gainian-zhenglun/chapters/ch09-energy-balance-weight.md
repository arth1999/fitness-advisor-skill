# 第9章 能量平衡与健康的体重

## 章节概述

本章深入探讨能量平衡——能量摄入与消耗的动态关系，以及它如何决定体重变化。从肥胖的流行现状到饮食紊乱（包括神经性厌食症和贪食症），覆盖体重管理的完整图谱。

## 核心知识点

### 1. 能量平衡的概念

- 能量平衡 = 能量摄入（食物/饮料） = 能量消耗（BMR+TEF+身体活动）
- 正能量平衡 → 体重增加
- 负能量平衡 → 体重减轻
- 约3500 kcal能量盈余/赤字 ≈ 约0.45 kg（1磅）体脂变化（这是简化估算，实际并非绝对精确）

### 2. 能量摄入

- 食物能量由CHO（4 kcal/g）、蛋白质（4 kcal/g）、脂肪（9 kcal/g）和酒精（7 kcal/g）提供
- 食欲调节是复杂的神经内分泌系统，涉及：下丘脑（整合中枢）、饥饿激素（Ghrelin）、饱腹激素（Leptin、PYY、CCK等）
- 现代食物环境（高能量密度、超级美味、无处不在、大份量）超越生物调节机制的能力

### 3. 能量消耗的组成

**基础代谢率（BMR）**：
- 占不爱活动者TDEE的约60%-75%
- 主要决定因素：瘦体重（最相关）、年龄（随年龄下降）、性别（女性BMR较低）
- 限制能量摄入（"节食"）可降低BMR——代谢适应（Metabolic Adaptation）

**食物生热效应（TEF）**：
- 约占总能量消耗的10%
- 不同营养素生热效应不同：蛋白质（20%-30%）> CHO（5%-10%）> 脂肪（0%-3%）

**身体活动（Physical Activity）**：
- 变化最大且意志可控的成分
- 包括：有目的的运动 + 非运动性活动生热作用（NEAT）+ 姿势维持

### 4. 体重指标与肥胖

**BMI（体重指数）**：
- BMI = 体重(kg) / 身高²(m²)
- 分类：<18.5低体重、18.5-24.9正常、25-29.9超重、≥30肥胖
- BMI的优点和局限：简单但不能区分脂肪和瘦体重

**腰围**：
- 反映腹部脂肪（内脏脂肪）的分布，是心血管和代谢疾病的独立危险因素
- 男性>102 cm（40英寸）、女性>88 cm（35英寸）为高风险

**肥胖的病因**：
- 多层次因素：遗传（约40%-70%遗传度）、环境（食物环境、身体活动环境）、行为、心理、社会经济
- 肥胖作为"基于肥胖的慢性疾病"（ABCD）的新概念
- 肠道微生物组在能量收获中的潜在作用

### 5. 减重策略

减重的核心原则：创造能量赤字。

**生活方式干预**（一线治疗）：
- 适中能量限制（减少约500-1000 kcal/d → 降低0.5-1 kg/周）
- 增加身体活动（每周≥150-300分钟中等强度）
- 行为改变策略：自我监测、目标设定、刺激管理、认知重建
- 饮食摄入记录可促进减重

**药物治疗**：BMI≥30或≥27+合并症时可考虑

**减重手术（Bariatric Surgery）**：
- BMI≥40或≥35+严重合并症时考虑
- 胃旁路术（RYGB）和袖状胃切除术是目前最常见的手术
- 效果：通常可减少初始体重25%-35%，并大幅改善或缓解2型糖尿病

### 6. 饮食紊乱

**神经性厌食症（Anorexia Nervosa）**：
- 极度限制能量摄入 → 显著低体重，对体重增加的强烈恐惧，体像障碍
- 健康后果：心血管（心动过缓、低血压）、骨丢失（骨质疏松）、内分泌紊乱（闭经）、电解质紊乱、死亡风险高

**神经性贪食症（Bulimia Nervosa）**：
- 反复暴食发作 + 不适当的代偿行为（催吐、泻药滥用、过度运动）
- 常维持正常体重或超重，使识别更困难
- 健康后果：电解质紊乱、牙釉质侵蚀（胃酸）、食道损伤、胃肠道问题

**暴食症（Binge Eating Disorder）**：
- 反复暴食发作但无代偿行为。最常见的饮食紊乱

### 争论9：流行的体重管理方案

探讨各种流行减重膳食（低碳水、低脂、地中海、间歇性禁食等）的证据：
- 大多数严格方案可实现短期减重
- 长期（>1年）效果差异通常不大——坚持性是最重要的预测因素
- 不损害营养充足性的方案（如地中海膳食）在长期健康结局上可能更有优势
- "最佳膳食"是个人能坚持的健康膳食
