# 第十二章 行为理论和运动促进策略

> ACSM's Guidelines Chapter 12: Behavioral Theories and Strategies for Promoting Exercise

## 核心概念

运动处方能否成功实施，不仅取决于科学的运动方案设计，更取决于个体的行为改变和长期依从性。

## 第一节 修正运动处方

### 提高运动依从性的原则
1. 运动处方必须个性化，考虑个体偏好、障碍、日程、社会环境
2. 运动计划应灵活可调（频率、强度、时间、方式均可适应个体变化）
3. 关注过程目标（如每周完成3次运动）而非仅结果目标（如减重5kg）
4. 识别并减少参与障碍（时间、成本、交通、身体不适、自我效能低）
5. 使用行为干预策略增强依从性

---

## 第二节 理解运动行为的理论基础

### 社会认知理论（Social Cognitive Theory, SCT）
- **自我效能感（Self-Efficacy）**：个体对自己完成运动任务能力的信心，是运动行为最强预测因子之一
- **结果期望（Outcome Expectations）**：对运动结果的预期（身体、心理、社交层面）
- **自我调节（Self-Regulation）**：目标设定、自我监控、计划能力

### 跨理论模型（Transtheoretical Model, TTM）
**行为改变阶段**：

| 阶段 | 特征 | 干预策略 |
|-----|------|---------|
| 前意向期（Precontemplation） | 未来6个月内无运动打算 | 提供信息，增加意识 |
| 意向期（Contemplation） | 有运动打算但尚未行动 | 激发动机，减少障碍 |
| 准备期（Preparation） | 开始有一些不规律运动 | 设定具体目标，制订计划 |
| 行动期（Action） | 规律运动但不足6个月 | 强化行为，预防反复 |
| 维持期（Maintenance） | 规律运动超过6个月 | 防止厌倦，处理中断 |

### 自我决定理论（Self-Determination Theory, SDT）
- **自主性（Autonomy）**：个体主动选择运动（而非被迫）
- **能力感（Competence）**：感知到进步和掌握感
- **关联性（Relatedness）**：与他人建立联系
- 内在动机比外在动机更能预测长期运动坚持

### 健康信念模型（Health Belief Model）
- 感知易感性（Perceived Susceptibility）
- 感知严重性（Perceived Severity）
- 感知获益（Perceived Benefits）
- 感知障碍（Perceived Barriers）
- 行动线索（Cues to Action）

## 第三节 增加体力活动的策略和方法

### 循证行为改变技术（BCTs）
| 策略 | 具体方法 |
|-----|---------|
| 目标设定 | SMART目标（具体、可测量、可达成、相关性、时限性） |
| 自我监控 | 运动日志、计步器、智能手机App、心率监测器 |
| 反馈 | 定期反馈进展，使用客观数据 |
| 社会支持 | 运动伙伴、团体运动课程、家庭支持 |
| 问题解决 | 识别障碍，预设计划（应对计划） |
| 预防反复 | 识别高风险情境，制订应对策略 |
| 激励 | 奖励、行为契约 |
| 环境重建 | 准备运动装备、预排运动时间、选择便利场地 |

### 有效的行为改变干预模式
- **动机性访谈（Motivational Interviewing）**：以客户为中心，探索和解决矛盾心理
- **认知行为疗法（CBT）技巧**：识别和挑战阻碍运动的负面思维
- **5A模式**：评估（Assess）→ 建议（Advise）→ 同意（Agree）→ 协助（Assist）→ 安排随访（Arrange）

### 科技辅助手段
- 可穿戴设备（计步器、加速度计、心率监测器）
- 智能手机应用程序
- 社交媒体支持群体
- 线上教练/远程健康指导
- 互动游戏（Exergaming/Active Video Games）

## 第四节 特殊人群的行为策略

### 老年人
- 注重社会支持和团体活动
- 强调运动的功能性获益（保持独立生活能力）
- 使用客观身体功能测试反馈进展
- 提供安全保障和适当监督
- 考虑交通和可达性

### 慢性疾病患者
- 对症状的恐惧教育（区分正常运动感与警示症状）
- 医学专业人士的推荐和支持至关重要
- 阶段式行为目标（从频率和持续时间开始，再增加强度）
- 团体心脏康复/肺部康复提供同伴支持

### 儿童和青少年
- 以游戏和趣味性为核心
- 多样化活动选择（避免过早专项化）
- 父母/家庭参与和榜样作用
- 减少屏幕时间作为干预目标之一
- 利用学校体育活动和课外运动

### 社会经济弱势群体
- 低成本或免费运动选择（步行、自重训练）
- 社区资源和公园利用
- 文化敏感的运动计划
- 关注基础需求（安全、交通、儿童照护）
