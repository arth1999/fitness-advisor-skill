# 第二章 运动前评估

> ACSM's Guidelines Chapter 2: Pre-Exercise Evaluation

## 核心概念

运动前评估的四个步骤：知情同意书 → 运动前健康筛查 → 健康史评估 → 心血管疾病危险因素分析。

## 1. 知情同意书（Informed Consent）

在所有体适能测试和运动参与前必须获得。必须包含：
- 测试目的和说明
- 可能出现的风险和不适
- 参与者的责任
- 预期获益
- 咨询渠道
- 医疗记录的用途（HIPAA合规）
- 自愿参与声明

## 2. ACSM运动前健康筛查流程

### 筛查目的
识别具有运动相关负性心血管事件风险的个体。ACSM不再推荐运动测试作为常规筛查的一部分。

### 两步筛查法

**步骤一**：用ACSM运动前健康筛查流程确定是否需要医学筛查（由运动专家或通过PAR-Q+自我筛查）。

**步骤二**：如有指征，转诊至医生进行医学筛查（筛查方式由医生临床判断）。

### 筛查流程核心要素

1. **确定当前体力活动水平**：是否有规律运动习惯（过去3个月，每周≥3天，每天≥30min中等强度）
2. **确定心血管/代谢/肾脏疾病的主要症状和体征**（表2.1）
3. **确定已确诊的心血管、代谢或肾脏疾病**
4. **确定预期运动强度**（低/中等/较高）

### 疾病症状和体征（表2.1关键项）
| 症状/体征 | 临床意义 |
|-----------|---------|
| 缺血性胸/颈/下颌/手臂疼痛 | 冠状动脉疾病主要表现 |
| 休息或轻度活动时呼吸困难 | 心肺功能失调 |
| 眩晕或晕厥 | 心脏功能失调（潜在致命） |
| 端坐呼吸或夜间阵发性呼吸困难 | 左心室功能紊乱 |
| 脚踝水肿 | 心力衰竭/静脉功能不全 |
| 心悸或心动过速 | 心律失常 |
| 间歇性跛行 | 外周动脉疾病 |
| 确诊的心脏杂音 | 瓣膜疾病（排除HCM和主动脉狭窄） |
| 日常活动时异常疲劳或呼吸困难 | 心血管/代谢性疾病可能 |

### 运动强度定义
| 强度 | %HRR或%VO₂R | METs | RPE (6-20) |
|-----|-----------|------|-----------|
| 低 | <30% | <2.0 | <9 |
| 较低 | 30-39% | 2.0-2.9 | 9-11 |
| 中等 | 40-59% | 3.0-5.9 | 12-13 |
| 较高 | 60-89% | 6.0-8.7 | 14-17 |
| 次最高到最高 | ≥90% | ≥8.8 | ≥18 |

### 筛查流程图结果分类（图2.2）
6类人群分类基于：运动习惯(有/无) × 确诊疾病(有/无) × 预期运动强度(中/高)

**关键决策规则**：
- 无症状+无疾病+低到中等强度运动 → 不需要医学筛查
- 有症状 → 需要医学筛查
- 无运动习惯+有疾病 → 推荐医学筛查
- 有运动习惯+有疾病 → 可继续低到中等强度运动，较高强度需医学筛查

## 3. 自我筛查工具

**PAR-Q+ (2020版)**：7个常规健康问题 + 补充疾病问题（第2-3页），覆盖关节炎、癌症、心脏病、高血压、代谢疾病、精神问题、呼吸系统疾病、脊髓损伤、中风、多病共存等10个方面。

## 4. 心血管疾病危险因素评估（表2.2）

| 危险因素 | 定义标准 |
|---------|---------|
| 年龄 | 男性≥45岁，女性≥55岁 |
| 家族史 | 一级亲属：男性<55岁或女性<65岁发生心梗、冠脉重建或猝死 |
| 吸烟 | 当前吸烟者或戒烟<6个月，或暴露于环境烟雾 |
| 久坐少动 | 每周<3天、每天<30min中等强度体力活动，持续至少3个月 |
| 肥胖 | BMI≥30 kg/m²或腰围男性>102cm、女性>88cm |
| 高血压 | 收缩压≥130mmHg或舒张压≥80mmHg，或服用降压药 |
| 血脂异常 | LDL≥130mg/dL或HDL<40mg/dL（男性）/HDL<50mg/dL（女性），或服用降脂药 |
| 糖尿病 | 空腹血糖≥126mg/dL或HbA1c≥6.5% |

## 5. 临床患者危险分层（AACVPR）

### 高危（任何一项）：
- LVEF <40%
- 心脏骤停或猝死史
- 安静或运动时严重心律失常
- 心梗/心脏手术并发心源性休克/心衰
- 运动时血流动力学异常
- 严重无症状缺血（ST段压低≥2mm）
- 低强度运动出现症状（<5 METs）
- 最大功能能力<5 METs
- 临床严重抑郁

### 中危：
- LVEF 40%-50%
- 中等强度运动出现心绞痛症状
- 轻到中度无症状缺血（ST段压低<2mm）

### 低危（全部满足）：
- LVEF >50%
- 无严重心律失常
- 无心力衰竭/心梗并发症
- 运动时血流动力学正常
- 功能能力≥7 METs
- 无临床显著抑郁
