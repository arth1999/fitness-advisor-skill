# 第五章 运动处方的基本原则

> ACSM's Guidelines Chapter 5: General Principles of Exercise Prescription

## FITT-VP原则概述

**Ex R_x**（Exercise Prescription，运动处方）应包含：

| 要素 | 英文 | 含义 |
|-----|------|------|
| F | Frequency | 频率：多久一次 |
| I | Intensity | 强度：费力程度 |
| T | Time | 时间：持续时间 |
| T | Type | 方式：方式或类型 |
| V | Volume | 总量：频率×强度×时间 |
| P | Progression | 进阶：如何逐渐增加 |

## 1. 运动训练课的基本组成

每次训练课三阶段：
1. **热身（Warm-up）**：5-10min低到中等强度动态活动，不超过15min
2. **健身训练（Conditioning）**：10-60min，包含有氧、抗阻、柔韧性
3. **整理活动（Cool-down）**：逐渐降低强度

## 2. 心肺耐力（有氧）运动FITT建议

| FITT | 建议 |
|------|------|
| 频率 | 每周≥3天；分散在3-5天最优 |
| 强度 | 中等（40%-59% HRR或VO₂R）或较高（60%-89% HRR或VO₂R） |
| 时间 | 每天30-60min中等强度（每周≥150min）；或每天20-60min较高强度（每周≥75min） |
| 方式 | 动用大肌群的持续性或间歇性有氧运动 |
| 总量 | ≥500-1,000 MET-min/wk；或每周≥1,000 kcal |
| 进阶 | "低起点，缓慢加"；每1-2周延长5-10min；4-6周后调整强度 |

### 有氧运动强度确定方法（表5.2）

| 强度 | %HRR/%VO₂R | %HRmax | %VO₂max | RPE(6-20) | METs |
|-----|-----------|--------|---------|----------|------|
| 低 | <30 | <57 | <37 | <9 | <2.0 |
| 较低 | 30-39 | 57-63 | 37-45 | 9-11 | 2.0-2.9 |
| 中等 | 40-59 | 64-76 | 46-63 | 12-13 | 3.0-5.9 |
| 较高 | 60-89 | 77-95 | 64-90 | 14-17 | 6.0-8.7 |
| 次最高到最高 | ≥90 | ≥96 | ≥91 | ≥18 | ≥8.8 |

### 强度计算公式（专栏5.2）
- **HRR法**：靶心率 = (HRmax/peak - HRrest) × 期望强度% + HRrest
- **VO₂R法**：靶VO₂R = (VO₂max/peak - VO₂rest) × 期望强度% + VO₂rest
- **HR法**：靶心率 = HRmax/peak × 期望强度%
- **MET法**：靶METs = [VO₂max/peak ÷ 3.5 mL/(kg·min)] × 期望强度%

### 最大心率预测公式（表5.3）
| 作者 | 公式 | 适用人群 |
|------|------|---------|
| Tanaka等 | 208 - 0.7 × 年龄 | 健康男女 |
| Gellish等 | 207 - 0.7 × 年龄 | 所有年龄段成年男女 |
| Gulati等 | 206 - 0.88 × 年龄 | 无症状中年女性 |

### 有氧运动类型分组（表5.4）
| 分组 | 特点 | 举例 |
|-----|------|------|
| A | 最少技能/体适能的耐力活动 | 步行、休闲自行车、水中有氧运动、慢舞 |
| B | 最少技能的高强度耐力运动 | 跑步、划船、有氧健身操、动感单车、爬台阶 |
| C | 需要技能的耐力运动 | 游泳、越野滑雪、滑冰 |
| D | 休闲运动 | 网球、篮球、足球、徒步旅行 |

### 间歇训练方案示例（专栏5.1）
- **HIIT (4×4方案)**：4组×90-95%峰值心率运动4min，组间休息3min
- **SIT (3×20方案)**：3组×全力运动20s，组间休息2min
- **抗阻间歇训练**：杠铃混合训练，每组5次重复，组间休息2min
- **HIFT**：3动作（400m跑+10卧推+10波比跳）×3轮

## 3. 抗阻训练FITT建议（表5.5）

| FITT | 建议 |
|------|------|
| 频率 | 初学者每周2-3天/肌群；有经验者可根据喜好选择（量比频率重要） |
| 强度 | 初学者60%-70% 1-RM，重复8-12次；有经验者根据目标调整 |
| 方式 | 多关节练习+单关节练习+核心训练；主动肌与拮抗肌结合 |
| 总量 | 初学者每个肌群每周<5组即可；有经验者与肌肥大/力量有剂量-反应关系 |

### 不同目标的抗阻训练建议
| 目标 | 强度 | 重复次数 | 组数 | 休息时间 |
|-----|------|---------|------|---------|
| 一般肌适能 | 60%-70% 1-RM | 8-12次 | 1-3组 | 60-120s |
| 最大力量 | ≥80% 1-RM | 1-6次 | 2-6组 | ≥2min |
| 肌肥大 | 30%-80% 1-RM（力竭） | 6-20次 | 3-6组 | 60-120s |
| 爆发力 | 上肢30%-60% 1-RM/下肢0-60% 1-RM | 3-6次（最快速度） | 1-3组 | ≥2min |
| 局部肌肉耐力 | 小负荷 | 15-25+次 | 2-4组 | <60s |

### 进阶原则：递增超负荷（Progressive Overload）
- 增加负荷5%
- 负荷不变，增加重复次数
- 增加每周训练组数或天数

## 4. 柔韧性练习FITT建议（表5.6）

| FITT | 建议 |
|------|------|
| 频率 | 每周2-3天，每天练习效果最佳 |
| 强度 | 拉伸到感觉紧绷或轻微不适 |
| 时间 | 静力性拉伸保持10-30s（老年人30-60s）；PNF：3-6s收缩(20%-75%MVC)+10-30s辅助拉伸 |
| 方式 | 静力性、动力性、弹振拉伸、PNF均有效 |

### 拉伸类型（专栏5.5）
1. **弹振拉伸**：利用肢体运动产生的势能快速拉伸
2. **动力性拉伸**：反复多次重复动作，逐步增加关节活动范围（推荐热身中使用）
3. **静力性拉伸**：缓慢拉伸到某位置后静止保持10-30s
4. **PNF**：等长收缩后对肌肉/肌腱群进行拉伸（收缩-放松法）

### 关键建议
- 每个柔韧性练习重复2-4次，累计拉伸时间达到90s
- 拉伸到紧绷或轻微不适即可，不超过疼痛范围
- 运动前推荐动力性拉伸；长时间静力性拉伸（>60s）可能影响运动表现
- 针对主要肌腱单元：肩带、胸部、颈部、躯干、下背部、臀部、大腿前后、脚踝
