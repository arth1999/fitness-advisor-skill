# 术语表 — ACSM运动测试与运动处方指南（第十一版）

> Glossary of Key ACSM Terms (English-Chinese)

## 核心概念

| ACSM英文术语 | 中文翻译 | 定义 |
|-------------|---------|------|
| Physical Activity (PA) | 体力活动 | 任何由骨骼肌收缩引起的导致能量消耗增加的身体动作 |
| Exercise (Ex) | 运动 | 有计划、结构化、重复性的体力活动，旨在改善或维持体适能 |
| Exercise Prescription (Ex R_x) | 运动处方 | 根据个体情况制定的，包含频率、强度、时间、方式、总量、进阶的运动计划 |
| Health-Related Physical Fitness | 健康相关体适能 | 与健康直接相关的体适能组成部分：心肺耐力、身体成分、肌肉适能、柔韧性 |
| Skill-Related Physical Fitness | 技能相关体适能 | 与运动表现相关的体适能：敏捷性、协调性、平衡、爆发力、反应时、速度 |
| Sedentary Behavior | 久坐行为 | 能量消耗≤1.5 METs的清醒状态下的坐、躺行为 |

## 运动处方原则

| ACSM英文术语 | 中文翻译 | 定义 |
|-------------|---------|------|
| FITT-VP | FITT-VP原则 | 频率(F)、强度(I)、时间(T)、方式(T)、总量(V)、进阶(P) — 运动处方六大要素 |
| Frequency (F) | 频率 | 每周执行运动计划的天数 |
| Intensity (I) | 强度 | 运动的费力程度，可用%HRR、%VO₂max、RPE、METs等表示 |
| Time (T) | 时间 | 每次运动训练的持续时间或每日/每周累计时间 |
| Type (T) | 方式 | 运动的方式或类型（有氧、抗阻、柔韧性等） |
| Volume (V) | 总量 | 频率×强度×时间，常用MET-min/wk或kcal/wk表示 |
| Progression (P) | 进阶 | 逐渐增加运动处方的FITT中各组成部分的过程 |

## 生理指标

| ACSM英文术语 | 中文翻译 | 定义 |
|-------------|---------|------|
| VO₂max (Maximal Oxygen Uptake) | 最大摄氧量 | 机体在最大强度运动中运输和利用氧气的最大能力，心肺耐力的金标准 |
| VO₂peak (Peak Oxygen Uptake) | 峰值摄氧量 | 在递增负荷运动测试中达到的最高摄氧量 |
| HRmax (Maximal Heart Rate) | 最大心率 | 最大运动强度时达到的最高心率 |
| HRrest (Resting Heart Rate) | 安静心率 | 坐位休息≥5min后的心率 |
| HRR (Heart Rate Reserve) | 储备心率 | HRmax - HRrest |
| VO₂R (VO₂ Reserve) | 储备摄氧量 | VO₂max - VO₂rest |
| RPE (Rating of Perceived Exertion) | 主观用力感觉 | 个体对运动用力程度的主观评分（Borg 6-20量表或CR10 0-10量表） |
| MET (Metabolic Equivalent) | 代谢当量 | 1 MET = 3.5 mL/(kg·min)，安静坐位时的能量代谢率 |
| RER (Respiratory Exchange Ratio) | 呼吸交换率 | VCO₂/VO₂，反映代谢底物利用，≥1.15提示接近最大努力 |
| AT (Anaerobic Threshold) | 无氧阈/乳酸阈 | 运动强度超过有氧代谢能力、开始积累乳酸的转折点 |
| SBP/DBP (Systolic/Diastolic Blood Pressure) | 收缩压/舒张压 | 心脏收缩/舒张时动脉血压 |
| RPP (Rate Pressure Product) | 心率血压乘积 | HR × SBP，间接反映心肌耗氧量 |
| SpO₂ | 外周血氧饱和度 | 无创脉搏血氧仪测量的动脉血氧饱和度百分比 |

## 训练术语

| ACSM英文术语 | 中文翻译 | 定义 |
|-------------|---------|------|
| 1-RM (One-Repetition Maximum) | 一次最大重复 | 某一特定动作只能完成一次的最大重量 |
| HIIT (High-Intensity Interval Training) | 高强度间歇训练 | ≥80%-100%峰值心率的高强度运动与休息交替 |
| SIT (Sprint Interval Training) | 短期冲刺间歇训练 | 全力或超极限运动（≥100% VO₂peak）与休息交替 |
| PNF (Proprioceptive Neuromuscular Facilitation) | 本体感觉神经肌肉促通术 | 等长收缩后进行拉伸的柔韧性训练方法 |
| Progressive Overload | 递增超负荷 | 随适应逐渐增加训练刺激以持续进步的原则 |
| LME (Local Muscular Endurance) | 局部肌肉耐力 | 参与运动的肌群维持重复收缩的能力 |

## 筛查与测试

| ACSM英文术语 | 中文翻译 | 定义 |
|-------------|---------|------|
| PAR-Q+ (Physical Activity Readiness Questionnaire) | 体力活动准备问卷升级版 | 运动前自我筛查工具，包含7个常规问题+补充疾病问题 |
| Informed Consent | 知情同意书 | 参与者了解测试/运动方案目的、风险、获益后的自愿同意文件 |
| CPET (Cardiopulmonary Exercise Testing) | 心肺运动测试 | 结合气体分析的运动测试，测量VO₂、VCO₂、通气量等 |
| GXT (Graded Exercise Test) | 递增负荷运动测试 | 逐级增加运动负荷至症状限制或达到预定终点的测试 |
| 6-MWT (6-Minute Walk Test) | 6分钟步行测试 | 6min内可行走的最远距离，评估功能能力 |
| SPPB (Short Physical Performance Battery) | 简易身体活动能力测试 | 下肢功能测试（步速+平衡+坐站），0-12分 |
| TUG (Timed Up and Go) | 起立行走测试 | 从椅子站起、行走3m、转身、返回、坐下的计时测试 |
| BESS (Balance Error Scoring System) | 平衡误差评分系统 | 3种站立姿势×2种表面，评估静态平衡 |

## 临床术语

| ACSM英文术语 | 中文翻译 | 定义 |
|-------------|---------|------|
| CVD (Cardiovascular Disease) | 心血管疾病 | 涉及心脏或血管的疾病 |
| CAD (Coronary Artery Disease) | 冠状动脉疾病 | 心脏动脉的动脉粥样硬化性疾病 |
| MI (Myocardial Infarction) | 心肌梗死 | 心肌组织缺血性坏死（心脏病发作） |
| HF (Heart Failure) | 心力衰竭 | 以劳力性呼吸困难和疲劳为特征的临床综合征 |
| LVEF (Left Ventricular Ejection Fraction) | 左心室射血分数 | 左心室每次收缩射出的血液百分比 |
| PAD (Peripheral Artery Disease) | 外周动脉疾病 | 心、脑血管以外的动脉血管疾病 |
| COPD (Chronic Obstructive Pulmonary Disease) | 慢性阻塞性肺病 | 以慢性气道炎症和不可逆气流受限为特征的疾病 |
| FEV₁.₀ (Forced Expiratory Volume in 1s) | 1秒用力呼气量 | 用力呼气第一秒呼出的气量 |
| CR (Cardiac Rehabilitation) | 心脏康复 | 为心血管疾病患者设计的综合性二级预防计划 |
| PR (Pulmonary Rehabilitation) | 肺部康复 | 为慢性肺部疾病患者设计的综合性康复计划 |
| AACVPR | 美国心肺康复协会 | American Association of Cardiovascular and Pulmonary Rehabilitation |
| ACSM | 美国运动医学学会 | American College of Sports Medicine |
| ISWT (Incremental Shuttle Walk Test) | 递增往返步行测试 | 症状限制性的渐进步行测试 |
| EIB (Exercise-Induced Bronchoconstriction) | 运动性支气管痉挛 | 运动诱发的支气管收缩，常见于哮喘患者 |
| AMS (Acute Mountain Sickness) | 急性高原病 | 快速上升至高海拔后出现的头痛、恶心等症状 |
| WBGT (Wet Bulb Globe Temperature) | 湿球温度指数 | 综合温度、湿度和辐射热的环境热应激评估指标 |
| WCTI (Wind Chill Temperature Index) | 风寒温度指数 | 结合风速和空气温度评估环境致冷能力的指数 |
