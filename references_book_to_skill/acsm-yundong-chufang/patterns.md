## 2. FITT-VP运动处方通用模式

### 核心模式：Ex R_x = F + I + T + T + V + P

```
运动处方（Ex R_x）
├── F (Frequency)  频率
│   └── 有氧：≥3-5天/周 | 抗阻：≥2-3天/周 | 柔韧：≥2-3天/周
├── I (Intensity)  强度
│   ├── 中等强度：40%-59% HRR/VO₂R, RPE 12-13, 3-5.9 METs
│   └── 较高强度：60%-89% HRR/VO₂R, RPE 14-17, 6-8.7 METs
├── T (Time)       时间
│   └── 有氧：≥150min/周中等强度 或 ≥75min/周较高强度
├── T (Type)       方式
│   ├── 有氧：步行、跑步、骑车、游泳等大肌群节律性运动
│   ├── 抗阻：自由重量、器械、弹力带、自重
│   └── 柔韧：静力性、动力性、PNF拉伸
├── V (Volume)     总量
│   └── ≥500-1,000 MET-min/wk 或 ≥1,000 kcal/wk
└── P (Progression) 进阶
    └── "低起点，缓慢加" → 先增加时间/频率 → 再增加强度
```

## 4. 运动强度转化模式

```
主观感觉 ────── RPE (6-20) ────── %HRR ────── METs ────── 强度分类

"很轻松"         <9               <30%        <2.0         低强度
"轻松"           9-11             30-39%      2.0-2.9      较低强度
"有些吃力"       12-13            40-59%      3.0-5.9      中等强度 ◄─ 推荐范围
"很吃力"         14-17            60-89%      6.0-8.7      较高强度 ◄─ 推荐范围
"非常吃力"       ≥18              ≥90%        ≥8.8         最高强度

谈话测试 (Talk Test): 能说话但不能唱歌 = 中等强度
情感效价 (Affective Valence): 愉悦感 > 不适感 = 更好的依从性
```

## 6. 处方进阶时序模式

```
时间线 ─────────────────────────────────────────────────────►

第1-4周         第5-8周         第9-12周        第13周+
│               │               │               │
│ 适应期        │ 提高期        │ 巩固期        │ 维持期
│               │               │               │
├─ F: 3天/周 → │─ F: 3-5天/周 →│─ F: 3-5天/周→│─ F: 3-5天/周
├─ I: 低→中等  │─ I: 中等      │─ I: 中等→较高 │─ I: 混合强度
├─ T: 10-15min │─ T: 20-30min  │─ T: 30-60min  │─ T: 个性化
├─ T: 首选类型 │─ T: 增加多样  │─ T: 多样化    │─ T: 偏好为主
└─ P: 优先增加T│─ P: 增加F和T  │─ P: 谨慎增加I │─ P: 周期性变化

警告信号：在任何阶段如出现过度疲劳、疼痛、持续运动不耐受 → 退回上一阶段
```

## 7. 环境-运动耦合模式

```
环境因素 ────────────────────────────────────────── 运动调整策略

高原 (≥1,500m)
├─ AMS风险 → 分阶段上升，≤600m/天
├─ 运动表现↓ → 保持靶心率，降低绝对速度/阻力
└─ 适应期 → 第1-4天减少运动量

热环境 (WBGT > 28°C)
├─ 脱水风险 → 补液匹配出汗率，体重丢失<2%
├─ 核心温度↑ → 热适应10-14天
└─ 热病风险 → 降低绝对负荷，延长休息

冷环境 (WCTI < -27°C)
├─ 冻伤风险 → 分层服装，暴露<30min
├─ EIB风险 → 运动前热身+药物预防
└─ 心血管风险 → 冠状动脉疾病者谨慎
```
