# 第13章 咀嚼与通气的肌动学原理

## 核心框架

### 1. 颞下颌关节 (TMJ)

#### 关节特征
- **类型**：改良的屈戌关节（双侧联动）
- **特殊结构**：关节盘将关节腔分为上下两腔
- 上腔：滑动（前伸/后缩/侧移）
- 下腔：旋转（张口/闭口）

#### TMJ运动
| 运动 | 描述 |
|------|------|
| 下压（张口） | 下颌骨下降——下颌头向前滚动+向前滑动 |
| 上提（闭口） | 下颌骨上提 |
| 前伸 | 下颌向前突出 |
| 后缩 | 下颌后退 |
| 侧移 | 下颌左右偏 |

#### 咀嚼肌
| 肌肉 | 功能 |
|------|------|
| 咬肌 | 强力闭口 |
| 颞肌 | 闭口+后缩下颌 |
| 翼内肌 | 闭口+辅助侧移 |
| 翼外肌 | 张口+前伸+侧移（唯一张口的主动肌） |
| 舌骨上肌群 | 辅助张口（舌骨固定时） |

#### 临床要点
- TMJ功能障碍：关节盘移位+弹响+疼痛+张口受限
- 偏侧咀嚼 → 双侧肌肉不平衡 → 面部不对称
- 颈部姿势影响TMJ——头部前伸姿势增加TMJ负荷

### 2. 通气的肌动学

#### 吸气（主动过程）
- **主要吸气肌**：膈肌（占潮气量的~75%）
  - 膈肌收缩 → 中心腱下降 → 胸腔垂直径增加 → 肺扩张
- **辅助吸气肌**：肋间外肌（抬高肋骨 → 前后径和横径增加）
- **颈辅助吸气肌**：（仅在用力呼吸时激活）斜角肌、胸锁乳突肌、上斜方肌
  - 临床意义：COPD患者长期使用颈辅助呼吸肌 → 颈部紧张

#### 呼气（安静时被动过程）
- 安静呼气：肺和胸廓弹性回缩力 → 被动呼气
- 用力呼气：腹肌（腹直肌、腹外斜肌、腹内斜肌、腹横肌）+ 肋间内肌

#### 胸廓运动 — 泵柄与桶柄机制
| 机制 | 肋骨运动 | 胸腔变化 |
|------|---------|---------|
| 泵柄 | 上位肋骨向上向前运动 | 前后径增加 |
| 桶柄 | 下位肋骨向外向上运动 | 横径增加 |

#### 临床要点
- 脊髓损伤水平决定呼吸受累程度（C3以上→膈肌麻痹→依赖呼吸机）
- 腹肌收缩 → 腹内压增加 → 辅助用力呼气和核心稳定
- 长期不良姿势 → 肋骨活动受限 → 潮气量减少 → 代偿性颈辅助呼吸

### 3. 咀嚼与通气的肌动学关联
- 口呼吸 → 舌骨位置下降 → 头前伸姿势 → 颈辅助呼吸肌过度使用
- 咀嚼+通气共用颅面-颈部肌群 → 功能障碍可能相互影响

### 4. 常见分析错误
- ❌ 认为张口是单纯的下旋转——实际上是旋转+前滑动的组合
- ❌ 认为正常呼吸是腹式或胸式二选一——正常呼吸两者协同
- ❌ 忽略膈肌的核心稳定角色——膈肌同时参与呼吸和核心稳定
