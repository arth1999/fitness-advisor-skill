# 第3章 骨骼肌的结构与功能

## 核心框架

### 1. 肌肉的基本本质
- 肌肉通过**拉动**而非推动来产生运动
- **基本原则**：肌肉收缩时，**最自由（较少受限）的身体部分**发生移动
- 远端自由 → 远端向近端移动（开链）；近端自由 → 近端向远端移动（闭链）

### 2. 三种收缩类型 — 临床区分法

| 收缩类型 | 长度变化 | 力矩对比 | 功能角色 | 识别口诀 |
|---------|---------|---------|---------|---------|
| **向心收缩** | 缩短 | 内部力矩 > 外部力矩 | 加速/举起 | "举起重物" |
| **离心收缩** | 拉长 | 内部力矩 < 外部力矩 | 控制下降/减震 | "下降" = 几乎100%离心 |
| **等长收缩** | 不变 | 内部力矩 = 外部力矩 | 稳定/维持姿势 | "保持不动" |

**离心收缩关键认识**：重力为运动提供动力，肌肉的离心收缩控制下降速度。所有"下降"类动作均由拮抗肌离心收缩控制。

### 3. 肌肉术语速查
- **原动肌（主动肌）** = 与特定运动最直接相关的肌肉
- **拮抗肌** = 对抗原动肌动作的肌肉 → 原动肌收缩时被拉长 → 拮抗肌过度僵硬会限制原动肌功能
- **协同收缩** = 原动肌+拮抗肌同时等长激活 → 稳定关节
- **稳定肌** = 固定身体节段以使其他肌肉更有效工作
- **协同肌** = 共同完成特定动作的肌群
- **力偶** = 2块+肌肉在不同线性方向产生力，但在同一旋转方向产生力矩（如斜方肌上束+下束+前锯肌 → 肩胛骨上旋）

### 4. 肌肉解剖层次（由外到内）
肌腹（肌外膜）→ 肌束（肌束膜）→ 肌纤维=细胞（肌内膜）→ 肌原纤维 → 肌节（肌动蛋白+肌球蛋白）

### 5. 肌丝滑行学说
- 肌节 = 基本收缩单位
- 肌球蛋白头 + 肌动蛋白结合位点 → 横桥 → 做功冲程 → 肌动蛋白滑过肌球蛋白 → 肌节缩短
- **肌力取决于横桥数量** → 与肌肉长度密切相关

### 6. 决定肌肉功能的3个因素

#### A. 横截面积
- 横截面积越大 → 产生力越大
- 通用常数：每平方厘米肌肉约产生22.68kg力
- 股四头肌横截面积~161.3cm² → 最大收缩力≈3658kg（几乎举起一辆汽车！）
- 但实际表现还需乘以力臂得出力矩

#### B. 形状 — 肌纤维排列决定功能倾向
| 形状 | 纤维排列 | 功能优势 | 示例 |
|------|---------|---------|------|
| 梭形 | 平行走行 | 大范围运动（张弛度大） | 肱桡肌 |
| 三角形 | 宽近端→小远端 | 稳定的力产生基础 | 臀中肌 |
| 菱形 | 大面积近端和远端 | 稳定/强力 | 菱形肌、臀大肌 |
| 羽状 | 斜向中央腱（单羽/双羽/多羽） | 力势最大化，但张弛度受限 | 股直肌、腓肠肌 |

#### C. 拉力线
- 肌肉拉力线 = 力矢量方向
- 拉力线相对关节旋转轴的位置 → 决定该肌肉产生什么动作（同Ch01分析框架）

### 7. 长度-张力关系

#### 主动长度-张力关系
- 肌肉在**静息长度**时产生最大主动力（横桥重叠最多）
- 过短 → 肌动蛋白丝重叠干扰横桥 → 力下降
- 过长 → 横桥数量减少 → 力下降

#### 被动长度-张力关系
- 肌肉被拉伸到超过静息长度时，结缔组织（肌内膜、肌束膜、肌外膜）产生被动张力
- 被动张力随拉伸增加呈指数增长

#### 总张力 = 主动张力 + 被动张力

### 8. 多关节肌的特殊问题

#### 主动不足
- 定义：多关节肌在一个关节缩短时，在另一个关节产生的力减少
- 机制：肌肉已处于缩短状态，肌节重叠过度，横桥减少
- **当要求多关节肌在已缩短的位置再产生大力量时** → 主动不足
- 例：屈膝位试图强力跖屈（腓肠肌已缩短）

#### 被动不足
- 定义：多关节肌在一个关节被拉长时，限制另一个关节的活动范围
- 机制：肌肉已被动拉伸到极限，不能再延展
- **当多关节肌被跨多个关节同时拉长时** → 被动不足 → 限制ROM
- 例：伸膝位屈髋受限（腘绳肌被动不足）

### 9. 力量-速度关系
- 向心收缩：速度↑ → 力↓（反比关系）
- 离心收缩：速度↑ → 力↑（直到平台期）
- 等长收缩：速度=0，力居中

### 10. 力量训练原则
- **超负荷原则**：逐步增加阻力以刺激适应
- **SAID原则**（Specific Adaptation to Imposed Demands）：训练方式决定适应方向 → 离心训练可针对性增强离心力量
- **肌肉肥大** = 肌纤维横截面积增大

### 11. 牵伸指南
- 低强度、长时间牵伸优于高强度、短时间
- 牵伸时感到轻度不适即可，不应剧烈疼痛
- 被牵伸肌肉应完全放松
- 热敷后牵伸效果更好

### 12. 常见分析错误
- ❌ 认为离心收缩时肌肉"放松"——实际上离心收缩是主动过程，肌肉消耗能量
- ❌ 混淆主动不足与被动不足：主动不足=力不够，被动不足=长度不够
- ❌ 忽略多关节肌的跨关节效应——单关节分析不够
- ❌ 认为横截面积大就一定功能强——力臂（力矩）同样重要
