# 第6章 腕关节的结构与功能

## 核心框架

### 1. 关节构成
- **桡腕关节**（椭圆关节）—— 2个自由度：桡偏/尺偏（冠状面）、屈曲/伸展（矢状面）+ 环转
- **腕中关节**（桡腕关节与腕中关节的联合运动）
- **远端桡尺关节**（车轴关节）—— 旋前/旋后

### 2. 腕骨排列
- 近端列：舟骨、月骨、三角骨、豌豆骨
- 远端列：大多角骨、小多角骨、头状骨、钩骨
- 舟骨骨折最常见 → 易漏诊，影响血供 → 骨不连风险

### 3. 腕关节运动范围（正常值）

| 运动 | 范围 |
|------|------|
| 屈曲 | ~80° |
| 伸展 | ~70° |
| 桡偏 | ~15° |
| 尺偏 | ~30° |

**尺偏 > 桡偏**：因为桡骨茎突比尺骨茎突更长更远，限制了桡偏。

### 4. 腕管 (Carpal Tunnel)
- 边界：腕骨为底、屈肌支持带（腕横韧带）为顶
- 内容：9条肌腱（指浅屈肌4、指深屈肌4、拇长屈肌1）+ 正中神经
- **腕管综合征** = 正中神经受压 → 拇指、示指、中指、环指桡侧半感觉异常 + 大鱼际萎缩
- 屈曲腕关节使腕管压力增加 → 加重症状

### 5. 关键肌肉
- **腕屈肌**：桡侧腕屈肌、尺侧腕屈肌（掌长肌薄弱屈腕）
- **腕伸肌**：桡侧腕长伸肌、桡侧腕短伸肌、尺侧腕伸肌
- **功能位置**：腕关节轻度伸展（10°~15°）+ 轻度尺偏 → 最有利于抓握力量

### 6. 临床关键应用
- 腕关节的功能性位置对抓握力量至关重要：伸腕位肌力最大（指屈肌被动长度-张力最优）
- 尺偏时腕关节更稳定（因为尺骨茎突的骨性限制）
- Colles骨折（桡骨远端骨折）的典型畸形 = "银叉畸形"

### 7. 常见分析错误
- ❌ 分析腕关节运动时忽略远端桡尺关节的活动
- ❌ 混淆尺偏/桡偏与旋前/旋后
- ❌ 认为腕管综合征必须手术——早期可保守治疗
