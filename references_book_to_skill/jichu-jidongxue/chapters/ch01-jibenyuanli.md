# 第1章 肌动学的基本原理

## 核心框架

### 1. 运动学 (Kinematics) — 不涉及力的运动描述
- **平移**：整个身体沿同一方向移动（直线平移/曲线平移）。步行时头顶的上下曲线运动是平移。
- **旋转**：身体关于旋转轴的弧线运动。几乎身体所有关节的功能性运动都是通过旋转实现的。整个人体的平移由旋转肢体的肌肉提供动力。
- **主动运动** = 肌肉激活产生；**被动运动** = 重力、韧带张力或外力产生。

### 2. 骨骼运动学 (Osteokinematics) — 骨的宏观运动

#### 运动平面 vs 旋转轴
| 平面 | 轴 | 常见运动 |
|------|-----|---------|
| 矢状面（左右分） | 内外轴（冠状轴） | 屈曲/伸展、背伸/跖屈 |
| 冠状面/额状面（前后分） | 前后轴（矢状轴） | 外展/内收、侧屈、尺偏/桡偏、外翻/内翻 |
| 水平面（上下分） | 垂直轴（纵轴） | 内旋/外旋、自转 |

**关键规则**：旋转轴始终垂直于运动平面。自由度 = 关节可运动的平面数（肩关节3、腕关节2、肱尺关节1）。

#### 解剖学姿势 — 所有描述的参考标准
- 前/后、内侧/外侧/中线、上/下、近端/远端、颅侧/尾侧、浅层/深层
- 起点 = 近端附着点；止点 = 远端附着点

#### 基本运动术语
- **屈曲/伸展**：矢状面上绕内外轴。屈曲 = 骨接近屈肌面的运动。
- **外展/内收**：冠状面上远离/靠近中线。
- **旋转**：绕纵轴（内旋 = 骨前侧表面向中线转，外旋 = 远离中线）。
- **环转**：经过2个平面的圆周运动，需≥2自由度。
- **前伸/后缩**：水平面上远离/靠近中线（常用于肩胛骨）。
- **水平内收/外展**：肩关节外展90°后在水平面上的运动。
- **旋前/旋后**：前臂旋转，旋前=掌心向后，旋后=掌心向前。
- **桡偏/尺偏**：腕关节冠状面运动。背伸/跖屈：踝关节矢状面运动。内翻/外翻：足底朝内/外。

#### 开链与闭链
- **开链运动** = 远端骨围绕相对固定的近端骨运动（如坐姿伸膝）。用于针对性肌群训练。
- **闭链运动** = 近端骨围绕相对固定的远端骨运动（如深蹲时股骨相对胫骨运动）。更具功能性、利用负重和生物力学优势。

### 3. 关节运动学 (Arthrokinematics) — 关节面之间的微观运动

#### 凹凸面规则（关节松动术的基础）
- **凸面在凹面上运动**：滚动与滑动方向**相反**（如肱骨头在关节盂上外展：向上滚动+向下滑动）
- **凹面在凸面上运动**：滚动与滑动方向**相同**
- **自转**：一个面上的单点在另一个面的单点上旋转（如桡骨头旋前旋后）

#### 临床应用
- 肩关节外展时，若缺少向下滑动的制衡 → 肱骨头向上平移 → 肩峰下撞击综合征（肌腱炎/滑囊炎）
- 关节松动术通过徒手施加滑动力的方向取决于凹凸面关系

### 4. 动力学 (Kinetics)

#### 力矩公式
```
力 × 力臂 = 力矩
内力 × 内部力臂 = 内部力矩
外力 × 外部力臂 = 外部力矩
```
- 力矩大小取决于：①作用力大小；②力臂（旋转轴到作用力的垂直距离）
- 肌力测试本质上是测力矩（包含肌肉力量+力臂长度两个因素）

### 5. 生物力学杠杆系统

| 杠杆类型 | 支点位置 | 力臂关系 | 优点 | 缺点 | 人体示例 |
|---------|---------|---------|------|------|---------|
| 第一类 | 力与阻力之间 | 可变 | 取决于支点位置 | 取决于支点位置 | 颈伸肌支撑头部 |
| 第二类 | 杠杆一端 | IMA > EMA | 省力（小力举大负荷） | 远端位移小、速度慢 | 跖屈肌提踵 |
| 第三类 | 杠杆一端 | IMA < EMA | 远端位移大、速度快 | 需要更大肌力 | 肱二头肌屈肘 |

**核心结论**：身体绝大多数杠杆为**第三类**——肌肉愿意支付高昂的"力量税"以换取远端的位移和速度优势。手需要快速大范围的移动来完成功能活动。

**肱二头肌 vs 肱桡肌 对比**（同为第三类杠杆）：
- 肱二头肌IMA小 → 需68kg力 → 但收缩1英寸举手15英寸（速度位移优势）
- 肱桡肌IMA大 → 仅需23kg力 → 但收缩1英寸举手仅5英寸（省力优势）
- 神经系统根据任务需求（力量 vs 速度/范围）选择性激活肌肉

### 6. 拉力线分析 — 肌肉功能判断核心方法

**三步推理法**：
1. 判断拉力线相对于关节旋转轴的位置
2. 拉力线在内外轴**前方** → 矢状面**屈曲**
   拉力线在内外轴**后方** → 矢状面**伸展**
3. 拉力线在前后轴**上方/外侧** → 冠状面**外展**
   拉力线在前后轴**下方/内侧** → 冠状面**内收**
4. 拉力线在垂直轴**前方** → 水平面**内旋**
   拉力线在垂直轴**后方** → 水平面**外旋**

**记忆原则**：不靠死记，靠分析拉力线与旋转轴的相对位置。

### 7. 矢量与合力
- 矢量表示力的大小（线长）和方向（箭头）
- 多块肌肉同时作用 → 取矢量合力 → 合力偏向更强壮的肌肉
- 临床意义：一侧肌肉无力时动作会被拉向强侧 → 姿势偏移 → 疼痛和运动紊乱
- 两条等大力矢量夹角越大，合力越小

### 8. 常见分析错误（Anti-patterns）
- ❌ 混淆运动平面与旋转轴（旋转轴垂直于运动平面，不是同一个东西）
- ❌ 把被动运动当成主动运动分析
- ❌ 闭链运动中使用开链的肌肉功能判断
- ❌ 忽略力臂变化（关节角度变化时力臂也随之变化）
- ❌ 将两块肌肉的拉线方向等同于其合力方向，忽略了力量大小差异
