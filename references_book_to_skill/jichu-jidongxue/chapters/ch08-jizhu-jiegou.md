# 第8章 脊柱的结构与功能

## 核心框架

### 1. 脊柱的区域划分与曲度

| 区域 | 节数 | 生理曲度 | 功能特征 |
|------|------|---------|---------|
| 颈椎 | C1-C7 | 前凸（lordosis） | 支撑头部、最大ROM |
| 胸椎 | T1-T12 | 后凸（kyphosis） | 肋骨连接、旋转、保护心肺 |
| 腰椎 | L1-L5 | 前凸（lordosis） | 承重、较大ROM |
| 骶椎 | S1-S5（融合）| 后凸 | 传递力至骨盆 |
| 尾椎 | 4块融合 | — | 韧带附着 |

### 2. 椎骨的基本结构
- **椎体**：承重，由松质骨+皮质壳构成
- **椎弓**：保护脊髓
- **关节突**：构成关节突关节（滑膜平面关节）→ 引导运动方向

### 3. 关节突关节 (Facet Joints) — 引导脊柱运动
- 颈段：关节面近水平 → 允许所有方向运动（屈伸+侧屈+旋转）
- 胸段：关节面近冠状 → 利于旋转，限制屈伸
- 腰段：关节面近矢状 → 利于屈伸，限制旋转

### 4. 椎间盘 (Intervertebral Disc) — 微动关节
- **纤维环**（外周）：同心胶原纤维层，抗拉伸
- **髓核**（中央）：含水的凝胶状，抗压缩、分散压力
- 功能：减震+允许椎体间小范围运动
- 椎间盘退变 → 高度丢失 → 关节突关节负荷增加 → 骨赘形成

### 5. 脊柱的主要韧带
- **前纵韧带**（ALL）：限制伸展
- **后纵韧带**（PLL）：限制屈曲
- **黄韧带**：连接相邻椎板，高弹性，限制屈曲
- **棘间韧带+棘上韧带**：限制屈曲
- **韧带复合体在屈曲时张力最大** → 保护椎间盘

### 6. 脊柱运动学术语
- 屈曲/伸展（矢状面）
- 侧屈（冠状面）
- 旋转（水平面）—— 注意耦合运动（侧屈+旋转常同时发生）

### 7. 核心稳定 (Core Stability)
- **局部稳定系统**：深层小肌肉（多裂肌、腹横肌）—— 节段间控制
- **整体稳定系统**：浅层大肌肉（腹直肌、竖脊肌）—— 大范围运动和力量
- 核心稳定 = 脊柱中立位的维持能力，不仅仅"腹肌力量"

### 8. 颅颈区 (Craniovertebral Region)
- **寰枕关节 (C0-C1)**：屈戌关节 → 点头（矢状面屈伸）
- **寰枢关节 (C1-C2)**：车轴关节 → 摇头（水平面旋转，占颈部旋转的~50%）
- 齿突骨折 → C1-C2不稳 → 严重神经风险

### 9. 常见姿势异常与病理
- **脊柱侧凸 (Scoliosis)**：冠状面侧弯+水平面旋转的三维畸形
- **椎间盘突出**：髓核突破纤维环 → 压迫神经根 → 放射性疼痛
- **椎管狭窄**：椎管直径减小 → 压迫脊髓/神经根

### 10. 临床关键应用
- 脊柱屈曲时椎间盘前部受压、后部张力增加 → 后外侧为椎间盘突出好发区
- 腰椎旋转+屈曲 → 椎间盘压力最大 → 最危险的举物姿势
- 颈椎旋转的50%来自C1-C2 → C1-C2功能障碍会显著限制头部旋转
- 胸椎灵活性下降 → 腰椎和颈椎代偿 → 过度使用和疼痛

### 11. 常见分析错误
- ❌ 把核心稳定等同于腹肌力量 → 应包括多裂肌、腹横肌、膈肌、盆底肌
- ❌ 所有腰痛都归因于椎间盘 —— 关节突关节、骶髂关节也可能是源
- ❌ 脊柱屈曲训练适用于所有腰痛患者 —— 椎间盘突出患者可能加重症状
