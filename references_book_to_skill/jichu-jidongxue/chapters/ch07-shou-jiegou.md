# 第7章 手的结构与功能

## 核心框架

### 1. 手弓系统

| 弓 | 构成 | 功能 |
|------|------|------|
| 近端横弓 | 腕骨（固定） | 腕管基底 |
| 远端横弓 | 掌骨头 | 可活动，适应抓握形状 |
| 纵弓 | 腕骨→掌骨→指骨 | 手的基本弓形 |

### 2. 拇指腕掌关节 (CMC Joint) — 手的核心
- **鞍状关节** → 2个自由度 + 独特的灵活性
- 运动：屈曲/伸展、外展/内收、对掌（拇指与小指指尖接触的复合运动）
- **对掌功能** 使人类能完成精细操作——是手最重要的功能

### 3. 掌指关节 (MCP Joints) — 2个平面上的运动
- 髁状关节（第1指是屈戌关节）
- 屈曲/伸展 + 外展/内收
- MCP伸展时侧副韧带松弛 → 外展内收灵活
- MCP屈曲时侧副韧带紧张 → 侧向稳定 → **功能位固定应在屈曲位**，防止侧副韧带挛缩

### 4. 指间关节 (IP Joints)
- 近端指间关节 (PIP) + 远端指间关节 (DIP)
- 均为屈戌关节 → 仅屈曲/伸展

### 5. 手指外在肌 — 前臂来源

#### 外在屈肌
| 肌肉 | 止点 | 功能 |
|------|------|------|
| 指深屈肌 | 第2~5指远节 | 屈DIP + PIP + MCP + 腕 |
| 指浅屈肌 | 第2~5指中节 | 屈PIP + MCP + 腕 |
| 拇长屈肌 | 拇指远节 | 屈拇指IP + MCP |

#### 外在伸肌
| 肌肉 | 止点 | 功能 |
|------|------|------|
| 指伸肌 | 第2~5指伸肌装置 | 伸MCP + PIP + DIP |
| 示指伸肌 | 示指伸肌装置 | 示指独立伸展 |
| 小指伸肌 | 小指伸肌装置 | 小指独立伸展 |
| 拇长伸肌 | 拇指远节 | 伸拇指IP |
| 拇短伸肌 | 拇指近节 | 伸拇指MCP |

### 6. 手内在肌 — 手内部来源
- **鱼际肌**（3块）：拇短展肌、拇短屈肌、拇对掌肌——拇指对掌+精细控制
- **小鱼际肌**（3块）：小指展肌、小指短屈肌、小指对掌肌——小指运动
- **骨间肌**（7块：4背侧+3掌侧）：背侧→外展手指；掌侧→内收手指
- **蚓状肌**（4块）：屈MCP + 伸PIP/DIP ——奇特的"反向"功能

### 7. 伸肌装置（Extensor Mechanism）
- 复杂的腱膜系统将外在伸肌和内在肌的力量协调到手指
- 骨间肌和蚓状肌通过侧束 → 伸PIP和DIP
- 外在伸肌通过中央束 → 伸PIP
- 此系统的病变导致手指畸形（如钮孔畸形、鹅颈畸形）

### 8. 抓握模式

| 模式 | 特征 | 示例活动 |
|------|------|---------|
| 力量抓握 | 手指屈曲包裹物体，拇指施加对抗力 | 握锤子、提箱子 |
| 精确抓握 | 指尖与拇指对掌 | 写字、捡针 |

### 9. 临床关键应用
- MCP关节在伸直位制动 → 侧副韧带挛缩 → 难以再做手指侧向运动
- 手内在肌无力 → 爪形手（MCP过伸 + PIP/DIP屈曲）
- 正中神经损伤 → "猿手"（大鱼际萎缩，拇指对掌丧失）
- 尺神经损伤 → 爪形手（第4、5指为主）+ 骨间肌萎缩
- 桡神经损伤 → "垂腕"（伸腕伸指肌麻痹）

### 10. 常见分析错误
- ❌ 只分析外在肌而忽略骨间肌/蚓状肌对手指精细控制的作用
- ❌ 长期MCP伸直位固定——应固定在MCP屈曲位以保持侧副韧带长度
- ❌ 评估拇指时忽略CMC鞍状关节的对掌功能
