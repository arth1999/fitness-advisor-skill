# 第10章 膝关节的结构与功能

## 核心框架

### 1. 膝关节基本特征
- **胫股关节**：髁状关节（改良的屈戌关节）——主要屈伸 + 少量旋转
- **髌股关节**：平面/鞍状关节
- 膝关节是人体**最大且最复杂的关节**

### 2. 半月板 (Menisci)
- 内侧半月板：C形，附着于内侧副韧带 → 活动度小 → 更易损伤
- 外侧半月板：O形，无韧带附着 → 活动度大
- 功能：分散压力（承担50%~70%的负荷）、减震、增加关节一致性、关节润滑

### 3. 交叉韧带 (Cruciate Ligaments)

| 韧带 | 主要功能 | 检查方法 |
|------|---------|---------|
| 前交叉韧带 (ACL) | 限制胫骨前移+限制过伸+限制内旋 | Lachman试验、前抽屉试验 |
| 后交叉韧带 (PCL) | 限制胫骨后移 | 后抽屉试验、Sag征 |

- ACL损伤常见于：急停、变向、落地时膝内扣（外翻+外旋）
- ACL与半月板损伤常合并发生（"不幸三联征"：ACL + 内侧半月板 + 内侧副韧带）

### 4. 侧副韧带
- **内侧副韧带 (MCL)**：限制膝外翻 → 损伤常见于外侧撞击
- **外侧副韧带 (LCL)**：限制膝内翻
- 稳定冠状面

### 5. 髌股关节 (Patellofemoral Joint)
- **髌骨的功能**（籽骨）：
  1. 增加股四头肌的内部力臂 → 增大伸膝力矩（平均增加~30%）
  2. 分散股四头肌腱与股骨髁间的压力和剪切力
- 髌骨在股骨滑车内滑动，屈膝时向下移动，伸膝时向上移动
- 髌股关节反作用力：下蹲时可达体重的7~8倍

### 6. 膝关节的"锁定"机制 (Screw-Home Mechanism)
- **终末伸展时**：胫骨在股骨上**外旋**约10° → 膝关节"锁定"在伸展位
- 由股骨内侧髁比外侧髁更长（前后径）驱动
- 功能：站立时无需股四头肌持续收缩即可维持膝伸直
- 解锁：腘肌收缩 → 胫骨内旋 → 从锁定位释放

### 7. 关键肌肉

#### 伸膝肌 — 股四头肌
| 头 | 特点 |
|------|------|
| 股直肌 | 跨双关节（屈髋+伸膝）——主动不足常见 |
| 股中间肌 | 深层单关节伸膝肌 |
| 股内侧肌 | 斜行纤维VMO → 内侧髌骨稳定 |
| 股外侧肌 | 最强大的头 |

#### 屈膝肌 — 腘绳肌
| 肌肉 | 特点 |
|------|------|
| 股二头肌（长头+短头） | 长头跨双关节（伸髋+屈膝） |
| 半腱肌 | 跨双关节 |
| 半膜肌 | 跨双关节 |

#### 其他重要肌肉
- **腘肌**：解锁膝关节（胫骨内旋）
- 腓肠肌：跨双关节，辅助屈膝 + 主要跖屈
- 髂胫束 (ITB)：外侧稳定，但过紧可导致外侧膝痛

### 8. 关节受限的常见模式
- **屈曲挛缩**：最常见 → 步态异常，能耗增加
- **伸展受限**：影响站立稳定和步态
- 关节固定于完全伸展位 → 步行能耗增加**23%~33%**
- 关节固定于45°屈曲位 → 步行能耗增加**37%**

### 9. 临床关键应用
- 闭链伸膝训练（如深蹲）：功能性+髌股关节压力小于开链伸膝
- 前膝痛 → 评估髌骨轨迹 + ITB紧张 + 髋外展肌/外旋肌无力
- 膝外翻往往源于**近端（髋外展肌无力）或远端（足过度旋前）**
- ACL重建后康复的核心：不仅是力量恢复，更需本体感觉和神经肌肉控制再训练

### 10. 常见分析错误
- ❌ 膝痛只检查膝关节 —— 检查髋和踝的生物力学链
- ❌ 开链伸膝和闭链伸膝混淆分析——股四头肌力矩因闭/开链大幅变化
- ❌ 忽略Q角对髌股关节的影响（正常Q角：男13°，女18°）
