## 强化剂证据速查

| 强化剂 | 效果 | 证据强度 | 安全性 | 合法 |
|--------|------|---------|--------|------|
| 碳水化合物 | 确定有效 | 强 | 安全 | 是 |
| 咖啡因（3-6mg/kg） | 提升耐力~3% | 中-强 | 基本安全* | 是 |
| 一水肌酸 | 提升高强度重复性表现 | 中 | 短期安全 | 是 |
| ω-3脂肪酸 | 抗炎（不一致） | 弱-中 | 高剂量出血风险 | 是 |
| 碳酸氢钠 | 不确定 | 弱 | 胃肠道不适 | 是 |
| 肉碱 | 无/极弱 | 弱 | 基本安全 | 是 |
| 中链脂肪酸(MCT) | 无体能益处 | 弱 | >30g腹泻 | 是 |
| 人参 | 无 | 无 | 基本安全 | 是 |
| 甘油（超水合） | 有效 | 中 | 基本安全 | **WADA禁用** |

## 体重管理策略速查

| 策略 | 效果 | 推荐 |
|------|------|------|
| 大幅减少热量 | 肌肉流失>脂肪流失 | ❌ 不推荐 |
| 少量多餐（6+次/日） | 维持肌肉，降体脂 | ✅ 强烈推荐 |
| 仅靠饮食不运动 | 不该损失的也损失 | ❌ 不推荐 |
| 低强度长时间运动 | 燃脂比例高但总量低 | ❌ 效率低 |
| >65%VO₂max运动 | 燃脂总量更高 | ✅ 推荐 |
| 赛前急性脱水减重 | 危险，影响表现 | ❌ 危险 |
| 长期渐进体重管理 | 安全，肌肉保留 | ✅ 最佳 |

## 不同环境营养调整速查

### 高温环境
- 🔼 液体摄入（每10-15分钟补水）
- 🔼 钠摄入（每小时约1g）
- 🔼 使用运动饮料而非纯水
- 🔽 降低运动强度/持续时间
- 📊 监测热指数

### 寒冷环境
- 🔼 碳水化合物摄入（寒战增加消耗）
- 🔼 总能量摄入
- 🔽 液体冻结问题 → 保温措施
- 注意呼吸道水分丢失

### 高海拔环境
- 🔼 碳水化合物优先（耗氧量更低）
- 🔼 铁摄入（18mg/d）
- 🔼 抗氧化剂（复合而非单一）
- 🔼 液体摄入
- 🔽 脂肪和蛋白质比例
- ⏰ 充足习服时间

## 常见误区纠正速查

| 误区 | 事实 |
|------|------|
| "蛋白质越多越好" | 最大利用率约1.5g/kg，过量=浪费+脱水风险 |
| "低强度运动最减脂" | 高强度运动燃脂总量更多 |
| "口渴才喝水" | 口渴时已脱水1.5-2L，太晚了 |
| "吃脂肪会变胖" | 关键是总能量平衡+进餐频率 |
| "不吃早餐能减重" | 导致肌肉流失+代谢下降+体脂率升高 |
| "维生素补剂多多益善" | 过量同样有害，食物优先 |
| "赛前强化训练" | 应递减训练量+增加碳水+休息 |
| "运动后等1小时再吃" | 应立即进食（抓住糖原合成酶峰值） |
