# 术语表 — 高级运动营养学（第2版）

## 能量代谢

| 术语 | 英文 | 定义 |
|------|------|------|
| 千卡 | kcal / Calorie | 营养学能量单位，1千卡=1000卡路里 |
| 腺苷三磷酸 | ATP (Adenosine Triphosphate) | 细胞的"高能化合物"，肌肉收缩的直接能量来源 |
| 磷酸肌酸 | PCr (Phosphocreatine) | 储存于肌肉中的高能磷酸化合物，用于快速再合成ATP |
| 糖酵解 | Glycolysis | 糖原分解为葡萄糖再转化为ATP的代谢过程，分有氧和无氧两种 |
| 有氧糖酵解 | Aerobic Glycolysis | 在充足氧气下进行的糖酵解，产生大量ATP且不生成乳酸 |
| 无氧糖酵解 | Anaerobic Glycolysis | 在缺氧条件下进行的糖酵解，副产物为乳酸（亦称乳酸系统） |
| 糖异生 | Gluconeogenesis | 通过非碳水化合物物质（乳酸、甘油、生糖氨基酸）生成葡萄糖的过程 |
| 科里循环 | Cori Cycle | 肌肉排出的乳酸在肝脏转化为葡萄糖后重新返回肌肉的过程 |
| β氧化 | Beta-Oxidation | 脂肪酸在线粒体中每次释放2个碳单位以产生ATP的代谢途径 |
| 食物热效应 | Thermic Effect of Food (TEF) | 进食导致的能量消耗额外增加的现象 |

## 碳水化合物相关

| 术语 | 英文 | 定义 |
|------|------|------|
| 血糖指数 | Glycemic Index (GI) | 衡量摄入的碳水化合物转化为血糖速率的指标（葡萄糖=100作为基准） |
| 血糖负荷 | Glycemic Load (GL) | GI × 碳水化合物克数 ÷ 100，反映实际血糖反应 |
| 单糖 | Monosaccharide | 碳水化合物最基本单位（葡萄糖、果糖、半乳糖） |
| 双糖 | Disaccharide | 2个单糖结合（蔗糖、乳糖、麦芽糖） |
| 多糖 | Polysaccharide | 含有多个（≥10）重复糖单元的复杂分子（淀粉、糖原、纤维素） |
| 葡萄糖聚合物 | Glucose Polymer | 由淀粉水解而成的长链葡萄糖分子，用于运动饮料和凝胶 |
| 碳水化合物负荷法 | Carbohydrate Loading | 赛前通过增加碳水摄入+减少训练量最大化糖原储备的策略 |
| 可溶性纤维 | Soluble Fiber | 可溶于水、呈凝胶状的纤维，与较低血糖水平相关 |
| 不溶性纤维 | Insoluble Fiber | 不溶于水、增加膳食体积的纤维，促进肠道健康 |

## 蛋白质与氨基酸

| 术语 | 英文 | 定义 |
|------|------|------|
| 必需氨基酸 | Essential Amino Acids (EAA) | 身体不能合成、必须从食物中获取的氨基酸（9种） |
| 非必需氨基酸 | Non-Essential Amino Acids | 身体可从其他氨基酸合成的氨基酸 |
| 条件必需氨基酸 | Conditionally Essential Amino Acids | 在特定代谢条件下变为必需的氨基酸 |
| 支链氨基酸 | Branched-Chain Amino Acids (BCAA) | 亮氨酸、异亮氨酸、缬氨酸，可在骨骼肌中被氧化 |
| 转氨基作用 | Transamination | 氨基酸的氮被用来生成另一种氨基酸的过程 |
| 脱氨基反应 | Deamination | 氨基从氨基酸脱去并转化为氨的过程 |
| 氮平衡 | Nitrogen Balance | 摄入氮与排出氮的平衡（正平衡=肌肉增长，负平衡=肌肉流失） |
| 优质蛋白质 | High Biological Value Protein | 含全部必需氨基酸且比例适合细胞蛋白质合成的蛋白质 |
| 谷氨酰胺 | Glutamine | 条件必需氨基酸，免疫细胞的主要燃料 |
| 色氨酸 | Tryptophan | 必需氨基酸，5-羟色胺（血清素）的前体 |

## 脂质与脂肪酸

| 术语 | 英文 | 定义 |
|------|------|------|
| 甘油三酯 | Triglyceride | 3个脂肪酸+1个甘油分子，人体脂肪的主要储存形式 |
| 必需脂肪酸 | Essential Fatty Acids | 亚油酸（ω-6）和亚麻酸（ω-3），身体不能自行合成 |
| ω-3脂肪酸 | Omega-3 Fatty Acids | 远端双键在第3位碳的脂肪酸，具有抗炎作用（EPA、DHA、ALA） |
| ω-6脂肪酸 | Omega-6 Fatty Acids | 远端双键在第6位碳的脂肪酸（亚油酸） |
| 中链甘油三酯 | Medium-Chain Triglycerides (MCT) | 碳链6-12个碳的脂肪酸，可直接吸收并快速氧化 |
| 饱和脂肪酸 | Saturated Fatty Acids | 碳链中无双键的脂肪酸，主要存在于动物脂肪 |
| 单不饱和脂肪酸 | Monounsaturated Fatty Acids | 碳链中有1个双键的脂肪酸（如橄榄油） |
| 多不饱和脂肪酸 | Polyunsaturated Fatty Acids | 碳链中有多个双键的脂肪酸（如植物油、鱼油） |
| 甘油 | Glycerol | 三碳单脂，代谢方式类似碳水化合物，强效保湿剂 |

## 维生素

| 术语 | 英文 | 定义 |
|------|------|------|
| 膳食参考摄入量 | Dietary Reference Intake (DRI) | 对营养素摄入的量化估计，用于健康人群膳食计划与评估 |
| 推荐膳食供应量 | Recommended Dietary Allowance (RDA) | 满足97-98%健康人群需求的每日摄入量 |
| 可耐受最高摄入量 | Tolerable Upper Intake Level (UL) | 不产生不良影响的每日最高摄入量 |
| 脂溶性维生素 | Fat-Soluble Vitamins | 维生素A、D、E、K，需要脂肪环境发挥作用 |
| 水溶性维生素 | Water-Soluble Vitamins | B族维生素和维生素C，需要水分环境发挥作用 |
| 抗氧化剂 | Antioxidants | 清除自由基、抑制氧化损伤的物质（维生素C、E、β-胡萝卜素、硒） |
| 视黄醇活性当量 | Retinol Activity Equivalent (RAE) | 维生素A的计量单位 |
| 植物化学物质 | Phytochemicals / Phytonutrients | 植物中的生物活性化合物（如番茄红素、芥子油苷） |

## 矿物质

| 术语 | 英文 | 定义 |
|------|------|------|
| 常量矿物质 | Macrominerals | 体内含量较多的矿物质：钙、磷、镁、钠、钾、氯化物 |
| 微量矿物质 | Trace Minerals | 体内含量极少的矿物质：铁、锌、铜、硒、碘等 |
| 铁蛋白 | Ferritin | 贮铁蛋白，血清铁蛋白可反映体内铁储备量 |
| 血红蛋白 | Hemoglobin | 红细胞内含铁的携氧蛋白 |
| 血细胞比容 | Hematocrit | 全血中红细胞的体积比例 |
| 运铁蛋白 | Transferrin | 负载铁并通过血液转运的蛋白质 |
| 钙稳态 | Calcium Homeostasis | 维持血液和组织中钙浓度稳定的调节机制 |
| 电解质 | Electrolytes | 在溶液中可导电的矿物质离子（钠、钾、钙、镁、氯化物） |

## 体液平衡

| 术语 | 英文 | 定义 |
|------|------|------|
| 水合正常 | Euhydration | 体内含水量正常的良好水合状态 |
| 低水合 | Hypohydration | 体内含水量低于正常水平 |
| 脱水 | Dehydration | 严重低水合状态 |
| 超水合 | Hyperhydration | 体内含水量高于正常水平 |
| 抗利尿激素 | Antidiuretic Hormone (ADH) | 促使肾脏重吸收更多水分的激素 |
| 醛固酮 | Aldosterone | 调节钠和水重吸收的激素 |
| 渗透压 | Osmolality | 溶液中溶质颗粒的浓度，影响水分的移动方向 |
| 低钠血症 | Hyponatremia | 血液钠浓度过低，可能导致危险的脑水肿 |

## 运动生理与医学

| 术语 | 英文 | 定义 |
|------|------|------|
| 最大摄氧量 | VO₂max | 最大强度运动时机体能够摄取和利用的最大氧气量 |
| 延迟性肌肉酸痛 | Delayed Onset Muscle Soreness (DOMS) | 运动后24-48小时达到高峰的肌肉钝痛和僵硬 |
| 中枢神经系统疲劳理论 | Central Fatigue Theory | 5-羟色胺合成增加导致中枢性疲劳的理论 |
| 运动诱发性哮喘 | Exercise-Induced Asthma (EIA) | 运动诱发的支气管收缩，寒冷干燥环境高发 |
| 稀释性假性贫血 | Dilutional Pseudoanemia | 运动导致血容量增加超过红细胞生成，引发类似贫血的症状 |
| 足底撞击性溶血 | Foot-Strike Hemolysis | 跑步时足底毛细血管中红细胞被反复撞击破坏 |
| 氧化应激 | Oxidative Stress | 活性氧生成超过抗氧化防御能力的状态 |
| 活性氧/自由基 | Reactive Oxygen Species (ROS) / Free Radicals | 高反应性氧分子，可损伤细胞结构 |
| 日內能量平衡 | Within-Day Energy Balance | 实时能量摄入与消耗的动态平衡（Benardot核心理念） |
| 女性运动员三联征 | Female Athlete Triad | 能量不足+月经紊乱+骨密度降低的综合征 |

## 高海拔与环境

| 术语 | 英文 | 定义 |
|------|------|------|
| 高住低练 | Live High, Train Low (LHTL) | 在高原居住但低海拔训练以提高携氧能力的训练模式 |
| 急性高山病 | Acute Mountain Sickness (AMS) | 高海拔最常见的轻度综合征（头痛、恶心、疲乏） |
| 高原脑水肿 | High-Altitude Cerebral Edema (HACE) | 精神状态改变和协调障碍，可危及生命 |
| 高原肺水肿 | High-Altitude Pulmonary Edema (HAPE) | 肺部积液导致呼吸困难，可危及生命 |
| 寒冷引起的多尿 | Cold-Induced Diuresis (CID) | 寒冷导致肾脏排尿增加的现象 |
| 冷诱导血管舒张 | Cold-Induced Vasodilation (CIVD) | 低温暴露约10分钟后末梢血管的脉冲性舒张 |
| 热指数 | Heat Index | 综合环境温度和相对湿度评估热应激风险的指标 |

## 强化剂相关

| 术语 | 英文 | 定义 |
|------|------|------|
| 强化剂 | Ergogenic Aid | 增强体力或精神（尤其消除疲劳）的物质 |
| 营养强化剂 | Nutritional Ergogenic Aid | 通过进入已知营养代谢途径发挥作用的强化剂 |
| 一水肌酸 | Creatine Monohydrate (CrM) | 肌酸的补剂形式，可增强高强度重复性运动表现 |
| 咖啡因 | Caffeine | 中枢神经系统兴奋剂，可改善耐力表现约3.2% |
| 合成代谢类固醇 | Anabolic Steroids | 禁用雄性激素类药物，增加肌肉但严重危害健康 |
