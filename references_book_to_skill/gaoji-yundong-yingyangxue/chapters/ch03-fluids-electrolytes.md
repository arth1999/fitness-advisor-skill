# 第3章 液体与电解质

## 章节概述

维持体液平衡可能是与保持高水平运动表现最相关的单一关键因素。然而，大多数运动员在训练和比赛中仍会出现脱水（导致血容量下降），从而降低运动能力。本章详述了体液平衡的生理机制、影响因素以及最佳补水补液策略。

## 核心概念

### 1. 体液分布与功能

| 指标 | 数值 |
|------|------|
| 人体总含水量 | 约占体重的57% |
| 细胞内液 | 占总水分的65% |
| 细胞外液 | 占总水分的35% |
| 充分水合的肌肉含水量 | 约75% |
| 脂肪组织含水量 | 约10% |
| 血液含水量 | 约93% |
| 一般男性含水量 | 约60% |
| 一般女性含水量 | 约50% |
| 运动员含水量 | 约70% |

**关键原则**：肌肉含量越高、体脂越少，体内水分占比越大。

### 2. 体液调节机制

- **抗利尿激素（ADH）**：当血液渗透压升高时，下丘脑通过垂体释放ADH，促使肾脏重吸收更多水分
- **醛固酮**：低钠时分泌以保留钠；高钠时停止分泌以排出多余钠
- **口渴机制延迟**：体内水分丢失1.5-2.0L后才会感到口渴——口渴是不可靠的补水信号

### 3. 出汗与散热

- 运动中几乎**100%的散热**通过汗液蒸发完成
- **高温环境下**剧烈运动每小时可丢失2.5L汗液
- 每丢失1L含50 mmol/L钠的汗液 = 丢失约3g氯化钠
- **湿热环境中**训练有素的运动员每小时液体丢失可**超过3L**
- 仅20-40%的能量代谢效率有效，60-80%以热能散失
- **1ml汗液可散发约0.5kcal热量**

### 4. 热指数风险分级

| 体表温度 | 热应激风险 |
|----------|-----------|
| 90-104°F (32-40°C) | 可能发生热痉挛或热衰竭 |
| 105-129°F (41-53°C) | 很可能发生热痉挛/热衰竭，可能中暑 |
| ≥130°F (≥54°C) | 极大可能发生中暑 |

### 5. 维持体液平衡的益处

- 使心率加快放缓
- 使体核温度上升放缓
- 提高每搏输出量和心排血量
- 改善皮肤血流
- 降低血浆钠浓度、渗透压和肾上腺素浓度
- 减少净肌糖原的使用

### 6. 胃排空影响因素

| 因素 | 影响 |
|------|------|
| 碳水化合物浓度 | >7%时胃排空减慢 |
| 碳水化合物类型 | 单糖（如葡萄糖）比多糖排空慢 |
| 液体总量 | 胃内液体量大则排空快 |
| 液体温度 | 运动时冷饮比体温液体排空更快 |
| 脱水状态 | 脱水会减慢胃排空 |
| 精神压力 | 焦虑显著减慢胃排空 |

### 7. 运动饮料策略

- **最佳碳水化合物浓度**：6-7%
- **推荐补液频率**：每10-15分钟摄入90-240ml
- **钠的作用**：唤起饮水的欲望，帮助维持血容量
- **饮料选择因素**：颜色、味道、气味、温度和质地均影响摄入意愿
- **葡萄糖聚合物优势**：在低渗透压下提供更多碳水化合物，加速胃排空

### 8. 电解质平衡

- 钠是主要细胞外电解质，直接影响血容量
- **血容量是衡量运动员能否保持较高水平运动表现的核心指标**
- "污染的解决方法是稀释"——更大液体池可缓冲pH变化，减少乳酸负面影响

## 实用要点

1. 不应以口渴作为补水信号——应按预定计划补水
2. 运动前摄入420-660ml液体达到水合状态
3. 运动中使用含6-7%碳水化合物+钠的运动饮料
4. 赛前7天逐渐减少训练、增加液体摄入以达到最佳水合
5. 可通过尿液颜色简单判断水合状态（浅色=水合良好）
