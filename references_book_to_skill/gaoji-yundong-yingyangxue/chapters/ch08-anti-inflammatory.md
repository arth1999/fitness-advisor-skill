# 第8章 抗炎与肌肉保健策略

## 章节概述

几乎所有运动员都经历过延迟性肌肉酸痛（DOMS）及其对运动表现的消极影响。本章系统评估了各种营养素和补剂对减轻DOMS的证据强度，结论是：目前研究数量有限，证据强度普遍偏弱，但某些营养素组合（ω-3脂肪酸+维生素E、充足水合、优质蛋白质）可能产生轻度改善效果。

## 核心概念

### 1. 延迟性肌肉酸痛（DOMS）

- **定义**：运动后24-48小时达到高峰的钝痛、僵硬和疼痛感
- **持续时间**：通常2-4天
- **机制**：与离心性运动（肌肉在伸展状态下收缩）最相关，导致肌肉细微结构破坏
- **伴随问题**：
  - 肌肉产生更多活性氧（ROS）
  - ATP利用效率降低 → 肌肉无力

### 2. 营养素与DOMS相关性证据总结

| 变量 | 可能有帮助的营养素 | 证据强度 |
|------|------------------|---------|
| 肌肉力量 | 维生素C(500mg/d)+维生素E(1200IU/d) | 弱 |
| DOMS疼痛感 | 维生素C(1-3g/d)、维生素E(400IU/d联合VC)、BCAA(5g/d，仅女性)、左旋肉碱(2g/d) | 弱 |
| 肌酸激酶(CK) | 维生素E(400-1200IU/d)、维生素C(1-3g/d)、HMB(1.5-3g/d)、左旋肉碱(2g/d) | 弱 |
| 细胞骨架破坏 | — | 无 |
| 关节活动度 | — | 无 |

### 3. ω-3脂肪酸与DOMS

**可能机制**：
- 与花生四烯酸竞争产生类花生酸
- 刺激大量抗炎因子产生
- 构成细胞膜一部分，改变炎症因子释放

**研究结论**：
- 补充ω-3脂肪酸（EPA 324mg + DHA 216mg/天）在离心运动后48小时有更好的疼痛评分
- 大腿围（炎症肿胀标志物）明显更小
- 效果需要运动前**30天**提前补充才显现

### 4. 维生素D与肌肉健康

**关键发现**：
- 所有血清维生素D水平低的受试者均存在肌肉疼痛；维生素D正常者均无肌肉疼痛
- 维生素D状态与握力评分显著正相关
- 室内运动员（体操、花样滑冰、篮球）日晒减少抑制维生素D生成 → 应检测血清维生素D
- 维生素D的抗炎作用：增加抗炎细胞因子，减少IL-6等炎症因子

### 5. 维生素E与维生素C

**维生素E**：
- 运动前14天至运动后7天补充800IU/天可降低肌肉酸痛评分
- 可能削弱人体正常的保护性抗氧化机制
- 从食物获取比补剂更安全

**维生素C**：
- 结果不一致：运动前补充14天（3g/d）可降低24小时后疼痛
- 运动前仅补充3天则无效果 → 补充持续时间是关键
- 3g/d是极大剂量（DRI的30-40倍），长期使用增加肾结石风险

### 6. 支链氨基酸（BCAA）

- 运动前15分钟摄入5g BCAA
- **女性**：24h、48h、72h肌肉酸痛评分明显降低
- **男性**：未感受到益处（可能是固定剂量问题，需要按体重调整）
- 10g可能是满足男性所需的剂量

### 7. 蛋白质摄入策略

- 运动（特别是不熟悉的离心运动）导致肌肉损伤和蛋白质分解
- 运动员蛋白质需要量约为非运动员的2倍（约1.5g/kg）
- **少量多次摄入**（每餐约30g）比一次性大剂量补充效果好得多
- 巧克力牛奶是促进运动后快速恢复的理想饮料（碳水+优质蛋白混合物）

### 8. 液体摄入与DOMS

- 脱水组肌肉酸痛程度比充分水合组高**44%**
- "解决污染的方法是稀释"——更大液体池可稀释代谢副产物
- 维持充分水合状态是减轻肌肉酸痛的基本策略

### 9. 酒精的影响（重要警告）

- 酒精是**抗营养素**：抑制维生素B转换为活性辅酶
- 糖原再合成被显著抑制：饮酒运动员需要**2倍时间**恢复糖原储备
- 是强力利尿剂：即使4%酒精含量也会促进尿液流失
- 影响中枢神经系统 → 运动技能和认知功能下降
- 这些影响可能持续数小时至数天

## 本章要点总结

1. 营养补剂对**消除**肌肉损伤无用处，但可能**减轻**某些症状（主要是疼痛）
2. 必须在运动前数天至数周持续补充才可能有效
3. 最佳剂量不确定，且不同营养素的最佳预处理期可能不同
4. 在非阻力训练者中效果最明显（已适应者耐受性更高）
5. **维持充分水合 + 规律摄入优质蛋白质 + 富含ω-3和维生素的食物**是最可靠的策略
