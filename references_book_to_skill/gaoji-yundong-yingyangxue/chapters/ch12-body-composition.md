# 第12章 身体成分与体重

## 章节概述

本章探讨运动员的身体成分评估、体重管理策略以及能量平衡与身体成分之间的关系。核心发现：传统的能量摄入不足策略不仅不能有效改善身体成分，反而可能导致去脂体重（肌肉）的丢失大于脂肪的丢失，使体脂率反而升高。

## 核心概念

### 1. 能量缺乏的悖论

**常见误区**：减少能量摄入 = 减少体脂 = 改善身体成分

**实际结果**：
- 能量摄入不足 → 身体代偿性减少代谢旺盛的组织（肌肉） → 代谢率下降
- 去脂体重的丢失量 > 脂肪的减少量
- 脂肪在体重中所占比例反而升高
- 恢复进食后，反弹增加的成分中脂肪多于肌肉

**极端案例**：
- 神经性厌食症患者死亡时体重极低、瘦体重明显减轻（心脏质量约为正常的50%），但体脂率却相对较高

### 2. 日内能量平衡与身体成分

**关键研究发现**：
- 4组国家级女性运动员（竞技体操、艺术体操、中距离跑、长跑）的比较：
  - 与最佳能量平衡偏差最大的运动员具有最高的体脂水平
  - 艺术体操运动员能量缺乏最严重（接近-800kcal/d），体脂率在4组中最高
  - 中距离赛跑运动员具有最佳的每日能量平衡状态和最低的体脂率

**吃零食实验**（每天增加3次×250kcal零食，总热量不变）：
- 2周后：体脂含量明显下降，去脂体重明显升高，无氧爆发力改善
  - 原因：自主减少其他餐次摄入量
  - 停用零食4周后效果消失 → 饮食习惯改变需持续强化

### 3. 身体成分评估方法

- **双能X线吸收法（DEXA）**：金标准，精确检测骨密度和身体成分
- **体重本身不能提供有用信息**——需要了解是脂肪还是肌肉的增减
- 体重稳定 ≠ 身体成分健康（可能肌肉减少、脂肪增加）

### 4. 减重策略的科学原则

**最佳减重策略**：轻微的能量不足 + 维持日内能量平衡 + 高进餐频率

| 策略 | 效果 |
|------|------|
| 大幅减少热量摄入 | 肌肉流失 > 脂肪流失，体脂率升高 |
| 少量多餐（每日6次+） | 去脂体重维持或增加，体脂下降 |
| 仅靠饮食减重（不运动） | 不该损失的身体成分同样减少 |
| 每日2餐（等热量） | 去脂体重明显下降 |

**运动强度选择**：
- 低强度运动（25%VO₂max）：脂肪供能比例高（~80%），但燃烧脂肪总量低
- 高强度运动（65-85%VO₂max）：脂肪供能比例较低（~60%），但燃烧脂肪总量更高
- **结论**：在规定时间内尽可能剧烈运动最有利于减脂

### 5. 高蛋白饮食与体重控制

- 高蛋白食物胃排空时间长 → 不推荐训练前/中摄入
- 运动饮料中添加蛋白质减少运动员真正需要的：液体、碳水化合物、电解质
- 蛋白质过量（>1.7g/kg）被转化为脂肪储存或燃烧供能 → 产生有毒含氮废物
- 氮废物排泄增加 → 水分流失增加 → 脱水风险增加

### 6. 按体重分级的运动项目的特殊问题

**高风险项目**：摔跤、举重、柔道、体操、花样滑冰、跳水等

**常见危险行为**：
- 摔跤运动员为降低体脂率和体重采取危险甚至可能致死的方法
- 长期摄入极低热量膳食 → 代谢率降低 → 去脂体重下降
- 脱水减重（赛前快速脱水以达到体重标准）

**正确策略**：
- 赛季前通过长期、渐进的体重管理达到目标体重
- 保持高进餐频率（每日6次+）
- 避免赛前急性脱水和极端节食
- 即使需要减重，也应保持轻微的能量不足（-300至-400kcal/d）

### 7. 代谢率的调节

- 低频率进食（每日2-3餐）→ 低代谢率 → 更易储存脂肪
- 高频率进食 → 更高的食物热效应 → 更快能量代谢率 → 不易发胖
- 长期低热量饮食 → 代谢率下降 → 正常进食后体重反弹加剧

### 8. 运动员的典型饮食问题

- **典型模式**：白天能量缺乏 + 晚上一次性大量进食
- **后果**：虽然24小时总能量平衡，但体重稳定而体脂高于理想水平
- **生理机制**：延迟进食 → 低血糖 → 肌肉丙氨酸动员 → 肝脏糖异生 → 肌肉流失 → 随后大餐 → 高胰岛素血症 → 脂肪生成增加

## 实用要点

1. **绝对不能感到饥饿**——饥饿=已经开始消耗肌肉
2. 早餐在起床后尽快摄入——醒来时血糖最低
3. 「重量」和「身体成分」是两个概念——需要DEXA等精确检测
4. 少量多餐是最有效的体重管理策略（不是大幅减少热量）
5. 总热量摄入充足是维持肌肉量和低体脂率的前提
