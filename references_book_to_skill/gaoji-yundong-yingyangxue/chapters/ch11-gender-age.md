# 第11章 性别与年龄

## 章节概述

本章阐述性别和年龄如何影响运动员的营养需求。不同性别和年龄段的运动员在能量代谢、激素环境、骨骼发育和营养需求方面存在显著差异，需要针对性的营养策略。

## 核心概念

### 1. 性别差异对营养需求的影响

#### 能量代谢差异
- **女性**：脂质氧化速率高于男性，蛋白质和碳水化合物氧化速率低于男性
- 女性在碳水化合物负荷法中可能不如男性有效——糖原储备和体能提升幅度较小
- **男性**：对碳水化合物的依赖性更高，糖原储备能力更强

#### 铁状态
- 女性运动员（尤其是育龄期）缺铁性贫血比男性更为普遍
- 原因：月经失血 + 摄入不足 + 运动性溶血
- 女运动员应特别关注富含铁的食物（尤其是红肉）或在医生指导下补充铁剂

#### 维生素D和骨密度
- 女性运动员闭经与低雌激素水平相关 → 骨密度下降 → 应力性骨折风险增加
- 日内能量缺乏时间较长的女性运动员闭经相关性更强

#### 蛋白质需求
- BCAA补充对女性DOMS的效果比男性更明显（男性可能需要更高剂量）
- 固定5g BCAA剂量对女性有效，但男性可能需要10g

### 2. 年龄差异

#### 青少年运动员
- 同时面临**训练和生长发育**双重需求——蛋白质和总能量需求相对更高
- 年轻男孩比成年男性对碳水化合物的依赖性更高
- 骨骼发育关键期，充足的钙和维生素D摄入至关重要
- 限制饮食的运动项目（体操、摔跤等）中青少年运动员营养不良风险更高

#### 老年运动员
- 老年人肌肉量减少 → 寒冷环境下的应对能力更差
- 从运动或寒战中获得热量的能力下降
- 需通过规律锻炼维持肌肉量以降低体温过低的主动防御能力
- 维生素D的DRI在70岁后提高到**20μg/d（800 IU/d）**
- 维生素B₁₂吸收可能因胃功能受损（内因子减少）而下降

### 3. 女性运动员三联征

三个相互关联的综合征：
1. **能量可用性不足**（有或无饮食紊乱）
2. **月经功能紊乱**（闭经）
3. **骨密度降低**（骨质疏松）

**核心驱动因素**：日内能量缺乏——即使24小时总能量平衡，日内长时间的能量缺乏也与闭经有最强相关性。

### 4. 青少年的特殊风险

- 骨骼过早闭合（合成代谢类固醇导致）→ 身材较矮小
- 过度训练损伤、营养不良和心理应激可断送有天赋的年轻运动员的前途
- 追求过低的体脂水平是危险行为——摔跤运动员甚至采取可能致死的方法
- 年轻运动员试图通过短期快速提高的做法仅适用于训练有素的专业运动员

### 5. 不同生命阶段的营养要点

| 生命阶段 | 关键营养素 | 特别注意事项 |
|---------|----------|------------|
| 青春期 | 钙、维生素D、铁、蛋白质 | 生长+训练双重需求；限制饮食项目风险高 |
| 成年女性 | 铁、钙、维生素D | 月经铁丢失；闭经影响骨密度 |
| 孕期 | 叶酸、铁、钙 | 需咨询医生，DRI可能高于常规值 |
| 老年 | 维生素D、B₁₂、蛋白质 | 胃功能减退影响B₁₂吸收；维持肌肉量 |

### 6. 素食运动员的特殊考量

- **维生素B₁₂缺乏风险**：植物性食物基本不含B₁₂
- **维生素B₂缺乏风险**：避免乳制品和大豆制品时风险更高
- **铁缺乏风险**：非血红素铁（植物来源）吸收率远低于血红素铁（动物来源）
- **蛋白质质量**：需要多样化植物蛋白来源以实现氨基酸互补

## 实用要点

1. 女性运动员应定期检测铁蛋白（不仅是血红蛋白）
2. 青少年运动员不应通过补剂寻求"快速增长"——平衡膳食和合理训练才是关键
3. 闭经不是训练的"正常现象"——是能量不足的警示信号
4. 素食运动员需特别注意B₁₂补充（强化食品或补剂）
5. 能量摄入不足时身体优先牺牲的是肌肉组织，而非脂肪
