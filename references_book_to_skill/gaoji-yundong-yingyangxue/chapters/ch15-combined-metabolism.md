# 第15章 爆发力—耐力型运动的代谢需求

## 章节概述

本章聚焦于既需要爆发力又需要耐力的综合型运动项目（如足球、篮球、曲棍球、网球、赛艇、综合格斗等），阐述这类运动的双重代谢特征以及相应的周期性营养策略。

## 核心概念

### 1. 爆发力—耐力型运动的代谢特征

这类运动的最大挑战：需要在**有氧和无氧系统之间频繁切换**。

| 运动阶段 | 主导供能系统 | 主要燃料 |
|---------|------------|---------|
| 冲刺/跳跃/爆发 | 无氧糖酵解 + PCr | 肌糖原、磷酸肌酸 |
| 持续跑动/巡航 | 有氧代谢 | 脂肪 + 碳水化合物 |
| 间歇性高强度 | 无氧→有氧循环 | 糖原为主 |
| 比赛最后阶段 | 混合（糖原耗竭时转向脂肪） | 取决于储备状态 |

### 2. 双重需求带来的营养挑战

**挑战1：糖原储备的双重压力**
- 无氧阶段大量消耗肌糖原
- 有氧阶段也依赖糖原（脂肪完全氧化需要碳水）
- 比赛持续时间长 → 糖原储备极易耗竭

**挑战2：间歇性恢复**
- 比赛中只有短暂的恢复窗口（如罚球、暂停、局间休息）
- 必须在这些短暂时间内快速补充能量和液体

**挑战3：体液大量丢失**
- 高强度间歇性运动产热量大
- 室内场馆（篮球场等）可能闷热
- 出汗速率高但补水机会有限

### 3. 比赛中的营养策略

#### 赛前
- 赛前90分钟完成高碳水膳食
- 持续啜饮运动饮料至比赛开始
- 确保良好的水合状态

#### 赛中（暂停/休息时间）
- 立即摄入运动饮料（6-7%碳水）
- 使用运动凝胶+水（提供浓缩能量）
- 抓住每一个可以补给的间隙

#### 赛后和比赛间歇期
- 立即摄入50-100g碳水化合物
- 接下来4小时内每小时1.0-1.2g/kg碳水
- 碳水+蛋白质混合摄入促进恢复

### 4. 对糖原依赖性的管理

- 爆发力—耐力型运动对糖原的依赖性**高于纯耐力运动**
- 比赛中补充碳水化合物是避免糖原耗竭的合理策略
- 许多运动员（如足球、曲棍球）在比赛期间只摄入水，失去了重要的提高运动能力的机会

**比赛期间补充碳水的益处**：
1. 维持血糖浓度，减少肝糖原消耗
2. 维持BCAA水平，避免中枢性疲劳
3. 抑制皮质醇产生（减少肌肉分解）
4. 持续向工作肌肉输送葡萄糖，减少肌糖原消耗

### 5. 周期性训练与营养周期化

**训练周期与营养匹配**：

| 训练阶段 | 营养重点 |
|---------|---------|
| 基础期（有氧为主） | 中等碳水（5-7g/kg），适量蛋白 |
| 专项期（增加强度和间歇） | 增加碳水（7-10g/kg），保证蛋白 |
| 赛前期 | 高碳水（8-12g/kg），糖原负荷 |
| 比赛期 | 赛中补充策略，赛后快速恢复 |
| 恢复期 | 逐步回归，维持营养质量 |

### 6. 赛季中的营养挑战

对于每周多场比赛的运动员（如篮球、曲棍球）：
- 无法执行完整的7天递减法
- **高碳水饮食和最佳水合状态变得更加重要**
- 必须在比赛间期的更短时间窗口内完成糖原恢复
- 需要更频繁的进餐计划（每日6-8次）

### 7. 团队项目中的常见问题

- **赛前过度训练**：教练可能在赛前要求多次排练 → 消耗糖原 + 增加心理压力
- **补剂滥用**：球员可能使用多种"能量"产品，相互作用的危险性未知
- **酒精消费**：团队社交活动中饮酒常见 → 糖原恢复受阻（需2倍时间）+ 脱水
- **睡眠不足**：旅行+比赛日程 → 影响恢复和免疫功能

### 8. 间歇性运动的能量消耗模式

- 虽总运动时间可能不长，但强度波动极大
- **高强度间歇**的单位时间能量消耗远高于稳态运动
- 比赛中无氧阶段产生的乳酸在有氧阶段可被重新利用（科里循环）
- 良好水合状态有助于缓冲乳酸——"污染的解决方法是稀释"

### 9. 赛前营养时间表（示例）

| 时间 | 行动 |
|------|------|
| 赛前7天 | 最后一次高强度训练，之后逐渐减少 |
| 赛前5-7天 | 开始增加碳水摄入 |
| 赛前3天 | 训练量大幅减少，碳水继续增加 |
| 赛前1天 | 轻量活动，大量摄入液体和碳水 |
| 比赛当天 | 赛前2-3小时完成主要膳食，赛前持续啜饮 |
| 赛中 | 每个暂停/休息：120-240ml运动饮料 |
| 赛后 | 立即摄入碳水+蛋白，开始糖原恢复 |

## 实用要点

1. 爆发力—耐力型运动员对糖原的依赖性最高
2. 赛中补碳水的机会虽有限但必须抓住每一个间隙
3. 赛前7天递减法是最佳策略，但若无法执行则高碳水饮食更为关键
4. 高进餐频率对维持整个赛季的表现至关重要
5. 团队运动中更需警惕酒精对恢复的负面影响
