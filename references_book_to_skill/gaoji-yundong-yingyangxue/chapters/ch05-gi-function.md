# 第5章 胃肠道功能与能量传递

## 章节概述

本章探讨运动中的胃肠道功能、胃排空的影响因素、能量和液体向工作肌肉的传递效率，以及常见的胃肠道问题（包括乳糖不耐受、谷蛋白不耐受/乳糜泻等）。核心理念是：胃肠道是将营养素输送给工作肌肉的关键通道，其功能的任何障碍都将直接影响运动表现。

## 核心概念

### 1. 胃排空与运动

胃排空是指单位时间内食物或饮料从胃排出的量。慢速胃排空意味着营养素和液体更慢进入小肠被吸收。

**影响胃排空的因素**（详见第3章）：
- 碳水化合物浓度（>7%减慢排空）
- 碳水化合物类型（单糖比多糖排空慢）
- 液体总量（胃内量越大排空越快）
- 液体温度（运动中冷饮排空更快）
- 脱水状态（脱水减慢排空）
- 精神压力和焦虑（显著减慢排空）

### 2. 运动中胃肠道不适的原因

运动中胃肠道问题极为常见，尤其在耐力型运动员中：

- **血液重分配**：运动时流向胃肠道的血液减少可达70-80%
- **机械冲击**：跑步等运动的反复震动
- **脱水**：加剧胃肠道缺血
- **高渗液体**：浓度过高的运动饮料
- **纤维摄入时机不当**：赛前高纤维食物
- **精神紧张**：比赛焦虑影响胃肠功能

### 3. 常见食物不耐受

#### 乳糖不耐受
- **原因**：乳糖酶合成不足，无法分解奶制品中的乳糖
- **症状**：腹胀、腹痛、腹泻、脱水
- **策略**：运动饮料极少以牛奶为基础；赛前和赛中避免含乳糖产品

#### 谷蛋白不耐受（乳糜泻）
- **越来越多运动员受影响**
- 需避免所有含小麦、大麦、黑麦的食品
- 需仔细检查运动补剂和能量棒成分

### 4. 优化胃肠道功能的策略

1. **训练胃肠道**：逐步让消化系统适应运动中摄入碳水和液体
2. **选择合适浓度**：6-7%碳水化合物溶液最易被吸收
3. **避免高纤维**：赛前和赛中选用低纤维淀粉类食物（如面食）
4. **分次少量摄入**：每10-20分钟摄入120-240ml液体
5. **葡萄糖聚合物的优势**：在低渗透压下提供更多碳水化合物
6. **避免高脂肪/高蛋白**：赛前和赛中这些食物胃排空时间长
7. **控制精神压力**：学习心理训练技巧减少焦虑对胃排空的影响

### 5. 能量传递效率

- **胃排空速率**决定能量可用性
- **肠道吸收**是第二个瓶颈——即使胃排空顺利，肠道吸收不良也会限制能量供应
- **溶液的渗透压**是关键：颗粒数量（而非颗粒大小）决定渗透压
- 葡萄糖聚合物可在低渗透压下输送更多碳水能量

### 6. 不同运动饮料的耐受性评估

- **果糖单独使用**比其他糖类更易引起胃肠道不适
- **大多数运动饮料**含多种类型的碳水化合物以平衡耐受性
- **含碳酸的饮料**可能增加饱腹感和降低饮水欲望（但科学证据显示对胃排空几乎没有影响）

## 实用要点

1. 在训练中测试比赛日将使用的饮料和食物——不要在比赛日尝试新品
2. 超级耐力运动员可"训练"消化系统耐受较高浓度碳水
3. 维持良好水合状态有助于维持胃排空速率
4. 有食物不耐受者应制作当地语言的饮食禁忌卡片
5. 运动后立即摄入碳水化合物可最大化糖原合成酶活性
