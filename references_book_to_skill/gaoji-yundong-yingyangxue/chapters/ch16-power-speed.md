# 第16章 爆发力—速度型运动项目

## 章节概述

本章为爆发力—速度型运动项目的专项营养计划，涵盖短跑、跳跃、投掷、举重、体操、跳水、短距离游泳等项目的具体营养策略和膳食计划示例。

## 核心概念

### 1. 爆发力—速度型项目的共同特征

- 运动持续时间短（通常<2分钟/组）
- 以无氧代谢（PCr系统和糖酵解）为主
- 对力量-体重比有严格要求（部分项目）
- 训练常分多组多次进行（间歇训练模式）
- 比赛/训练间有休息期

### 2. 营养核心策略

#### 碳水化合物
- 推荐量：5-7 g/(kg·d)（日常训练），高强度训练期可7-10 g/(kg·d)
- 确保训练前糖原储备充足
- 训练后快速补充糖原（利用糖原合成酶活性峰值窗口）

#### 蛋白质
- 推荐量：1.6-1.7 g/(kg·d)
- 分次摄入（每日6次，每餐约30g）
- 训练后立即摄入碳水+蛋白质混合膳食

#### 液体
- 即使运动时间短，出汗量也可能很大
- 保持良好水合以缓冲乳酸
- 训练前摄入420-660ml液体

### 3. 一日六餐模式的优势

传统的每日2-3餐模式导致：
- 白天长时间能量缺乏 → 肌肉分解
- 晚餐大餐 → 高胰岛素血症 → 脂肪生成

**六餐制**使运动员更容易达到能量平衡：
- 每3小时左右进餐一次
- 避免血糖大幅波动
- 维持稳定的能量和营养素供应
- 提高食物热效应

### 4. 膳食计划示例原则

膳食计划展示如何避免24小时内能量过剩和不足的急剧波动，目标是将能量偏差控制在**±400kcal**以内。

**关键设计原则**：
- 根据训练时间安排餐次
- 早晨训练前必须进食（醒来时血糖最低）
- 训练后立即摄入碳水化合物
- 总热量分配到6餐而非3餐

### 5. 体重敏感型项目（体操、跳水等）

**特殊挑战**：
- 追求低体重和高力量-体重比
- 容易能量摄入不足
- 肌腱和韧带损伤风险高

**策略**：
- 采用"轻微能量不足"而非大幅减食
- 频繁进餐以维持代谢率和血糖稳定
- 保证充足的钙和维生素D（骨健康）
- 通过运动后立即补充碳水减少肌肉分解

### 6. 一水肌酸在爆发力项目中的应用

- 适合：短跑、跳跃、投掷等需要重复性最大强度输出的项目
- 补充方案：5天用+5天停的周期性模式
- 注意：会增加体重（水潴留），可能对体重敏感项目不利

### 7. 训练后恢复窗口

- **0-2小时**：高GI碳水化合物 + 少量优质蛋白质
- **2-4小时**：中GI碳水化合物 + 均衡营养
- **当天剩余时间**：中至高GI碳水化合物，维持能量平衡

### 8. 常见问题与解决方案

| 问题 | 解决方案 |
|------|---------|
| 训练前没时间吃早餐 | 起床后立即摄入小份易消化食物（如香蕉+运动饮料） |
| 训练中感到疲劳 | 检查训练前血糖状态，确保碳水储备充足 |
| 难以增加肌肉 | 检查总能量摄入是否充足（通常是不足） |
| 体脂不降 | 提高进餐频率，而非减少总能量摄入 |
| 肌肉僵硬 | 检查糖原负荷是否过度（每g糖原储存3g水） |

## 实用要点

1. 爆发力型运动员也需要充足的碳水化合物（不是只有蛋白质重要）
2. 按体重分级的运动项目应通过长期渐进减重而非赛前急性脱水
3. 碳水化合物负荷法对灵活性要求高的项目（体操等）可能适得其反
4. 六餐制优于三餐制——无论总热量摄入多少
5. 训练后立即进食是当天最重要的营养时机之一
