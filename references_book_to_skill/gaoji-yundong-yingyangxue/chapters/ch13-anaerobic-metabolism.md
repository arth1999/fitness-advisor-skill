# 第13章 高强度爆发力—力量型运动员的无氧代谢

## 章节概述

本章聚焦于以无氧代谢为主的运动项目（如举重、短跑、投掷、跳跃、摔跤、体操等），阐述这类运动员独特的能量系统特征、营养需求以及优化无氧表现的营养策略。

## 核心概念

### 1. 无氧运动的能量系统特征

#### 磷酸肌酸系统（PCr系统/ATP-CP系统）
- **特点**：储存的磷酸肌酸以无氧方式生成ATP
- **持续时间**：用于最大强度活动，仅维持**数秒**
- **燃料来源**：磷酸肌酸储备
- **限制因素**：PCr储备非常有限
- **营养策略**：一水肌酸补充可使肌肉PCr含量增加约50%

#### 无氧糖酵解（乳酸系统）
- **特点**：通过糖原分解以无氧方式生成ATP，副产物为乳酸
- **持续时间**：不超过**2分钟**
- **燃料来源**：肌糖原
- **限制因素**：乳酸累积引起pH下降
- **缓冲策略**：良好水合（更大液体池稀释乳酸）；碳酸氢钠可能有效但证据不确定

### 2. 能量代谢系统的层次

| 系统 | 供能方式 | 持续时间 | 适用强度 |
|------|---------|---------|---------|
| PCr系统 | 无氧 | 数秒 | 最大强度 |
| 无氧糖酵解 | 无氧 | <2分钟 | 极高强度 |
| 有氧糖酵解 | 有氧 | 持续 | 高强度 |
| 有氧代谢系统 | 有氧 | 长时间 | 低强度 |

在现实运动中，这些系统并非孤立运行，而是以不同比例同时供能。

### 3. 力量型运动员的营养重点

#### 碳水化合物
- 高强度运动严重依赖碳水化合物供能（短跑95%来自碳水）
- 即使力量型运动以无氧为主，也需要碳水化合物维持血糖和神经功能
- 推荐摄入量：5-7 g/(kg·d)日常，高强度训练期可更高

#### 蛋白质
- 力量型运动员推荐量：1.6-1.7 g/(kg·d)
- 实际摄入常远超需求（达推荐量的300-775%）
- 关键：**蛋白质的日内摄入模式**比总摄入量更重要
- 每餐约30g优质蛋白，少量多次摄入

#### 一水肌酸（CrM）
- 对高强度重复性运动表现有微小但显著的改善
- 补充方案：每日10-28g分4次，连续5天→停5天，每月5天即可维持
- 可能增加体重（水潴留）
- 能量摄入充足的运动员可能无法从中获益

### 4. 健美运动员的营养误区

| 常见做法 | 问题 |
|---------|------|
| 蛋白质摄入>3.0 g/(kg·d) | 超出最高利用率1.5g/kg的部分被燃烧或转为脂肪 |
| 单次摄入>50g蛋白质 | 超过利用率，加重氮废物排泄 |
| 能量摄入不足 | 无法增加肌肉量——蛋白质被用于供能而非构建 |
| 依赖氨基酸补剂 | 1oz肉提供约7000mg氨基酸 vs 补剂仅500-1000mg，且补剂更贵 |

### 5. 按体重分级项目的特殊问题

- **摔跤、举重、柔道**等需要严格控制体重
- 长期极低热量饮食 → 代谢率下降 → 减重越来越困难
- 追求过低体脂水平可能采取危险甚至致死的方法
- **正确策略**：通过长期渐进体重管理 + 高进餐频率 + 轻微能量不足

### 6. 爆发力运动中的能量平衡

- 许多力量型运动员能量摄入不足——这是一个主要问题
- 能量不足 → 肌肉蛋白质作为燃料被消耗 → 力量下降
- 增加能量摄入（尤其碳水化合物）可能是最有效的"强化"策略

### 7. 碳酸氢钠（小苏打）的争议

- **理论**：中和无氧代谢产生的乳酸的酸性
- **实际**：
  - 体内水分充足的运动员似乎无法获得理论预期的益处
  - 钠可能比碳酸氢盐更重要（增加血容量=更大缓冲空间）
  - 可能引起严重胃肠道不适和恶心
  - 多数研究未发现对跑步速度有显著提高

## 实用要点

1. 力量型运动员同样需要充足碳水化合物（不是只有蛋白质重要）
2. 蛋白质摄入量超过1.7g/kg无额外合成代谢益处
3. 分次摄入蛋白质（每餐约30g）比一次性大剂量更有效
4. 保持能量摄入充足是最基础、最有效的"强化"策略
5. 一水肌酸补充应周期性使用（5天用+5天停）而非持续使用
