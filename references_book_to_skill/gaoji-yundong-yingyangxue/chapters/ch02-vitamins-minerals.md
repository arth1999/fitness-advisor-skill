# 第2章 维生素与矿物质

## 章节概述

本章详细介绍各类维生素和矿物质在运动代谢中的功能、运动员的特殊需求、缺乏与过量的风险、以及最佳补充策略。核心观点是："超量不如恰好足量"——过量补充维生素和矿物质同样会导致生理系统紊乱，从食物中获取微量营养素优于补剂。

## 核心概念

### 1. 膳食参考摄入量（DRI）体系

| 标准 | 定义 |
|------|------|
| RDA（推荐膳食供应量） | 满足97-98%健康人群需求的每日摄入量 |
| AI（适宜摄入量） | 当RDA无法确定时的近似估算值 |
| UL（可耐受最高摄入量） | 不产生不良影响的每日最高摄入量 |
| EAR（平均需要量） | 满足半数健康人群需求的摄入量 |

### 2. 水溶性维生素

#### 维生素B₁（硫胺素）
- **DRI**：男1.2 mg/d，女1.1 mg/d
- **运动员推荐量**：1.5-3.0 mg/d（按每1000kcal摄入0.5mg计算）
- **功能**：碳水化合物代谢的关键辅酶（TPP）
- **注意**：运动员很少缺乏，除非限制热量摄入

#### 维生素B₂（核黄素）
- **DRI**：男1.3 mg/d，女1.1 mg/d
- **运动员推荐量**：1.1 mg/1000 kcal
- **功能**：参与能量代谢（FAD和FMN辅酶）
- **注意**：素食运动员风险较高

#### 烟酸（B₃）
- **DRI**：男16 mg/d，女14 mg/d
- **功能**：参与糖酵解和脂肪合成（NAD/NADP辅酶）
- **重要警告**：过量补充会降低脂肪代谢，增加对碳水化合物的依赖，导致耐力下降

#### 维生素B₆
- **DRI**：1.3-1.7 mg/d
- **运动员推荐量**：1.5-2.0 mg/d
- **核心功能**：蛋白质代谢、糖原分解、神经递质合成
- **需求与蛋白质摄入量相关**：每摄入1g蛋白质需要0.016mg维生素B₆
- **毒性**：长期500-5000 mg/d可导致永久性周围神经病变

#### 维生素B₁₂（钴胺素）
- **DRI**：2.4 μg/d
- **功能**：红细胞生成、神经系统健康
- **风险人群**：纯素食运动员（植物性食物几乎不含B₁₂）
- **重要**：无证据表明大剂量注射B₁₂能提高运动表现

#### 叶酸
- **DRI**：400 μg/d
- **功能**：DNA合成、红细胞生成
- **来源**：深绿色蔬菜、豆类、强化谷物

#### 维生素C
- **DRI**：男90 mg/d，女75 mg/d
- **运动员推荐量**：200 mg/d
- **功能**：胶原蛋白合成、促进铁吸收、抗氧化、肉碱合成
- **UL**：2000 mg/d，长期>1g/d增加肾结石风险

### 3. 脂溶性维生素

#### 维生素A
- **DRI**：男900 μg RAE/d，女700 μg RAE/d
- **毒性阈值**：3000 RAE/d，潜在毒性高
- **β-胡萝卜素**：作为维生素A前体和抗氧化剂，毒性低

#### 维生素D
- **DRI**：15 μg/d（600 IU/d）
- **UL**：100 μg/d（4000 IU/d）
- **核心功能**：促进钙磷吸收、肌肉收缩、蛋白质合成代谢、改善免疫功能、抗炎作用
- **关键发现**：日晒对骨密度的影响大于膳食钙摄入量；室内运动员维生素D缺乏风险高

#### 维生素E
- **DRI**：15 mg/d
- **功能**：强抗氧化剂，保护细胞膜
- **警示**：每天补充≥400IU可能增加全因死亡率

#### 维生素K
- **DRI**：男120 μg/d，女90 μg/d
- **功能**：凝血、强化骨骼
- **注意**：服用华法林者需特别注意维生素K摄入

### 4. 常量矿物质要点

- **钙**（DRI: 1000 mg/d）：与维生素D协同维持骨密度，日晒比补钙对骨密度影响更大
- **铁**：最易缺乏的矿物质，运动员因溶血和血尿更易缺铁，需检测铁蛋白而不只是血红蛋白

### 5. 植物化学物质（七彩系统）

- **红色**（番茄红素）：清除自由基，与抗癌相关
- **绿色**（芥子油苷、叶酸）：与抗癌作用相关
- **橙色/黄色**（类胡萝卜素）：维生素A前体，抗氧化

## 实用要点

1. 从食物中获取维生素和矿物质优于补剂
2. 过量补充单一抗氧化剂可能破坏抗氧化剂池的平衡
3. 24%的营养补剂样本被检出违禁物质（诺龙），运动员应警惕
4. 仅当确实存在缺乏风险时才考虑补充，且应在专业人士指导下进行
