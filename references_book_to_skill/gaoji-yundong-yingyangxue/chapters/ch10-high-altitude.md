# 第10章 高海拔

## 章节概述

本章阐述在高海拔（寒冷+低氧）环境下，人体所经历的生理压力和营养压力，并提出有效的应对策略。核心论点是：高海拔环境的低氧条件会显著增加能量需求、改变底物利用模式、增加氧化应激，运动员需通过特殊的营养策略来应对这些挑战。

## 核心概念

### 1. 高海拔的界定

| 分类 | 海拔范围 |
|------|---------|
| 高海拔 | 5000-8200 ft (1500-2500 m) |
| 超高海拔 | 8200-18000 ft (2500-5500 m) |
| 极限海拔 | ≥18000 ft (≥5500 m) |

### 2. "高住低练"（LHTL）模式

- **高住高练**的替代方案——在高原训练会导致速度下降、输出功率下降、氧气流量降低
- **LHTL目标**：建立组织适应性以提高携氧能力，改善脂肪代谢，降低碳水依赖
- **要求**：至少在海抜6600-8200 ft (2000-2500m)居住**4周**

### 3. 高海拔的生理适应机制

| 适应机制 | 表现 |
|---------|------|
| 通气量增加 | 呼吸从14次/分钟增至20余次/分钟（缺氧通气反应） |
| 心率加快 | 儿茶酚胺介导 |
| 心输出量增加 | 儿茶酚胺介导 |
| 红细胞增多 | 提高携氧能力最有效的方式 |
| 冷诱导血管舒张（CIVD） | 暴露约10分钟后启动，脉冲性调节末梢血管 |

### 4. 寒冷环境中的能量代谢变化

- 寒战使总能量消耗增加**2.5倍**，主要来自碳水化合物氧化的增加
- 工作肌肉消耗的能量中30-40%用于运动，60-70%以热的形式散失
- 血浆儿茶酚胺水平升高 → 肌糖原消耗增加
- 维持充足碳水化合物摄入对于在寒冷环境中的表现至关重要

### 5. 高海拔疾病（HAI）综合征

| 疾病 | 核心症状 | 关键治疗 |
|------|---------|---------|
| 急性高山病（AMS） | 头痛+胃肠道问题/疲乏/眩晕/入睡困难 | 停止攀升、休息、降低海拔>1600ft |
| 高原脑水肿（HACE） | 精神状态改变或共济失调 | 立即降低海拔>3300ft、吸氧、地塞米松 |
| 高原肺水肿（HAPE） | 静息时呼吸困难、咳嗽、胸闷 | 吸氧、降低海拔1600-3300ft |

**预防措施**：缓慢攀升（最快600 m/d）、在低海拔地区睡眠、避免用力过度、出发前1天至抵达后2天口服乙酰唑胺。

### 6. 增加高原反应风险的因素

- 海拔攀升过快
- **高脂、高蛋白、低碳水化合物膳食**
- 长期滞留在高海拔
- 高活动水平
- 肥胖者更易发生急性高山病

### 7. 高海拔的能量与营养需求

**能量需求**：
- 攀登珠穆朗玛峰的能量消耗比平原高**2.5-3倍**
- 但食物摄入通常降低10-50%（取决于海拔升高速度）
- 体重下降是普遍现象，包括肌肉质量流失

**碳水化合物优先**：
- 相对于脂肪和蛋白质，碳水化合物代谢的耗氧量更低
- 有利于糖原储备、减少蛋白质消耗
- 碳水化合物摄入不足 → 低血糖 → 精神紊乱和定向障碍

**铁的需求**：
- 每天需要约**18mg**铁
- 但高海拔常伴随食欲减退，难以满足
- 提前确保铁状态良好（补充铁剂需数月才见效）

**抗氧化剂需求**：
- 高海拔增加氧化应激水平
- 建议补充**复合抗氧化剂**（维生素C + β-胡萝卜素 + 硒 + 维生素E），而非单一抗氧化剂
- 只补充一种抗氧化剂效果不如复合补充

### 8. 液体需求

- 穿着保暖冬衣进行中高强度运动时出汗速率接近**2 L/h**
- 低温环境中的脱水原因：
  - 难以获得足够饮用水
  - 水分丢失过多（厚重衣物）
  - 经呼吸丢失水分
  - 寒冷引起的多尿（CID）

- 融化足够冰雪以维持液体需要需超过**6小时**和约**2L汽油**
- 液体容易冻结——需要合适的保温方法

### 9. 高海拔烹饪的挑战

- 海拔每升高**5000ft（1500m）**，做饭时间增加**1倍**
- 包装好的高碳水化合物零食或食物是最佳代餐品
- 好的选择：低脂全谷物能量棒、燕麦棒、高蛋白棒

## 实用要点

1. 缓慢攀升：每天不超过600m，每攀升600-1200m休息1天
2. 优先保证总热量摄入——在食欲减退时"被迫进食"
3. 高碳水饮食优于高脂/高蛋白饮食（耗氧量更低）
4. 提前确保铁状态良好（出发前数月开始）
5. 准备足够的液体和防冻措施
6. 补充复合抗氧化剂而非单一抗氧化剂
