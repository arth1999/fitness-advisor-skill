# 第14章 耐力型运动的有氧代谢

## 章节概述

本章聚焦于以有氧代谢为主的耐力运动项目（如马拉松、长距离自行车、铁人三项、越野滑雪等），阐述有氧运动的能量系统特征、耐力运动员的特殊营养需求以及优化耐力表现的营养策略。

## 核心概念

### 1. 有氧运动的能量系统特征

#### 有氧代谢系统
- **燃料来源**：碳水化合物+脂肪（少量蛋白质）
- **特点**：可以持续生成大量ATP，不会产生限制性的副产物
- **关键限制因素**：碳水化合物储备（糖原和血糖）有限
- **蛋白质供能**：占总能量需求的3-5%，糖原低时可>5%

#### 耐力训练对代谢的适应
- 增加线粒体体积和数量 → 提高脂肪氧化能力
- 在更高运动强度下仍能以脂肪为主要燃料
- **节约糖原** → 延缓疲劳 → 提高耐力

### 2. 耐力运动员的碳水化合物需求

**最高需求水平**：
- 中高强度耐力训练日常：7-12 g/(kg·d)
- 极限训练（>4h/d）：10-12 g/(kg·d)或更多
- **例如**：70kg运动员按12g/kg计算需摄入840g碳水（3360kcal）

**碳水需求最高的原因**：
- 训练消耗大量糖原
- 即使以脂肪为主要燃料，仍需碳水化合物参与脂肪完全氧化
- 维持血糖水平（中枢神经系统功能）
- 运动后糖原再合成

### 3. 耐力运动员的蛋白质需求

- **推荐量**：1.2-1.4 g/(kg·d)
- 与力量型运动员（1.6-1.7）几乎相等甚至更高
- **原因**：
  - 耐力训练中蛋白质供能占3-5%
  - 糖原/血糖水平低时蛋白质供能比例增加
  - 可能存在少量蛋白尿
- 但占总能量比例仅9-10%（远低于力量型的14-15%），因为总能量摄入更高

### 4. 碳水化合物负荷法（赛前糖原最大化）

#### Sherman等（1983）推荐方法（安全版）：
1. **赛前7天开始**逐渐减少训练强度和时间
2. 同时逐步增加碳水化合物摄入
3. 无需先耗竭糖原（旧方法危险）
4. 保证良好水合状态（每储存1g糖原需3g水）

#### 碳水化合物负荷法的运动适用性：

| 适合 | 不适合 |
|------|--------|
| 马拉松 | 体操（灵活性重要） |
| 长距离自行车 | 赛车（灵活性重要） |
| 铁人三项 | 跳水（力量-体重比） |
| 越野滑雪 | 花样滑冰（动作精准性） |

### 5. 运动中碳水摄入策略

- **运动饮料**：6-7%碳水溶液，每10-20分钟摄入120-240ml
- **运动凝胶**：在更小包装中包含更多能量，需与水同服
- **维持血糖**可避免：
  - 肝糖原过度消耗
  - BCAA下降导致中枢性疲劳
  - 皮质醇（分解代谢激素）升高
  - 肌糖原过度消耗

### 6. 耐力运动员常见的营养问题

| 问题 | 原因 | 后果 |
|------|------|------|
| 能量摄入不足 | 高训练量 + 食欲抑制 + 过度关注体重 | 肌肉流失、表现下降、免疫抑制 |
| 铁缺乏 | 足底撞击性溶血 + 出汗丢失 + 胃肠道出血 + 摄入不足 | 携氧能力下降、耐力下降 |
| 胃肠道问题 | 运动中胃肠缺血 + 机械冲击 + 高渗饮料 | 营养吸收障碍、被迫降速 |
| 低钠血症 | 过度饮水 + 钠摄入不足 + 长时间出汗 | 危险——可危及生命 |

### 7. 水合策略对耐力运动员的重要性

**核心指标**：血容量
- 血容量降低 → 每搏输出量降低 → 心率代偿性加快
- 出汗速率降低 → 体温升高 → 中暑风险
- 即使有可用液体，运动员也会发生自发性脱水

**马拉松实际情况**：
- 在50-54°F（10-12°C）下比赛：体重下降1-5%
- 在温暖天气下比赛：体重下降约8%（体内水分丢失12-15%）

### 8. 超级耐力运动中的特殊考量

- 运动中需额外碳水化合物（因能量消耗极大）
- "训练"消化系统耐受较高浓度碳水化合物
- 钠的补充至关重要（长时间出汗丢失大量钠）
- **低钠血症风险**：只补水不补钠的危险远大于轻微脱水
- 赛前超水合策略（甘油已被WADA禁用）

## 实用要点

1. 耐力运动员对碳水需求是所有运动员类型中最高的
2. 蛋白质需求（每kg体重）与力量运动员相当甚至更高
3. 运动中定时补充碳水+液体对维持耐力至关重要
4. 定期检测铁状态（铁蛋白）——耐力运动是铁缺乏的最高风险运动
5. 赛前7天递减训练+增加碳水摄入是安全的糖原最大化策略
6. 长时间赛事中注意补钠，防范低钠血症
