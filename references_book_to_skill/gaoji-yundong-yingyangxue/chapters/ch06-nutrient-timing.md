# 第6章 营养素和液体的摄入时机

## 章节概述

本章为全书核心章节之一，提出"日内能量平衡"的革命性概念——机体的能量和液体摄入与消耗应保持动态的实时平衡，而非以24小时为单位的粗略平衡。这是Dan Benardot近20年深入研究的核心理念，也是当今运动营养领域的热点议题。

## 核心概念

### 1. 日内能量平衡（Within-Day Energy Balance）

**传统误区**：能量和营养素的供给以24小时为单位。

**正确认知**：
- 机体以实时、动态的方式运转，内分泌系统和中枢神经系统持续对生物指标（血糖、胰岛素等）进行微调
- 血糖在进食后约1小时达到峰值，约3小时后恢复至空腹水平
- 因此每隔3小时左右需要补充能量，避免日内能量缺乏

**研究发现**：无论能量偏差表现为过剩还是不足，与最佳能量平衡偏差最大的运动员都具有最高的体脂水平。艺术体操运动员的能量缺乏最为严重（接近800kcal/d），其体脂率在这4组中也最高。

### 2. 进餐频率的关键影响

| 进餐频率 | 效果 |
|---------|------|
| 每日2餐（等热量） | 去脂体重明显下降 |
| 每日6餐（等热量） | 维持去脂体重，甚至增加 |
| 少量多餐 | 更高食物热效应，更快能量代谢率 |
| 增加零食（总热量不变） | 体脂下降，去脂体重升高，无氧爆发力改善 |

**机制解释**：
- 高进餐频率 → 较低胰岛素反应 → 减少脂肪生成
- 更快的代谢速率 → 单位时间燃烧更多热量
- 更有效脂肪利用 → 更容易维持肌肉量和较低体脂率

### 3. 避免日内能量波动过大的益处

1. **改善身体成分**：减少脂肪、维持或增加肌肉
2. **提升运动表现**：维持稳定的血糖供应
3. **改善血脂**：频繁进餐者血脂水平更低
4. **保护免疫功能**：避免低血糖削弱免疫细胞活性
5. **降低闭经风险**：日内能量缺乏时间越长，闭经相关性越强

### 4. 营养素时机与免疫系统

- 免疫细胞以葡萄糖和谷氨酰胺为燃料
- 血糖快速降低会削弱巨噬细胞和淋巴细胞功能
- **剧烈运动后立即摄入碳水+蛋白质**可预防免疫功能退化（延迟1小时则无此效果）
- 禁食状态下运动更严重——削弱免疫和抗氧化系统

### 5. 赛前7天递减法

**三个主要目标**：

1. **逐渐转换到休息状态**：避免赛前过度训练（增加患病和受伤风险）
2. **逐步增加肌糖原储备**：降低训练强度、增加碳水摄入
3. **具备良好的水合状态**：减少运动量、大量摄入液体

**关键禁忌**：
- 赛前强化训练是极大错误——会传达给运动员不信任的信息
- 应在得到充分休息的情况下参赛

### 6. 运动前/中/后的碳水摄入策略

#### 运动前（约90分钟前）
- 完成高碳水化合物膳食摄入
- 之后持续啜饮含碳水运动饮料（每10-15分钟60-120ml）或吃低纤维淀粉类零食

#### 运动中
- 6-7%碳水溶液每10-20分钟摄入120-240ml
- 避免仅含果糖的饮料（更易引起胃肠道不适）
- 葡萄糖聚合物在低渗透压下提供更多能量

#### 运动后
- 立即摄入碳水50-100g（200-400kcal）
- 运动后2小时内：高GI碳水化合物
- 运动后2-4小时：中等GI碳水化合物
- 当天剩余时间：中到高GI碳水化合物

### 7. 碳水化合物负荷法

**Bergstrom等（1967）旧方法**：
- 先通过低碳水饮食+剧烈运动消耗糖原（危险：可导致突发致命性血压降低）
- 再高碳水+减量训练

**Sherman等（1983）改进方法**：
- 直接高碳水+减量训练，无需耗竭阶段
- 效果相当，安全性更高

### 8. 液体摄入时机

- 运动前：摄入420-660ml液体
- 运动中：每10-15分钟摄入90-240ml含6-7%碳水+钠的饮料
- 运动后：补充丢失体液的150%（丢失1L补充1.5L）
- 绝对不要等口渴才喝——口渴已是延迟信号（丢失1.5-2L后才出现）

## 实用要点

1. **绝对不能感到饥饿**——饥饿意味着已处于能量缺乏状态
2. 采用分食进餐计划：每餐分两次吃（如早餐的一部分在10点左右吃）
3. 运动后不要拖延进食——拖延会降低糖原恢复效率
4. 赛前7天是"递减"而非"加强"训练的时期
5. 碳水化合物负荷法不适用于所有运动（体操等需要灵活性的项目慎用）
