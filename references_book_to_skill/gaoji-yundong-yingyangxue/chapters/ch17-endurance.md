# 第17章 耐力型运动项目

## 章节概述

本章为耐力型运动项目（马拉松、长距离自行车、铁人三项、越野滑雪、长距离游泳等）提供专项营养计划。重点在于如何在长时间、高能量消耗的运动中维持能量供应、水合状态和电解质平衡。

## 核心概念

### 1. 耐力项目的能量需求特征

**能量消耗量级**：
- 马拉松：约2600-3000kcal/场
- 铁人三项：约8000-10000kcal/场
- 环法自行车赛：约5000-8000kcal/天

**代谢特征**：
- 以有氧代谢为主（脂肪+碳水化合物）
- 糖原储备是限制因素
- 低强度阶段脂肪供能比例高
- 冲刺/爬坡阶段碳水供能比例急剧上升

### 2. 耐力运动员的三大营养支柱

#### 碳水化合物——第一支柱
- 日摄入量：7-12 g/(kg·d)（高强度耐力训练）
- 极限训练（>4h/d）：10-12 g/(kg·d)以上
- 赛前：碳水化合物负荷法（7天递减法）
- 赛中：每小时30-60g碳水化合物（运动饮料+凝胶）
- 赛后：立即开始糖原恢复（1.0-1.2g/kg/h持续4小时）

#### 液体和电解质——第二支柱
- 出汗速率：1-3L/h（取决于环境条件）
- 钠丢失：每小时可丢失3-7g氯化钠
- 补液策略：每10-20分钟120-240ml含钠运动饮料
- 防范低钠血症：不能只补水不补钠

#### 蛋白质——第三支柱
- 日摄入量：1.2-1.4 g/(kg·d)
- 运动中供能占3-5%
- 训练后碳水+蛋白质促进恢复
- 巧克力牛奶作为恢复饮料的天然选择

### 3. 赛前7天递减法的执行

| 天数 | 训练 | 膳食 |
|------|------|------|
| 第7天 | 最后一次完整训练 | 高碳水开始 |
| 第6天 | 训练量减少约30% | 继续高碳水 |
| 第5天 | 训练量减少约50% | 进一步增加碳水比例 |
| 第4天 | 轻量训练 | 碳水占总能量70%+ |
| 第3天 | 极轻量或休息 | 高碳水+大量液体 |
| 第2天 | 轻松活动 | 最大化糖原储备 |
| 第1天 | 休息或极轻量 | 赛前餐+赛中补给计划就绪 |
| 比赛日 | 比赛 | 执行赛中补给计划 |

### 4. 赛中补给策略

#### 短于1小时的赛事
- 主要依靠赛前储备
- 赛中可不补或少补碳水

#### 1-2.5小时的赛事
- 每小时30g碳水化合物（约500-600ml标准运动饮料）
- 注意维持水合

#### 超过2.5小时的赛事
- 每小时30-60g碳水化合物（最多90g，使用多种糖源）
- 必须补充钠（每小时约0.5-1g）
- 使用运动凝胶+水的组合
- 在补给站持续补给

### 5. 特殊环境下的耐力策略

#### 高温环境
- 预冷策略（赛前冰背心等）
- 增加钠摄入（高温出汗更多）
- 更频繁的液体补给
- 使用热指数评估风险

#### 寒冷环境
- 碳水化合物需求更高（寒战增加消耗）
- 液体不易冻结的补给方式
- 注意呼吸道水分丢失

#### 高海拔环境
- 适应期至少4周
- 红细胞生成需充足铁、叶酸、B₁₂
- 碳水代谢耗氧量低于脂肪——更依赖碳水

### 6. 耐力运动的常见胃肠道问题

- **原因**：运动中胃肠道血流减少70-80% + 机械震动 + 高渗溶液
- **策略**：
  - 训练胃肠道（长期适应性训练）
  - 选择等渗或低渗运动饮料
  - 使用葡萄糖聚合物（降低渗透压）
  - 避免赛前高纤维和高脂肪食物

### 7. 低钠血症的预防

- **机制**：过度补水+钠摄入不足 → 血液钠浓度危险下降
- **症状**：头痛、恶心、意识模糊 → 可发展为脑水肿和死亡
- **预防**：
  - 按计划补水（不要过量）
  - 使用含钠运动饮料（而非纯水）
  - 赛事中补充含钠食品（椒盐饼干等）
  - 赛后不大量摄入纯水

### 8. 膳食计划要点

- 将每日食物分为6份较小份额（而非3份大份额）
- 重点不是晚餐——训练前和训练中的能量供应更重要
- 起床后立即进食（过夜禁食后血糖最低）
- 训练后立即补充碳水（抓住糖原合成酶峰值）

## 实用要点

1. 耐力运动是所有运动类型中对碳水需求最高的
2. 赛中每小时30-60g碳水是维持表现的关键
3. 钠的补充与碳水同等重要——不能只补水不补钠
4. 赛前7天递减法是最佳大赛准备策略
5. 个性化补给方案需在平时训练中测试和磨合
