# 第4章 强化剂

## 章节概述

强化剂（Ergogenic Aids）是指能增强体力或精神、消除疲劳症状的物质。本章系统评估了各类合法营养强化剂的作用机制、功效证据和安全性，强调"平衡膳食比药水和药片更有效、更安全、更经济"。核心警示：多数宣称具有功能强化特性的物质根本无此功效；若运动员规律摄入满足需要量的能量和营养素，那些有效的物质也将失去其功能强化特性。

## 核心概念

### 1. 强化剂分类

| 类别 | 示例 |
|------|------|
| 物理强化剂 | 力量训练器材、轻量跑鞋、鼻贴 |
| 药理强化剂 | 合成代谢类固醇（禁用）、高剂量营养素 |
| 生理强化剂 | 血液回输（禁用）、桑拿浴、按摩 |
| 营养强化剂 | 碳水化合物负荷、运动饮料、咖啡因、肌酸 |
| 心理强化剂 | 催眠、放松技巧、想象技巧 |

### 2. 碳水化合物作为强化剂

- 碳水化合物是最典型且被充分验证的营养强化剂
- **碳水化合物负荷法的目标**：最大化糖原储备
- **注意**：每储存1g糖原伴随储存3g水，可能造成肌肉僵硬（对体操、赛车等运动不利）
- 女性可能不如男性有效——女性脂质氧化速率更高

### 3. 一水肌酸（CrM）

- **机制**：由精氨酸、甘氨酸和甲硫氨酸构成，与磷酸结合生成磷酸肌酸（PCr），用于维持高强度运动中的ATP水平
- **补充方案**：每日10-28g，分4次，持续5天后肌肉肌酸饱和；每月补充5天即可
- **效果**：对高强度运动表现有微小但显著的改善；可增加体重（水潴留）
- **安全性**：无儿童/青少年长期安全性数据；能量摄入充足的运动员可能无法从中获益

### 4. 咖啡因

- **来源**：咖啡、茶、可乐、巧克力等
- **机制**：中枢神经系统兴奋剂；促进脂肪酸动员（理论上节约糖原）
- **推荐剂量**：运动前约1小时摄入3-6 mg/kg；耐力运动最后阶段1-2 mg/kg
- **效果**：对耐力表现的平均提升率约3.2%±4.3%
- **策略**：赛前至少7天不摄入咖啡因以获得最大效果
- **安全性**：正常剂量无利尿作用；过量引起易激惹、失眠、腹泻、焦虑

### 5. ω-3脂肪酸

- **潜在益处**：降低红细胞黏稠度以改善氧输送、增强有氧代谢、增加生长激素释放、减轻组织炎症
- **实际证据**：对力量和耐力的持续改善未被证实；对减轻肌肉酸痛的研究结果不一致
- **推荐**：每周吃一次冷水鱼即可显著降低心脏病和卒中风险

### 6. 其他强化剂评估

| 强化剂 | 证据强度 | 注意事项 |
|--------|---------|---------|
| 甘油（丙三醇） | 有效（超水合） | WADA已列为禁用物质 |
| 碳酸氢钠 | 效果不确定 | 可能引起严重胃肠不适 |
| 肉碱 | 效果甚微 | 素食运动员可能受益 |
| 中链脂肪酸（MCT） | 无体能益处 | 单次>30g增加腹泻风险 |
| 人参 | 缺乏证据 | 对自行车运动表现无影响 |
| 槲皮素 | 初步研究 | 需定期摄入维持血浆浓度 |
| 白藜芦醇 | 仅在动物实验有效 | 人体研究不足 |

### 7. 购买补剂前的关键考量

1. 孕妇或哺乳期妇女应谨慎
2. 同时服用的药物和补剂可能产生危险相互作用
3. ω-3脂肪酸等可能抑制凝血功能
4. 选择有NSF、USP或Consumer Lab认证的品牌
5. 多达25%的营养补剂含有未标注的禁用物质

## 实用要点

1. 补充额外碳水化合物是最简单、最有效的"强化"策略
2. 蛋白质补剂若用量超过1.5g/kg，多余的将被燃烧供能或以脂肪储存
3. 运动前补充葡萄糖聚合物溶液比运动中同量摄入更有效
4. 绝对不要使用来源不明的补剂产品
5. 能量摄入充足的运动员通常不需要任何强化剂
