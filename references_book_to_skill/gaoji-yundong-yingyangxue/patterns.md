# 核心模式与原则 — 高级运动营养学（第2版）

## 1. 日内能量平衡模式（Benardot核心框架）

**原则**：机体不是以24小时为单位运转，而是以实时、动态的方式持续调节。能量摄入与消耗应保持动态平衡。

**实践模式**：
- 进餐频率≥每日6次
- 约每3小时进食一次（匹配血糖波动周期：进食后1小时达峰→3小时后恢复空腹水平）
- 能量偏差控制在±400kcal以内
- 绝对不准感到饥饿

**底层机制**：
```
高频率进食 → 低胰岛素反应 → 减少脂肪生成
            → 高食物热效应 → 更快代谢率
            → 稳定血糖 → 减少肌肉分解
            → 更有效脂肪利用 → 更好身体成分
```

## 2. 碳水化合物窗口模式

**原则**：碳水化合物的储备有限，必须在正确的时间窗口补充。

| 时间窗口 | 策略 | 碳水类型 |
|---------|------|---------|
| 运动前90分钟 | 完成高碳水主餐 | 低→中GI |
| 运动前30分钟至运动开始 | 持续啜饮运动饮料 | 中→高GI |
| 运动中 | 每10-20分钟补充 | 6-7%碳水+钠溶液 |
| 运动后0-2小时 | 立即补充50-100g | 高GI |
| 运动后2-4小时 | 持续补充 | 中GI |
| 当天剩余 | 正常进餐 | 中→高GI |

## 3. 运动强度-燃料比例模式

```
步行（25%VO₂max） → 碳水40% ：脂肪60%
慢跑（65%VO₂max） → 碳水50% ：脂肪50%  
长跑（85%VO₂max） → 碳水80% ：脂肪20%
短跑（95%VO₂max+）→ 碳水95% ：脂肪5%
```

**关键推论**：高强度运动虽然脂肪供能比例较低，但单位时间内燃烧的脂肪总量更多。

## 4. 营养素剂量-效应天花板模式

**原则**：超过一定阈值后，额外摄入的营养素不再产生益处，甚至有害。

| 营养素 | 最大有效剂量 | 超过后的后果 |
|--------|------------|------------|
| 蛋白质 | 1.5-1.7 g/(kg·d) | 燃烧供能或储存为脂肪；氮废物排泄增加（脱水风险） |
| 蛋白质/餐 | ~30g | 超出部分利用率低 |
| 一水肌酸 | 约160 mmol/kg干肌肉（≈5天补充） | 不再增加储备；水潴留增加体重 |
| 维生素C | 200 mg/d（UL: 2000） | 耐度下降；肾结石风险 |
| 维生素E | 15 mg/d（RDA） | ≥400IU/d可能增加全因死亡率 |
| 烟酸 | 14-20 mg/d | 过量抑制脂肪代谢→耐力下降 |
| 咖啡因 | 3-6 mg/kg（运动前） | 耐受性增加；焦虑、失眠 |
| 中链甘油三酯 | 单次<30g | 超量→腹泻和胃肠不适 |

## 5. 糖原-水共储存模式

**原则**：每储存1g糖原，伴随储存约3g水。

**含义**：
- 糖原负荷→体重增加约1-2kg（主要为水）
- 肌肉含水量75% vs 脂肪10%
- 脱水时糖原恢复困难
- 对灵活性要求高的项目（体操等）：过度糖原负荷可能不利
- 对炎热环境运动：额外的水储存是有利因素

## 6. 乳酸代谢与缓冲模式

**原则**：乳酸是"暂时的能量储存形式"，而非有害废物。缓冲乳酸的关键是充分水合。

**机制链**：
```
充足水合 → 更大血容量 → 更大液体池稀释乳酸 → 更好pH缓冲 → 延缓疲劳
脱水     → 更小血容量 → 乳酸浓度更高  → pH快速下降 → 加速疲劳
```

**科里循环**：肌肉排出的乳酸 → 肝脏 → 转化为葡萄糖 → 返回肌肉供能

## 7. 中枢神经系统疲劳模式

```
低血糖 + 肌糖原下降
    ↓
BCAA分解代谢增强（用于糖异生）
    ↓
血浆BCAA水平降低
    ↓
色氨酸竞争更多血脑屏障受体载体
    ↓
大脑色氨酸增加 → 5-羟色胺合成增加
    ↓
中枢神经疲劳 → 肌肉疲劳
```

**干预关键**：维持血糖和BCAA水平 → 控制色氨酸/BCAA比例

## 8. 三阶段肌肉保护模式

| 阶段 | 策略 | 目的 |
|------|------|------|
| 运动前 | 确保糖原储备充足 | 减少运动中蛋白质消耗 |
| 运动中 | 持续补充碳水 | 维持血糖+BCCA水平；抑制皮质醇 |
| 运动后 | 立即摄入碳水+蛋白质 | 利用糖原合成酶峰值；启动修复 |

## 9. 身体成分双悖论模式

**悖论1：吃得少≠瘦得快**
```
大幅减少能量摄入 → 身体代偿减少肌肉（高代谢组织）
                 → 代谢率下降
                 → 体脂率反而升高
```

**悖论2：吃得多≠体重增加（当进餐频率高时）**
```
增加零食（总热量不变）→ 自主减少其他餐次
                      → 更高食物热效应
                      → 体脂下降 + 去脂体重增加
```

## 10. 赛前7天递减优化模式

| 维度 | 第7天 | 第5天 | 第3天 | 第1天 |
|------|-------|-------|-------|-------|
| 训练量 | 100% | ~50% | ~20% | 极轻/休息 |
| 碳水比例 | 60% | 65% | 70%+ | 70%+ |
| 液体摄入 | 正常 | 增加 | 大量 | 最大化 |
| 心理状态 | 自信练习 | 巩固信心 | 积极肯定 | 放松备战 |

## 11. 营养素时序-免疫保护模式

**原则**：剧烈运动后立即摄入碳水+蛋白质可保护免疫功能。

```
运动后立即补充碳水+蛋白 → 维持血糖 → 保护免疫细胞功能
运动后延迟1小时补充     → 免疫抑制已发生 → 感染风险增加
空腹状态下运动           → 免疫和抗氧化系统双重受损
```

## 12. 热环境风险决策模式

**热指数评估 → 对应措施**：

| 体表温度 | 风险等级 | 对应措施 |
|---------|---------|---------|
| <90°F（32°C） | 相对安全 | 常规补水 |
| 90-104°F（32-40°C） | 中等风险 | 增加补水频率；监测症状 |
| 105-129°F（41-53°C） | 高风险 | 考虑取消/推迟；强制补水 |
| ≥130°F（≥54°C） | 极高风险 | 取消户外运动 |

## 模式应用原则

1. **个性化**：所有模式均需根据运动员的运动项目、训练阶段、体重、性别、年龄和环境条件调整
2. **训练中测试**：任何新的补给策略必须在训练中充分测试，绝不在比赛日尝试
3. **食物优先**：补剂应在食物无法满足需求时才考虑
4. **周期性**：营养策略应与训练周期同步变化
