# 第6章 维生素和矿物质

## 章节概述

微量营养素虽不直接供能，但作为酶的辅因子在能量代谢、氧气运输和抗氧化中起关键作用。本章综述维生素和矿物质对运动表现的影响。

## 核心原则
- 微量营养素摄入量以mg\~μg计，远低于宏量营养素（数百克）
- 人体无法自身合成，必须通过饮食摄取
- DRI（膳食参考摄入量）中仅1/3的营养素考虑了身体活动的影响
- RDA满足约98%健康人群，AI是基于有限科研数据的充足值
- 超过RDA的补充不提升运动表现，但缺乏时补充可恢复

## B族维生素（水溶性）

| 维生素 | RDA（男/女） | 运动相关功能 | 运动影响 |
|--------|-------------|-------------|---------|
| B₁（硫胺素） | 1.2/1.1mg | 丙酮酸→乙酰辅酶A，支链氨基酸脱羧 | 限制饮食运动员有低摄入风险 |
| B₂（核黄素） | 1.3/1.1mg | 电子传递链（FMN、FAD） | 训练增加需求；缺乏者补后运动表现改善 |
| B₃（烟酸） | 16/14mg | NAD/NADP电子载体 | 运动前>50mg阻碍脂肪动员，降低有氧耐力 |
| B₆（吡哆醇） | 1.3/1.3mg | 氨基酸转化、糖原磷酸化 | 5%\~60%运动员缺乏；耐力训练者需1.5\~2.3mg/天 |
| B₉（叶酸） | 400μg DFE | DNA合成、红细胞形成 | 8%\~33%女性有氧运动员血浆叶酸低 |
| B₁₂（钴胺素） | 2.4μg | 血红蛋白合成、甲基转移 | 不偏食者通常充足；补充不提升运动表现 |
| 泛酸 | 5mg（AI） | 辅酶A组成，三羧酸循环 | 补充1.8g/天×7天对50km计时赛无益 |
| 生物素 | 30μg（AI） | 氨基酸代谢、糖异生 | 无数据证明补充提升运动表现 |
| 胆碱 | 550/425mg（AI） | 乙酰胆碱、肌酸、卵磷脂形成 | 剧烈运动降低血浆胆碱；定期补充可能有帮助 |

## 维生素C（抗坏血酸）
- RDA：男90mg/天，女75mg/天
- 功能：合成儿茶酚胺和肉碱、促进铁吸收、抗氧化
- 10%\~30%运动员摄入不足，15%血浆浓度较低
- 缺乏者补充后跑步效率和有氧功率显著提升
- 生理应激增加维生素C需求

|--------|--------|------|---------|

## 关键矿物质

### 宏量元素

| 矿物质 | 日需量 | 运动相关功能 |
|--------|--------|-------------|
| 钠 | 1.5g（AI） | 体液平衡、神经传导、肌肉收缩 |
| 钾 | 4.7g（AI） | 体液调节、葡萄糖转运、糖原存储、ATP产生 |
| 钙 | 1000mg（AI） | 骨骼形成、神经传导、肌肉收缩、糖原代谢 |
| 磷 | 700mg（RDA） | ATP、磷酸肌酸、2,3-DPG组成 |
| 镁 | 400/310mg（RDA） | 300多种酶的辅因子，ATP酶活性 |

**镁特别关注**：
- 女性运动员摄入常仅60%\~65% RDA
- 高强度无氧运动后尿镁丢失增加21%
- 补充镁可降低肌酸激酶、增加腿部力量

### 微量元素

| 矿物质 | RDA（男/女） | 核心功能 |
|--------|-------------|---------|
| **铁** | 8/18mg | 血红蛋白/肌红蛋白形成，氧气运输 |
| 锌 | 11/8mg | 100多种金属酶，碳酸酐酶、乳酸脱氢酶 |
| 铜 | 900μg | 铁代谢、能量产生、抗氧化 |
| 硒 | 55μg | 谷胱甘肽过氧化物酶抗氧化 |
| 铬 | 35/25μg（AI） | 促进胰岛素作用 |

**铁缺乏三个阶段**：
1. 铁储备下降（血清铁蛋白<12μg/L）：15%\~20%女性，20%\~37%女运动员
2. 功能性铁缺乏/IDNA（sTfR>8.5mg/L）：非贫血性铁缺乏
3. 缺铁性贫血（Hb男<130g/L，女<120g/L）：5%\~12%女运动员

- 有氧训练增加铁损耗约3mg/天，应增加30%\~70%铁摄入
- 功能性铁缺乏女性补铁后计时赛成绩改善、肌肉疲劳感下降
- 锌缺乏：低锌饮食→血清锌↓→上下肢力量↓、通气率↑、做功效率↓

## 专业应用提示
- 持续限制饮食的运动员（体操、芭蕾、摔跤、啦啦队）面临更大缺乏风险
- 补充多种维生素矿物质的长期研究：澳大利亚体育学院国家运动员7\~8个月补充RDA的100\~5000倍后，运动表现无显著改善
- 平衡膳食（瘦肉、水果、蔬菜、全谷物、奶制品）是基础
- 仅在明确缺乏时通过补剂纠正，可恢复运动表现
- BMI评估和体成分测试是发现营养缺乏风险的重要工具
