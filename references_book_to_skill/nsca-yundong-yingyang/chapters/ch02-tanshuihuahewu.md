# 第2章 碳水化合物

## 章节概述

碳水化合物是运动中最关键的能源物质。本章系统阐述碳水化合物的分类、体内代谢调节以及在不同运动类型中的作用。

## 碳水化合物的四大功能

1. 神经细胞和红细胞的代谢能量燃料（大脑几乎只使用血糖）
2. 骨骼肌收缩的代谢能量燃料
3. 作为脂肪进入三羧酸循环的引物（"脂肪在碳水化合物的火焰中燃烧"）
4. 节省蛋白质（避免蛋白质被分解供能）

## 碳水化合物的分类

### 单糖（己糖，C₆H₁₂O₆）
- **葡萄糖（血糖/右旋糖）**：人体最重要的单糖，可直接被细胞使用
- **果糖（左旋糖）**：最甜的糖，存在于水果和蜂蜜中，需经肝脏转化为葡萄糖
- **半乳糖**：与葡萄糖结合形成乳糖，需经肝脏转化为葡萄糖

### 双糖（低聚糖）
- **蔗糖**：葡萄糖+果糖，"餐桌上的糖"，占美国总能量消费的1/4
- **乳糖**：葡萄糖+半乳糖，甜度最低
- **麦芽糖**：葡萄糖+葡萄糖，常见于谷物制品

### 多糖
- **淀粉**：直链淀粉（消化慢）和支链淀粉（消化快），占碳水化合物摄入的50%
- **纤维**：膳食纤维（非水溶性如纤维素、半纤维素）和功能性纤维（水溶性如β-葡聚糖、果胶），推荐男性38g/天、女性25g/天
- **糖原**：人体内碳水化合物的储存形式，肝脏浓度高但骨骼肌总量大（约400g）

## 血糖指数（GI）
- 反映摄入50g碳水化合物后血糖上升速率的指标
- **高GI食物**：运动饮料、精制糖、土豆泥——快速提升血糖和胰岛素
- **低GI食物**：糙米、全麦面包、多谷物面包——缓慢释放
- 运动前/中适合高GI（维持血糖），运动间隙适合低GI（稳定血糖）

## 体内糖的调节

### 血糖稳定
- 成人5升血液中仅约5克葡萄糖
- **胰岛素**（β细胞分泌）：降低血糖，促进糖原合成和葡萄糖摄取
- **胰高血糖素**（α细胞分泌）：升高血糖，促进肝糖分解和糖异生

### 糖原合成
葡萄糖 → 葡萄糖-6-磷酸 → 葡萄糖-1-磷酸 → UDP-葡萄糖 → 糖原（由糖原合成酶催化）

### 糖原分解（糖原→葡萄糖-1-磷酸→葡萄糖-6-磷酸→进入糖酵解）
由磷酸化酶催化，从非还原端逐一移除糖基

### 糖酵解
- 10步酶控反应链：1个葡萄糖（6C）→ 2个丙酮酸（3C）
- ATP生成速率最快的方式，发生在细胞质中
- 丙酮酸两种命运：转化为乳酸（无氧糖酵解）或进入线粒体（三羧酸循环）

### 乳酸
- pH值下降导致肌肉灼烧感和疲劳
- 乳酸根可被非运动肌、心肌甚至运动肌本身用作燃料
- 适当的训练可使运动员在高强度运动时乳酸处于相对较低水平

## 碳水化合物与运动表现

### 有氧运动
- 1小时高强度运动使肌糖原减少55%，2小时几乎耗尽
- "撞墙"现象：肌糖原耗竭导致疲劳
- 运动前优化糖原（>150 mmol/kg肌肉）可使耗竭时间延长20%
- 运动中补充0.5\~2g/min（30\~120g/h）碳水化合物维持血糖
- 推荐摄入：55%\~65%总能量，5\~7g/kg体重/天（高强度可达8\~10g）

### 无氧运动
- 糖原使用速率与运动强度成正比
- 肌糖原<30 mmol/kg时，依赖血糖供能（速率较慢）
- 高强度间歇运动中糖原对重复冲刺至关重要
- 推荐摄入：55%\~65%总能量，5\~7g/kg体重/天

### 力量训练
- 碳水化合物是抗阻运动最重要的能量来源
- IIx型快肌纤维糖原消耗是疲劳的重要原因
- **胰岛素促合成效应**：促进蛋白质合成、降低蛋白质分解、促进葡萄糖摄入和储存
- 建议运动前/中/后摄入液体碳水化合物促进恢复和瘦体重增加

## 专业应用策略
- 一天多场比赛时选择高GI食物快速恢复糖原
- 力量训练日常选择低GI，运动后选高GI以促进胰岛素反应
- 有氧项目：5\~7g/kg体重/天（高强度8\~10g）
- 无氧项目：5\~7g/kg体重/天
- 总能量3500kcal、65%来自碳水≈570g碳水/天（70kg运动员≈8g/kg）
