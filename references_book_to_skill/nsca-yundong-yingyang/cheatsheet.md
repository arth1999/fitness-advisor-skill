# 运动营养速查表 / Quick Reference Cheatsheet

## 每日宏量营养素推荐摄入量

| 运动员类型 | 碳水化合物 | 蛋白质 | 脂肪 |
|-----------|-----------|--------|------|
| **有氧耐力** | 8\~10g/kg（55%\~65%总热量） | 1.2\~1.4g/kg | ≥1g/kg（≥15%总热量，约30%最佳） |
| **无氧/间歇** | 5\~7g/kg（55%\~65%总热量） | 1.5\~2.0g/kg | 同上 |
| **力量/爆发力** | 5\~8g/kg（55%\~65%总热量） | 1.5\~2.0g/kg（上限） | 同上 |
| **一般健身** | 5\~7g/kg | 0.8\~1.0g/kg | 同上 |
| **减重期** | 不低于训练所需 | 1.5\~2.0g/kg（保持） | 减少，不低于15% |
| **增肌期** | 40%\~50%总热量 | 1.5\~2.0g/kg（上限） | 20%\~30%总热量 |

## 营养补充时机速查

| 时间窗口 | 有氧耐力 | 抗阻训练 |
|---------|---------|---------|
| **运动前4h** | 1\~4g/kg碳水 | — |
| **运动前2h** | 1\~2g/kg碳水+0.15\~0.25g/kg蛋白 | 6g必需氨基酸+35g碳水 |
| **运动前30\~60min** | 1.2\~1.5g/kg碳水 | 20g乳清蛋白（可选） |
| **运动中** | 每15\~20min：240\~480mL 6%\~8%碳水溶液 | 6%碳水+6g必需氨基酸 |
| **混合碳水** | 葡萄糖+蔗糖+麦芽糖糊精（氧化速率↑） | — |
| **运动后30min内** | 1.5g/kg碳水 | 30\~40g碳水+20\~25g蛋白（4:1） |
| **运动后2h** | 如4h内有下一场：0.6\~1.0g/kg/h碳水 | 同上剂量 |

## 运动饮料电解质标准

| 成分 | 推荐含量 | 作用 |
|------|---------|------|
| 碳水化合物 | 6%\~8%（6\~8g/100mL） | 供能，维持血糖 |
| 钠 | 460\~1150mg/L（20\~50mEq/L） | 补充流失、刺激口渴、保留水分 |
| 钾 | 78\~195mg/L（2\~5mEq/L） | 补充少量流失 |
| 渗透压 | 等渗（与血液相同） | 最易吸收 |

## 水合状态快速评估

| 指标 | 水合良好 | 轻度缺水 | 中度缺水 |
|------|---------|---------|---------|
| 体重变化 | <1% | 1%\~2% | >2% |
| 尿液颜色 | 浅黄色（柠檬水色） | 深黄色 | 橙色/棕色 |
| 尿量 | 正常/多 | 少 | 很少 |
| 口渴感 | 无 | 轻微 | 明显 |

## 常见场景营养方案

### 晨练者（6:00训练）
- 起床后立即：240\~360mL水 + 香蕉/能量棒（0.5g/kg碳水）
- 训练中：运动饮料（如>60min）
- 训练后：正常早餐（碳水+蛋白）

### 一天两练（上午+下午）
- 第一练后30min内：1.5g/kg碳水+0.3\~0.4g/kg蛋白
- 两练之间：高碳水午餐+补水
- 第二练前2h：加餐碳水1\~2g/kg
- 第二练后：同上补充

### 比赛日（下午场比赛）
- 赛前3\~4h：主餐 2g/kg碳水+0.15\~0.25g/kg蛋白+少量健康脂肪
- 赛前1h：小加餐 0.5g/kg碳水
- 赛中（如>60min）：运动饮料或凝胶+水
- 赛后：立即碳水+蛋白（4:1）

### 长途旅行比赛
- 随身携带熟悉的食物和补剂
- 提前了解目的地餐饮条件
- 旅行中每小时补水150\~200mL（飞机空气干燥）
- 到达后按当地时间调整进餐

## 危险信号（需立即转介专业人员）

- 女性连续3个月闭经 → 三联征风险
- 应力性骨折反复发生 → 骨密度问题
- 体重快速下降(>1kg/周) → 能量严重不足
- 催吐/滥用泻药证据 → 饮食紊乱
- 脱水导致意识模糊 → 低钠血症/热病紧急情况
- 运动后血尿/严重肌肉疼痛 → 横纹肌溶解
