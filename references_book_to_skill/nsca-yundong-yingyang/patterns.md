# 运动营养决策模式 / Decision Patterns

本文档提炼NSCA运动营养指南中的可复用决策模式和营养处方逻辑。

---

## 模式1：宏量营养素配比三段式

```
运动类型 ──→ 碳水化合物需求 ──→ 蛋白质需求 ──→ 脂肪需求

有氧耐力    → 8\~10g/kg/天        → 1.2\~1.4g/kg/天   → 剩余热量（≥1g/kg，≥15%总热量）
            → 55%\~65%总能量                              → 约30%总热量
                                                           → 饱和:多不饱和:单不饱和≈各10%

无氧/间歇   → 5\~7g/kg/天         → 1.5\~2.0g/kg/天   → 同上
            → 55%\~65%总能量

力量/爆发力 → 5\~8g/kg/天         → 1.5\~2.0g/kg/天   → 同上
            → 55%\~65%总能量       （应摄入上限值）

减重期      → 不低于训练所需       → 1.5\~2.0g/kg/天   → 减少总脂肪供能
                                   （保持防丢肌肉）

增肌期      → 40%\~50%总热量       → 30%总热量         → 20%\~30%总热量
            → +15%总能量摄入       → 1.5\~2.0g/kg/天
```

## 模式2：营养补充时机决策树

```
赛前/训练前多久？ ──→ <30min：0.5g/kg碳水
                 ──→ 1h：1g/kg碳水
                 ──→ 2h：1\~2g/kg碳水
                 ──→ 3h：1.5\~2g/kg碳水
                 ──→ 4h：2\~4g/kg碳水 + 0.15\~0.25g/kg蛋白

运动时长？ ──→ <30min：仅补水即可
          ──→ 30\~60min：考虑碳水补充
          ──→ 60\~90min：每10\~20min补充90\~240mL 6%\~8%碳水电解质饮料
          ──→ >90min：同上 + 考虑碳水+蛋白(4:1)混合物

运动后恢复窗口？ ──→ 立即(<30min)：1.5g/kg碳水（有氧）或30\~40g碳水+20\~25g蛋白（抗阻）
                ──→ 2h内：0.6\~1.0g/kg碳水/小时
                ──→ 4\~6h内：每2h补充等量碳水
                ──→ >24h：高碳水饮食(8\~10g/kg/天)即可
```

## 模式3：水合状态评估与补水决策

```
评估水合状态 ──→ 尿液颜色深？→ 缺水 → 运动前4h：5\~7mL/kg
            ──→ 尿比重高？  → 缺水 → 运动前2h：3\~5mL/kg
            ──→ 晨体重波动>1%？→ 缺水 → 调整日常补水

运动中补水 ──→ 每10\~20min补充90\~240mL
          ──→ 选择含6%\~8%碳水 + 460\~1150mg/L钠 + 78\~195mg/L钾的饮料

运动后补水 ──→ 称运动前后裸体重差
           ──→ 补充量 = 丢失体重 × 150%
           ──→ 每减重0.45kg → 补充600\~720mL液体
           ──→ 含钠食物/饮料促进保留水分
```

## 模式4：补剂评估三问与选择矩阵

```
评估任何补剂前：
Q1: 是否安全合法？
Q2: 是否有科学证据表明影响健康或运动表现？
Q3: 是否有可靠的科学原理？

选择矩阵（按运动目标）：

目标：增加最大力量和肌肉量
  ├─ 首选：单水合肌酸（填充20g/天×5\~7天，维持2\~5g/天）
  ├─ 次选：蛋白质补剂（乳清+酪蛋白混合物，1.5\~2.0g/kg/天）
  └─ 备选：HMB（3g/天，初练者或训练量骤增期）

目标：提升高强度运动耐力/延缓疲劳
  ├─ 首选：β-丙氨酸（3\~6g/天，单次≤800mg防刺痛）
  └─ 次选：碳酸氢钠/柠檬酸钠（注意肠胃不适）

目标：提升有氧耐力/长时运动表现
  ├─ 首选：运动饮料（6%\~8%碳水+电解质）
  ├─ 次选：咖啡因（3\~9mg/kg，运动前60min）
  ├─ 次选：BCAA（6\~12g，减轻中枢疲劳和肌肉损伤）
  └─ 备选：专利HMW碳水（快速糖原再合成）

目标：减少身体脂肪
  ├─ 基础：能量负平衡（每天-500kcal）
  ├─ 辅助：高蛋白膳食（增加饱腹感和产热效应）
  └─ 可选：鱼油（EPA+DHA 1\~2g/天，抗炎+潜在减脂）

目标：改善恢复/减少肌肉损伤
  ├─ 首选：运动后碳水+蛋白(4:1)
  ├─ 次选：BCAA（6\~12g）
  ├─ 备选：谷氨酰胺（免疫功能）
  └─ 备选：鱼油（抗炎）
```

## 模式5：体成分改变的分阶段策略

```
阶段判断：当前处于赛季什么时期？
  ├─ 非赛季 → 可激进改变体成分（增肌或减脂）
  ├─ 赛季前期 → 可适度调整
  └─ 赛季期间 → 尽量维持，不做大幅改变

减脂阶段：
  1. 确定目标体脂率（根据项目需求）
  2. 测量当前体成分（皮褶法/DEXA/Bod Pod）
  3. 设定能量缺口（每天-500kcal，每周-0.45kg）
  4. 保持蛋白质1.5\~2.0g/kg/天
  5. 调整碳水摄入（不低于训练需求）
  6. 减少脂肪供能（但不低于15%总热量）
  7. 维持抗阻训练（防止肌肉流失）
  8. 每2\~4周测量体成分调整
  9. 监控能量水平、睡眠、情绪（防过度训练）
  10. 达到目标后逐步恢复至维持热量

增肌阶段：
  1. 确定目标去脂体重
  2. 测量当前体成分
  3. 设定能量盈余（+15%，约+350\~500kcal/天）
  4. 蛋白质1.5\~2.0g/kg/天（摄入上限）
  5. 碳水40%\~50%总热量
  6. 脂肪20%\~30%总热量
  7. 结合周期性抗阻训练
  8. 训练前后补充蛋白+碳水
  9. 考虑肌酸补充
  10. 每2\~4周测量体成分监控脂肪增加
```

## 模式6：饮食紊乱风险筛查与转介流程

```
筛查信号（任一出现需关注）:
  ├─ 体脂率异常低（女性<12%，依项目而定）
  ├─ 闭经/月经过少（≥3个月）
  ├─ 应力性骨折史/骨密度低
  ├─ 快速体重下降（>1kg/周持续）
  ├─ 能量摄入明显不足（<基础代谢率）
  ├─ 食物回避行为（完整食物类别排除）
  ├─ 过度关注体重/体形
  ├─ 催吐/滥用泻药或利尿剂证据
  └─ 过度运动（远超训练计划）

分层响应：
  低风险（饮食失调可能性）→ 监控+教育+加强沟通
  中风险（亚临床饮食紊乱）→ 转介心理健康专业人员+营养干预
  高风险（临床饮食紊乱）  → 立即转介精神科/心理治疗+多学科团队

多学科团队组成：
  精神科医生/心理治疗师 + 注册营养师 + 队医 + 体能教练 + 专项教练
```

```
   │           │           │        │        │
   ▼           ▼           ▼        ▼        ▼

```

## 模式8：运动饮料选择决策树

```
运动时长 < 60min？
  ├─ 是 → 纯水即可（常温或凉爽环境）
  └─ 否 → 进入下一步

环境温度高/湿度大？
  ├─ 是 → 优先含电解质饮料（钠460\~1150mg/L）
  └─ 否 → 标准运动饮料

运动强度？
  ├─ 低强度（<50%VO₂max）→ 低浓度碳水溶液（≤4%）
  ├─ 中等强度（50%\~70%VO₂max）→ 标准6%\~8%碳水
  └─ 高强度（>70%VO₂max）→ 6%\~8%碳水+混合糖类+电解质

有无肠胃敏感史？
  ├─ 有 → 避免高果糖、避免高浓度(>8%)、小口多次
  └─ 无 → 标准配方

运动后快速恢复需求（<4h内有下一场）？
  ├─ 是 → 碳水+蛋白(4:1)+电解质，立即补充
  └─ 否 → 碳水+电解液+正常餐食即可
```
