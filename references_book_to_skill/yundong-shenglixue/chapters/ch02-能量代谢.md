# 第二章: 能量代谢

## Core Idea
ATP是生命活动的唯一直接能源。机体通过磷酸原系统、糖酵解系统、有氧氧化系统三条途径再合成ATP，三者并非按顺序而是相互整合协调供能，不同运动强度和持续时间下各系统的贡献比例不同。

## Frameworks Introduced
- **三大供能系统整合模型**: 磷酸原系统（ATP-CP）→糖酵解系统（乳酸能）→有氧氧化系统，容量递增但功率递减
  - 当需要：分析特定运动项目的能量供应特征或制定训练方案时使用
  - 如何：磷酸原供能≤7.5s（功率最高56 J/kg·s），糖酵解供能≈33s（功率29.3 J/kg·s），有氧氧化供能无限（功率仅15 J/kg·s）

- **ATP稳态概念**: 细胞在能量转换过程中维持ATP恒定含量的现象
  - 当需要：理解运动时能量代谢调节的核心目标时使用
  - 如何：ATP浓度过低则供能不足，过高则反馈抑制分解代谢

- **运动节省化**（Exercise Economy）: 同等负荷下达到更大功率输出或更高摄氧量水平的能力
  - 当需要：评估优秀运动员训练效果时使用——最大摄氧量进入平台期后，运动节省化成为成绩提升关键
  - 如何：通过改善骨骼肌能量代谢效率、优化运动单位募集、降低通气和心率实现

## Key Concepts
- **磷酸原系统**: ATP和CP分解供能，不需氧、不产乳酸，极量运动≤7.5s的供能基础
- **糖酵解系统**: 糖原/葡萄糖无氧分解生成ATP同时产生乳酸，供能约33s
- **有氧氧化系统**: 糖、脂肪、蛋白质彻底氧化，供能总量最大但速率最低
- **呼吸商(RQ)**: 单位时间CO2生成量与耗氧量的比值，反映底物利用类型（糖=1.0，脂肪=0.71）
- **基础代谢率(BMR)**: 基础状态下单位时间的能量代谢率
- **氧热价**: 某种食物氧化时消耗1L氧气所产生的热量
- **最大摄氧量(VO2max)**: 反映有氧工作能力的金标准指标

## Mental Models
- 将三大供能系统理解为汽车的"氮气加速（磷酸原）→涡轮增压（糖酵解）→巡航模式（有氧）"——都在运转但比例随需求变化
- 将ATP稳态理解为水库水位——进水（ATP合成）必须等于出水（ATP分解），水位太低则干涸，太高则溢堤
- 运动时能量代谢不是"切换"而是"混合"，即使是6秒极量运动也有有氧系统参与（13%-44%）

## Anti-patterns
- **误认为有氧系统响应慢**: 实际上即使是6s极量运动有氧系统也参与供能，30s Wingate实验中有氧供能占13%-44%
- **仅用最大摄氧量评估有氧能力**: 优秀运动员VO2max趋于稳定后，运动节省化水平的提高是成绩提升关键
- **忽略能量代谢测定的影响因素**: 肌肉活动、环境温度、食物特殊动力效应、精神紧张均显著影响代谢率

## Key Takeaways
1. ATP稳态是能量代谢调节的核心，ATP同时是能量受体和供体
2. 运动中三种供能系统整合供能，非按顺序切换，各系统贡献比例取决于运动强度和持续时间
3. 慢性运动训练可上调能量代谢酶活性、改善运动节省化、加速恢复过程
4. Wingate实验评价糖酵解能力，最大摄氧量评价有氧能力，磷酸原能商法评价ATP-CP系统
5. 最大功率输出出现在力和速度最佳匹配时（约为各自最大值的35%）
