# 第一章: 肌肉活动

## Core Idea
骨骼肌是运动系统的动力来源（占体重40%-45%），其收缩与舒张由生物电活动触发，通过兴奋-收缩耦联实现化学能向机械能的转换，不同类型的肌纤维在力量、速度和抗疲劳能力上存在根本差异。

## Frameworks Introduced
- **肌丝滑行理论**（Huxley, 1950s）: 肌肉收缩不是肌丝本身缩短，而是细肌丝向粗肌丝中央滑行，明带变短、暗带不变、H区变窄
  - 当需要：解释肌肉收缩的分子机制时使用
  - 如何：Ca2+与肌钙蛋白结合 → 原肌球蛋白构型改变暴露结合位点 → 横桥与肌动蛋白结合 → ATP水解供能 → 横桥摆动牵引细肌丝滑行

- **运动单位募集原则**（Size Principle）: 小α运动神经元（I型纤维）先被募集，大α运动神经元（II型纤维）后被募集
  - 当需要：设计训练强度以针对性发展不同类型肌纤维时使用
  - 如何：低强度运动→I型纤维优先；强度增大→IIa型纤维动员；最大强度→IIb型纤维为主

- **肌纤维类型三分法**: I型（慢缩氧化型）、IIa型（快缩氧化酵解型）、IIb型（快缩酵解型）
  - 当需要：分析运动项目的肌纤维需求或进行运动员选材时使用
  - 如何：依据ATP酶染色区分，I型有氧能力强抗疲劳，II型无氧能力强易疲劳

## Key Concepts
- **静息电位**: 细胞未受刺激时膜内负外正的电位差（-70~-90mV），主要由K+外流形成
- **动作电位**: 细胞兴奋时发生的迅速短暂可扩布的电位波动，是兴奋的标志（Na+内流形成上升支，K+外流形成下降支）
- **肌节**: 两相邻Z线间的一段肌原纤维，是肌肉收缩的基本功能单位
- **横桥**: 肌球蛋白头部突出，可与肌动蛋白结合并摆动，是力量产生的分子基础
- **兴奋-收缩耦联**: 膜电变化与肌丝滑行之间的中介过程，Ca2+是耦联媒介物
- **缩短收缩/等张收缩**: 张力大于阻力，肌肉缩短，做正功
- **拉长收缩/离心收缩**: 张力小于阻力，肌肉被拉长，做负功（制动减速作用）
- **等长收缩**: 张力等于阻力，肌肉长度不变，维持姿势

## Mental Models
- 将肌纤维募集理解为"先动用省力型慢纤维，再调用爆发型快纤维"的渐进策略
- 将力量-速度关系理解为跷跷板：负荷越大速度越慢，要获得最大功率需在力和速度间取最佳匹配（约最大值的35%）
- 将肌纤维类型分布理解为：遗传设定"默认值"（遗传度96.5%），训练可微调亚型间比例

## Anti-patterns
- **等张收缩训练发展力量不均衡**: 非等动收缩只有关节最弱点得到最大锻炼，应配合等动训练使全范围都得到锻炼
- **忽视离心收缩的训练价值**: 离心收缩做负功可产生更大张力，牵张-缩短环节可增强后续向心收缩的力量输出

## Key Takeaways
1. 动作电位遵循"全或无"原则，有绝对不应期，不衰减传导
2. 肌肉收缩的初长度影响张力，存在最适初长度使横桥作用数最多
3. 力、速度、功率三者存在依赖关系，最大功率出现在力与速度均为最大值35%处
4. 肌纤维类型主要由遗传决定但训练可造成选择性肥大（力量训练→II型肥大，耐力训练→I型氧化能力增强）
5. 肌电图可评价肌肉协调性、局部疲劳度，并辅助预测肌纤维类型
