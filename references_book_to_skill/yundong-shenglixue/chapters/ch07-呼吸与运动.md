# 第七章: 呼吸与运动

## Core Idea
呼吸是机体与外界环境间气体交换的全过程（外呼吸→气体运输→内呼吸）。肺通气是呼吸的起点，但肺通气功能通常不是限制最大摄氧量的主要因素——心泵功能储备远小于肺通气储备（20倍 vs 7-8倍），心脏才是有氧运动能力的关键限制环节。

## Frameworks Introduced
- **呼吸全过程三环节模型**: 外呼吸（肺通气+肺换气）→ 气体血液运输 → 内呼吸（组织换气）
  - 当需要：系统分析呼吸功能的瓶颈环节时使用
  - 如何：依次检查肺通气效率、肺换气匹配度(V/Q)、血液携氧能力、组织用氧能力

- **氧解离曲线三段分析法**: S形曲线反映Hb在不同PO2下的O2结合/释放特性
  - 当需要：评估运动时组织供氧能力时使用
  - 右上段（60-100mmHg）：结合段，平坦，PO2下降不影响载氧——高原运动安全保障
  - 中段（40-60mmHg）：释放段，较陡——安静组织供氧
  - 左下段（15-40mmHg）：陡峭段——剧烈运动时大量释放O2，氧利用系数可达77%

- **运动时呼吸调节的多因素假说**: 神经调节主导快速反应，体液调节主导慢速调整
  - 当需要：解释运动各阶段呼吸变化时使用
  - 如何：运动前通气增加=条件反射；运动起始骤升=大脑皮质和本体感受器传入；慢速增加=化学感受性反射（CO2、H+、温度）

## Key Concepts
- **肺活量(VC)**: 最大吸气后尽力呼出的气量，受年龄性别训练水平影响
- **肺泡通气量**: 每分钟吸入肺泡的新鲜空气量 = (潮气量-无效腔)×呼吸频率
- **通气/血流比值(V/Q)**: 正常安静约0.84，过大或过小均降低气体交换效率
- **氧解离曲线**: PO2与Hb氧饱和度关系曲线，受PCO2、pH、温度、2,3-DPG影响
- **P50**: Hb氧饱和度为50%时的PO2值，增大=曲线右移=亲和力降低=利于释放O2
- **氧利用系数**: (动脉氧含量-静脉氧含量)/动脉氧含量×100%，安静约26%，运动可达77%
- **氧脉搏**: 每次心跳摄取的氧量 = VO2/HR，反映心肺综合功能
- **氧通气当量(VE/VO2)**: 每分通气量与摄氧量的比值，越小说明摄氧效率越高

## Mental Models
- 将深慢呼吸vs浅快呼吸理解为：同等肺通气量下，深慢呼吸少"浪费"在无效腔的比例，如高速公路vs城市拥堵——同样车速但效率天差地别
- 将氧解离曲线S形理解为闸门：高原平坦段=闸门半开可保证基本流量；陡峭段=闸门灵敏全开，需求量增加立即大量释放
- 将V/Q比值理解为供需匹配度：V/Q=0.84=供需恰好；>0.84=有气无血（浪费通气）；<0.84=有血无气（部分静脉血未氧合就回心）

## Anti-patterns
- **运动时呼吸频率过快而不加深**: 浅快呼吸导致解剖无效腔通气占比增加，肺泡通气量反而降低
- **憋气时间过长**: 胸膜腔内压升高阻碍静脉回流，心输出量减少，尤其儿童和老年人危险
- **认为肺通气是耐力限制因素**: 实际上肺通气储备（20倍）远大于心泵储备（7-8倍），心脏才是瓶颈

## Key Takeaways
1. 呼吸过程的三个环节：外呼吸、气体血液运输、内呼吸，任一环节障碍均影响供氧
2. 深慢呼吸优于浅快呼吸——增加潮气量减少无效腔比例，提高肺泡通气效率
3. 运动时呼吸调控是多因素共同作用，神经调节主导快速变化，体液调节主导慢速变化
4. 氧解离曲线右移（P50增大）有利于组织获氧，运动时体温升高和CO2增多均促进右移
5. 呼吸肌训练可提升肺功能（肺活量、MVV）和耐力水平
