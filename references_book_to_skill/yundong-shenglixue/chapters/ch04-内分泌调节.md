# 第四章: 内分泌调节

## Core Idea
内分泌系统通过激素经血液运输实现对机体功能的缓慢、广泛、持久调控。内分泌功能轴（下丘脑-垂体-靶腺轴）是激素分泌的层级控制结构，运动应激下激素的分泌模式发生根本改变，参与对能量代谢和水盐平衡的精密调控。

## Frameworks Introduced
- **内分泌功能轴模型**: 下丘脑→垂体→靶腺三级调控
  - 当需要：理解激素分泌的层级控制和反馈调节时使用
  - 如何：下丘脑释放促激素释放激素→垂体分泌促激素→靶腺分泌效应激素→负反馈抑制上级腺体

- **激素对运动时能量代谢的调控**: 交感-肾上腺系统为主、胰岛激素为辅的双向调控
  - 当需要：分析运动时能量底物动员的机制时使用
  - 如何：儿茶酚胺促进糖原分解和脂肪动员，胰岛素降低促进脂肪酸利用，胰高血糖素升高维持血糖

- **激素对运动时水盐代谢的调控**: 抗利尿激素(ADH)和醛固酮协同保水保钠
  - 当需要：理解运动时体液平衡维持机制时使用
  - 如何：运动出汗→血浆渗透压升高→ADH分泌增加→肾远曲小管水重吸收增加；同时肾素-血管紧张素-醛固酮系统激活→Na+重吸收增加

## Key Concepts
- **激素**: 内分泌细胞分泌的经血液运输到靶组织发挥调节作用的化学信息物质
- **允许作用**: 某种激素本身不直接引起效应，但其存在是另一种激素发挥作用的前提
- **下丘脑-垂体-肾上腺皮质轴(HPA轴)**: 应激反应的核心内分泌轴
- **儿茶酚胺**: 肾上腺素和去甲肾上腺素，交感-肾上腺髓质系统的主要激素
- **胰岛素**: 促进合成代谢、降低血糖的激素，运动时分泌受抑制
- **胰高血糖素**: 促进糖原分解和糖异生、升高血糖
- **生长激素**: 促进蛋白质合成和脂肪分解，运动时显著升高

## Mental Models
- 将内分泌功能轴理解为军事指挥链：下丘脑（司令部）→垂体（前线指挥所）→靶腺（作战部队），上级下达指令，下级执行并反馈战况
- 将运动时的激素调控理解为"踩油门松刹车"：交感-肾上腺髓质兴奋（踩油门加速分解代谢），胰岛素降低（松刹车解除合成抑制）
- 将允许作用理解为"钥匙和锁"的关系：激素B是锁，激素A是打开锁的钥匙，只有钥匙在场锁才能发挥作用

## Anti-patterns
- **将运动时所有激素变化理解为同步**: 实际上不同激素有不同的反应时程，生长激素在运动后30-45分钟达到峰值，而儿茶酚胺几乎即刻升高
- **忽视激素受体敏感性的变化**: 长期训练可能改变靶组织对激素的敏感性，而非仅仅改变激素浓度

## Key Takeaways
1. 内分泌调控具有缓慢、广泛、持久的特点，与神经调节互补
2. 三大内分泌功能轴：下丘脑-垂体-肾上腺皮质轴、下丘脑-垂体-甲状腺轴、下丘脑-垂体-性腺轴
3. 运动时交感-肾上腺系统激活是能量代谢调控的主导机制
4. 激素通过细胞膜受体或细胞内受体两种机制发挥作用
5. 长期运动训练可改变内分泌腺的分泌能力和靶组织的受体敏感性
