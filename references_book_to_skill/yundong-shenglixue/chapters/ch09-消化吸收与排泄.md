# 第九章: 消化、吸收与排泄

## Core Idea
消化系统将食物中的糖、脂肪、蛋白质分解为可吸收的小分子物质供机体利用，肾则清除代谢废物并维持水盐和酸碱平衡。运动时消化系统血流量大幅减少，长期不当运动可能损害消化功能；运动对肾功能有显著影响，可导致运动性蛋白尿。

## Frameworks Introduced
- **三大营养物质消化吸收路径**: 
  - 糖：多糖→双糖→单糖（葡萄糖等）→小肠上皮吸收→门静脉→肝
  - 脂肪：脂肪→脂肪酸+甘油一酯→乳糜微粒→淋巴→血液循环
  - 蛋白质：蛋白质→多肽→氨基酸→小肠上皮吸收→门静脉→肝
  - 当需要：制定运动前中后营养补充策略时使用

- **肾的尿液生成三过程模型**: 肾小球滤过 → 肾小管和集合管重吸收 → 肾小管和集合管分泌
  - 当需要：理解运动对肾功能的影响时使用
  - 如何：运动时肾血流量减少→滤过率下降→尿量减少；同时ADH增加→水重吸收增加→尿浓缩

## Key Concepts
- **消化**: 食物在消化道内被分解为可吸收小分子物质的过程（机械消化+化学消化）
- **吸收**: 消化产物经消化道黏膜上皮细胞进入血液或淋巴的过程
- **胃排空**: 食物由胃排入十二指肠的过程，糖最快、蛋白质次之、脂肪最慢
- **运动性蛋白尿**: 运动后尿中出现蛋白质的现象，一般为良性暂时性改变
- **运动对消化功能的影响**: 适中运动促进消化，大强度运动抑制消化（因胃肠道血流量减少）
- **肾单位**: 肾的基本功能单位，由肾小体和肾小管组成

## Mental Models
- 将运动时消化系统血流减少理解为"非核心部门预算削减"——血液优先分配给运动肌，消化功能暂时降级
- 将肾的滤过-重吸收机制理解为"筛子+回收站"：先全部筛过去，再把有用的物质回收，废物随尿排出
- 将运动性蛋白尿理解为"滤网超负荷"现象——剧烈运动使肾小球滤过膜通透性暂时增加

## Anti-patterns
- **运动后立即大量进食**: 运动时胃肠道血流量减少，消化功能抑制，运动后立即大量进食可导致消化不良
- **长时间运动中不补充水分和电解质**: 持续出汗导致脱水和电解质紊乱，加重肾负担
- **忽视运动前排空时间**: 不同食物胃排空时间不同，运动前应根据运动强度合理安排进食时间

## Key Takeaways
1. 运动中消化系统血流量可减少至安静时的20%，合理安排运动与进食时间至关重要
2. 三大营养物质中糖排空最快（约2h），脂肪最慢（4-6h），运动前饮食应据此调整
3. 运动性蛋白尿一般为良性暂时性改变，但持续蛋白尿应警惕肾损伤
4. 适中运动促进消化吸收功能，大强度长时间运动抑制消化
5. 运动时ADH和醛固酮分泌增加，是机体保水保钠的重要适应机制
