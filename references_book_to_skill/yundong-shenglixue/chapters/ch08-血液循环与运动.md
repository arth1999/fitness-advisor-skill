# 第八章: 血液循环与运动

## Core Idea
心脏是血液循环的泵动力源，心输出量=搏出量×心率。运动时心输出量从安静5 L/min增至25-40 L/min，血液循环系统通过心率加快、搏出量增加、血流重新分配、血压调控等机制保证运动肌充足供氧。心泵功能储备是限制最大有氧运动能力的关键因素。

## Frameworks Introduced
- **心脏泵血功能的异长调节与等长调节**: 
  - 异长调节（Frank-Starling定律）：静脉回流量增加→心室舒张末期容积增大→心肌初长度增加→搏出量增加
  - 等长调节：交感兴奋→心肌收缩能力增强→搏出量增加（与初长度无关）
  - 当需要：分析运动时搏出量增加的机制时使用——低强度以异长调节为主，大强度以等长调节为主

- **血流重新分配模型**: 运动时血液从内脏器官向运动肌和心脏大幅度转移
  - 当需要：理解运动时各器官血供策略时使用
  - 如何：骨骼肌血管舒张（局部代谢产物作用）+ 内脏血管收缩（交感神经作用），总外周阻力变化不大，保证血压稳定

- **运动性心脏肥大分类**:
  - 离心性肥大：耐力运动员，心腔扩大为主+室壁轻度增厚（容量负荷刺激）
  - 向心性肥大：力量运动员，左室壁增厚为主+心腔扩大不明显（压力负荷刺激）
  - 当需要：根据运动项目预测心脏适应性改变或评估运动员心脏功能时使用

## Key Concepts
- **心动周期**: 一次收缩和舒张构成一个机械活动周期，心率75bpm时约0.8s
- **搏出量(SV)**: 一侧心室一次搏动射出的血量，安静约70mL，运动员可达160-200mL
- **心输出量(CO)**: 每分钟一侧心室输出血量 = SV×HR，安静约5 L/min
- **射血分数(EF)**: SV/心室舒张末期容积，55%-65%，比SV更准确反映泵血效率
- **心指数(CI)**: 每平方米体表面积的心输出量，安静约3.0-3.5 L/m²/min
- **心搏峰**: 搏出量达到峰值时的心率水平（110-130 bpm）
- **最佳心率范围**: 心输出量维持在较高水平的心率范围（120-180 bpm）
- **运动性心动徐缓**: 长期训练导致安静心率低于60 bpm，反映心泵效率提高
- **减压反射**: 动脉血压升高→压力感受器兴奋→心率减慢+血管舒张→血压回落（负反馈）
- **肌肉泵**: 骨骼肌节律性舒缩挤压静脉+静脉瓣导向，促进血液回心

## Mental Models
- 将心脏储备理解为发动机转速表和扭矩的乘积储备——心率储备类似转速余地，搏出量储备类似扭矩余地
- 将血流重新分配理解为战场上的资源调度——非必要部门（内脏）削减预算，核心战区（运动肌）增加投入
- 将运动性心脏肥大理解为专业工具适配——耐力运动员=大排量长跑车（大腔大流量），力量运动员=高压泵（厚壁强压）

## Anti-patterns
- **剧烈运动后骤然停止站立不动**: 肌肉泵突然失效，大量血液淤积下肢，回心血量骤减可导致重力性休克
- **误用心率绝对值比较训练水平**: 安静心率受个体差异影响大，心率储备（HRmax-HRrest）和运动后心率恢复速率更能反映训练水平
- **忽视舒张期缩短的影响**: 心率过快时舒张期缩短更显著，心室充盈不足是心输出量下降的主因

## Key Takeaways
1. Frank-Starling机制在低强度运动中主导搏出量增加，心肌收缩力增强在较大强度运动中更关键
2. 运动时血流重新分配的生理意义：保证运动肌获得充足供血同时维持必要血压
3. 耐力运动员心脏为离心性肥大（腔大），力量运动员为向心性肥大（壁厚），均属功能性代偿
4. 运动性心动徐缓增加了心率储备，延长了心脏舒张期休息时间
5. 肌肉泵和呼吸泵是促进静脉回流的两大辅助动力，运动后应逐步降低强度而非骤停
