# 第六章: 血液与运动

## Core Idea
血液是心血管内循环流动的液体组织（占体重7%-8%），由血浆和血细胞组成。血液承担物质运输、内环境稳态维持和防御保护三大功能，运动对血液各成分产生显著影响，急性运动可引起血液浓缩，长期训练可导致运动性贫血等适应性改变。

## Frameworks Introduced
- **血液组成与功能模型**: 血浆（55%）+ 血细胞（45%），以血细胞比容(HCT)衡量
  - 当需要：评估血液携氧能力和运动适应性时使用
  - 如何：正常男性HCT=40%-50%，女性37%-48%，运动时脱水可致HCT升高（血液浓缩）

- **血浆蛋白功能分类**: 白蛋白（维持胶体渗透压）、球蛋白（免疫防御）、纤维蛋白原（凝血）
  - 当需要：理解血浆功能时使用
  - 如何：白蛋白降低可导致组织水肿，球蛋白降低提示免疫功能下降

## Key Concepts
- **血细胞比容(HCT)**: 血细胞在血液中所占的容积百分比，反映血液浓缩或稀释状态
- **血浆**: 血液的液体成分，含93%水、电解质、血浆蛋白、血脂等
- **血红蛋白(Hb)**: 红细胞内运输O2和CO2的蛋白质，1g Hb可结合1.34-1.39mL O2
- **运动性贫血**: 长期大强度训练导致Hb和红细胞数量低于正常水平
- **运动对血浆的影响**: 急性运动导致血浆容量减少（出汗和液体转移），长期耐力训练可致血浆容量增加
- **运动对血细胞的影响**: 急性运动可引起白细胞增多（儿茶酚胺动员边缘池白细胞释放），血小板数量和活性增加

## Mental Models
- 将血液理解为"物流系统"：血浆是运输介质（公路），红细胞是运氧卡车，白细胞是安保巡逻，血小板是维修队
- 将血细胞比容变化理解为"溶液浓度"变化——运动出汗=水分减少=溶液变浓（HCT升高），补水=稀释（HCT降低）
- 将运动性贫血理解为"供不应求"状态：高强度训练加速红细胞破坏，而新生速度跟不上破坏速度

## Anti-patterns
- **仅凭单次HCT判断贫血**: 急性运动后血液浓缩可掩盖真实贫血状态，应在安静清晨空腹状态下评估
- **忽视血浆容量扩张的适应性**: 耐力训练后HCT降低不一定是贫血，可能是血浆容量增加导致稀释性降低，是适应性表现

## Key Takeaways
1. 血液的核心功能：物质运输、内环境稳态维持、防御保护
2. 运动时血浆容量变化是影响HCT和血液黏稠度的关键因素
3. 长期耐力训练导致血浆容量扩张是血液稀释性适应，有利于降低血流阻力
4. 运动性贫血需区分真性贫血（红细胞绝对减少）和假性贫血（血浆稀释）
5. 高原训练和低氧训练可刺激红细胞生成增加，提升血液携氧能力
