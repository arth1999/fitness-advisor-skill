# 运动营养核心推理模式 — ACSM运动营养学

## 模式1：能量可用性优先原则

### 核心公式
```
能量可用性(EA) = (能量摄入 - 运动能量消耗) / 去脂体重(FFM)
健康阈值：EA ≥ 30 kcal/kg FFM/d
风险阈值：EA < 30 kcal/kg FFM/d → RED-S
```

### 推理逻辑
1. 运动员无论从事何种运动，首要问题总是：**能量摄入是否动态匹配能量消耗？**
2. 不能等到24小时结束再评估——内分泌系统实时工作
3. 能量不足的后果按严重程度递进：表现下降 → 激素紊乱 → 骨密度丧失 → 多系统功能障碍
4. 解决方案不是简单"多吃"，而是优化能量摄入的**数量、类型和时机**

### 应用模式
```
问题：运动员出现不明原因疲劳、月经失调、反复损伤
→ 第一步：评估能量可用性（EA）
→ 第二步：检查24小时内能量缺口模式（不仅仅是总量）
→ 第三步：优化进食频率和时机（每3-4小时进食）
→ 第四步：确保碳水化合物、蛋白质和脂肪的平衡
→ 第五步：监测恢复正常
```

## 模式2：宏量营养素优先序位

### 需求层级
```
优先级1（始终优先）：碳水化合物 5-10 g/kg/d
    ↓ 基础能量底物、中枢神经系统燃料、限制性因素
优先级2（配合碳水）：蛋白质 1.2-2.0 g/kg/d
    ↓ 组织修复、但单餐>30-40g利用率不增
优先级3（长期稳定）：脂肪 占总能量20-35%
    ↓ 必需脂肪酸、脂溶性维生素、长时间低强度燃料
```

### 典型错误模式识别
| 错误模式 | 识别特征 | 纠正方向 |
|---|---|---|
| "高蛋白偏执" | 蛋白质>3g/kg/d，碳水摄入不足，尿液深黄 | 降低蛋白质至1.2-2.0g/kg，增加碳水化合物 |
| "碳水恐惧" | 低碳水饮食，训练疲劳、大脑模糊 | 增加碳水化合物至5+g/kg，尤其训练前后 |
| "脂肪恐惧" | 脂肪<15%总能量，皮肤干燥、激素问题 | 增加到20-35%，确保必需脂肪酸 |
| "液体忽略" | 训练不补水、不喝运动饮料、体重快速下降 | 制定个性化补水方案 |

## 模式3：血糖稳定性三角

### 三要素
```
1. 进食频率：每3-4小时（匹配血糖波动周期）
2. 碳水化合物质量：低GI/GL为主 + 运动前后中高GI
3. 与蛋白质/脂肪配合：减缓葡萄糖吸收，稳定胰岛素反应
```

### 血糖紊乱的后果链
```
不规律进食/跳餐
    → 血糖下降(<70 mg/dL)
    → 皮质醇分泌增加
    → 糖异生激活（分解肌肉和骨组织获取氨基酸）
    → 瘦体重和骨量减少
    → 体脂率相对上升 + 运动表现下降
    → 长期：RED-S/三联征/应力性骨折
```

### 运动员饮食频率建议
- **训练日**：5-6餐（3主餐+2-3加餐/零食）
- **休息日**：3-4餐（根据总能耗调整）
- **绝对避免**：跳早餐或长时间空腹训练
- **关键窗口**：运动后30分钟内开始恢复补充

## 模式4：铁状态分阶段评估

### 评估框架
```
第1层：筛查层
  → 血红蛋白(Hb) + 血细胞比容(Hct)
  → 目的：识别临床贫血
  → 局限：正常不等于铁状态良好

第2层：确认层
  → 血清铁蛋白
  → 目的：评估铁储备
  → 关键：最低正常阈值不统一(10-35 ng/mL)
  → 运动员应保持在正常中高水平

第3层：机制层
  → 转铁蛋白 + 血清铁 + TIBC
  → 目的：理解铁缺乏的原因
  → 低转铁蛋白：蛋白质/能量营养不良
  → 高TIBC：铁储存不足

第4层：综合判断层
  → 结合训练负荷、饮食评估、月经史（女性）
  → 确定是多因素重叠还是单一因素
```

### 干预决策树
```
血清铁蛋白低于阈值
    ├─ Hb正常、Hct正常 → 铁储备不足（无贫血）
    │   ├─ 饮食调整：增加血红素铁来源 + 维生素C
    │   └─ 考虑低剂量铁补充剂（需医学监督）
    │
    ├─ Hb低、Hct低 → 缺铁性贫血
    │   ├─ 医学评估：排除消化道出血、月经过多等
    │   ├─ 铁补充剂治疗（医学监督）
    │   └─ 饮食策略配合
    │
    └─ Hb低、Hct正常/高 → 可能的其他原因贫血
        └─ 检查：B12、叶酸、溶血标志物、训练阶段
```

## 模式5：水合状态评估与干预

### 水合状态判断矩阵
```
运动前后体重变化
    ↓
体重下降<1%：水合维持良好
体重下降1-2%：轻度脱水——需加强赛中补水
体重下降>2%：显著脱水——表现受影响，需立即纠正补水策略
体重下降>3%：严重脱水——生命安全风险

尿液颜色辅助判断：
  浅黄/透明：水合良好
  深黄：需要补水
  棕色/深褐：严重脱水
```

### 补水策略制定步骤
```
第1步：计算个人出汗率
  (运动前体重 - 运动后体重 + 运动中摄入液体) / 运动分钟数

第2步：确定赛中补水目标
  目标 = 出汗率 - 可耐受胃排空量(通常300-800 mL/h)
  最低目标：体重下降<2%

第3步：选择饮料成分
  运动<1h且强度不高：水可能足够
  运动>1h或高强度：碳水(30-60g/h) + 钠(0.5-0.7g/L) + 水

第4步：运动后补液
  每失重0.5kg → 补充16-24oz(473-709 mL)液体
  包含钠以促进水分保留
```

## 模式6：运动营养周期化

### 与训练周期对应的营养调整

| 训练阶段 | 训练特征 | 营养重点 |
|---|---|---|
| 准备期（基础） | 大运动量、中低强度 | 碳水4-6g/kg、蛋白质1.2-1.6g/kg、能量平衡 |
| 专项期（强化） | 高强度+专项耐力 | 碳水6-10g/kg、蛋白质1.6-2.0g/kg、重点赛中营养 |
| 赛前减量 | 训练量减少、强度维持 | 碳水加载（如需）、电解质平衡、抗氧化食物 |
| 比赛期 | 比赛为主、恢复训练 | 赛前/中/后精确策略、熟悉食物、避免新尝试 |
| 过渡期（休赛） | 主动恢复、低负荷 | 能量适度下调、维持营养质量、体重管理窗口 |

### 营养周期化的三个层次
1. **年度周期化**：对应赛季和非赛季的能量需求波动
2. **月度周期化**：对应不同训练阶段的宏量营养素比例调整
3. **日周期化**：一天的训练前中后营养策略

## 模式7：抗氧化平衡原则

### 核心悖论
```
运动 → ↑ 活性氧(ROS)产生 → ↑ 抗氧化需求
                          BUT
高剂量抗氧化剂补充 → 削弱训练适应 → ↓ 运动表现
```

### 正确策略
```
优选层级：
1. 多样化食物中的抗氧化剂 → 协同保护、不过量
2. 充足基础营养素（硒、锌、VC、VE来自食物） → 支持内源性抗氧化酶
3. 避免单一高剂量补充 → 防止破坏抗氧化剂间平衡
4. 只在确诊缺乏时补充 → 医学监督下进行
```

### 抗氧化剂协同作用矩阵
```
维生素C → 循环再生维生素E和谷胱甘肽
维生素E → 保护脂质膜
谷胱甘肽 → 中和多种ROS（水基）
硒 → GPx酶活性所需
β-胡萝卜素 → 脂基抗氧化 + 维生素A前体
类黄酮 → 浓度极高(100-1000× VC)
```

## 模式8：补充剂循证评估框架

### 四阶梯评估
```
阶梯1：安全性
  ├─ 成分是否已知且安全？
  ├─ 有无第三方认证（NSF/Informed-Sport）？
  └─ 有无WADA禁用物质风险？

阶梯2：有效性
  ├─ 有无高质量随机对照试验支持？
  ├─ 效应量是否临床/运动有意义？
  └─ 是否优于食物方案？

阶梯3：效力
  ├─ 产品剂量是否达到有效水平？
  └─ 生物利用率如何？

阶梯4：必要性
  ├─ 饮食能否满足需求？
  ├─ 是否有明确缺乏证据？
  └─ 风险收益比是否有利？
```

### 补充剂决策流程
```
运动员提出补充剂需求
    │
    ├─ 是否有确诊缺乏（血液检测证实）？
    │   └─ 是 → 针对性补充+医学监督
    │
    ├─ 是否有特定运动营养需求？
    │   ├─ 运动>1h → 运动饮料（碳水+电解质）
    │   ├─ 高强度重复运动 → 考虑一水肌酸（有循证支持）
    │   └─ 其他 → 评估个体情况
    │
    └─ 是否仅为"保险"或"提升表现"？
        └─ 否 → 优化食物营养 → 监测效果 → 如无改善再评估
```
