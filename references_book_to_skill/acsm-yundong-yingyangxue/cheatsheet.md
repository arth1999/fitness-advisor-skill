# 运动营养速查手册 — ACSM运动营养学

## 关键数字速查

### 宏量营养素推荐

| 营养素 | 能量密度 | AMDR | 运动员推荐 | 一般成人RDA |
|---|---|---|---|---|
| 碳水化合物 | 4 kcal/g | 45-65% | 5-10 g/kg/d | 130 g/d（脑最低需求） |
| 蛋白质 | 4 kcal/g | 10-35% | 1.2-2.0 g/kg/d | 0.8 g/kg/d |
| 脂肪 | 9 kcal/g | 20-35% | 20-35%（不推荐高脂） | — |
| 膳食纤维 | — | — | 男38g/d 女25g/d | 14g/1000kcal |

### 运动中补充

| 参数 | 推荐值 |
|---|---|
| 碳水化合物（赛中） | 30-60 g/h（>3h可至90g/h多源） |
| 液体 | 按出汗率，不超过出汗率 |
| 运动饮料碳水浓度 | 等渗或低渗（通常4-8%碳水溶液） |
| 钠（赛中高温>2h） | 0.5-0.7 g/L |

### 运动后恢复

| 参数 | 推荐值 | 时机 |
|---|---|---|
| 碳水化合物 | 1.0-1.2 g/kg/h | 前4小时 |
| 蛋白质 | 0.25-0.3 g/kg 或 10g EAA | 运动后0-2小时 |
| 液体 | 每失重0.5kg → 16-24 oz (473-709 mL) | 尽快开始 |

### 脱水与表现

| 脱水程度（%体重） | 影响 |
|---|---|
| <1% | 基本无影响 |
| 1-2% | 口渴感增强 |
| >2% | 认知功能损害、有氧表现下降（炎热天气更明显） |
| 3-5% | 无氧能力下降 |
| >6% | 严重水合不足——心肺功能下降、排汗减少 |

## 水合状态评估速查

| 评估方法 | 水合良好 | 需补水 | 严重脱水 |
|---|---|---|---|
| 尿颜色 | 浅黄/透明 | 深黄 | 棕色/深褐 |
| 尿渗透压 | <700 mOsm/kg | 700-900 | >900 mOsm/kg |
| 体重变化（运动中） | <1% | 1-2% | >2% |

## 运动饮料设计速查

| 成分 | 推荐范围 | 注意事项 |
|---|---|---|
| 碳水化合物 | 4-8% (w/v) | 多糖渗透压低，优于等量单糖 |
| 钠 | 0.5-0.7 g/L (20-30 mmol/L) | 促进水分吸收，维持血容量 |
| 钾 | 2-4 mmol/L | 补充汗液丢失 |
| 温度 | 10-15°C | 较低温度促进自愿饮水和胃排空 |
| 渗透压 | 等渗或低渗 | 高渗延缓胃排空→胃肠不适 |

## 关键激素对能量代谢的调控

| 激素 | 来源 | 触发 | 作用 |
|---|---|---|---|
| 胰岛素 | 胰腺β细胞 | 血糖>120 mg/dL | ↓ 血糖；促进细胞摄取葡萄糖 |
| 胰高血糖素 | 胰腺α细胞 | 血糖<80 mg/dL | ↑ 血糖；分解肝糖原 |
| 肾上腺素 | 肾上腺 | 应激 | ↑↑ 血糖（快速）；肝糖原分解 |
| 皮质醇 | 肾上腺 | 低血糖/慢性应激 | 糖异生；分解肌肉/骨组织 |
| 加压素(ADH) | 下丘脑/垂体 | 低血容量/高渗透压 | 保水；血管收缩 |
| 醛固酮 | 肾上腺皮质 | 低血压 | 保钠保水；↑血容量 |

## 训练前后餐设计原则

| 参数 | 训练前（2-4h前） | 训练前（30-60min） | 训练后（0-2h） |
|---|---|---|---|
| 碳水化合物 | 高（维持血糖+填充糖原） | 中高（易消化） | 高（1.0-1.2 g/kg/h×4h） |
| 蛋白质 | 中 | 低 | 中（0.25-0.3 g/kg） |
| 脂肪 | 低 | 低 | 中 |
| 纤维 | 低 | 低 | 中 |
| 液体 | 充足 | 少量 | 补足缺失 |

## 蛋白质摄入模式速查

| 参数 | 推荐 |
|---|---|
| 单餐最佳摄入量 | 约30g（约30-40g达到利用上限） |
| 频率 | 每3-4小时 |
| 每日总量 | 1.2-2.0 g/kg/d |
| 运动后补充 | 10g必需氨基酸 或 0.25-0.3 g/kg |
| 超过40g/餐的后果 | 剩余部分→能量来源或脂肪储存 |
| 高蛋白饮食风险 | BUN升高→脱水倾向；骨密度可能降低 |

## 常见运动补液情景决策树

```
运动时长 < 30分钟
  └─ 通常不需额外补充（常温常湿）

运动时长 30-60分钟 + 中等强度
  ├─ 常温常湿：水可能足够
  └─ 高温高湿或高强度间歇：运动饮料

运动时长 1-3小时
  └─ 运动饮料：碳水30-60g/h + 电解质

运动时长 > 3小时
  └─ 运动饮料 + 可能补充食物：碳水60-90g/h多源 + 钠

运动后
  └─ 液体：每失重0.5kg→16-24oz + 碳水 + 蛋白质
```

## 环境适应时间参考

| 适应类型 | 大致时间 | 关键策略 |
|---|---|---|
| 热适应 | 7-14天 | 逐步增加高温训练时间 |
| 高海拔适应（短期） | 3-7天 | 降低训练强度；保证铁储备充足 |
| 高海拔适应（红细胞生成） | 3-6周 | 进入前确保铁蛋白正常 |
| 时差适应 | 每1h时差≈1天 | 提前调整作息；抵达后尽快适应当地时间 |
| 新食物适应 | 至少3-5天 | 赛前不要尝试新食物 |

## RED-S警告信号清单

- [ ] 不明原因的疲劳和训练恢复不良
- [ ] 反复发生的损伤（尤其是应力性骨折）
- [ ] 频繁生病（免疫功能下降）
- [ ] 月经功能紊乱或闭经（女性）
- [ ] 体重过度波动或持续下降
- [ ] 训练反应减弱（同样训练刺激不再产生适应）
- [ ] 情绪变化：易怒、抑郁、对训练的强迫性态度
- [ ] 关注食物/体重/身体形象的强迫行为
- [ ] 运动表现不升反降
- [ ] 骨密度检测下降

出现3个或以上信号 → 需要进行运动员专项医学和营养评估
