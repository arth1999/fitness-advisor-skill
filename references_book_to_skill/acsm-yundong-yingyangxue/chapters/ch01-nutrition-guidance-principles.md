# 第1章 运动员营养指导原则

## 章节概述

本章构建运动营养学的认知框架，阐明营养与体力活动的双向互动关系，并破除12项常见营养谬见，为全书奠定科学方法论基础。

## 核心理论框架

### 营养学的三条基本规则

1. **过量不如适量好**（剂量决定毒性）：任何营养素超出生理需求量不仅无益，反而可能造成问题。例如蛋白质摄入超过1.7g/kg/d不会增强合成代谢，反而减少碳水化合物摄入空间。
2. **广泛饮食确保营养全面**：不存在"完美食物"。单一食物重复摄入（如每日相同早餐）会导致某些营养素过剩而其他匮乏。食用多种食物是维持营养良好的核心策略。
3. **实时动态匹配能量消耗**：能量摄入应与消耗实时匹配。不应等到一天结束才评估能量平衡——内分泌系统（如胰腺）持续监测血糖并实时调节胰岛素分泌。

### 体力活动引发的连锁营养需求变化

| 体力活动效应 | 营养后果 |
|---|---|
| 能量消耗率增加 | 碳水化合物需求增加（强度越高依赖越大） |
| 产热增加（运动效率仅20-40%） | 出汗增加，需补充液体和电解质 |
| 能量底物类型变化 | 脂肪只能有氧代谢，碳水化合物可无氧+有氧 |
| 代谢维生素需求增加 | 尤其是B族维生素，帮助能量底物代谢 |

## 关键概念

### 能量可用性（Energy Availability）

传统"热量输入-热量输出"模型已被修正。即使24小时内满足总能量需求，如果能量消耗时机导致严重的能量平衡缺口，运动员仍可能面临激素和身体成分失调风险。国际奥委会共识声明将运动中相对能量不足（RED-S）定义为影响全身多系统的综合征。

### 必需营养素 vs. 非必需营养素

不可将"非必需"误解为"不重要"。必需营养素（如必需氨基酸、必需脂肪酸）是指人体无法自行合成、必须从食物获取的营养素。非必需营养素虽可由人体合成，同样不可或缺。

### 营养不良的多重成因

1. **组织机构因素**：赛事赞助商提供的产品可能不适合所有运动员；国家管理机构监督有限，导致不良营养行为长期存在
2. **知识缺乏**：将营养学误认为"理念系统"而非科学；错误归因（如将肌肉增长归因于蛋白质而非综合训练和能量供给）
3. **运动传统**：教练沿用其运动员时期的营养策略；过度关注体重而非身体成分
4. **食物限制**：过敏（IgE介导免疫反应）、不耐受（消化酶不足）、敏感（非IgE介导的炎症反应）

## 营养指南核心参数

| 参数 | 推荐值 |
|---|---|
| 碳水化合物 | 5-10g/kg/d（运动强度决定） |
| 蛋白质 | 1.2-2.0g/kg/d |
| 脂肪 | 总能量的20-35% |
| 赛中碳水化合物补充 | 30-60g/h |
| 运动后糖原补充 | 1.0-1.2g/kg/h（前4小时） |
| 运动后蛋白质补充 | 10g必需氨基酸或0.25-0.3g/kg（运动后0-2h） |
| 严重脱水阈值 | 体重下降>2-3% |

## DRI体系说明

- **EAR（平均需要量）**：满足50%健康人群的需求——不适合作为个人目标
- **RDA（推荐供给量）**：EAR+2SD，满足约98%人群——适用于个人
- **AI（适宜摄入量）**：无法确定RDA时的估计值——适用于个人和团体
- **UL（可耐受最高摄入量）**：超过此量不良反应风险增加——超过UL比低于RDA风险更大

## 执业范围

美国50州中47州制定了营养学实践法律。未持证人员可讨论《饮食指南》中的一般营养原则，但不可针对已确诊疾病提供临床营养建议。注册营养师（RD/RDN）拥有最大执业范围。

## 12项常见营养谬见

1. 严格限制热量可减肥 → 实际：瘦体重可能比脂肪减少更多
2. 3500kcal=1磅体重 → 人类不适用此线性关系
3. 深夜进食会发胖 → 避免低血糖更重要
4. 多吃蛋白质增强肌肉 → 需要综合能量+营养+运动刺激
5. 不含胆固醇的食物有益心脏 → 心脏病由多重因素引起
6. 运动员不会骨密度低 → 能量不足可导致骨质流失
7. 食物渴望反映营养需求 → 多为环境刺激和饮食限制导致
8. 草药产品天然即安全 → 需独立实验室验证成分
9. 水是完美补液饮料 → 运动1小时以上需含电解质和碳水化合物的饮料
10. 所有脂肪都有害 → 必需脂肪酸是维持健康所必需
11. 碳水化合物使人发胖 → 全谷物和蔬菜中的碳水化合物降低肥胖风险
12. 定期禁食清除毒素 → 健康人体系统可持续有效排毒
