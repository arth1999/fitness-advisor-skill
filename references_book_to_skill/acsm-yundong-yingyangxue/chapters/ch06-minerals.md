# 第6章 矿物质——对身体健康和运动表现的重要性

## 章节概述

本章阐述常量矿物质和微量矿物质的功能、运动员需求特点、缺乏风险评估，以及矿物质间相互作用对吸收的影响。

## 矿物质分类

### 常量矿物质（每日需求>100mg）

| 矿物质 | 核心功能 | 缺乏风险人群 | UL | 食物来源 |
|---|---|---|---|---|
| 钙 | 骨骼健康、神经传导、肌肉收缩、凝血 | 女性运动员、闭经者、青少年 | 2500 mg/d | 乳制品、豆制品、绿叶蔬菜 |
| 磷 | 骨骼矿化、ATP组成、酸碱缓冲 | 罕见（摄入普遍充足） | 4000 mg/d | 肉类、奶制品、全谷物 |
| 镁 | 300+酶反应辅因子、神经肌肉功能、蛋白质合成 | 大量出汗运动员 | 350 mg/d(补充剂) | 坚果、种子、绿叶蔬菜、全谷物 |
| 钠 | 细胞外液渗透压调控、血容量维持 | 长时间高温运动者 | 2300 mg/d | 食盐、加工食品 |
| 钾 | 细胞内液渗透压、神经肌肉功能、降血压 | 利尿剂使用者 | 未设定 | 水果、蔬菜（香蕉、土豆） |
| 氯 | 细胞外液电解质、胃酸成分 | 罕见 | 未设定 | 食盐 |

### 微量矿物质

| 矿物质 | 核心功能 | 运动员特殊关注 | 缺乏症状 | 食物来源 |
|---|---|---|---|---|
| 铁 | 血红蛋白&肌红蛋白组分、氧运输、电子传递 | 最关键微量矿物质，尤其女性/耐力运动员 | 缺铁性贫血：小细胞低色素性贫血 | 红肉、肝脏、豆类、强化谷物 |
| 锌 | 300+酶辅因子、免疫功能、蛋白质合成 | 素食运动员、大量出汗者 | 生长迟缓、免疫功能下降 | 肉类、贝类、全谷物 |
| 硒 | 谷胱甘肽过氧化物酶组分、抗氧化 | 地区土壤含量差异大 | 克山病（心肌病） | 巴西坚果、鱼类、全谷物 |
| 铜 | 铜蓝蛋白组分、铁转运至血红蛋白 | 锌过量摄入可能干扰吸收 | 贫血、神经系统疾病 | 贝类、肝脏、坚果 |
| 碘 | 甲状腺激素组分、代谢率调控 | 对代谢率有间接影响 | 甲状腺肿、甲减 | 碘盐、海产品 |

## 运动员铁状态：最重要的微量矿物质

### 为何运动员缺铁风险更高

| 风险因素 | 机制 |
|---|---|
| 运动性贫血 | 血容量增加快于红细胞增加→稀释效应（有益适应） |
| 足行溶血 | 跑步时红细胞在足底毛细血管中被压碎 |
| 血管内溶血 | 肌肉挤压、循环加速导致红细胞分解加快 |
| 尿/汗/粪便失铁 | 剧烈运动后尿中可能出现红细胞（血尿）；汗液失铁 |
| 胃肠道出血 | 耐力运动可诱发消化道微量出血 |
| 月经失血 | 女性运动员额外风险 |
| 饮食摄入不足 | 限制能量摄入、素食者尤为常见 |

### 铁状态评估指标

| 指标 | 正常值（男/女） | 意义 |
|---|---|---|
| 血红蛋白 | 13.8-17.2 / 12.1-15.1 g/dL | 携氧能力——受多种因素影响，不敏感 |
| 血细胞比容 | 42-52% / 36-48% | 红细胞体积比——同样不敏感 |
| 血清铁蛋白 | >10-35 ng/mL | **储存铁**——最敏感的早期缺铁指标 |
| 转铁蛋白 | 200-400 mg/dL | 铁转运蛋白——低值与蛋白质营养不良相关 |
| 血清铁 | 75-175 / 65-165 μg/dL | 循环铁含量 |

### 关键警示
血红蛋白正常≠铁状态良好。身体优先维持血红蛋白——如果缺铁，会先消耗肌红蛋白和含铁酶中的铁来维持血红蛋白。因此即使血常规正常，运动员仍可能因酶铁不足导致运动表现下降。

## 矿物质间相互作用

- **钙过量→抑制铁、镁、锌吸收**（服用钙补充剂者需关注）
- **锌过量→干扰铜吸收**（可能导致缺铜性贫血）
- **草酸盐（蔬菜）和植酸盐（谷物麸皮）→与铁、锌、钙、镁结合，降低吸收**
- **维生素C→促进非血红素铁吸收**
- **钠和钾的平衡**→共同调控细胞内/外液分布

## 运动员实践策略

1. 铁状态每年定期检查（血红蛋白+血细胞比容+铁蛋白）
2. 重视食物来源铁（血红素铁吸收率高于非血红素铁）
3. 女性运动员和青少年生长期需额外关注钙和铁
4. 素食运动员需密切关注铁、锌、钙、B12状态
5. 避免钙补充剂与含铁餐同时摄入（间隔2-3小时）
6. 多汗环境训练时注意电解质（尤其是钠）补充
7. 不要随意服用高剂量矿物质补充剂（如铁过量导致血色素沉着症/含铁血黄素沉着症）
