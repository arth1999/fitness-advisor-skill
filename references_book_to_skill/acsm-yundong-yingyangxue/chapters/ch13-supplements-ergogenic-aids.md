# 第13章 膳食补充剂、食物及增补剂提高运动表现的预想与现实

## 章节概述

本章在维生素和矿物质章节基础上，进一步回顾旨在提高运动表现的膳食补充剂和增补剂的循证评估，揭露谬见，阐明潜在风险和实际效果。

## 膳食补充剂的定义与法规

### 美国《膳食补充剂健康与教育法》(DSHEA)定义
> "维生素、矿物质、草药、植物性药材、氨基酸、代谢物、构成要素、提取物或任何这些成分的组合"

### 关键法规特点
- **与药品不同**：补充剂上市前无需证明安全性或有效性
- **标签监管宽松**：标签上列出的成分可能与实际内容不符
- **污染风险**：研究发现部分补充剂含有国际奥委会和世界反兴奋剂机构(WADA)禁用物质，但这些物质未列在标签上
- **运动员风险**：误食含有禁用物质的补充剂可能面临兴奋剂处罚

### 运动员选择补充剂的考量因素
1. **安全性**：成分是否已知且安全？
2. **有效性**：是否有可靠科学证据支持其所声称的功效？
3. **效力**：实际剂量是否达到有效水平？
4. **合法性**：是否含有WADA禁用物质？是否通过第三方认证（如NSF Certified for Sport）？

## 常见补充剂的循证评估

### 有较强证据支持的

| 补充剂 | 证据等级 | 适用情境 | 注意事项 |
|---|---|---|---|
| 一水肌酸 | 高 | 高强度重复性爆发力运动 | 需良好营养状态为前提；营养不良者效果有限 |
| 咖啡因 | 高 | 耐力运动、警觉性维持 | 剂量依赖；个体反应差异大；高剂量可能增加焦虑和脱水 |
| β-丙氨酸 | 中高 | 1-4分钟高强度运动 | 可能引起皮肤感觉异常（无害但不适） |
| 硝酸盐（甜菜根汁） | 中 | 耐力运动（降低氧耗） | 剂量和时机需精确控制 |
| 碳酸氢钠 | 中 | 高强度短时运动 | 胃肠道不适副作用常见 |
| 运动饮料（碳水+电解质） | 高 | 运动>1h或高强度间歇 | 基础营养补充，非典型"补充剂" |
| 蛋白粉 | 适用 | 食物摄入不足时的便捷补充 | 食物优先；注意第三方认证 |

### 证据不足或无效的

| 补充剂 | 评估 |
|---|---|
| 支链氨基酸(BCAA) | 单独补充对肌肉合成的增益有限（完整蛋白质更有效） |
| 谷氨酰胺 | 健康运动员中缺乏效能证据 |
| 维生素E高剂量 | 研究显示可能增加氧化应激和炎症（Kona铁三研究） |
| 多种抗氧化剂组合 | 可能削弱训练适应 |
| 多数"燃脂"产品 | 缺乏可靠科学证据 |

### 有潜在风险的

| 风险类型 | 说明 |
|---|---|
| WADA禁用物质污染 | 无第三方认证产品可能含未标明的禁用物质 |
| 药物相互作用 | 草药与处方药可能相互作用 |
| 毒性累积 | 脂溶性维生素和某些矿物质过量风险 |
| 标签虚假 | 实际成分与标签不符 |
| 未知长期效应 | 许多产品缺乏长期安全性数据 |

## 运动员使用补充剂的常见误区

1. **"天然=安全"**：很多有毒物质是天然的——天然不代表安全
2. **"如果有用，多吃更好"**：违背剂量-毒性原则
3. **"冠军都在用"**：非因果关联——冠军的成功更可能来自训练、基因和全面的营养策略
4. **"补充剂可弥补不良饮食"**：补充剂不能替代食物的全面营养素和植物化学物质
5. **"标签上有的就是瓶子里有的"**：标签和实际内容可能不符

## 补充剂安全实践建议

1. **食物优先原则**：首先从多样化食物中获取营养
2. **仅在明确需要时使用**：确诊缺乏、医学需要或特定运动营养需求
3. **选择第三方认证产品**：NSF Certified for Sport、Informed-Sport等
4. **记录所有服用的补充剂**：品牌、批次号——以备兴奋剂检测需要
5. **了解WADA禁用清单**：定期检查最新禁用物质列表
6. **咨询专业人士**：使用前咨询注册营养师或有资质的运动医学医生
7. **报告所有补充剂**：在兴奋剂检测表格上如实报告

## 关键立场文件结论

- **ACSM/AND/DC联合立场（2016）**：如果运动员从各种食物中摄入足够能量来保持体重，就不需要补充维生素和矿物质
- **国际奥委会共识（2018）**：膳食补充剂对高性能运动员存在风险，应基于循证评估后谨慎使用
- 一般性建议：在考虑补充剂之前，优化训练、营养和恢复
