# 第7章 运动中的脱水问题

## 章节概述

本章阐述水分作为最重要营养素的地位，包括体液分布、电解质调控、脱水机制与后果，以及运动前中后的补水策略。

## 体液分布基础

### 体液分区

| 分区 | 占体重% | 主要电解质 | 容量（70kg男性） |
|---|---|---|---|
| 细胞内液（ICF） | ~40% | 钾（K+） | ~28L |
| 细胞外液（ECF） | ~20% | 钠（Na+） | ~14L |
| - 血管内液（IVF/血浆） | ~5% | 钠 | ~3.5L |
| - 组织液（ISF） | ~15% | 钠 | ~10.5L |

### 水分含量与身体成分

| 人群 | 体水分% |
|---|---|
| 男性（正常体脂） | ~60% |
| 女性（正常体脂） | ~50% |
| 运动员（高瘦体重） | ~70% |
| 肥胖人群 | ~40% |

- 肌肉含水量约75%
- 骨骼含水量约22%
- 脂肪组织含水量仅10-20%（脂肪本身不含水）
- 血液含水量约83%

## 运动与体液流失

### 运动产热与散热
- 运动效率仅20-40% → 每消耗100kcal约60-80kcal产热
- 热量通过汗液蒸发散发
- 活动强度越大→产热越多→出汗率越高
- **每天液体流失**：静息时约2.5L，高温工作时可达12L，超长距离自行车可达12.7L

### 影响出汗率的因素
1. 环境温度（越高出汗越多）
2. 环境湿度（越高汗液蒸发越难→出汗率代偿增加）
3. 服装类型（吸汗材料降低蒸发效率→应选吸湿排汗面料）
4. 身体条件（训练良好者出汗率更高、钠丢失更少）
5. 年龄（老年人和青春期前儿童体温调节能力可能低于年轻成人）

### 汗液电解质浓度

| 电解质 | 汗液(mmol/L) | 血液(mmol/L) | 细胞内液(mmol/L) |
|---|---|---|---|
| 钠 | 20-80 | 130-155 | 10 |
| 钾 | 4-8 | 3.2-5.5 | 150 |
| 氯 | 20-60 | 96-110 | 8 |
| 钙 | 0-1 | 2.1-2.9 | 0 |
| 镁 | <0.2 | 0.7-1.5 | 15 |

## 水合状态定义

| 状态 | 定义 | 尿渗透压 |
|---|---|---|
| 水合正常 | 正常排汗能力、良好体温控制 | <700 mOsm/kg |
| 脱水 | 正在丢失水分的过程（液体流失>摄入） | — |
| 水合不足 | 机体缺水约6-10%体重（严重） | >900 mOsm/kg |
| 低钠血症 | 血钠<135 mmol/L | — |

## 脱水对运动表现的影响

- **液体赤字>2%体重**：认知功能损害、有氧运动表现降低（尤其炎热环境）
- **液体赤字3-5%体重**：无氧能力下降
- **液体赤字>6%体重**：严重水合不足，心肺功能下降、汗液分泌减少、肌肉组织血流降低
- **比赛前刻意脱水减重**：对运动表现产生负面影响
- **无法适应脱水**：不存在"脱水适应"——这是必须纠正的教练误区

## 加重脱水的因素

1. **高蛋白饮食**：尿素氮（BUN）升高→增加尿液生成
2. **盐摄入不足**：低钠→血容量下降→体液流失增加
3. **腹泻**：胃肠道体液流失可达正常的100倍
4. **利尿物质**：某些药物（利尿剂）和食物（高钾食物）可能增加排尿
5. **咖啡因/酒精**：乙醇干扰正常能量代谢过程，增加脱水可能性

## 低钠血症

### 定义
血钠<135 mmol/L，导致血容量减少、肌肉血流不畅、汗液减少、头痛、恶心、平衡能力丧失。

### 运动员主要原因
- 摄入液体超过排汗和排尿量（过度补水）
- 补充无钠或低渗液体（通常是纯水）
- 运动中钠因排汗流失却未有效补充
- 女性因体型较小，风险可能更大

### 严重后果
脑水肿→意识混乱→昏迷（如果水肿发生在脑部）

## 体液平衡调控激素

| 激素 | 触发条件 | 作用 |
|---|---|---|
| 加压素（ADH） | 低血容量、低血压、高渗透压 | 减少尿液生成、血管收缩 |
| 醛固酮 | 低血压（肾素→血管紧张素系统） | 减少肾脏钠流失→升高血压和血容量 |

## 运动补水策略

### 出汗率计算公式

1. 运动前体重(A) - 运动后即刻体重(B) = 汗液丢失重量(C)
2. C × 16oz/lb = 汗液丢失体积(D)
3. D + 运动中补充液体(E) = 总出汗量
4. 总出汗量 ÷ 运动分钟数 = 每分出汗率

### 补水建议

| 时机 | 建议 |
|---|---|
| 运动前 | 确保水合正常状态开始运动 |
| 运动中 | 目标：防止脱水>2%体重；补水不应超过出汗率 |
| 运动后 | 体重每下降0.5kg→补充16-24oz（473-709mL）液体 |
| 饮料成分 | 运动>1h：含碳水化合物(30-60g/h)+电解质（尤其是钠）的运动饮料 |
| 饮料温度 | 较低温度有助于促进自愿饮水和胃排空 |

### 运动饮料设计原则
- 渗透压适中（等渗或低渗）
- 含钠（促进水分吸收和维持血容量）
- 碳水化合物浓度适合（避免过高浓度导致胃排空延迟）
- 多种糖源（葡萄糖+果糖可增加总吸收率）
