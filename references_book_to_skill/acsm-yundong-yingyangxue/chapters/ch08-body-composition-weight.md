# 第8章 运动员的体重管理与身体成分

## 章节概述

本章剖析体重与身体成分的区别，阐述能量平衡的动态本质，以及如何通过科学的身体成分评估指导运动员的营养与训练干预。

## 核心概念辨析

### 体重(Body Weight) ≠ 健康或表现指标

体重由四种成分构成：

| 成分 | 理想方向 | 说明 |
|---|---|---|
| 瘦体重（LBM） | 越多越好 | 肌肉、器官、骨骼（非脂肪部分） |
| 骨量 | 越多越好 | 骨密度和骨骼强度 |
| 身体水分 | 越多越好 | 水合良好是运动表现的前提 |
| 脂肪量 | 越少越好（但非无限） | 男性<25%、女性<32%为肥胖阈值 |

### BMI的局限性

- BMI = 体重(kg) / 身高²(m²)
- 原始设计：评估人口的肥胖患病率
- 不适用于个体运动员：肌肉发达的运动员可能BMI>30被"误诊"为肥胖
- 可能漏诊"瘦胖子"：体重正常但体脂高、肌肉低者
- **运动员评价应基于身体成分而非BMI**

### 两个关键案例

1. **足球前锋增重**：250lb→275lb，但全部增重为脂肪→用原来的肌肉移动更重的身体→更快疲劳→表现下降
2. **形体运动减肥**：限制热量导致肌肉减少幅度大于脂肪→更虚弱、脂肪恢复更快→骨密度降低和饮食失调风险

## 能量平衡的现代理解

### 传统模式（已过时）
> 热量输入 - 热量输出 = 体重变化

### 现代模式
> 实时能量可用性(Energy Availability) 动态匹配能量消耗

关键要点：
- 24小时能量平衡≠健康——一天内严重的能量缺口仍可导致激素紊乱
- 内分泌系统实时工作——胰腺不会"等到一天结束"才评估需要分泌多少胰岛素
- 长时间能量负平衡→适应性产热（代谢率过度降低）→更难减脂

### 能量可用性(EA)公式

$$
EA = \frac{\text{能量摄入} - \text{运动能量消耗}}{\text{去脂体重(FFM)}}
$$

国际奥委会共识：EA<30 kcal/kg FFM/d 为低能量可用性，增加RED-S风险。

## 相对能量不足(RED-S)的后果

### 表现影响
- 肌肉力量下降
- 肌糖原储备减少
- 有氧耐力下降
- 恢复能力降低
- 训练反应减弱
- 协调能力受损
- 伤病风险增加

### 健康影响
- 月经功能紊乱（女性）
- 骨密度降低→应力性骨折风险增加
- 免疫功能下降→反复感染
- 心血管风险增加
- 胃肠功能紊乱
- 代谢率下降
- 心理问题（抑郁、饮食紊乱）

## 身体成分评估方法

| 方法 | 优点 | 局限性 |
|---|---|---|
| DXA（双能X线） | 金标准、分区测量 | 成本高、辐射（微量） |
| 皮褶厚度 | 成本低、便携 | 操作者依赖性高 |
| 生物电阻抗(BIA) | 快速、非侵入 | 水合状态影响结果 |
| 水下称重 | 准确 | 设备要求高、不便 |
| 腰围 | 简单 | 仅反映腹部脂肪分布 |

**最佳实践**：体脂评估应结合多种方法，关注趋势而非单次数值，由有资质的运动营养师监督。

## 体重管理策略

### 错误做法
1. 严格限制热量→瘦体重和骨量损失>脂肪损失
2. 不吃早餐→导致长时间低血糖→皮质醇升高→瘦体重分解
3. 晚上19:00后不吃→就寝前可能出现低血糖→皮质醇产生→降低瘦体重和骨量
4. 仅关注体重→忽视身体成分变化

### 正确做法
1. 以身体成分而非体重为目标
2. 小幅能量负平衡（300-500 kcal/d），避免过度限制
3. 高频率、小份量的进食模式维持血糖稳定
4. 减脂最佳窗口：休赛期
5. 确保充足的蛋白质摄入保护瘦体重
6. 结合抗阻训练维持/增加肌肉质量
7. 在合格营养师监督下进行体重管理

### 特殊人群警示
- 体重分级运动（摔跤、举重）：ACSM有专门立场声明[27]
- 审美运动（体操、花样滑冰、跳水）：饮食紊乱高风险
- 女性运动员三联征：能量不足→闭经→骨质疏松
- 青少年运动员：能量限制严重影响生长发育
