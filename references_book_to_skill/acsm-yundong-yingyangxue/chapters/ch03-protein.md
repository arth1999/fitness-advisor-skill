# 第3章 蛋白质

## 章节概述

本章系统阐述蛋白质的结构与分类、消化吸收代谢、运动员实际需求，并重点纠正"蛋白质万能论"的常见误区。

## 蛋白质结构基础

### 氨基酸分类

| 分类 | 定义 | 示例 |
|---|---|---|
| 必需氨基酸 | 人体无法合成，必须从食物获取 | 亮氨酸、异亮氨酸、缬氨酸(BCAA)、赖氨酸、蛋氨酸、苯丙氨酸、苏氨酸、色氨酸、组氨酸 |
| 非必需氨基酸 | 人体可以从其他物质合成 | 丙氨酸、天冬氨酸、谷氨酸等 |
| 条件性必需氨基酸 | 特定条件下需求超过合成能力 | 谷氨酰胺（应激/损伤时） |

### 蛋白质质量评价

- **完全蛋白质**：含所有必需氨基酸（动物性食品）
- **不完全蛋白质**：缺少一种或多种必需氨基酸（多数植物性食品）
- **互补蛋白质**：两种不完全蛋白质组合可提供所有必需氨基酸（如谷物+豆类）

## 蛋白质的功能

1. 组织构建与修复（肌肉、骨骼、皮肤、器官）
2. 酶和激素的合成
3. 免疫功能（抗体产生）
4. 液体平衡（维持渗透压）
5. 酸碱平衡（缓冲作用）
6. 运输功能（脂蛋白、转铁蛋白、血红蛋白）
7. 能量来源（4kcal/g，但非首选）

## 关键数字与阈值

| 参数 | 数值 |
|---|---|
| 蛋白质能量值 | 4 kcal/g |
| 成人RDA | 0.8 g/kg/d |
| 运动员推荐 | 1.2-2.0 g/kg/d |
| 单餐最大合成代谢利用量 | 约30-40g/餐 |
| 每餐超过40g蛋白质后果 | 剩余部分作为能量来源或储存为脂肪 |

## 消化、吸收与代谢

- **消化起始**：胃（胃蛋白酶+盐酸变性蛋白质）
- **主要消化场所**：小肠（胰蛋白酶、胰凝乳蛋白酶等）
- **吸收形式**：氨基酸、二肽、三肽
- **脱氨作用**：去除氨基后，碳骨架可转化为碳水化合物或脂肪
- **含氮废物**：尿素（BUN），通过肾脏排出——蛋白质过量增加脱水风险

## 蛋白质过量摄入的问题

1. **脱水风险**：含氮废物（尿素）排出需增加尿液生成
2. **低骨密度风险**：高蛋白摄入与尿钙增加相关
3. **碳水化合物摄入不足**：蛋白质替代碳水化合物，降低运动表现
4. **肾脏负担**：已有肾病者尤须谨慎
5. **饱和脂肪伴随摄入**：动物蛋白来源常伴随高饱和脂肪

## 运动员蛋白质策略

### 最佳摄入模式
- **分散摄入**：每餐约30g，而非一餐摄入80g+"浪费"40-50g
- **频率**：每3-4小时一次，维持氨基酸可用性
- **运动后窗口**：运动后0-2小时摄入约10g必需氨基酸或0.25-0.3g/kg
- **与能量平衡配合**：蛋白质合成代谢需要良好的能量（尤其是碳水化合物）环境

### 食物来源建议
- 优先选择完整蛋白质来源：瘦肉、禽肉、鱼类、蛋类、乳制品
- 植物蛋白组合：谷物+豆类（确保必需氨基酸齐全）
- 无需依赖蛋白质补充剂：食物可满足1.2-2.0g/kg/d需求
- 素食运动员需额外关注：蛋白质总量、铁、钙、维生素D、维生素B12、锌

## 常见误区纠正

| 误区 | 事实 |
|---|---|
| 蛋白质越多肌肉越多 | 肌肉增长需运动刺激+充足能量+营养平衡，非单纯增加蛋白质 |
| 蛋白质补充剂优于食物 | 食物提供全面的伴随营养素，补充剂无法替代 |
| 低碳水高蛋白是最佳运动饮食 | 碳水化合物是运动表现的限制性能量底物 |
| 蛋白质不会转化为脂肪 | 过量蛋白质经脱氨后碳骨架可合成为脂肪 |
