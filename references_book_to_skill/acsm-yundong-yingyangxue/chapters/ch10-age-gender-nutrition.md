# 第10章 根据年龄和性别最优化营养策略

## 章节概述

本章探讨年轻运动员、女性运动员和老年运动员的差异化营养需求，强调忽视年龄和性别差异可能损害运动潜能并增加健康风险。

## 一、年轻运动员的营养需求

### 生长发育关键数据

| 指标 | 女性 | 男性 |
|---|---|---|
| 生长突增停止年龄 | 约16岁 | 约19岁 |
| 青少年期获得的骨量 | 约50%的骨量在青少年期获得 | — |
| 钙推荐摄入量 | 1300 mg/d（9-18岁） | 1300 mg/d（9-18岁） |

### 年轻运动员的特殊营养风险

1. **能量需求高但胃容量小**：难以通过少量食物满足高能量需求→需营养密度高的食物
2. **钙需求峰值**：青少年期骨量积累决定一生峰值骨量
3. **铁需求增加**：生长突增期血液量扩张+女性月经初潮
4. **维生素D**：室内训练运动员（游泳、体操等）风险高
5. **饮食紊乱风险**：参与审美运动（体操、花样滑冰）的年轻运动员

### 危险信号
- 生长速度减缓或停止
- 青春期延迟（女性>16岁仍无月经初潮）
- 反复发生的应力性骨折
- 持续疲劳和训练恢复不良
- 频繁生病（免疫功能下降）

## 二、女性运动员的营养需求

### 女性运动员三联征(Female Athlete Triad)

依次或同时出现的三个相互关联的问题：
1. **低能量可用性**（有或无饮食紊乱）
2. **月经功能障碍**
3. **低骨密度**

### 月经功能分类

| 状态 | 定义 |
|---|---|
| 月经正常 | 规律月经周期 |
| 月经稀少 | 周期不规律或频率降低 |
| 继发性闭经 | 曾有过正常月经周期，但连续3个月以上未出现 |
| 原发性闭经 | 从未发生过月经（>16岁） |

### 闭经的关键原因与后果
- **主要原因**：能量摄入不足→雌激素分泌下降→破骨细胞抑制减弱
- **骨密度后果**：雌激素是破骨细胞抑制剂→雌激素下降→骨质流失→应力性骨折风险增加
- **恢复可逆性**：恢复正常能量摄入和月经后，部分骨质可恢复，但不是全部

### 女性运动员营养关键参数

| 参数 | 特别要求 |
|---|---|
| 铁 | 育龄女性EAR比男性高约70%（约13.8mg/d vs 8.1mg/d） |
| 钙 | 闭经运动员需特别关注 |
| 维生素D | 室内运动项目女性运动员风险高 |
| 能量摄入 | 最低目标：维持月经功能和骨健康 |

### 女性运动员低钠血症风险
- 女性体型通常较男性小
- 过量补充无钠或低渗液体（纯水）的风险更大
- 建议：运动饮料应含适宜浓度的钠

## 三、老年运动员的营养需求

### 衰老相关的生理变化

| 变化 | 营养影响 |
|---|---|
| 出汗率降低 | 热应激风险增加→主动监测水合状态 |
| 新陈代谢率下降 | 需更高营养密度饮食（更高营养/热量比） |
| 胃肠道活动性降低 | 稍增加纤维摄入+充足液体；胃肠功能可能影响维生素和矿物质吸收 |
| 免疫功能可能下降 | 合理运动+良好饮食+充分休息 |
| 尿频增加 | 可能抑制液体摄入意愿→脱水风险 |
| 适应新环境时间延长 | 旅行后减少运动强度和频率数天 |
| 损伤愈合和疾病康复时间延长 | 避免过度训练；适当补充剂可能需在医生指导下使用 |

### 特别关注的营养素

| 营养素 | 原因 |
|---|---|
| 钙 | 骨质疏松风险随年龄增加 |
| 铁 | 吸收能力可能下降 |
| 锌 | 免疫功能维持 |
| 维生素B6 | 能量代谢 |
| 维生素B12 | 吸收可能因内因子减少而受损（60岁以上） |
| 维生素D | 皮肤合成能力下降 |

### 老年运动员实践策略
1. 主动制定液体摄入策略，不要被动等渴
2. 定期咨询医生评估营养补充需求
3. 新鲜水果蔬菜+全谷物产品提供纤维和碳水化合物
4. 频繁生病可能是免疫功能下降的信号
5. 避免过度训练，给身体足够恢复时间

## 通用实践原则

1. 不同年龄和性别的DRI不同——需针对性调整
2. 所有年龄段的骨骼强化体力活动都重要
3. 育龄女性铁需求远高于同龄男性
4. 青少年钙需求成人化，但胃容量有限→需策略性选择高钙食物
5. 素食运动员在所有年龄段都需额外关注铁、钙、维生素D、B12、锌
6. 营养教育应辐射到运动员的家人、教练和队友
