core_frameworks:
  - name: OPT Model (5 Phases)
    description: >
      Phase 1 (Stabilization Endurance) → Phase 2 (Strength Endurance) →
      Phase 3 (Muscular Development) → Phase 4 (Maximal Strength) → Phase 5 (Power).
      Each phase has specific acute variable prescriptions for reps, sets, intensity,
      rest, and tempo, plus matched progressions across all training components.
  - name: Corrective Exercise Continuum
    description: >
      Inhibit (SMR) → Lengthen (Stretch) → Activate (Strengthen) → Integrate (Re-pattern).
      Applied to address muscle imbalances identified during assessments.
  - name: Integrated Training (7 Components)
    description: >
      Flexibility + Cardiorespiratory + Core + Balance + Plyometric + SAQ + Resistance.
      All components are trained concurrently with phase-appropriate progressions.
  - name: Three-Zone Cardiorespiratory Model
    description: >
      Zone 1 (below VT1, fat-burning, talk comfortably) → Zone 2 (VT1–VT2,
      mixed fuel, talk in short sentences) → Zone 3 (above VT2, carb-burning, cannot talk).
  - name: Stage Training for Cardio
    description: >
      Stage I (Aerobic Base, Zone 1) → Stage II (Aerobic Efficiency, Zone 1–2) →
      Stage III (Aerobic Power, Zone 2–3).
  - name: Flexibility Continuum
    description: >
      SMR (foam rolling) → Static Stretching → Active Stretching → Dynamic Stretching.
      Phase-dependent application from corrective to performance.

key_assessments:
  - PAR-Q+ (Physical Activity Readiness Questionnaire)
  - Static Postural Assessment (anterior, lateral, posterior views)
  - Overhead Squat Assessment (OHSA)
  - Single-Leg Squat Assessment
  - Pushing and Pulling Assessments
  - Push-up Test (muscular endurance)
  - 1RM / Estimated 1RM (strength)
  - Vertical Jump (power)
  - 40-Yard Dash (speed)
  - Pro Agility / 5-10-5 (agility)
  - Body composition: skinfold, bioelectrical impedance, circumference, BMI
  - Cardiorespiratory: YMCA step test, Rockport walk test, 1.5-mile run, 12-min Cooper test

three_postural_distortions:
  - Upper Crossed Syndrome (forward head, rounded shoulders, thoracic kyphosis)
  - Lower Crossed Syndrome (anterior pelvic tilt, lumbar lordosis)
  - Pes Planus Distortion Syndrome (flat feet, knee valgus, internally rotated femurs)

scope_boundaries:
  can_do:
    - Design individualized, evidence-based exercise programs
    - Conduct fitness and movement assessments
    - Provide general nutrition guidance to healthy populations
    - Coach behavior change and goal setting
    - Refer to appropriate healthcare professionals
    - Offer basic hydration and supplementation education
  cannot_do:
    - Diagnose or treat illness, injury, or disease
    - Prescribe specific meal plans
    - Prescribe supplements for disease treatment
    - Provide physical therapy or medical rehabilitation
    - Advise against healthcare professional recommendations
