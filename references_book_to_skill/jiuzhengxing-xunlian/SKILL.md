core_frameworks:
  - name: 纠正性训练三步骤 (3-Step Corrective Exercise Continuum)
    description: >
      自我肌筋膜放松 (SMR, 恢复再生) → 拉伸训练 (Stretch, 增大活动范围) →
      强化训练 (Strengthen, 重塑运动模式)。三步必须按此顺序执行，不可颠倒。
      SMR 解决肌筋膜受限与扳机点，拉伸改善肌肉长度与关节活动度，
      强化重建神经肌肉协调与正确运动模式。

  - name: 弹力绳模型 (Elastic Cord Model)
    description: >
      肌肉并非仅通过向心收缩产生运动，更重要的是在张力下离心拉长以减缓关节受力
      （如同弹力绳/蹦极绳）。肌肉通过离心拉长产生张力→储存势能→收缩释放能量。
      功能失调的肌肉无法有效发挥弹力绳作用，导致关节过度受压和代偿模式。

  - name: 从足到头结构评估 (Foot-to-Head Structural Assessment)
    description: >
      足部和踝部 → 膝关节 → 腰椎—骨盆髋带 (LPHC) → 胸椎和肩胛带 → 颈部和头部。
      每个区域通过问诊→视诊→触诊三步收集信息。身体各部位互为因果，
      下方失衡向上传导，上方失衡向下代偿。

  - name: 全身失衡传导链 (Whole-Body Compensation Chain)
    description: >
      足过度内旋 → 小腿内旋 → 膝外翻 → 大腿内旋 → 髋臼后移 → 骨盆前倾 →
      腰椎过度前凸 → 胸椎过度后凸 → 肩胛带前伸 → 头部前倾 → 颈椎过度前凸。
      下方结构失衡向上传导，上方结构前倾破坏全身对线。

five_regional_dysfunctions:
  - region: 足部和踝部
    dysfunctions:
      - name: 过度内旋 (Overpronation)
        signs: [足弓塌陷, 足外翻, 大足趾外翻/囊肿, 小足趾畸形, 距骨内移, 足底筋膜紧张]
      - name: 背屈不足 (Dorsiflexion Insufficiency)
        signs: [小腿肌肉紧张, 跟腱紧张, 踝前部受压]

  - region: 膝关节
    dysfunctions:
      - name: 侧向对齐问题/膝外翻 (Knee Valgus)
        signs: [髌骨内移, 单腿下蹲时膝过度内移, Q角增大]
      - name: 屈伸运动轨迹问题 (Patellar Tracking Dysfunction)
        signs: [髌骨摩擦音/爆裂音, 膝前部疼痛]

  - region: 腰椎—骨盆髋带
    dysfunctions:
      - name: 骨盆前倾 (Anterior Pelvic Tilt)
        signs: [腰带前低后高, 腰椎曲度增大, 髋臼后移]
      - name: 腰椎过度前凸 (Lumbar Hyperlordosis)
        signs: [下背部过度弓起, 靠墙时背部可容整手/臂]

  - region: 胸椎和肩胛带
    dysfunctions:
      - name: 胸椎过度后凸 (Thoracic Hyperkyphosis)
        signs: [C7与胸骨切迹高度差>3.8cm, 上背部拱起]
      - name: 肩胛带前伸 (Scapular Protraction)
        signs: [肩胛骨椎缘远离脊柱, 肩胛骨突出, 肩峰前于耳屏]
      - name: 内旋臂 (Internally Rotated Arms)
        signs: [站立时可见全部手指/手背]
      - name: 高位肩胛 (Elevated Scapula)
        signs: [一侧或双侧肩胛骨偏高, 斜方肌上部紧张]

  - region: 颈部和头部
    dysfunctions:
      - name: 头部前倾 (Forward Head Posture)
        signs: [颧骨明显前于锁骨, 头部重量加倍效应]
      - name: 颈椎过度前凸 (Cervical Hyperlordosis)
        signs: [靠墙时视线向上倾斜, 颈部过度弯曲]

scope_boundaries:
  can_do:
    - 进行从足到头的结构评估（问诊、视诊、触诊）
    - 识别常见肌肉骨骼失衡问题（非医疗诊断）
    - 选择和推荐纠正性训练（SMR/拉伸/强化）
    - 设计个性化纠正性训练计划
    - 提供姿势改善和生活方式建议（站姿、坐姿、睡姿、鞋类选择）
    - 教授客户寻找身体中立位置
    - 通过纠正性训练缓解常见肌肉骨骼功能障碍与疼痛
    - 与医疗专业人员建立转诊合作关系
  cannot_do:
    - 诊断疾病或损伤（属于医疗行为）
    - 治疗急性损伤、骨折、韧带撕裂
    - 开具医疗处方或药物治疗
    - 替代物理治疗师或医生的专业工作
    - 对禁忌症客户不经医生许可即进行训练
    - 对癌症、严重心血管疾病、急性关节炎患者擅自开展训练
