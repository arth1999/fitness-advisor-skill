# 评估与训练模式库 (Patterns)

> 可复用的结构评估流程、失衡干预映射、训练进退阶决策树和常见代偿模式修复路径。

---

## 模式1：从足到头结构评估流程

```
步骤1: 足部和踝部评估
├── 问诊: 疼痛位置、关节炎、伤病史、活动水平、职业、功能限制、关联症状、加重/缓解因素
├── 视诊: 足弓高度 → 足部朝向(外翻/内翻) → 大足趾(外翻/囊肿) → 小足趾(畸形)
├── 触诊: 距骨位置 → 小腿肌肉(扳机点) → 足底筋膜(压痛点)
└── 关联说明: 足踝失衡如何影响膝

步骤2: 膝关节评估
├── 问诊: (同上模式 + 膝特有问题)
├── 视诊: 外观(肿胀/疤痕) → 静态髌骨中心线 → 动态单腿下蹲 → 后方臀部观察
├── 触诊: 髌骨轨迹(屈伸膝感受摩擦/爆裂声)
├── 中立位指导: 足中立位+臀肌收缩+骨盆后倾
└── 关联说明: 膝如何受上下结构影响

步骤3: LPHC 评估
├── 问诊: (同上模式 + 睡眠影响 + LPHC特有问题)
├── 视诊: 腰带前/后高度(骨盆倾斜) → 腰椎曲度(侧面)
├── 触诊: 靠墙手测(下背部-墙间距) → 仰卧评估
├── 中立位指导: 手掌法(ASIS上双手看指尖)
└── 关联说明: LPHC与下肢和胸椎的关系

步骤4: 胸椎和肩胛带评估
├── 问诊: (同上模式 + 压力水平 + 胸椎肩胛特有问题)
├── 视诊: 肩胛骨位置 → 肩峰-耳屏对齐 → 手臂位置
├── 触诊: C7-胸骨切迹高度差 → 肩胛骨突出 → 肩胛骨高度 → 斜方肌紧张度
├── 中立位指导: 靠墙挺胸收肩胛+手臂外旋
└── 关联说明: 胸椎肩胛与腰椎和头颈的关系

步骤5: 颈部和头部评估
├── 问诊: (同上模式 + TMJ + 颈头特有问题)
├── 视诊: 颧骨-锁骨垂线 → 靠墙视线评估
├── 中立位指导: 靠墙全身对齐(足跟-臀-肩-头触墙,下背部仅容两指节)
└── 总结: 全身对齐与代偿链概述
```

## 模式2：全身失衡传导链 (代偿路径)

### 自下而上传导链 (Bottom-Up Chain)
```
足过度内旋 + 背屈不足
    ↓
小腿过度内旋
    ↓
膝外翻 (侧向对齐问题)
    ↓
大腿内旋 → 髋臼后移
    ↓
骨盆前倾
    ↓
腰椎过度前凸
    ↓
重心前移
    ↓
胸椎过度后凸
    ↓
肩胛带前伸 + 内旋臂 + 高位肩胛
    ↓
头部前倾
    ↓
颈椎过度前凸
```

### 自上而下传导链 (Top-Down Chain)
```
头部前倾 (重量倍增效应)
    ↓
重心前移
    ↓
胸椎过度后凸 (上身前倾代偿)
    ↓
腰椎过度前凸 (保持躯干直立代偿)
    ↓
骨盆前倾
    ↓
髋臼后移 + 大腿内旋
    ↓
膝外翻
    ↓
小腿内旋
    ↓
足过度内旋 + 背屈不足
```

### 关键枢纽点
- **骶髂关节 (SI Joint):** 上半身与下半身的力传导枢纽
- **骨盆位置:** 决定上下半身对齐的核心变量
- **C7-T1:** 颈胸交界，头部与躯干的力线转折点

## 模式3：五大区域失衡 → 肌肉 → 训练映射

### 足部和踝部失衡干预
```
过度内旋 (Overpronation)
├── 需SMR: 足底筋膜(高尔夫球)、小腿(泡沫轴/手)
├── 需拉伸: 小腿(腓肠肌/比目鱼肌)、足部和足趾
├── 需强化: 大足趾下推(踇长屈肌/踇展肌)、胫骨前肌激活
└── 生活方式: 鞋类选择、赤足在不稳定表面行走

背屈不足 (Dorsiflexion Insufficiency)
├── 需SMR: 小腿(比目鱼肌/腓肠肌)、跟腱周围
├── 需拉伸: 小腿拉伸(多角度)
├── 需强化: 大足趾下推(间接改善)
└── 注意: 背屈不足与过度内旋通常共存，同时处理
```

### 膝关节失衡干预
```
膝外翻 (Knee Valgus)
├── 需SMR: ITB(泡沫轴)、大腿内侧(泡沫轴)、臀肌(泡沫轴)、髋屈肌(泡沫轴/网球)
├── 需拉伸: 髋内收肌、臀肌、腘绳肌
├── 需强化: 鸭式站立 → 侧卧抬腿(臀中肌/臀小肌) → 仰撑提臀(臀大肌) → 单腿支撑转动(运动链)
└── 因果处理: 同时解决足过度内旋(下方)和骨盆前倾(上方)

髌骨运动轨迹问题 (Patellar Tracking)
├── 需SMR: 股四头肌(泡沫轴)、ITB
├── 需拉伸: 股四头肌
├── 需强化: 等长股四头肌激活 → 向心/离心单关节膝伸展
└── 注意: 髌骨轨迹问题常因膝外翻导致，需先纠正对齐
```

### LPHC 失衡干预
```
骨盆前倾 + 腰椎过度前凸
├── 需SMR: 髋屈肌(泡沫轴/网球)、竖脊肌(网球)、臀肌(泡沫轴)、腰方肌(泡沫轴)
├── 需拉伸: 髋屈肌(矢状面/额状面/水平面三方向)、下背部、梨状肌
├── 需强化: 仰卧骨盆倾斜 → 仰撑提臀 → 瑞士球臀肌激活 → 球上仰身屈伸(离心腹肌)
├── 关键拮抗关系:
│   紧张需放松: 髋屈肌(腰大肌/髂肌)、竖脊肌
│   无力需强化: 臀大肌、腹肌群(离心功能)、腘绳肌
└── 下方原因: 同时检查是否有足内旋和膝外翻需要处理
```

### 胸椎和肩胛带失衡干预
```
胸椎过度后凸 (Thoracic Hyperkyphosis)
├── 需SMR: 胸椎(泡沫轴/双网球)、背阔肌(泡沫轴)、胸部(网球)
├── 需拉伸: 门框拉伸(胸肌)、胸部拉伸、肱二头肌、"为什么"拉伸(背阔肌)
├── 需强化: 地面缩肩 → 坐姿划船 → 直臂上举(仰卧→站姿) → 弓步手臂上举
└── 呼吸影响: 胸椎后凸压迫膈肌和肋间肌 → 需改善呼吸模式

肩胛带前伸 (Scapular Protraction)
├── 需SMR: 网球放松肩胛骨周围(菱形肌区)、胸部按摩(胸小肌)
├── 需拉伸: 胸部拉伸、门框拉伸
├── 需强化: 地面缩肩(菱形肌) → 坐姿划船 → 挥手再见式 → "曝光式"练习
└── 注意: 胸小肌紧张→肩胛骨前伸→菱形肌被拉长无力

内旋臂 (Internally Rotated Arms)
├── 需SMR: 背阔肌(泡沫轴)、胸大肌(网球)、肩前部(网球)、肱二头肌(网球)
├── 需拉伸: 胸肌、肱二头肌、仰卧肩前部拉伸
├── 需强化: "曝光式"练习(肩袖外旋) → 挥手再见式(肩外旋+肩胛下降) → 坐姿划船
└── 注意: 冈下肌和小圆肌(外旋袖肌)在长期圆肩姿势中被过度拉长

高位肩胛 (Elevated Scapula)
├── 需SMR: 按摩杖按摩斜方肌上部、网球放松肩胛骨
├── 需拉伸: 斜方肌拉伸
├── 需强化: 挥手再见式(肩胛下降) → 直臂下拉(背阔肌+肩胛下降)
└── 注意: 斜方肌下部通常被拉长无力，需特别强化
```

### 颈部和头部失衡干预
```
头部前倾 + 颈椎过度前凸
├── 需SMR: 按摩杖/网球按摩后颈、按摩杖按摩斜方肌上部
├── 需拉伸: 颈伸肌拉伸、斜方肌拉伸
├── 需强化: 颈部屈曲(颈长肌/头长肌)
├── 因果处理: 
│   上方: 直接处理颈部和头部对齐
│   下方: 胸椎过度后凸必须先得到改善(上背部挺直后头部自然回位)
└── 姿势提示: 靠墙全身对齐练习(每日练习)
```

## 模式4：训练计划构建模板

### 基于主要抱怨的区域聚焦计划

```
客户主要抱怨: [具体部位疼痛/功能障碍]

1. 结构评估 (从足到头)
   → 发现所有关联失衡，不仅聚焦疼痛部位

2. 确定关键失衡链
   → 画出自下而上或自上而下的传导路径

3. 按三步骤选择训练
   每次课程:
   ├── SMR (5-15min):
   │   ├── 最受限区域的一般SMR (泡沫轴)
   │   └── 精确扳机点的特殊SMR (网球/按摩杖)
   ├── 拉伸 (5-15min):
   │   ├── 被动拉伸 (初期)
   │   ├── 主动拉伸 (拮抗肌收缩)
   │   └── 动态拉伸 (多区域整合)
   └── 强化 (10-20min):
       ├── 等长激活 (目标肌肉能否独立激活?)
       ├── 向心单关节 (建立控制)
       ├── 离心多关节 (弹力绳减速训练)
       └── 运动链/多平面 (整合到功能动作)

4. 家庭练习 (选择3-5个最实用动作)
   ├── 1-2个SMR (如足底网球滚动、小腿自我按摩)
   ├── 1-2个拉伸 (如小腿拉伸、髋屈肌拉伸)
   └── 1-2个强化 (如鸭式站立、仰撑提臀)

5. 生活方式建议
   ├── 站姿/坐姿/睡姿改善
   ├── 鞋类选择建议
   └── 工作环境调整
```

## 模式5：进退阶决策树

### 通用进退阶判断
```
客户执行训练时:
├── 出现疼痛?
│   ├── 是 → 立即停止 → 退阶到无痛训练或SMR
│   └── 否 → 继续评估
├── 能正确完成技术?
│   ├── 否 → 退阶 (简化姿势/减少关节/增加支撑)
│   └── 是 → 继续评估
├── 在正确部位感受到效果?
│   ├── 否 → 检查技术 → 调整或退阶
│   └── 是 → 继续评估
├── 已熟练掌握且不再进步?
│   ├── 是 → 进阶 (更复杂/更重/更不稳定/更快)
│   └── 否 → 保持当前水平继续训练
└── 训练耗时过长?
    ├── 是 → 用复合训练替代多个单一训练
    └── 否 → 维持当前计划
```

### SMR 进退阶路径
```
特殊SMR (网球精确扳机点) ← 退阶 ← 一般SMR (泡沫轴大面积)
     ↓ 进阶
拉伸训练 (同一肌肉的被动拉伸)
     ↓
主动拉伸 → 动态拉伸 → 强化训练
```

### 拉伸进退阶路径
```
卧姿被动拉伸 ← 退阶 ← 站姿被动拉伸
     ↓ 进阶
主动拉伸 (加入拮抗肌收缩)
     ↓ 进阶
动态拉伸 (加入其他身体部位)
     ↓ 进阶
强化训练 (增加负荷)
```

### 强化进退阶路径
```
等长单关节 ← 退阶 ← 向心单关节
     ↓ 进阶
离心单关节 → 向心/离心多关节
     ↓ 进阶
开链运动链 → 闭链运动链
     ↓ 进阶
多平面多维度协调训练
     ↓ 进阶
运动专项模拟 (增加速度和外部负荷)
```

## 模式6：常见客户情景应对

### 情景1: 客户只有膝痛，拒绝做足部训练
```
沟通策略:
1. 使用比喻: "房子的地基歪了，墙就会出现裂缝。膝盖就像墙，足就像地基。"
2. 展示评估证据: 让客户亲眼看到足内旋→小腿内旋→膝外翻的连接
3. 短期承诺: "先试2周足部训练，然后我们再看膝的变化"
```

### 情景2: 客户没有时间做所有推荐训练
```
策略:
1. 精简到3个最核心训练 (1 SMR + 1 拉伸 + 1 强化)
2. 选择可以融入日常生活的训练 (如洗澡时的足底按摩、等电梯时的小腿拉伸)
3. 先用复合训练替代 (如用单腿支撑转动同时训练臀肌+核心+平衡)
```

### 情景3: 客户过于积极，训练过度
```
策略:
1. 强调正确执行 > 完成更多次数
2. 解释离心训练过量容易疲劳和受伤
3. 给具体的"上限"(如每天最多2组x10次)
4. 教育他们识别身体信号(疼痛 vs 适度的训练感)
```

### 情景4: 客户抱怨训练没有效果
```
排查路径:
1. 检查技术正确性 (是否在正确部位感受到效果?)
2. 检查训练频率和坚持度
3. 重新评估 — 失衡是否有新的变化?
4. 是否SMR不足导致拉伸和强化无法有效执行?
5. 是否需要调整训练选择?
```
