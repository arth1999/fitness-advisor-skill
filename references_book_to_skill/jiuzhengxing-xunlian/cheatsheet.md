## 六、纠正性训练三步骤总览

```
步骤1: 自我肌筋膜放松 (SMR)
├── 目标: 恢复和再生受损软组织
├── 时机: 每次训练开始前 / 家庭练习
├── 一般工具: 泡沫轴
├── 特殊工具: 网球、高尔夫球、按摩杖
├── 进阶信号: 不再感到压迫感 → 换更硬工具 或 过渡到拉伸
└── 退阶信号: 疼痛/不适/无法正确执行 → 换更软工具/坐姿/卧姿/手指按摩

步骤2: 拉伸训练 (Stretch)
├── 目标: 增大活动范围 / 恢复正确运动模式
├── 阶段1: 被动拉伸 (安全可控，适合初期)
├── 阶段2: 主动拉伸 (拮抗肌收缩，适合进阶)
├── 阶段3: 动态拉伸 (多部位整合，过渡到强化)
├── 进阶信号: 技术达标/不再进步 → 加入拮抗肌收缩/站姿/多部位
└── 退阶信号: 疼痛/无法正确执行/无法平衡 → 坐姿/卧姿/加支撑/退回SMR

步骤3: 强化训练 (Strengthen)
├── 目标: 重塑神经肌肉协调和正确运动模式
├── 阶段1: 等长 (激活)
├── 阶段2: 向心单关节 → 离心单关节
├── 阶段3: 向心/离心多关节
├── 阶段4: 开链运动链 → 闭链运动链
├── 阶段5: 多平面多维度协调
├── 进阶信号: 技术达标/正确次数 → 增加负荷/移除支撑/加快速度/更不稳定
└── 退阶信号: 疼痛/神经肌肉混乱/过度酸痛 → 减少负荷/加支撑/卧姿/退回SMR
```

## 七、静态姿势改善速查

| 情境 | 关键建议 |
|------|---------|
| 站姿 | 目视前方→收回下巴→挺胸收肩→骨盆后倾→重心压足跟 |
| 计算机前坐姿 | 显示器于视线水平、肩放松后收、骨盆中立、足平放、30-60分钟起身 |
| 开车 | 座椅位置使膝略低于髋、双手在方向盘上不耸肩 |
| 使用手机 | 抬起到视线水平、双手交替、不用单肩夹手机 |
| 仰卧睡姿 | 膝下垫枕（减轻腰椎压力） |
| 侧卧睡姿 | 膝间夹枕（保持骨盆对齐）、枕头高度使颈椎中立 |
| 鞋类 | 逐步减少鞋跟高度、逐渐切换平底鞋 |
| 提重物 | 身体两侧交替负载、使用双肩包替代单肩包 |
