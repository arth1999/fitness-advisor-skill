# 术语表 (Glossary) — 基于生物力学的纠正性训练

> 中英双语术语表，按类别分组。每个术语包含中文名称、英文名称及简要定义。

## 一、结构评估术语 (Structural Assessment)

| 中文 | English | 定义 |
|------|---------|------|
| 结构评估 | Structural Assessment | 有组织地收集客户肌肉骨骼系统信息的过程，包括问诊、视诊、触诊 |
| 问诊 | Interview | 通过结构化提问获取客户肌肉骨骼健康史、疼痛、活动水平等信息 |
| 视诊 | Visual Assessment | 通过肉眼观察客户身体各部位的静态排列和对称性 |
| 触诊 | Palpation | 通过手部接触评估骨性标志位置、软组织张力和关节运动质量 |
| 客户评估表 | Client Assessment Diagram (CAD) | 记录评估结果的标准化表格，包含解剖图、问诊结果和评估发现 |
| 从足到头评估 | Foot-to-Head Assessment | 从足部和踝部开始，沿身体向上依次评估各区域的方法论 |

### 足部和踝部评估
| 中文 | English | 定义 |
|------|---------|------|
| 过度内旋 | Overpronation | 足部在正常活动中向内过度收缩或转动幅度过大，导致足弓塌陷、踝内移 |
| 背屈不足 | Dorsiflexion Insufficiency | 踝关节无法在全范围内适当前屈（足背向小腿靠拢），通常伴随过度内旋 |
| 距骨 | Talus | 踝关节处的关键骨骼，位于胫骨和腓骨下方、跟骨上方 |
| 距骨中立位置 | Talus Neutral Position | 距骨在踝内外侧压力均匀时的位置，为足踝正确对齐位置 |
| 内侧纵弓 | Medial Longitudinal Arch | 沿足部内侧从足跟到大足趾底部的弓形，为主要减震器 |
| 外侧纵弓 | Lateral Longitudinal Arch | 沿足部外侧的弓形，更为稳定，运动能力弱 |
| 横弓 | Transverse Arch | 横跨足中段的弓形，消除侧向冲击力 |
| 足底筋膜 | Plantar Fascia | 足底宽而致密的结缔组织，从足跟延伸至足趾 |
| 跟腱 | Achilles Tendon | 连接腓肠肌和比目鱼肌至跟骨的强大弹性肌腱 |
| 外翻（足） | Out-toeing | 双足从身体中线向外旋，常见于过度内旋的代偿 |
| 踇外翻 | Hallux Valgus (Bunion) | 大足趾向小足趾方向偏移，第一跖趾关节内侧突出 |
| 槌状趾 / 爪形趾 | Hammer Toe / Claw Toe | 小足趾畸形，因重量无法正常分布所致 |

### 膝关节评估
| 中文 | English | 定义 |
|------|---------|------|
| 膝外翻 | Knee Valgus | 膝关节向身体中线过度内移的侧向对齐问题 |
| 膝内翻 | Knee Varus | 膝关节向身体外侧偏移（弓形腿） |
| 髌骨运动轨迹问题 | Patellar Tracking Dysfunction | 膝屈伸时髌骨不能在股骨槽上流畅滑动，产生摩擦/爆裂声 |
| Q角 | Q-Angle | 股骨相对于膝中心的角度，男性>15°、女性>18°视为异常 |
| 单腿下蹲评估 | Single-Leg Squat Assessment | 动态评估膝外翻的核心方法，客户单腿浅蹲约30°观察髌骨位置 |
| 萨尔萨舞臀部 | Salsa Dance Hip | 单腿下蹲时髋部远离中线向外的代偿运动，提示膝外翻 |
| 半月板 | Meniscus (pl. Menisci) | 股骨和胫骨之间的C形软骨盘，起减震作用 |
| 侧副韧带 | Collateral Ligaments | 膝内外侧韧带（内侧MCL、外侧LCL），提供侧向稳定性 |
| 交叉韧带 | Cruciate Ligaments | 膝内前后交叉韧带（ACL/PCL），提供旋转和前后稳定性 |
| 髂胫束 | Iliotibial Band (ITB) | 沿大腿外侧延伸的密集结缔组织，连接臀大肌/TFL至胫骨 |

### LPHC 评估
| 中文 | English | 定义 |
|------|---------|------|
| 腰椎—骨盆髋带 | Lumbo-Pelvic-Hip Complex (LPHC) | 腰椎、骨盆和髋部汇合的功能区域 |
| 骨盆前倾 | Anterior Pelvic Tilt | 骨盆前部过度向前/向下旋转（>正常约10°前倾） |

| 髂前上棘 | Anterior Superior Iliac Spine (ASIS) | 骨盆前部的骨突，腰带前部所在位置 |
| 髂后上棘 | Posterior Superior Iliac Spine (PSIS) | 骨盆后部的骨突，腰带后部所在位置 |
| 骶髂关节 | Sacroiliac Joint (SI Joint) | 骶骨与骨盆髂骨之间的关节，为上下半身力传导枢纽 |
| 髋臼 | Acetabulum | 骨盆形成的杯状凹陷，与股骨头组成髋关节 |
| 手掌法 | Hand-on-Pelvis Method | 客户将手掌放在骨盆前部，低头看手指来判断骨盆中立位的方法 |
| 靠墙手测 | Wall Hand Test | 靠墙站立，手滑入下背部与墙之间评估腰椎前凸程度 |

| 胸骨切迹 | Suprasternal Notch (Jugular Notch) | 胸骨上端、锁骨中间的凹槽 |
| 耳屏 | Tragus | 盖住耳朵前部开口的肉质突起，用于肩峰对齐评估的参照点 |
| 肩峰 | Acromion | 肩胛骨向外延伸的顶部骨突出 |
| 盂肱关节 | Glenohumeral Joint | 肩关节，肱骨与关节盂相连处 |
| 胸锁关节 | Sternoclavicular Joint | 锁骨与胸骨连接处 |
| 肩锁关节 | Acromioclavicular Joint (AC Joint) | 锁骨与肩峰连接处 |

### 颈部和头部评估

| 项韧带 | Ligamentum Nuchae | 从颅底延伸至C7的结缔组织，支撑头部防止过度屈曲 |

## 二、功能解剖学术语 (Functional Anatomy)

| 中文 | English | 定义 |
|------|---------|------|
| 弹力绳模型 | Elastic Cord Model | 肌肉在张力下离心拉长以减速关节运动、储存势能、然后向心释放的模型 |
| 实际功能解剖学 | Functional Anatomy in Real Life | 区别于传统尸体解剖学，强调肌肉在负重、重力、地面反作用力下的实际功能 |
| 离心收缩 | Eccentric Contraction | 肌肉在张力下拉长的动作，产生减速效应 |
| 向心收缩 | Concentric Contraction | 肌肉缩短，起点和止点相互靠近的动作 |
| 等长收缩 | Isometric Contraction | 肌肉激活但长度不变的动作 |
| 起点 | Origin | 肌肉附着在相对固定的骨骼上的端点 |
| 止点 | Insertion | 肌肉附着在相对活动的骨骼上的端点 |
| 弹力绳效应 | Elastic Cord Effect | 肌肉离心拉长产生张力→减速关节→储存势能→向心释放的完整功能链 |
| 相互抑制 | Reciprocal Inhibition | 一块肌肉收缩时，其拮抗肌受到神经抑制而放松 |
| 地面反作用力 | Ground Reaction Force (GRF) | 地面以大小相等、方向相反的力回馈身体（牛顿第三定律） |
| 骨骼肌 | Skeletal Muscle | 受意识控制的肌肉类型，通过肌腱附着在骨骼上，有弹性、收缩性和延展性 |
| 平滑肌 | Smooth Muscle | 不受意识控制的肌肉类型，存在于消化道等器官 |
| 心肌 | Cardiac Muscle | 仅存在于心脏的肌肉类型 |
| 肌腱 | Tendon | 弹性坚韧结缔组织，连接肌肉至骨骼 |
| 韧带 | Ligament | 坚韧带状结缔组织，连接骨骼至骨骼 |
| 软骨 | Cartilage | 坚固有弹性的组织，包裹骨骼两端保护关节 |
| 滑液 | Synovial Fluid | 关节间隙中的透明液体，润滑关节并维持活动 |

## 三、筋膜系统术语 (Fascia Systems)

| 中文 | English | 定义 |
|------|---------|------|
| 筋膜 | Fascia | 三维结缔组织网络，包裹、支撑和分离身体每个结构 |
| 浅筋膜 | Superficial Fascia | 皮肤与肌肉之间的筋膜层 |
| 深筋膜 | Deep Fascia | 连接不同层次肌肉、保持神经血管器官位置的筋膜层 |
| 肌筋膜 | Myofascia | 与肌肉相关的特殊筋膜系统 |
| 肌筋膜网络 | Myofascial Network | 围绕肌肉的筋膜系统，帮助身体在多平面协调运动 |
| 前表线 | Superficial Front Line | 身体前方的筋膜线（胸锁乳突肌→腹直肌→股四头肌→胫骨前肌） |
| 后表线 | Superficial Back Line | 身体后方的筋膜线（足底筋膜→腓肠肌→腘绳肌→竖脊肌→颅顶） |
| 前深线 | Deep Front Line | 身体核心深层的筋膜线（颈长肌→横膈膜→髋屈肌→髋内收肌→胫骨后肌） |
| 外侧线 | Lateral Line | 身体两侧的筋膜线（腓骨肌→ITB→TFL/臀大肌→腹斜肌→肋间肌） |
| 螺旋线 | Spiral Line | 从前到后缠绕身体的筋膜线，协调旋转运动 |
| 胸腰筋膜 | Thoracolumbar Fascia | LPHC区域的三层筋膜集合，提供核心稳定性 |
| 扳机点 | Trigger Point | 肌肉或筋膜中高度敏感、可触及的紧张结节 |
| 粘连 | Adhesion | 疤痕组织在筋膜层之间的异常连接 |
| 受限 | Restriction | 筋膜因紧张、僵硬或粘连而失去正常滑动能力 |

## 四、纠正性训练术语 (Corrective Exercise)

| 中文 | English | 定义 |
|------|---------|------|
| 纠正性训练三步骤 | 3-Step Corrective Exercise Continuum | SMR → 拉伸 → 强化 (Inhibit → Lengthen → Activate) |
| 自我肌筋膜放松 | Self-Myofascial Release (SMR) | 对肌筋膜受限区域施加持续压力以恢复和再生的自我按摩技术 |
| 一般SMR技术 | General SMR Technique | 使用大表面积工具（如泡沫轴）处理大面积肌群 |
| 特殊SMR技术 | Specific SMR Technique | 使用小表面积工具（网球/按摩杖）精确定位单块肌肉或扳机点 |
| 被动拉伸 | Passive (Static) Stretching | 保持固定姿势增大活动范围的拉伸，身体其他部位被动放松 |
| 主动拉伸 | Active Stretching | 在拮抗肌收缩的同时保持目标肌拉伸 |
| 动态拉伸 | Dynamic (Integrated) Stretching | 多部位协调的、精确可控的运动，整合之前掌握的正确动作模式 |
| 单关节训练 | Single-Joint Exercise | 仅涉及一个关节运动的强化训练 |
| 多关节训练 | Multi-Joint Exercise | 涉及多个关节的强化训练 |
| 运动链训练 | Kinetic Chain Exercise | 一系列连续相互依赖的动作组成的训练 |
| 开链运动 | Open Kinetic Chain | 运动链末端（手/足）可自由移动 |
| 闭链运动 | Closed Kinetic Chain | 运动链末端（手/足）受约束或固定 |
| 多平面训练 | Multi-Planar Exercise | 在不同方向（前后/左右/旋转）进行的训练 |
| 多维度训练 | Multi-Dimensional Exercise | 在不同稳定性条件下（如不稳定表面）进行的训练 |
| 进阶 | Progression | 增加训练难度和挑战 |
| 退阶 | Regression | 降低训练难度，使客户能正确完成 |
| 训练替代方案 | Exercise Alternative | 解决相同失衡问题的相似训练 |
| 纠正性训练计划 | Corrective Exercise Program | 基于评估结果的个性化SMR→拉伸→强化训练方案 |
| 家庭练习 | Home Exercise | 客户在家独立完成的训练任务 |
| 禁忌症 | Contraindication | 不应使用特定训练技术的情况 |
| 弹振式拉伸 | Ballistic Stretching | 涉及弹跳或跳跃的拉伸方式，有慢性问题的客户应避免 |
| 泡沫轴 | Foam Roller | 一般SMR的主要工具，大表面积圆柱体 |
| 按摩杖 | Thera Cane | 特殊SMR工具，弯曲杖形设计用于颈部、肩部等难以触及的部位 |
| 客户依从性 | Client Compliance | 客户坚持执行推荐训练计划的程度 |
