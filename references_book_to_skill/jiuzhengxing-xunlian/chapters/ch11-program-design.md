# 第11章 纠正性训练计划制订

## 概述

有效的纠正性训练计划必须：
1. 基于结构评估结果确定需要解决的失衡问题
2. 按照正确的顺序选择训练（SMR → 拉伸 → 强化）
3. 满足客户的生理需求和心理需求
4. 结合沟通策略提高参与度和依从性
5. 包含家庭练习和生活方式建议

## 客户关系建立

### 了解基本动机

客户寻求帮助的原因可能是：
- 减轻特定部位的疼痛
- 恢复某种活动能力（如打网球、园艺）
- 改善姿势和外观
- 提升整体健康

**关键策略：** 在问诊中收集客户的动机信息（"您的疼痛是否限制了您做某件事？"），将这些动机与训练计划关联。

### 角色认同

- 客户可能认同自己是"病人"、"伤者"或"能力受限者"
- 纠正性训练的过程之一是帮助客户从这些角色过渡到"有能力者"
- 通过逐步成功的训练体验改变自我认知

### 作为引导者

纠正性训练专家的角色不是"治疗师"（那是医疗专业人员），而是**引导者**——引导客户通过评估了解身体问题，通过训练逐步改善，最终恢复功能。

## 咨询流程

### 第一步：初步接触

- 首选电话直接交流（比邮件/短信更能缓解焦虑）
- 提供基本简介：评估过程、着装要求、时间、费用
- 发送详细的场所位置和停车信息
- 保持简单清晰，不要信息过载

### 第二步：问诊过程

**核心目标：**
1. 了解客户的肌肉骨骼健康历史
2. 收集损伤、手术、疾病信息
3. 了解身体活动水平和类型
4. 确定职业相关负荷
5. 发现客户的动机和期望

**问诊技巧：**
- 由你来提问并记录（不是让客户自己填表）
- 观察肢体语言
- 营造团队合作交流的氛围
- 给客户时间回忆和补充信息

### 第三步：视诊和触诊过程

- 从足部开始（威胁性最小）
- 持续向客户解释每个步骤和发现
- 记录结果到CAD
- 如果客户感到疼痛或不适，立即停止
- 评估结果直接关联到后续训练选择

### 第四步：成交

- 基于评估结果解释训练计划的价值
- 将训练建议与客户的个人目标关联
- 讨论训练频率、时长和预期进展
- 回答所有疑问

## 计划构建

### 训练课程结构

每次训练课程的建议结构：

```
1. SMR (5-15分钟)
   - 针对评估中发现的受限区域
   - 一般技术（泡沫轴）+ 特殊技术（网球/按摩杖）

2. 拉伸训练 (5-15分钟)
   - 从被动拉伸开始
   - 进阶到主动拉伸（加入拮抗肌收缩）
   - 最后到动态拉伸（多部位整合）

3. 强化训练 (10-20分钟)
   - 从等长→向心单关节→离心→运动链→多平面
   - 正确执行优先于次数
   - 疲劳时停止，不鼓励"再坚持一下"
```

### 家庭练习设计

- 选择客户可以独立正确完成的训练
- 训练应实用（器材易得、场地不限）
- 给客户选择权（几项可选的替代训练）
- 每次课程检查家庭练习的执行情况

### 训练计划周期

| 阶段 | 时间框架 | 重点 |
|------|---------|------|
| 初期 | 1-4周 | SMR为主，轻拉伸，建立信心和习惯 |
| 中期 | 4-8周 | 增加主动/动态拉伸，引入等长和向心强化 |
| 进阶期 | 8-12周 | 离心强化、运动链、多平面训练 |
| 维持期 | 12周+ | 综合训练、运动专项模拟 |

## 案例研究模式

### 客户背景信息收集

- 年龄、性别、职业
- 主要抱怨（疼痛部位、时长、性质）
- 日常活动和运动习惯
- 伤病史和手术史
- 客户目标

### 从评估到计划

**示例流程（膝内侧疼痛客户）：**
1. 问诊：右膝内侧疼痛，跑步时加重，久坐也痛
2. 评估发现：
   - 足部：右侧过度内旋，足外翻
   - 膝：右侧膝外翻（单腿下蹲时明显）
   - LPHC：骨盆前倾，腰椎过度前凸
   - 胸椎：轻度后凸
   - 头颈：轻度前倾
3. 失衡分析：足过度内旋 → 小腿内旋 → 膝外翻 →（可能原因而非膝本身问题）
4. 训练策略：
   - SMR：足底（高尔夫球）、小腿、ITB、臀肌
   - 拉伸：小腿、腘绳肌、髋屈肌
   - 强化：鸭式站立、侧卧抬腿、仰撑提臀、单腿支撑转动

### 整理课后练习

- 为客户的每次课程提供书面总结
- 列出家庭练习及简要执行说明
- 提供替代训练选项（如果客户没有特定器材）
- 记录客户进展以便下次课程参考

## 有效沟通策略

1. **避免术语：** 用"自我按摩"替代"自我肌筋膜放松"
2. **关联动机：** 每个训练都解释它如何帮助客户回到想做的活动
3. **表扬进步：** 当客户完成过去无法完成的动作时，明确指出来
4. **提供选择：** 给客户2-3个可行的替代训练，让他们有控制感
5. **建立连接：** 解释不同训练之间的关系——"因为你放松了X，所以现在可以做Y"
6. **处理焦虑：** 观察困惑、过度热情、不提问等焦虑信号，及时调整策略

## 常见问题处理

| 问题 | 应对策略 |
|------|---------|
| 客户不理解为何治膝却训练足 | 解释失衡传导链，用比喻说明"基石歪了上面的东西都会歪" |
| 客户觉得训练太多没时间 | 用复合动态训练替代多个单关节训练 |
| 客户偷懒不做家庭练习 | 选择客户最喜欢的1-2个训练，先养成习惯再增加 |
| 客户训练无进展 | 检查技术正确性 → 调整变量 → 退阶重新准备 |
| 客户抱怨训练后疼痛 | 退阶到SMR和被动拉伸，评估是否过度训练 |
