# 第7章 拉伸训练技术

## 拉伸训练概述

**定义：** 拉伸是伸展并拉长肌肉纤维（及周围肌腱和筋膜）的过程，以恢复血液流动、修复或再训练受损组织、提高软组织弹性。

**核心目的：**
- 打破"疼痛→肌肉收紧→不动→更多限制→更多损伤"的恶性循环
- 安全增大身体活动范围
- 让神经系统形成正确的传导方式
- 为强化训练阶段奠定基础

## 三种拉伸类型

### 1. 被动拉伸 (Passive/Static Stretching)

**特征：**
- 保持固定姿势（借助手、墙壁、地面或他人帮助）
- 身体其他部位被动放松，不需控制
- 在预定时间内增大关节活动范围

**适用阶段：** 拉伸训练最初始阶段

**优势：**
- 安全可控
- 客户感到安全，不需要控制身体其他部位
- 不会引起新运动的焦虑

**示例：** 卧姿股四头肌拉伸

### 2. 主动拉伸 (Active Stretching)

**特征：**
- 基于"相互抑制"原理——肌肉收缩时拮抗肌受到抑制
- 在拮抗肌收缩的同时保持目标肌拉伸姿势

**适用阶段：** 被动拉伸之后，作为进阶

**优势：**
- 促进神经肌肉协调
- 教会身体同时拉伸和收缩肌肉
- 让神经系统重新放松之前受伤或疼痛的部位
- 为动态拉伸和日常活动做准备

**关键要求：** 必须了解肌肉解剖学——知道哪块肌肉收缩能使拮抗肌放松

**示例：** 髋屈肌拉伸同时收缩臀大肌

**相互抑制对照：**

| 目标拉伸肌 | 需主动收缩的拮抗肌 |
|-----------|-----------------|
| 髋屈肌（腰大肌、髂肌） | 臀大肌 |
| 腘绳肌 | 股四头肌 |
| 股四头肌 | 腘绳肌 |
| 胸大肌/胸小肌 | 菱形肌、斜方肌中部 |
| 腓肠肌/比目鱼肌 | 胫骨前肌 |

### 3. 动态拉伸 (Dynamic / Integrated Stretching)

**特征：**
- 将个别部位的正确动作整合为涉及身体多个部位的协调功能性动作
- 强调精确和可控的运动（不是弹跳或剧烈运动）
- 逐步增大活动范围或速度

**适用阶段：** 拉伸训练的最后阶段

**优势：**
- 提升动态柔韧性
- 整合之前掌握的个别部位正确动作
- 理想过渡到强化训练或体育活动

**注意：** 许多动态拉伸训练同时也被视为强化训练。

**示例：** 在腘绳肌拉伸中加入躯干旋转——同时拉伸腘绳肌 + 强化站立足的稳定肌 + 强化斜方肌

## 教学方法

1. **拉伸时长：** 拉伸30-60秒与拉伸超过60秒相比，通常不会产生更大的活动范围增加。但每个人的体能条件最终决定拉伸应保持的时间和频率。

2. **告知感受位置：** 告诉客户应该在哪个部位感受到拉伸。这有助于：
   - 客户自我监控训练
   - 减少伤痛发生可能性
   - 提升内在动机
   - 帮助识别技术问题

   **示例：** 梨状肌拉伸——应感受到臀部后方从脊椎底部到髋部外侧的拉伸。若客户感到腹股沟不适，可能技术不正确。

3. **建立与评估的关联：** 强调每项拉伸训练如何改善评估中发现的失衡。

4. **为强化训练铺路：** 让客户知道当前拉伸训练为他们即将进行的强化训练做好了什么准备。

## 安全注意事项

- **禁止弹振式拉伸：** 弹跳或跳跃式拉伸会对慢性肌肉骨骼问题的客户造成进一步伤害
- **尊重保护性收紧：** 长期受限的肌肉可能因保护受伤部位而无法拉伸——强行拉伸会使肌肉进一步收紧
- **避免过度拉伸：** 客户信心增强后可能过度拉伸 → 肌肉组织损伤 → 第二天过度酸痛
- **必须热身：** 拉伸前应进行SMR热身（或在家洗热水浴）
- **充分补水：** 软组织含大量水分，脱水的客户应推迟训练
