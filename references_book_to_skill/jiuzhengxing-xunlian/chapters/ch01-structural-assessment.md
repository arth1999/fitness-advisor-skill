# 第1章 结构评估方法论

## 概述

结构评估是基于生物力学的纠正性训练的核心基础。它通过有组织、有次序、有条理的方式收集客户肌肉骨骼系统状况的信息。评估从足部和踝部开始，沿身体向上依次进行，最终到达颈部和头部。这一"从足到头"的方法论基于一个核心理念：**身体各部位相互关联，任一环节的失衡都会通过运动链影响整体**。

## 评估三要素

结构评估包含三个不可分割的部分：

### 1. 问诊 (Interview)

在接触客户身体之前，通过结构化提问收集信息。

**核心问题清单：**
1. 该部位是否有疼痛？疼痛的具体位置和性质？
2. 是否患有关节炎或其他影响肌肉骨骼健康的疾病？
3. 过去的损伤或手术史？
4. 身体活动水平（类型、频率、强度、持续时间）？
5. 职业（日常身体负荷情况）？
6. 该部位问题是否限制了某些活动或功能？
7. 疼痛是否与其他部位的症状相关联？
8. 什么情况会加重或缓解症状？
9. 疼痛是否影响睡眠？（针对LPHC及以上部位）

### 2. 视诊 (Visual Assessment)

通过肉眼观察客户身体各部位的静态排列和对称性。

**通用原则：**
- 客户需穿着便于观察的服装（男性：短裤+背心；女性：短裤+运动内衣或背心）
- 脱掉鞋袜
- 从正面、侧面、后面多角度观察
- 检查肿胀、老茧、不对称、颜色变化、畸形等

### 3. 触诊 (Palpation)

通过手部接触验证视诊发现，感知骨性标志位置、软组织张力和关节运动质量。

**关键原则：**
- 必须先征得客户同意
- 从足部开始（威胁性最小）
- 触诊时持续与客户沟通感受
- 如客户感到疼痛，立即停止

## 从足到头评估流程

### 区域一：足部和踝部

| 检查项目 | 方法 | 正常/异常判断 |
|---------|------|-------------|
| 足弓高度 | 视诊内侧纵弓 | 足弓下降或无足弓 = 过度内旋 |
| 足部朝向 | 视诊双足位置 | 足外翻（鸭式）常见于过度内旋 |
| 大足趾 | 视诊第一跖趾关节 | 外翻、囊肿、老茧 = 过度内旋 |
| 小足趾 | 视诊形态 | 锤状趾、爪形趾 = 重量分布异常 |
| 距骨位置 | 触诊踝内外侧凹陷 | 拇指与食指压力不均 = 距骨偏移 |
| 小腿肌肉 | 触诊腓肠肌/比目鱼肌 | 扳机点、结节、过度紧张 |
| 足底筋膜 | 触诊足底从跟到趾 | 柔软或疼痛区域（跟骨前/大足趾后最明显） |

**主要失衡：过度内旋 (Overpronation)、背屈不足 (Dorsiflexion Insufficiency)**

**中立位指导：** 客户站立，评估者拇指食指置于踝内外侧凹陷，客户缓慢内外转动踝，直到两指感受到的压力均匀。此即距骨中立位置。

### 区域二：膝关节

| 检查项目 | 方法 | 正常/异常判断 |
|---------|------|-------------|
| 外观 | 视诊膝周围 | 肿胀、疤痕、肌肉不对称 |
| 静态髌骨位置 | 视诊髌骨中心-踝中线-股骨中线对齐 | 交点过度向内或向外 = 侧向对齐问题 |
| 动态髌骨位置 | 单腿浅蹲(约30°)视诊 | 膝过度靠近/远离中线 = 膝外翻/内翻 |
| 后方动态评估 | 后方视诊单腿下蹲时臀部 | "萨尔萨舞臀部" = 膝外翻代偿 |
| 髌骨轨迹 | 触诊屈伸膝时髌骨滑动 | 摩擦/嘎吱/爆裂声 = 运动轨迹问题 |

**主要失衡：侧向对齐问题/膝外翻 (Knee Valgus)、屈伸运动轨迹问题 (Patellar Tracking)**

**Q角参考：** 男性>15°、女性>18°视为异常（女性骨盆更宽，角度自然更大）。

**中立位指导：** 足踝中立位 + 臀肌收缩使大腿小腿外旋 + 骨盆后倾，髌骨中心对齐第二足趾。

### 区域三：腰椎—骨盆髋带 (LPHC)

| 检查项目 | 方法 | 正常/异常判断 |
|---------|------|-------------|
| 骨盆倾斜 | 视诊腰带前部 vs 后部高度 | 腰带前低后高 = 骨盆前倾 |
| 腰椎曲度 | 侧面视诊下背部曲线 | C形/反向C形大弧 = 腰椎过度前凸 |
| 靠墙触诊 | 站立靠墙，手掌滑入下背部与墙之间 | 整只手/臂可进入 = 腰椎过度前凸 |
| 仰卧评估 | 仰卧双腿伸直，手滑入下背部下方 | 所有手指/整只手可进入 = 腰椎过度前凸 |

**主要失衡：骨盆前倾 (Anterior Pelvic Tilt)、腰椎过度前凸 (Lumbar Hyperlordosis)**

> 骨盆前倾与腰椎过度前凸总是同时出现，因为腰椎和骨盆通过骶髂关节连接。

**中立位指导（手掌法）：** 双手手掌置于骨盆前部（髂前上棘），手指并拢与地面平行，食中指尖相触。低头看手，若看不到指尖，则骨盆后倾至能看到指尖为止。

### 区域四：胸椎和肩胛带

| 检查项目 | 方法 | 正常/异常判断 |
|---------|------|-------------|
| 胸椎后凸 | 一指置C7棘突，一指置胸骨切迹，比较高度差 | >3.8cm = 胸椎过度后凸 |
| 肩胛骨位置 | 视诊肩胛骨椎缘与脊柱距离 | 远离脊柱 = 肩胛带前伸 |
| 肩胛骨突出 | 触诊肩胛骨是否平贴胸腔 | 可轻易摸到边缘 = 肩胛带前伸 |
| 肩胛骨高度 | 触诊双侧肩胛骨上缘高度 | 不等高 = 高位肩胛 |
| 斜方肌张力 | 捏触斜方肌上部纤维 | 过紧、结节 = 高位肩胛可能 |
| 肩峰位置 | 视诊肩峰与耳屏的矢状面对齐 | 肩峰明显前于耳屏 = 肩胛带前伸 |
| 手臂位置 | 视诊站立时手部可见范围 | 可见全部手指/手背 = 内旋臂 |

> 正常排列时，C7棘突比胸骨切迹高约3.8cm。差值越大，后凸越严重。

**主要失衡：胸椎过度后凸 (Thoracic Hyperkyphosis)、肩胛带前伸 (Scapular Protraction)、内旋臂 (Internally Rotated Arms)、高位肩胛 (Elevated Scapula)**

**中立位指导：** 背靠墙站立，挺直胸椎至肩部触墙，收回肩胛骨，手臂外旋。注意避免耸肩或拱起下背部代偿。

### 区域五：颈部和头部

| 检查项目 | 方法 | 正常/异常判断 |
|---------|------|-------------|
| 头部前倾 | 颧骨最突出点与锁骨下方垂线比较 | 颧骨明显前于锁骨 = 头部前倾 |
| 颈椎前凸 | 靠墙站立，头部后部轻触墙后评估视线 | 视线向上倾斜 = 颈椎过度前凸 |

**主要失衡：头部前倾 (Forward Head Posture)、颈椎过度前凸 (Cervical Hyperlordosis)**

> 头部每前移约2.5cm（1英寸），头部对身体的相对重量加倍。

**中立位指导：** 靠墙站立，骨盆后倾至下背部与墙之间仅容两指关节，头部后部轻触墙，肩部位于耳屏下方，下背部不弯曲，颈部不拱起。

## 全身关联性总结

```
足过度内旋 → 小腿内旋 → 膝外翻 → 大腿内旋 → 髋臼后移
→ 骨盆前倾 → 腰椎过度前凸 → 重心前移
→ 胸椎过度后凸 → 肩胛带前伸 → 内旋臂/高位肩胛
→ 头部前倾 → 颈椎过度前凸
```

此传导链说明：**身体某一部位的失衡会沿运动链向上和向下传导，引起全身代偿**。

## 客户评估表 (CAD)

CAD 是记录评估结果的核心工具，包含：
- 每个身体部位的疼痛标记（在解剖图上用×标记）
- 问诊记录（关节炎、伤病史、活动水平、职业等）
- 视诊和触诊发现
- 客户是否知晓中立位置的记录

## 沟通策略

- 评估过程中持续向客户解释每个步骤和发现
- 使用简单易懂的语言（避免术语）
- 帮助客户理解各部位之间的关联
- 解释为何疼痛部位不一定是问题的根源
- 良好的沟通能提高客户对后续训练计划的依从性
