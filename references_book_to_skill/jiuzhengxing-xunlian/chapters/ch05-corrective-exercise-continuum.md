# 第5章 纠正性训练三步骤体系

## 核心理念

纠正性训练计划的三大步骤必须**严格按照顺序**执行：

```
自我肌筋膜放松 (SMR) → 拉伸训练 (Stretch) → 强化训练 (Strengthen)
   恢复与再生           增大活动范围         重塑运动模式
```

**不可颠倒顺序的原因：**
1. 先让受损结构恢复和再生，才能进行拉伸
2. 先恢复软组织的柔韧性和活动范围，才能安全负重强化
3. 跳跃步骤 → 代偿模式加重 → 训练无效甚至有害

## 三步骤详解

### 第一步：自我肌筋膜放松 (SMR)

**目的：** 恢复和再生受到慢性肌肉骨骼失衡影响的软组织

**"肌筋膜"词源：** myo=肌肉 + fascial=筋膜

**作用机制：**
- 对受限、紧张、僵硬、粘连、扳机点的肌筋膜区域施加持续压力
- 刺激局部血液循环，缓解血液流动不畅和毒素积聚
- 提高组织弹性和柔韧度
- 释放内啡肽减轻疼痛

**两种技术：**

| 类型 | 工具 | 适用范围 | 用途阶段 |
|------|------|---------|---------|
| 一般技术 | 泡沫轴（大表面积） | 大面积肌群 | 计划初期、热身/放松 |
| 特殊技术 | 网球/高尔夫球/按摩杖（小表面积） | 单块肌肉、精确扳机点 | 计划进阶、精确定位 |

### 第二步：拉伸训练 (Stretch)

**目的：** 提升肌肉和筋膜的柔韧性，增大关节活动范围，重新训练运动模式

**三种拉伸类型（由易到难）：**

| 类型 | 特征 | 适用范围 |
|------|------|---------|
| 被动拉伸 | 保持固定姿势，身体被动放松 | 拉伸训练初期，安全可控 |
| 主动拉伸 | 拮抗肌收缩促进目标肌放松（相互抑制） | 进阶阶段，神经肌肉协调 |
| 动态拉伸 | 多部位协调运动，整合模式 | 最后阶段，过渡到强化训练 |

**相互抑制原理：** 一块肌肉收缩时，其拮抗肌受到抑制而放松。例如：臀大肌收缩→髋屈肌抑制→髋部更有效地伸展。

### 第三步：强化训练 (Strengthen)

**目的：** 在负重或张力下挑战肌肉骨骼和神经肌肉系统，激活、协调并持续强化正确运动模式

**七种强化类型（由易到难）：**

| 类型 | 特征 | 适用阶段 |
|------|------|---------|
| 1. 等长 | 肌肉激活但长度不变，神经协调最容易 | 最初始，肌肉重新激活 |
| 2. 向心 | 肌肉缩短，起点止点靠近 | 早期，安全性高 |
| 3. 离心 | 肌肉拉长减速，弹力绳效应关键 | 需要良好控制力时 |
| 4. 单关节 | 仅移动一个关节 | 强化单个肌群 |
| 5. 多关节 | 涉及多个关节 | 肌群整合 |
| 6. 运动链 | 一系列连续相互依赖的动作 | 开链先进阶，闭链需要更多协调 |
| 7. 多平面多维度 | 多方向+不稳定表面等 | 模拟现实，最终目标 |

## 训练评估

选择每个训练前，需通过以下6个维度评估：

| 评估维度 | 核心问题 |
|---------|---------|
| **结构** | 训练针对哪些结构（肌肉、筋膜、关节、骨骼）？ |
| **失衡** | 训练改善哪些肌肉骨骼失衡问题？ |
| **力学** | 正确力学机制是什么？客户能正确执行吗？ |
| **实用** | 训练是否实用？（器材易得、场地可行、时间合理） |
| **水平** | 需进阶还是退阶以满足客户需求？ |
| **替代** | 是否有相似的替代训练可以提高客户依从性？ |

## 进退阶决策框架

### 自我肌筋膜放松 (SMR)

**何时退阶：**
1. 客户感到不舒服或身体滑落
2. 出现任何类型疼痛
3. 无法正确执行技术
4. 导致身体其他部位疼痛/不适
5. 没有合适的器材（或不愿购买）

**退阶方法：**
- 使用更软的工具（密度更低的泡沫轴/更软的球）
- 从站姿改为坐姿或卧姿
- 用手指按摩替代硬工具
- 加入电热垫加热后按摩

**何时进阶：**
1. 目标区域不再感到压迫感或敏感
2. 训练耗时过长（改为更大表面积工具）
3. 组织已达到期望的活动范围

**进阶方法：**
- 换更硬的工具
- 同时整合拉伸（如泡沫轴放松臀肌+拉膝到胸）
- 调整身体角度增加压力（如叠放双腿增加受力）
- 直接进阶到拉伸训练

### 拉伸训练

**何时退阶：**
1. 客户感到疼痛或不适
2. 动作太难，无法正确执行
3. 站立/跪姿无法保持平衡
4. 训练器材不实用

**退阶方法：**
- 使用平衡辅助工具
- 改站姿/跪姿为坐姿或卧姿
- 退回到SMR重新准备组织

**何时进阶：**
1. 可以正确完成但看不到益处
2. 已能熟练掌握，但训练过于耗时
3. 达到所需活动范围
4. 已不再进步

**进阶方法：**
- 加入拮抗肌主动收缩（被动→主动拉伸）
- 从坐姿/跪姿转到站姿
- 移除平衡辅助工具
- 结合多个身体部位（如加入躯干旋转 → 动态拉伸）
- 直接进阶到强化训练

### 强化训练

**何时退阶：**
1. 任何疼痛出现（退到无痛训练或SMR/拉伸）
2. 无法正确执行或协调动作
3. 第二天或当天严重疼痛
4. 客户表现出焦虑不适的非语言信号（呼吸速率增加、咬嘴唇等）

**退阶方法：**
- 使用平衡辅助工具（如另一只足轻触地面）
- 改站姿为卧姿
- 退回到SMR或拉伸训练重新准备

**何时进阶：**
1. 技术达标，能成功完成所需重复次数
2. 训练耗时过长（用复合训练替代多个单关节训练）
3. 客户身体和精神都准备好了参加特定活动/运动

**进阶方法：**
- 等长→向心单关节→离心多关节→运动链→多平面多维度
- 增加外部负荷（重量）
- 移除支撑/触觉反馈（如离开墙壁）
- 加快到现实生活中的速度
- 模拟运动专项动作

## 禁忌症汇总

### SMR 禁忌
- 身体肿胀/水肿部位
- 瘀血块上或附近
- 急性类风湿性关节炎
- 开放伤口
- 动脉瘤
- 皮肤过敏/荨麻疹/湿疹/皮疹
- 高血压（需谨慎，咨询医生）
- 身体过度紧张/痉挛部位（保守使用）
- 孕晚期避免长时间俯卧/仰卧
- 高烧
- 严重疾病或传染病
- 疝气或近期腹部手术
- 骨折处
- 骨质疏松症（保守，避免直接压力）
- 静脉曲张处（不直接施压）
- 某些癌症（先咨询医生）

### 拉伸训练禁忌
- 目标部位或周围剧烈疼痛
- 骨折部位或正在愈合的骨折
- 灵活性过大的关节（如脱臼/撕裂史）
- 近期急性损伤（撕裂/瘀伤）
- 急性炎症关节
- 类风湿性关节炎/骨质疏松症（谨慎）
- 抗凝药物使用史（谨慎）
- 类固醇药物使用史（谨慎）
- 受伤或手术后不久

### 强化训练禁忌
- 客户抱怨疼痛或不适
- 正在愈合的骨折部位
- 冠心病/高血压/类风湿性关节炎（需医生许可）
- 持续创伤的关节
- 近期急性损伤
- 急性炎症关节
- 骨质疏松症（谨慎保守）
- 受伤/手术后或慢性神经肌肉疾病（需医生许可）

## 训练变量设置原则

| 变量 | 指导原则 |
|------|---------|
| 组数/次数 | 正确执行 > 完成特定数量 |
| 频率 | 根据客户个人需求和能力调整 |
| 强度 | 循序渐进，避免过度疲劳 |
| 休息 | 确保每次训练之间有足够恢复时间 |

## 客户信心建设

1. **语言坦率真诚：** 避免技术术语（"自我肌筋膜放松" → "自我按摩"）
2. **以简单训练开始：** 客户容易成功的训练能增强信心
3. **循序渐进：** 心理和生理层面同步进阶
4. **观察焦虑信号：** 困惑、过于热情/沉默、不提问 = 可能不理解或缺乏信心
5. **倾听回应：** 花时间倾听顾虑，诚实回答问题
6. **建立联系：** 解释每个训练与评估结果的具体关系
