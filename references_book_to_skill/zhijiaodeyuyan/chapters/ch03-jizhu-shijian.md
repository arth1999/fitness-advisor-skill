# 第3章 记住时间

## 核心内容

记忆是学习的终点站。教练要植入三类记忆：运动知识（规则/战术）、运动技能知识（技术信息）、运动技能执行知识（实际运动模式）。理解了记忆如何编码、巩固和提取，教练才能让语言转化为持久的动作改变。

## 工作记忆（短期记忆）

巴德利的多成分模型：语音循环系统（语言/听觉）+视觉空间模板系统（视觉/空间/运动）+情节缓冲区（绑定各系统+长期记忆）+中央执行系统（指挥协调）。

### 关键发现
- 米勒(1956)：口头记忆广度 7±2 个记忆单位
- 现代共识：4±2 个记忆单位，但组块化后可无限扩展
- 记忆游戏证明：嵌入故事情节中的信息（有语义关联）比随机词语列表的记忆率高得多
- **信息的框架决定大脑记忆的持久性**

### 记忆宫殿（轨迹法）
将新信息放在已知空间情景中——利用情节缓冲区将新信息与长期记忆绑定。教练应用：用类比/隐喻，将技术概念嵌入运动员熟悉的场景（如"像坐在不稳定的公园长椅上一样下蹲"优于"髋部向后伸至脚跟之后"）。

### 工作记忆的教练启示
- 过度指导/过度提示增加运动员工作记忆负担 → 分析性瘫痪
- 精英教练在运动员执行动作前仅用8.9%的时间提供指导，但整体过度依赖指导仍是执教文化中最常见的现象
- 视觉空间工作记忆与注意力共同预测运动学习效果

## 长期记忆

### 多重记忆系统

| 类型 | 子类别 | 功能 | 运动示例 |
|------|--------|------|----------|
| **外显记忆**（有意识：知道什么） | 情节记忆 | 个人经历的时间旅行 | 回忆上次比赛中的关键进球 |
| | 语义记忆 | 事实/想法/概念知识 | 知道比赛规则和战术 |
| **内隐记忆**（无意识：做什么） | 触发效应 | 之前接触加快后续知觉 | 看过对手录像后更快识别其偏好 |
| | 程序性记忆 | 复杂运动技能的自动执行 | 走路、骑车、挥杆的自动模式 |

> 亨利·莫莱森的案例：海马体切除后无法形成新外显记忆，但仍能学习新运动技能——证明外显和内隐记忆是分离的系统。

## 记忆形成三阶段

### 1. 编码
- 将当前关注的信息与已有记忆中的信息关联
- 海马体 → 整合和提炼感觉信息（"什么"路径+ "在哪里"路径）→ 新记忆产生
- **一起被激活的神经元会连接在一起（赫布理论）**

### 计划回路 vs 动作回路

| 回路 | 核心结构 | 功能 | 功能障碍示例 |
|------|----------|------|-------------|
| 计划回路 | 运动皮层+基底神经节（多巴胺调节） | 引导运动意图/计划 | 帕金森病：自发运动困难 |
| 动作回路 | 初级运动皮层+小脑 | 将意图转化为动作+实时调整 | 小脑损伤：运动时序失调 |

### 2. 巩固
- **化学巩固**：杏仁核调节去甲肾上腺素 → 情感价值高的信息被优先巩固。教练应使用情感价值高的示例和积极语言
- **结构巩固**：记忆从海马体转移到大脑皮层，不再依赖海马体。程序性记忆在基底神经节纹状体稳定，小脑参与反应运动的长期巩固

### 3. 提取
- **编码特异性**：编码方式决定提取条件。语言提示进入运动皮层——"踢"字激活腿部运动区域
- **环境特异性**：学习环境影响记忆提取。在比赛环境中训练的运动员更能适应比赛场景
- **情绪特异性**：积极情绪下更容易提取积极记忆。教练应使用积极语言编码提示（"集中精力蹬地"优于"你没有充分伸展髋部"）

## 应用要点

1. 精简信息：一环节一提示，避免超出工作记忆
2. 使用类比/隐喻激活记忆宫殿——将新信息嵌入已知场景
3. 用情感价值高的语言和示例传递关键提示——杏仁核参与记忆巩固
4. 积极语言编码：描述"要做什么"而非"不要做什么"
5. 训练环境尽可能模拟比赛环境——保证环境特异性
6. 动词在运动前传递（激活运动系统），运动中保持安静
