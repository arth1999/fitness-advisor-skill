# 第10章 速度型动作提示

## 核心内容

速度型动作包括直线速度（起跑→加速→绝对速度）和多方向速度（切步、变向、转身冲刺、后退步）。速度训练中提示的特殊挑战：动作速度快、决策时间短，提示必须在极短时间内传递清晰信息。节奏提示（在运动中发声）在此尤为重要。

## 直线速度 — 提示序列

### 三点式起跑

| 3D要素 | 示例提示 |
|--------|----------|
| 基本3D提示 | "像被猛击后背一样冲出起跑线" |
| 精简提示 | "炸出！" |
| 类比（场景） | "像被人拿枪追赶一样——不顾一切地冲出去" |
| 姿势提示 | "让背部的两条胶带靠在一起"（脊柱中立） |
| 姿势提示 | "把鞋带向上拉——脚踝保持勾脚尖姿势" |

### 加速阶段（0-9米）

| 3D要素 | 示例提示 |
|--------|----------|
| 基本3D提示 | "用力将地面推向身后的墙壁" |
| 精简提示 | "推地" |
| 远距离外部 | "冲出赛道——朝终点线爆发" |
| 类比（物体） | "像斧头砍树一样撞击地面" |
| 类比（场景） | "像被人追赶着跑上一座陡山" |
| 类比（物体） | "像汽车轮胎抓住地面——每次蹬地都有推进力" |
| 方向调整 | "推入地面"（加大蹬地力量）vs "离开地面"（缩短触地时间） |
| 节奏提示 | 前3步"加油-加油-加油" |

### 绝对速度阶段（18+米）

| 3D要素 | 示例提示 |
|--------|----------|
| 基本3D提示 | "每一步都要蹬离地面——像火柴划过火柴盒一样划过地面" |
| 类比（约束） | "每一步的最后都像把一杯茶放在膝盖上保持平衡" |
| 类比（物体） | "像喷气式飞机起飞——加速并逐渐抬升" |
| 精简提示 | "弹-弹-弹" |
| 姿势提示 | "顶天立地——从头顶到脚底保持修长笔直" |
| 节奏提示 | "轻——快——轻——快"（代表触地轻+频率快） |

## 速度型提示的通用原则

### 节奏训练法（运动中唯一允许的教练发声）

| 动作类型 | 节奏提示形式 | 示例 |
|----------|-------------|------|
| 起跑 | 单音爆发 | "砰！""炸！" |
| 加速 | 连续重音 | "蹬-蹬-蹬" 或 "加油-加油-加油" |
| 绝对速度 | 交替轻快 | "轻-快-轻-快" |
| 变向 | 阶段标记 | "刹——转——冲！" |
| 跳跃 | 重音+拖长 | "下——爆——！" |

### 力学调节的动词策略
- 需要**加大蹬地力量** → "推入地面"
- 需要**缩短触地时间** → "离开地面" 或 "弹离地面"
- 需要**加大步幅** → "蹬离地面，向前飞出"
- 需要**加快步频** → "像热锅上的蚂蚁一样快速触地弹回"

### 速度的注意力焦点转移

| 运动员水平 | 推荐焦点 | 原因 |
|-----------|----------|------|
| 新手 | 动作形态+外部参照（"每步步幅一致""膝盖推向终点"） | 需要建立基本运动模式 |
| 中级 | 外部结果（"冲出跑道""把地面向后推"） | 模式已建立，强化输出 |
| 高手 | 远距离目标/结果（"冲过终点线""最快的总时间"） | 自动化完成，关注结果 |

### 胶带在速度训练中的应用
- 膝盖贴胶带 → "把胶带推向终点线"（加强前摆）
- 鞋带贴胶带 → "把胶带贴到云端"（脚踝背屈）
- 背部上下各一条 → "让两条胶带靠在一起"（起跑时脊柱中立）

## 应用要点

1. 速度训练中节奏提示是唯一在运动员运动时发声的时机——使用单字或拟声词
2. 起跑和加速 → 力量导向提示（"推/蹬/炸"）；绝对速度 → 频率和轻快导向（"弹/轻/快"）
3. 变向动作 → 清晰标记每个阶段的动词（急刹→蹬离→加速）
4. 后退步 → 姿势类比（"头顶一碗汤"）是最有效的稳定策略
5. 视觉目标（锥桶、终点线、标志物）在速度训练中尤其重要——运动员本能地会追逐可见目标
