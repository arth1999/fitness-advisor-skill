# 第8章 力量型动作提示

## 核心内容

本章提供了27个动作序列中的第1组——力量型动作的3D提示示例。核心理念：提示不是告诉你该做什么，而是帮助运动员**感觉**该做什么。

## 力量型动作的提示特点

力量型动作通常为多关节复合运动，需要协调的肌肉发力模式。提示需同时传达：
- **姿势稳定性**（近端固定/躯干控制）
- **力量产生**（发力方向和速度）
- **节奏控制**（离心-向心转换的时序）

## 力量型动作提示的通用原则

1. **起始姿势** → 使用衣物参照（拉链、腰带、鞋带）确保身体对齐
2. **离心阶段** → 用"触"类动词（轻触、控制、放慢）+ 外部参照物（箱子、地面、墙壁）
3. **向心阶段** → 用爆发类动词（推、蹬、击、打）+ 方向指示
4. **整个过程** → 一个环节一个提示；在描述中解释技术，在提示中瞄准结果
5. **精简节奏词** → 双音节头韵/尾韵可同时标记动作的两个阶段（如"触-蹬"、"下-上"）
