# 第7章 路线图

## 核心内容

掌握了前面6章的理论和工具还不够——真正的挑战在于将新的执教语言变成**习惯**。本章提供了从"我将要改变"到"我就是这样执教的"的行为改变路线图。

## 为什么提示改变很难

### 习惯的本质
- 习惯 = 曾经经过深思熟虑的选择，现在变成无思考的自主行为
- 习惯形成的神经网络与运动技能学习的程序性记忆共享相同机制
- 一旦习惯形成，它深深隐藏在潜意识中——直到我们决定挖掘、审视、完善它

### 两个教练的故事
| 教练A | 教练B |
|-------|-------|
| 学完课程后热情尝试 | 学完课程后热情尝试 |
| 几周后未能设计出合适提示 | 也感到吃力和不适 |
| 回到熟悉的内部提示 | 坚持——因为知道收益>成本 |
| 约一个月后完全恢复旧习惯 | 几个月后提示已能无需思考脱口而出 |

**差异的原因**：教练B有系统的方法支撑行为改变，而非仅靠一时的热情。

## 习惯循环模型（克利尔+杜希格）

### 四阶段循环

```
提示(Cue) → 渴望(Craving) → 反应(Response) → 回报(Reward)
   ↑                                                  |
   └──────────────────────────────────────────────────┘
```

| 阶段 | 含义 | 提示习惯中的应用 |
|------|------|-----------------|
| **提示** | 触发行为的信号 | 看到某个特定动作/错误 → "设计外部提示的时刻到了" |
| **渴望** | 行为背后的驱动力 | 希望看到运动员眼睛发亮、立即理解并改进动作时的满足感 |
| **反应** | 将习惯付诸行动 | 在大脑中搜索3D要素 → 说出精心设计的外部提示/类比 |
| **回报** | 满足欲望的结果 | 运动员动作改变；运动员说"我明白了！"；表现提升 |

> 四阶段缺一不可。缺少提示 → 习惯不启动；渴望减弱 → 无动力行动；行为困难 → 无法实施；回报不足 → 不会重复。

## 提示习惯养成3步法

### 第1步：选择锚点（建立提示）
- 在现有训练流程中找到一个固定的"触发点"
- 示例：每次热身结束后的第一个正式训练动作 = 提示设计时刻
- 示例：每次看到运动员走神 → 四类显著性触发器 → 吸引注意力
- 不要让习惯依赖记忆——用环境设计让触发点不可忽略

### 第2步：刻意实践（反应+渴望）
- 按DDCDD循环系统地使用外部提示/3D模型/类比
- 前2-4周：有意识地、费力地每次停下思考："这个提示是外部还是内部？3个D都包含了吗？"
- 2-8周：越来越自然——不再需要刻意检查
- 允许试错——最初失败的提示是收集运动员语言偏好的机会
- 从运动员的反馈（语言+表情+动作）中学习

### 第3步：迭代优化（回报强化）
- 训练结束后记录：哪个提示有效？为什么？运动员的反应是什么？
- 整理个人的"提示语言库"——按动作分类，积累可用的外部提示和类比
- 无效提示 → 使用3D模型调整（改变距离/方向/描述）
- 有效提示 → 精简、升级（头韵/尾韵）、归档
- 定期回顾自己的提示语言库，淘汰僵化的内部提示

### 时间框架参考
| 阶段 | 时间 | 重点 |
|------|------|------|
| 初期 | 第1-2周 | 感到吃力和不适——正常！有意识地使用外部提示 |
| 中期 | 第3-8周 | 提示设计越来越快，外部提示比例显著增加 |
| 自动化 | 第2-3个月 | 外部提示已成为默认，无需思考即可脱口而出 |
| 精进 | 持续 | 不断更新类比库，适应新运动员和新世代 |

## 克服常见障碍

### 障碍1："关键时刻想不到外部提示"
- **应对**：提前准备——训练前花5分钟预览今天要教的1-3个动作，预写3D提示
- **应对**：使用提示语言库——训练间隙快速查阅

### 障碍2："运动员对我的新提示没反应"
- **应对**：回3D模型——改变一个D要素（先换动词）
- **应对**：升级为类比——可能是文字提示不够生动

### 障碍3："老运动员习惯了内部提示"
- **应对**：渐进过渡——在描述动作时保留内部提示的熟悉感，在提示动作时替换为外部
- **应对**：解释"为什么"——告诉运动员外部专注研究结果，让他们成为合作伙伴

### 障碍4："回到旧习惯的诱惑"
- **应对**：让运动员成为责任伙伴——告诉他们"如果我又开始说'弯曲膝盖'，提醒我"
- **应对**：环境设计——在训练区贴3D提示语模型图/外部提示清单

## 应用要点

1. 改变执教语言就是培养新习惯——需要系统方法而非仅仅意愿
2. 利用习惯循环：设计明确的提示信号 → 培养对运动员"顿悟时刻"的渴望 → 通过3D模型简化反应 → 记录运动员改变作为回报
3. 从"有意识不舒服"（前2-4周）→"越来越自然"（1-3个月）→"我就是这样执教的"（自动化）
4. 建立个人提示语言库——这是积累型资产，随时间增值
5. 教练的成长路径：从模仿他人的提示 → 设计自己的提示 → 即时创造个性化提示
