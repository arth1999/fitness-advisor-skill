# 筋膜与解剖列车术语词汇表

## 核心概念

| 术语（中文）        | 术语（英文）              | 定义                                                                                  |
| ------------------- | ------------------------- | ------------------------------------------------------------------------------------- |
| 肌筋膜              | Myofascia                 | 肌肉组织（myo-）和伴随结缔组织网（fascia）的不可分割整体。作者使用单数形式强调其单一性 |
| 肌筋膜经线          | Myofascial Meridian       | 一连串肌腱与肌肉的连接线，传递张力与弹力，协助运动并提供稳定性                         |
| 肌筋膜连接          | Myofascial Continuity     | 结构网中连接两个纵向毗邻区域的部分。例如：前锯肌与腹外斜肌之间的筋膜连接               |
| 解剖列车            | Anatomy Trains            | 对整个肌筋膜经线系统的比喻性称呼，一个描述性词汇                                       |
| 张拉整体结构        | Tensegrity                | 骨骼（受压元件）漂浮在肌筋膜（受拉元件）构成的张力网络中                               |
| 纤维体              | Fibrous Body              | 整个胶原网组成的系统，包含骨骼、软骨、肌腱、韧带、皮肤中的胶原和肌筋膜群               |

## 轨道术语

| 术语（中文）| 术语（英文）    | 定义                                                                      |
| ----------- | --------------- | ------------------------------------------------------------------------- |
| 轨道        | Track           | 肌筋膜经线中一段连续的筋膜结构                                             |
| 车站        | Station         | 肌筋膜轨道附着于骨骼的位置——骨性标志，拉力在此作用于骨骼                      |
| 特快列车    | Express Train    | 跨越多个关节的浅层肌筋膜单元。例：竖脊肌（跨越脊柱全长）、股二头肌长头（髋+膝）|
| 普通列车    | Local Train      | 单关节的深层肌筋膜单元，在特快列车下方提供细调。例：股二头肌短头、横突间肌    |
| 脱轨        | Switch           | 解剖列车规则的例外——只在特定姿势条件下经线才连续。例：SBL在膝伸直时连续       |
| 机车库      | Roundhouse       | 多条经线力学矢量交汇和/或交叉的骨性标志。例：髂前上棘、耻骨、肩胛骨          |
| 道岔        | Crossover        | 两条经线之间的交叉连接，提供额外的灵活性和稳定性（多见于臂线）                |

## 姿势与代偿术语

| 术语（中文）| 术语（英文）          | 定义                                                                                  |
| ----------- | --------------------- | ------------------------------------------------------------------------------------- |
| 闭锁缩短    | Locked Short          | 组织长期处于向心性收缩状态，肌筋膜张力过高且长度缩短                                     |
| 闭锁延长    | Locked Long           | 组织长期处于离心负荷状态，被过度牵拉——张力高但长度过长                                      |
| 倾斜        | Tilt                  | 偏离垂直线或水平线——结构的一边比另一边高。如"骨盆前倾"、"头右倾"                          |
| 弯曲        | Bend                  | 一连串的倾斜所形成的曲线，通常用于脊柱。如"腰椎侧弯"                                      |
| 旋转        | Rotate/Rotation       | 在水平面上绕垂直轴出现，以结构前端指向方向命名。如"骨盆左旋"、"股骨内旋"                   |
| 偏移        | Shift                 | 身体部位重心的位移（左-右/前-后/上-下）。如"胸腔相对于骨盆后移"                             |
| 点头        | Nutation              | 骶骨在髂骨之间向前下倾斜（骶骨岬向前下移动）                                              |
| 反点头      | Counternutation       | 骶骨在髂骨之间向后上倾斜（骶骨岬向后上移动）                                              |

## 功能概念

| 术语            | 英文                        | 说明                                                       |
| --------------- | --------------------------- | ---------------------------------------------------------- |
| 纵向解剖        | Longitudinal Anatomy        | 整体概括肌肉组织内长的、可延展的带与吊索的系统性观点         |
| 肌肉孤立论      | Muscle Isolation Theory     | 仅通过肌肉起点与止点定义单块肌肉功能的标准分析方法           |
| 筋膜健康        | Fascial Fitness             | Schleip提出的筋膜训练概念——关注筋膜特性和反应的训练方式      |
| 伸展-收缩循环   | Stretch-Shortening Cycle    | 筋膜/肌肉在收缩前通过反向动作预先拉长以储存和释放势能        |
| 螺旋网格模式    | Lattice/Grid Pattern        | 适当负荷下筋膜内形成的规则螺旋网格胶原排列                   |
| 卷曲度          | Crimp                      | 筋膜内分子层面的波浪结构，提供初始弹性+帮助GTO感知负荷        |
| 动觉能力        | Kinesthetic Literacy        | 精确感知身体在空间中的位置和动作方式的能力                    |
| 近端启动远端延迟| Proximal Initiation          | 力量从中心向末端的顺序传递——长运动链的标志性协调模式           |
| 惊恐反应        | Startle Response            | 跨文化的SFL突然收缩——保护脊柱和身体前侧所有敏感部分             |
| 协调的筋膜张拉整体结构| Coordinated Fascial Tensegrity | 经线间均衡的弹性稳定正中状态                               |
