## 经线间拮抗关系速查

| 拮抗对            | 失衡表现                                         | 评估要点                             |
| ----------------- | ------------------------------------------------ | ------------------------------------ |
| SBL ↔ SFL         | "军姿"、头前移、骨盆前移、惊恐反应               | 矢状面侧面观；前屈测试；前表线是否被向下拉 |
| LL左 ↔ LL右       | 脊柱侧弯、肩髋倾斜、一侧躯干短缩                 | 冠状面前后观；双手悬吊观察；躯干"X"评估  |
| DFL ↔ LL          | X型腿/O型腿、足旋前/旋后、膝轨迹异常              | 膝内外翻观察；足弓形态；内收/外展肌群张力 |
| SPL左 ↔ SPL右     | 旋转不对称、肩部高低、肋弓至对侧ASIS距离差异       | 四个警示信号；旋转测试；胸骨VS耻骨方向    |
| DFAL ↔ DBAL       | 圆肩、肩胛骨位置异常、胸小肌短缩                  | 肩胛骨"X"评估；手臂线拉伸测试            |
| FFL ↔ BFL         | 投掷/击球动作不平衡、肩袖损伤                     | 俯卧抬臂+对侧腿；仰卧对侧肢带协调         |

## 筋膜训练核心原则速查

| 原则               | 关键要点                                                   | 时间/频率参考                    |
| ------------------ | ---------------------------------------------------------- | -------------------------------- |
| 适当负荷重组结构   | 规律负荷→螺旋网格；不活动→"毡子"状                          | 每周3-5次                        |
| 训练长运动链       | 孤立训练不发展筋膜功能；多角度训练不同负荷方向               | 每次训练使用不同角度               |
| 温柔坚持           | 筋膜重建比肌肉慢；大强度后24h胶原减少→72h恢复               | 间隔至少48小时再大强度训练         |
| 利用筋膜弹性       | 伸展-收缩循环；袋鼠腱等弹性结构                              | 运动中应用预拉长-收缩循环           |
| 关注本体感觉       | 筋膜感受器是肌肉的6倍；精细感觉训练延长运动寿命               | 每次训练关注内在感觉                |

## 最快分析流程

1. **侧面观**（矢状面平衡）：SBL ↔ SFL关系 → 骨盆倾斜 → 头位 → 膝
2. **后面观/前面观**（冠状面平衡）：LL左右 → 肩髋倾斜 → 脊柱弯曲 → 膝内外翻
3. **前屈测试**：SBL连续性和各节段活动度
4. **旋转评估**：SPL对称性（四个警示信号）
5. **足弓和步态**：DFL+下SPL支撑功能
6. **呼吸模式**：DFL中段（横膈）+SFL（腹直肌深层）

> **提示**：此理论是标准解剖学的补充视角，不可替代对单块肌肉功能的分析。应结合使用。
