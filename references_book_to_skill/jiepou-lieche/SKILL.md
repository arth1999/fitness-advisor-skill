# 解剖列车——筋膜补充视角

## 核心框架

**解剖列车**（Anatomy Trains）是 Thomas W. Myers 提出的肌筋膜经线理论，将人体视为由筋膜网络连接的张拉整体结构。其核心是 **12条肌筋膜经线（myofascial meridians）**——一连串肌筋膜连接构成的纵向拉力线。

### 理论定位

- **补充视角**：此理论补充（而非替代）传统的肌肉孤立论分析
- **基于标准解剖学**：所有结构均可在标准解剖学中验证
- **临床导向**：为徒手治疗师和动作教练提供整体评估框架
- **筋膜是整体**：全身肌筋膜是一个不可分割的整体结构（作者使用单数 myofascia）

### 核心概念

| 概念               | 定义                                                                                            |
| ------------------ | ----------------------------------------------------------------------------------------------- |
| 肌筋膜经线         | 一连串肌腱与肌肉的连接线，传递张力与弹力                                                       |
| 肌筋膜连接         | 结构网中连接两个纵向毗邻区的部分（如：前锯肌与腹外斜肌之间的筋膜连接）                         |
| 轨道（Track）      | 肌筋膜经线中一段连续的筋膜结构                                                                  |
| 车站（Station）    | 轨道附着于骨骼的位置——骨性标志                                                                  |
| 特快列车（Express）| 跨越多个关节的浅层肌筋膜单元                                                                    |
| 普通列车（Local）  | 单关节的深层肌筋膜单元，在特快列车下方提供细调                                                 |
| 脱轨（Switch）     | 解剖列车规则中的例外——特定条件下经线才连续                                                     |
| 机车库（Roundhouse）| 多条经线力学矢量交汇的骨性标志（如：髂前上棘、耻骨）                                          |
| 张拉整体结构       | 骨骼（受压元件）漂浮在肌筋膜（受拉元件）构成的张力网中                                         |
| 闭锁缩短/闭锁延长  | 姿势性代偿模式：某些肌筋膜长期处于向心收缩（缩短）或离心负荷（延长）状态                       |

## 12条肌筋膜经线概览

### 主线（Cardinal Lines）——维持直立姿势

| 经线                          | 缩写 | 位置         | 主要功能                           |
| ----------------------------- | ---- | ------------ | ---------------------------------- |
| 后表线（Superficial Back Line）| SBL  | 身体后表面   | 维持直立伸展，防止向前蜷曲         |
| 前表线（Superficial Front Line）| SFL | 身体前表面   | 维持躯干屈曲平衡，保护前侧敏感结构 |
| 体侧线（Lateral Line）         | LL   | 身体两侧     | 调整前后与左右平衡，侧屈与旋转制动  |
| 螺旋线（Spiral Line）          | SPL  | 环绕身体     | 调整扭转与旋转，稳定躯干和下肢      |

### 臂线（Arm Lines）——上肢功能

| 经线                              | 缩写  | 位置             | 主要功能                   |
| --------------------------------- | ----- | ---------------- | -------------------------- |
| 臂前表线（Superficial Front Arm Line）| SFAL | 手臂前侧，浅层   | 抓握、内收、大范围推拉动作 |
| 臂前深线（Deep Front Arm Line）       | DFAL | 手臂前侧，深层   | 拇指侧稳定性与控制         |
| 臂后表线（Superficial Back Arm Line） | SBAL | 手臂后侧，浅层   | 外展、伸展、拉力传递       |
| 臂后深线（Deep Back Arm Line）        | DBAL | 手臂后侧，深层   | 小指侧稳定性与精细调节     |

### 功能线（Functional Lines）——跨躯干运动链

| 经线                              | 缩写 | 走向                         | 主要功能                       |
| --------------------------------- | ---- | ---------------------------- | ------------------------------ |
| 前功能线（Front Functional Line）  | FFL  | 肱骨→胸→对侧耻骨→对侧大腿   | 增加对侧推力，如投掷           |
| 后功能线（Back Functional Line）   | BFL  | 肱骨→背→对侧骶骨→对侧大腿   | 增加对侧拉力与运动减速         |
| 同侧功能线（Ipsilateral Functional Line）| IFL | 背阔肌外缘→同侧髂前上棘→膝内侧 | 同侧稳定，如单杠悬吊时扭转 |

### 核心线（Core Line）

| 经线                      | 缩写 | 位置                         | 主要功能                                       |
| ------------------------- | ---- | ---------------------------- | ---------------------------------------------- |
| 前深线（Deep Front Line）  | DFL  | 身体最深层的核心             | 支撑内在弧度、稳定核心、呼吸-行走耦合、内脏支撑 |

## 使用方法

### 姿势评估中的经线思维

当观察到一个姿势偏差时，追踪该偏差可能涉及哪些经线，沿着经线寻找"沉默"的短缩或延长区域。

### 治疗中的经线思维

不要只治疗疼痛部位，沿经线向远端探索可能的张力来源。先松解外周代偿模式，再处理核心问题。

### 动作训练中的经线思维

训练长运动链而非孤立肌肉。训练整条经线的协调性，使"近端启动和远端延迟"的力学传导更加有效。

## 与其他知识体系的关系

- **标准解剖学**：此理论是补充视角，不可替代标准解剖学
- **针灸经络**：肌筋膜经线与针灸经络有部分重叠，但不是同一概念（见附录3）
- **肌肉链（chaines musculaires）**：Mézière/Busquet的肌肉链基于功能连接，解剖列车基于筋膜直接连接
- **筋膜悬带（myofascial slings）**：Vleeming/Lee的前后斜悬带与功能线相似
- **生物力学**：补充生物力学，将相互关系论引入传统力学分析
