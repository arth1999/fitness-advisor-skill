# 第8章：功能线（Functional Lines）

## 概述

功能线从臂线开始，跨过躯干表面延伸到对侧骨盆和下肢（或反向）。左右两条线跨过躯干呈"X"型，几乎不参与调控静止站姿——主要在体育运动或其他活动中借助对侧力量补充稳定性和平衡，或增加推力。

## 三条功能线

### 后功能线（Back Functional Line, BFL）

| 车站             | 轨道                           |
| ---------------- | ------------------------------ |
| 肱骨             | 背阔肌（远端起）               |
| 腰骶结合部（跨中线）| 腰骶筋膜→骶筋膜（丛层）       |
| 骶骨             | 臀大肌（下部纤维）              |
| 股骨干（下1/3）  | 股外侧肌                      |
| 髌骨             | 髌下韧带                      |
| 胫骨粗隆         | （可继续下行至胫前肌→内侧足弓）|

### 前功能线（Front Functional Line, FFL）

| 车站                 | 轨道                       |
| -------------------- | -------------------------- |
| 肱骨                 | 胸大肌（下缘纤维）          |
| 第5/6肋软骨          | 腹直肌外鞘/半月线          |
| 耻骨结节/耻骨联合    | 跨耻骨（纤维软骨连接）      |
| 股骨粗线             | 对侧长收肌                 |

### 同侧功能线（Ipsilateral Functional Line, IFL）

| 车站                 | 轨道                       |
| -------------------- | -------------------------- |
| 肱骨                 | 背阔肌（外侧缘）            |
| 第10-12肋外侧        | 腹外斜肌（后部纤维）        |
| 髂前上棘             | 缝匠肌                     |
| 胫骨内侧髁/鹅足      | （悬吊或扭转动作时使用）    |

## 功能

- **后功能线**：从背阔肌→对侧臀大肌→股外侧肌→胫骨粗隆。在投掷动作后段发挥减速（刹车）作用，防止关节损伤
- **前功能线**：从胸大肌→腹直肌→对侧长收肌。为肢体提供附加速度和驱动力，如投标枪、凌空截击网球
- **同侧功能线**：在吊环运动、高空杂技的吊丝带、自由泳划水时稳定躯干

## 关键特征

- 功能线很少发挥调控静止站姿的作用
- 在运动中主要借助对侧力量补充而发挥稳定和平衡功能
- 在跨栏等动作中将对侧肩部力量增加至跨栏腿
- 瞬间动作会摇动地穿过不同动作单位，形成与螺旋线和体侧线的多种连结组合
- 腹外斜肌被三条经线共用：体侧线（外侧纤维→同侧髋）、功能线（中部纤维→耻骨→对侧）、螺旋线（上部纤维→对侧腹内斜肌→对侧髋）

## 常见姿势代偿

功能线与静止站姿关系较小，但当出现明显的旋转倾向（特别是与惯用手或特定动作相关）时，功能线可能会参与到限制中：
- 一侧肩反复向对侧髋拉近的动作模式
- 惯用手侧的旋转倾向

## 运动中的功能线应用

### 投掷项目（标枪、棒球）
- 向后引臂时：BFL缩短，FFL延长
- 投出瞬间：FFL缩短，BFL延长
- 动作末尾：BFL扮演刹车角色，拉住FFL和手臂以免过度前甩
- 长期康复需增强BFL的肌力和精确时间控制

### 网球
- 正面上手发球：FFL快速推动力（胸大肌+腹肌+长收肌防止耻骨上提）
- 正手击球：臂前表线→同侧胸肌→对侧胸肌→对侧臂前表线
- 反手击球：一侧背阔肌→对侧背阔肌+向下至骨盆

### 皮划艇
- 划桨手臂→臂后深线→BFL→稳定到对侧下肢
- 推力从一只脚传到另一只脚，模拟步行

## 功能线与其他经线的关系

- 功能线呈螺旋形分布，总是以螺旋模式起作用
- 可视为螺旋线的补充线，或臂线在躯干的延续
- 与螺旋线的相似性：都是螺旋环绕身体的模式
- 关键区别：功能线是跨躯干的臂到对侧腿的力传送，螺旋线是全身的旋转调控

## 功能线训练原则

- 俯卧同时抬高一侧手臂+对侧下肢→BFL协调训练
- 仰卧协调对侧肢带肌→FFL整体训练
- 瑜伽三角式/反三角式→BFL延展
- 跪姿向上向后伸手臂旋转→FFL拉伸
- 训练关键是使两条互补路线（如BFL和FFL）之间的关系从对抗变为协调
