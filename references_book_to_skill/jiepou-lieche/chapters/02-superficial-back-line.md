# 第3章：后表线（Superficial Back Line, SBL）

## 概述

后表线连接并保护整个身体的后表面，像一个从脚底到头顶的盔甲，可分为脚趾到膝盖以及膝盖到额头两部分。站立膝盖伸直时，后表线成为一条连续的肌筋膜线路。

## 轨道与车站

| 车站（骨性标志）        | 轨道（肌筋膜结构）             |
| ----------------------- | ------------------------------ |
| 趾骨跖面                | 足底筋膜（含5条腱束+外侧束）  |
| 跟骨                    | 跟腱/腓肠肌                   |
| 股骨髁                  | 腘绳肌（股二头肌、半腱肌、半膜肌）|
| 坐骨结节                | 骶结节韧带                    |
| 骶骨                    | 腰骶筋膜/竖脊肌               |
| 枕骨嵴                  | 帽状腱膜/头皮筋膜             |
| 眉弓/眶上嵴             | 枕肌-额肌（终点）             |

## 功能

### 姿势功能
- 在完全直立伸展状态下支撑身体，避免身体像胎儿般蜷缩屈曲
- 长期保持姿势需要慢收缩、耐力型肌纤维和加厚的筋膜束带（跟腱、腘绳肌肌腱、骶结节韧带、胸腰筋膜、竖脊肌条索、后头脊）

### 动作功能
- 除膝关节被后表线肌肉牵拉向后以外，所有运动功能都是产生伸展与过度伸展
- 限制向前屈曲的动作（矢状面）
- 功能障碍时强化和维持过度后伸

### 发育功能
- 在发育过程中，后表线的缩短促使婴儿从胎儿屈曲状态转变为直立姿势
- 后表线进一步缩短产生过伸动作

## 常见姿势代偿模式

- 踝背屈受限
- 膝关节过伸
- 腘绳肌缩短（以弥补深层旋转肌功能不足）
- 骨盆前移
- 骶骨下垂（点头/nutation）
- 脊柱前凸
- 胸椎前屈时伸肌扩大
- 枕骨下方受限导致上段颈椎过伸
- 枕骨在寰椎上向前移位或旋转
- 眼-脊运动不连续

## 关键临床要点

### 脱轨点
当膝盖弯曲时，后表线上部和下部相对分离——比较容易在髋关节屈曲。膝盖伸直时后表线连接为一个整体，前屈动作可能变得困难。

### 足底筋膜-跟腱连接
后表线下段如同弓弦：足底筋膜和跟腱相关筋膜是弓弦，足跟是箭头。后表线长期过度紧张时可将跟骨向前推进距下关节，或张力牵拉胫腓骨复合体向后。

### 骶结节韧带连接
腘绳肌与骶结节韧带之间有清晰的纤维性筋膜连接，这是后表线连续的核心环节。臀大肌不属于后表线（方向与层次不同）。

### 枕下肌群
位于中央的枕下肌群（头直肌、头斜肌）虽小，却是后表线的功能核心——是眼球和脊椎间动作的媒介。这些肌肉容易导致头颈部过伸和旋转。

### 脊柱曲线
后表线为脊柱和腿部原始和次生的弯曲提供了功能连接。原始弯曲（胸曲、骶曲）反映胎儿发育的弯曲姿势；次生弯曲（颈曲、腰曲）是后表线发展的结果。

## 利用后表线的评估方法

**体前屈测试**：保持膝盖伸直触碰脚趾，注意背部两侧SBL的轮廓以及手触及的位置。处理其中一个区域（如足底筋膜）就可以影响整条线上各处的活动度与长度。

## 常见治疗要点

- 足底筋膜放松：从足底筋膜→跟腱→腓肠肌→腘绳肌的连续处理
- 骶结节韧带松解对骨盆前移的改善
- 竖脊肌筋膜的浅层滑动
- 枕下肌群释放对颈部和眼-脊协调的改善
- 头皮筋膜/帽状腱膜的浅层操作

## 拉伸建议

- 婴儿式：拉伸竖脊肌和头皮筋膜
- 下犬式/前屈式：整体拉伸SBL
- 坐姿体前屈：分离SBL腿部
- 足背屈+脚趾过伸：拉伸足底→小腿
- 弓步推墙：拉伸小腿后侧

## 与其他经线的关系

- 与前表线互为拮抗——如同帆船的索具：SBL在后侧向下牵拉背部，SFL由颈部至骨盆向上牵拉前侧
- 与螺旋线在骶结节韧带和竖脊肌共享轨道
- 与臂后表线在斜方肌和头皮筋膜交汇
