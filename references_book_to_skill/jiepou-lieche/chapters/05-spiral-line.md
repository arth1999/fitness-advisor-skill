# 第6章：螺旋线（Spiral Line, SPL）

## 概述

螺旋线有左右两条，以螺旋反向环绕身体。从颅骨两侧→穿过上背部→连接对侧肩部→环绕肋部到身体前面→在肚脐水平交叉回到同侧髋→沿大腿前外侧向下→越过胫骨到内侧足弓→通过足底向上→经下肢后外侧到坐骨→进入竖脊肌筋膜→回到接近起点的颅骨位置。

## 轨道与车站（全程）

| 车站（骨性标志）        | 轨道（肌筋膜结构）               |
| ----------------------- | -------------------------------- |
| 枕骨嵴/乳突             | 头夹肌→颈夹肌                  |
| 下颈椎/上胸椎棘突       | 跨棘突筋膜→对侧大小菱形肌       |
| 肩胛骨内缘              | 菱形肌-前锯肌（肩胛骨悬带）     |
| 外侧肋骨（第5-9肋）     | 前锯肌下部                      |
| 肋部→髂前上棘           | 腹外斜肌→腹白线→对侧腹内斜肌   |
| 髂前上棘/髂嵴           | 阔筋膜张肌→髂胫束（前缘）       |
| 胫骨外侧髁              | 胫骨前肌（直接筋膜连接至ITT）   |
| 第1跖骨基部/第1楔骨     | 腓骨长肌（足底"马镫"的另一端）  |
| 腓骨头                  | 股二头肌（外侧腘绳肌）          |
| 坐骨结节                | 骶结节韧带→骶筋膜→对侧竖脊肌   |
| 枕骨嵴                  | 竖脊肌（回到起点附近）          |

## 功能

### 姿势功能
- 将身体用两个螺旋环绕，维持所有平面上的平衡
- 连接足弓和骨盆角
- 有效帮助确定行走时膝关节的运动轨迹
- 在不平衡时参与引发、代偿和维持身体扭曲、旋转和侧移

### 动作功能
- 引起并调整身体的扭转和旋转
- 在离心和等长收缩时稳定躯干和下肢以避免旋转崩溃
- 涉及多数其他主线（SBL、SFL、LL）和臂后深线的筋膜——功能失调时其他经线的基础功能也受影响

## 关键结构：菱形-前锯肌

- 菱形肌和前锯肌共同形成肩胛骨的肌筋膜悬带——肩胛骨悬挂在两者之间
- 连接极其厚实有力（不低于1厘米），使肩胛骨如同嵌入菱形-前锯肌中
- 内外侧不平衡：常见模式为菱形肌闭锁延长+前锯肌闭锁缩短→肩胛骨远离脊柱（驼背倾向）
- 相反模式：菱形肌闭锁缩短+前锯肌闭锁延长→肩胛骨靠近棘突（平背倾向）

## 足弓"马镫"机制

- 胫骨前肌和腓骨长肌共同形成足弓的"马镫"（stirrup/sling）
- 胫前肌向上牵拉内侧足弓薄弱部位，腓骨长肌拉住骰骨（外侧足弓）
- 两者你松我紧：胫前肌短缩+腓骨肌松弛→足内翻（旋后）；反之为足外翻（旋前）
- 此关系向上延伸至骨盆：骨盆前倾→胫前肌上方张力消失→内侧足弓降低；骨盆后倾→内侧足弓上抬

## 下螺旋线"跳绳"

阔筋膜张肌→髂胫束（前缘）→胫骨前肌→腓骨长肌→股二头肌，形成从髋部→下肢前外侧→足弓→下肢后外侧→回到髋部的连续"跳绳"。

## "第四腘绳肌"

股二头肌长头（特快列车）下方有两列重要普通列车：
- **股二头肌短头**：腓骨头→股骨粗线。慢性膝屈曲或胫骨外旋时可能过度活跃。
- **大收肌中部**：股骨粗线→坐骨支。与骨盆前倾或不能正常屈髋有关。

## 常见姿势代偿模式

- 踝旋前/旋后
- 膝扭转（膝关节轨迹内偏/外偏）
- 骨盆相对双脚旋转
- 胸腔相对骨盆旋转
- 单肩耸起或向前移位
- 头部倾斜、位移或旋转

## 四个警示信号（螺旋线失衡）

1. 头部相对于胸腔的倾斜或移动
2. 一侧肩部比对侧前移
3. 胸腔相对于骨盆向外侧移动
4. 一侧肋弓至对侧髂前上棘的距离改变（胸骨与耻骨方向不同）

## 上螺旋线与躯干姿势旋转

- 上螺旋线（枕骨→对侧肩→同侧髂前上棘）是调控上半身旋转的最佳线路
- 核心旋转模式（前深线）通常被外周螺旋线代偿——先松解外周再处理核心
- 当外周代偿被解除后，核心旋转模式会显得更突出——需教育当事人这是正常过程

## 步行中的骶骨交叉

在步态中，骶骨两侧的韧带通过螺旋线筋膜组织连接：右侧骶结节韧带+左侧骶髂背侧长韧带→横跨骶骨的连续筋膜，在行进中稳定骨盆。

## 头前移姿势与胸带

前锯肌中部纤维→穿过胸大肌下→胸骨底部中线交叉→形成"胸带"→经菱形肌→头夹肌→到达对侧颅骨。此线紧张导致头前移时胸骨后勒感、呼吸受限。

## 螺旋线与膝关节轨迹

螺旋线将膝关节外侧拉向从ASIS到内踝的线。此线收缩导致膝内旋。评估方法：双脚第2趾平行站立，双膝向前弯曲观察"头灯"是否向内或向外偏移。

## 上螺旋线与下螺旋线

由于髂前上棘机车库的多方向牵拉竞争，螺旋线的上下两部分常常是单独工作的：
- 上螺旋线：调控上半身旋转
- 下螺旋线：足弓-骨盆-膝关节轨迹调控
- 在步行等交替运动中可和谐协作
