# 第5章：体侧线（Lateral Line, LL）

## 概述

体侧线位于身体两侧，起自足内侧与外侧的中点，从踝外侧上行，经小腿和大腿外侧面，以"篮纹编织状"或"鞋带交叉"方式上至躯干，由肩部下方上行至头颅的耳部区域。

## 轨道与车站

| 车站（骨性标志）        | 轨道（肌筋膜结构）               |
| ----------------------- | -------------------------------- |
| 第1跖骨基部、第5跖骨基部| 腓骨长肌肌腱、腓骨短肌           |
| 腓骨头                  | 腓骨头前韧带                    |
| 胫骨外侧髁              | 髂胫束（ITT）                   |
| 股骨大转子/髂嵴外缘     | 阔筋膜张肌（前缘）/臀大肌上部（后缘）/臀中肌 |
| 髂嵴外缘→肋骨           | 腹外斜肌+腹内斜肌（"篮纹编织"） |
| 下位肋骨→上位肋骨       | 肋间外肌+肋间内肌               |
| 第1肋→乳突              | 胸锁乳突肌（锁骨头）+头夹肌（"X"交叉）|

## 功能

### 姿势功能
- 调整身体前后平衡和左右平衡
- 对其他表层线（SFL、SBL、臂线、螺旋线）之间的力量进行调节
- 以协调方式固定躯干和下肢，防止上肢活动时身体结构变形扭曲

### 动作功能
- 参与身体侧弯（躯干侧弯、髋部外展、足外翻）
- 对躯干侧向和旋转运动有可调性"刹车"作用
- 在步行中调整和控制旋转动作——肋间肌如同手表发条，储存和释放旋转势能

## "X"交叉模式

体侧线在躯干和颈部以"鞋带交叉"方式走行：
- **躯干"X"**：腹外斜肌（向前下）与腹内斜肌（向后下）交叉
- **肋骨"X"**：肋间外肌（向前下）与肋间内肌（向后下）交叉
- **颈部"X"**：胸锁乳突肌（向前上）与头夹肌（向后上）交叉

## 常见姿势代偿模式

- 踝关节旋前或旋后
- 踝背屈受限
- 膝内翻（O型腿）或膝外翻（X型腿）
- 大腿内收受限/慢性外展肌挛缩
- 腰椎侧弯或腰椎受压（双侧LL缩短时）
- 胸腔在骨盆上侧移
- 胸骨与骶骨间的短缩
- 头前倾姿势时肩关节受限

## 关键临床要点

### 深层体侧线
虽然按解剖列车规则不属于体侧线，但两组结构与LL功能紧密相关：
- **腰方肌（QL）**：唯一针对侧屈进行调节的肌肉，与前斜角肌共同悬吊胸腔（QL→第12肋→胸腔→斜角肌→颈椎）
- **前斜角肌+枕下肌群**：前斜角肌牵拉下颈椎屈曲，枕下肌群牵拉上部过伸→"头前倾"姿势

### 体侧线与步行
- 正常人步行时头部和胸部适当向前移动，大部分"左右摆动"运动发生在腰以下
- 过度"左右摆动"（如年长老兵体）提示体侧线调节能力下降
- 过多的"上下起伏"（如青少年体）提示SFL/SBL问题

### 躯干整体X评估法
设想一"X"：一条腿从C7棘突到耻骨，另一条腿从胸骨颈静脉切迹到骶骨顶点。胸骨至骶骨线明显短于C7至耻骨线是西方人中极其常见的负重体型特征。

### 体侧线的心理表达
- 前表线=正面="是"（信任/开放）
- 后表线=背面="不"（保护/防御）
- 体侧线=侧面="也许"（诱惑/暧昧）

## 利用体侧线的评估方法

- 站立位从前面/后面观察肩部和髋部倾斜
- 让被检查者双手悬吊在单杠上，观察身体左右不对称处
- 站在门口，观察躯干两侧与门框的距离关系

## 拉伸与运动建议

- 三角式及其变体：对LL下部进行拉伸
- 踝关节内翻+背屈：拉伸腓骨肌群
- 交叉前屈站立：拉伸髂胫束
- 侧向爬行/鳗鱼式摆动：整合身体侧屈运动
