# 第4章：前表线（Superficial Front Line, SFL）

## 概述

前表线连接身体前表面，从足背到颅骨侧面（乳突），沿身体前侧上行。与后表线整体连接的特征不同，前表线各部分之间较多独立分离的功能。

## 轨道与车站

| 车站（骨性标志）      | 轨道（肌筋膜结构）               |
| --------------------- | -------------------------------- |
| 趾骨背侧              | 短伸肌肌腱、长伸肌肌腱           |
| 足背/踝前支持带       | 小腿前侧间隔（胫骨前肌、趾长伸肌、拇长伸肌）|
| 胫骨粗隆              | 股四头肌/髌骨/髌下韧带          |
| 髂前下棘（AIIS）      | 股直肌（严格来说仅股直肌属SFL）  |
| 第5肋骨               | 腹直肌（含前、中、后三层腹直肌鞘）|
| 胸骨                   | 胸骨肌/胸肌筋膜（胸肋关节筋膜）   |
| 乳突/星点              | 胸锁乳突肌（SCM）                |

## 重要支线

- 从胫骨前肌沿髂胫束（ITT）前缘上行→大腿外侧→髂前上棘（与腹内斜肌机械连接）——进入螺旋线
- 缝匠肌从胫骨→大腿内侧→髂前上棘（连接腹外斜肌）——进入同侧功能线

## 髂前上棘的机车库

多条经线的拉力在ASIS竞争交汇。腿部各轨道经过此处向上分离的选择包括：
- 直线向上：股直肌→髂前下棘→机械连接至骨盆
- 外侧：阔筋膜张肌→髂胫束→螺旋线
- 内侧：缝匠肌→同侧功能线
- 深层：髂肌→前深线

## 功能

### 姿势功能
- 与后表线共同维持矢状面平衡——如同帆船索具：SFL从颈部至骨盆向上牵拉前侧，SBL向下牵拉后侧
- 前表线在人类直立姿势中将所有最敏感脆弱区域沿前侧分布

### 动作功能
- 收缩引起：脚趾伸直、踝关节背屈、膝关节伸直、髋关节和躯干屈曲
- 但此时上颈部过伸（因SCM向后上方走行至乳突）

## 常见姿势代偿模式

- 前表线前面被向下拉，而后表线在背侧被向上拉（常见模式）
- 头前倾位：SCM将胸腔顶端向下牵拉→腹直肌下降肋骨→限制呼吸
- "站军姿"模式：后表线中段闭锁缩短，前表线被拉伸或离心性拉长
- 腹部凸出但腹直肌深层筋膜张力高（影响呼吸或牵拉背部）

## 关键临床要点

### 股直肌与骨盆的机械连接
前表线在髋部没有直接的肌筋膜连接——股直肌与髂肌邻近但无筋膜相连。在矢状面运动中髋骨形成机械连接，使SFL作为一趟列车运作。但在旋转运动时则分离。

### 舌骨下肌群不属于前表线
常见错误是将舌骨下肌群归于SFL。它们从胸骨后方附着，是颈部内脏柱的一部分，属于前深线（DFL）。

### 胸锁乳突肌的特殊走向
SCM将肌筋膜拉力从身体前侧突然跳至头颅后侧（乳突/星点），导致一个反常现象：拉紧前表线可使躯干屈曲，但会同时引起颈部顶端的过伸。

### 惊恐反应与前表线
惊恐反应是跨文化可观察到的前表线突然收缩——小腿僵直、躯干和胳膊屈曲、上颈部过伸。该模式伴随快速剧烈呼气，持续状态常与沮丧情绪相关。

## 利用前表线的评估方法

- "桥式"伸展：中等程度伸展SFL上部
- 下腰动作：最完全的SFL拉伸（需足够力量和弹性）
- 跪坐姿势：检查SFL最下端伸展能力
- 站姿观察：前表线各部分在躯干过伸时连接为整体

## 拉伸建议

- 跪位脚趾朝后，臀部坐足跟——SFL最下端
- 身体向后倒，髋关节伸直（需充分支撑）——膝以上至髋部SFL
- "桥式"——SFL上部中等伸展
- 下腰动作——SFL最完全伸展（初学者不推荐）

## 与后表线的相互作用

- SFL过短→躯干/髋屈曲、膝盖伸直受限、上颈椎过伸
- SBL过短→过度伸展、"军姿"、骨盆前移
- 评估模式：注意哪些经线段处在闭锁缩短/闭锁延长状态
- 常见失衡：腹直肌向下牵拉胸腔（而非正常地向上牵拉耻骨），导致肋骨下降限制呼吸
