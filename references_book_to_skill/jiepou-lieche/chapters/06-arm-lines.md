# 第7章：手臂线（Arm Lines）

## 概述

四条手臂线起始于中轴骨，穿过肩部的四个层面，止于手臂的四个象限和手的四个"边"（拇指、小指、手掌、手背）。手臂线依据跨过肩部的位置深浅命名，上下肢在筋膜连接上并不平行（因为胚胎发育中肩部外旋而下肢内旋）。

## 命名为四个面的原则

手臂外展、手掌向前、鹰嘴朝地时：
- **臂前表线（SFAL）**：手臂前侧——掌肌群→屈肌群→内侧肌间隔→胸大肌/背阔肌
- **臂后表线（SBAL）**：手臂背侧——斜方肌→三角肌→外侧肌间隔→伸肌群
- **臂前深线（DFAL）**：手臂内旋后前侧——鱼际肌→桡骨→肱二头肌→胸小肌
- **臂后深线（DBAL）**：手臂内旋后背侧——小鱼际肌→尺骨→肱三头肌→肩袖/菱形肌/肩胛提肌

## 臂前深线（DFAL）

| 车站         | 轨道                           |
| ------------ | ------------------------------ |
| 第3/4/5肋    | 胸小肌/胸锁筋膜                |
| 喙突         | 肱二头肌短头                   |
| 桡骨粗隆     | 桡骨骨膜（前缘）               |
| 桡骨茎突     | 桡侧副韧带/大鱼际肌群          |
| 舟状骨/大多角骨| 至大拇指外侧                  |

### 关键特征
- 手臂下垂时DFAL近心端短缩，形成"圆肩"
- 手臂外展水平以上时（如引体向上），胸小肌和肱二头肌短头才连接为经线
- 肱二头肌特快列车跨越三个关节（盂肱、肱尺、桡尺）
- 下方三列普通列车：喙肱肌（肩）、肱肌（肘）、旋后肌（前臂旋转）
- **拇指线**：指压按摩操作者需注意DFAL保持松弛延长，疼痛不仅来自拇指本身，更来自上臂-喙突-肋骨连接区域

## 臂前表线（SFAL）

| 车站                 | 轨道                       |
| -------------------- | -------------------------- |
| 锁骨/胸骨/肋骨/髂嵴  | 胸大肌+背阔肌（扇形连接）  |
| 肱骨内上髁           | 内侧肌间隔                |
| 手指掌面             | 屈肌群→腕管→手指          |

### 关键特征
- 起点为宽的扇形：从锁骨到中部肋骨（胸大肌）+从胸椎下部到髂嵴（背阔肌）
- 控制手臂在身体前侧和侧面的大范围动作
- 提供游泳划水、击球等大动作推动力
- **SFAL与DFAL的区别**：手臂外旋（掌心向上）拉伸SFAL→牵拉胸大肌和屈肌；手臂内旋（拇指向上）拉伸DFAL→牵拉胸小肌和肱二头肌

## 臂后深线（DBAL）

| 车站                 | 轨道                               |
| -------------------- | ---------------------------------- |
| C7/T1-T5棘突         | 菱形肌+肩胛提肌                   |
| 肩胛骨内缘           | 肩袖肌群（冈下肌、小圆肌、肩胛下肌+冈上肌）|
| 肱骨大结节/小结节    | 肱三头肌                           |
| 尺骨鹰嘴             | 尺骨骨膜                           |
| 尺骨茎突             | 尺侧副韧带/小鱼际肌群              |
| 三角骨/钩骨          | 至小指外侧                         |

### 关键特征
- 肩袖肌群四块肌肉控制肱骨头＝眼球肌肉控制眼球——精准的肌肉排列规则
- 菱形肌→前锯肌（螺旋线）与菱形肌→肩袖肌群（DBAL）在肩胛骨脊柱缘分叉
- **柔道翻滚沿DBAL走行**：从小指侧→尺骨→肱三头肌→肩背侧→后功能线
- 大致等同于下肢的体侧线，与DFAL共同调整肘关节角度

## 臂后表线（SBAL）

| 车站                 | 轨道                       |
| -------------------- | -------------------------- |
| 枕骨嵴→C7→T1-T12     | 斜方肌                     |
| 肩胛冈/肩峰/锁骨外1/3| 三角肌                     |
| 三角肌粗隆           | 外侧肌间隔                |
| 肱骨外上髁           | 伸肌群→手指背侧            |

### 关键特征
- 斜方肌-三角肌可视为一大块三角形肌肉——头骨后方→肩前方→手臂后方
- 颈部斜方肌↔三角肌中部；枕部斜方肌↔三角肌前部；胸部斜方肌↔三角肌后部
- **筋膜/肌肉的转换**：肩部浅层为肌肉→上臂为筋膜→前臂为肌肉→手部为肌腱
- 深层线则相反：肩部深层为筋膜→上臂为肌肉（肱二头肌/肱三头肌）→前臂为筋膜→手部为肌肉

## 关键交叉（道岔）

手臂额外的旋转和活动需要大量交叉连接来提供灵活性和稳定性：
- 肱二头肌长头（DBAL→DFAL）——冈上肌机械连接
- 肱二头肌腱膜（DFAL→SFAL）——单侧负重时的力转移路径
- 大圆肌（DBAL→SFAL）——肩胛骨到肱骨前侧面
- 肱桡肌（SBAL→DFAL）——外侧肌间隔到桡骨
- 旋前圆肌（DFAL→SFAL）

## 肩胛骨"X"与肩部稳定

肩胛骨稳定由两组拮抗关系形成的"X"决定：
- **第一支**：菱形肌（向内上）↔前锯肌（向外前）——影响肩胛骨在胸腔上的内外侧位置
- **第二支**：下斜方肌（向内下）↔胸小肌（向前下）——影响肩胛骨倾斜/前倾

## 常见姿势代偿

- 肩部前缩、后缩、上抬
- "圆肩"（肩胛骨内旋及前倾）
- 腕管综合征、肘部撞击、肩部撞击
- 慢性肩部肌肉或扳机点疼痛

## 使用臂线进行评估

- **臂前线**：受试者身体前倾，手臂外旋（检查SFAL）或内旋（检查DFAL）
- **臂后线**：受试者身体后仰，手腕外旋（检查SBAL）或内旋（检查DBAL）
- 注意各类运动习惯造成的交叉肌肉过度参与

## 与下肢线路的不平行关系

手臂与下肢在骨骼上有对应性，在肌筋膜轨道上却不同：
- 臂前深线（DFAL）↔下肢前深线（DFL）
- 臂后深线（DBAL）↔下肢体侧线（LL）
- 臂前表线（SFAL）↔下肢后表线（SBL）
- 臂后表线（SBAL）↔下肢前表线（SFL）
这种"前后反转"反映了胚胎发育中上下肢的不同旋转方向。
