# 第10章 运动员铁损耗及铁缺乏的预防、诊断和治疗

## 章节概述

本章系统阐述铁在运动员中的代谢特征、铁缺乏的分期与诊断、运动员高风险的生理原因以及铁补充的策略与风险。铁缺乏是运动员最常见的营养素缺乏问题之一，尤其影响女运动员和耐力运动员。

## 核心知识点

### 1. 铁的生理功能

铁的核心功能包括：
- 血红蛋白（Hb）中的铁负责氧气运输
- 肌红蛋白中的铁负责肌肉内氧储存
- 线粒体呼吸链的细胞色素含铁参与能量代谢
- 铁是多种酶（如过氧化氢酶、过氧化物酶）的辅因子

### 2. 铁缺乏的分期

**第1期 — 铁储存耗竭**：血清铁蛋白降低（<12-20 μg/L），但血红蛋白和功能指标正常。无贫血，但可能影响运动能力。

**第2期 — 缺铁性红细胞生成**：运铁蛋白饱和度下降，红细胞原卟啉增加。血红蛋白功能开始受影响。

**第3期 — 缺铁性贫血**：血红蛋白浓度显著下降（男性<130 g/L，女性<120 g/L），红细胞形态异常。运动能力受到显著影响。

### 3. 运动员铁缺乏的高风险机制

- **铁丢失增加**：胃肠道出血（大量跑步后的冲击性溶血）、汗液铁丢失（虽少量但高温训练时累积）、月经失血（女性）
- **铁摄入不足**：限制膳食（尤其能量摄入受限的运动员）、素食者
- **铁需求增加**：训练引起红细胞质量增加、生长发育期
- **铁吸收障碍**：某些饮食成分（如茶中的鞣酸、植酸）抑制铁吸收
- **"运动性贫血"（伪贫血）**：血浆容量扩张稀释了血红蛋白，但实际铁状态可能正常

### 4. 铁状况的评价

核心评价指标包括：
- 血清铁蛋白（储存铁的金标准）
- 血红蛋白和红细胞比容（诊断贫血）
- 运铁蛋白饱和度
- 可溶性运铁蛋白受体（区分功能性铁缺乏与储存铁耗竭）
- C-反应蛋白（CRP）（排除炎症对铁蛋白的影响）

### 5. 铁补充策略

**膳食铁补充**：优先推荐。血红素铁（动物来源）吸收率约15%-35%；非血红素铁（植物来源）吸收率约2%-20%。增加铁吸收的策略：与维生素C同食、避免与茶/咖啡同食。

**口服铁补充剂**：需在医生监督下进行。常用剂量为50-100 mg元素铁/日。不良反应包括胃肠道不适（便秘、恶心）。不合理的铁补充有铁过载风险。

## 实用应用

- 高风险运动员应定期（每6-12个月）进行铁生化指标筛查
- 解释铁指标时需考虑血浆容量变化和训练阶段
- 优先通过膳食干预改善铁状态
- 补充铁只在确认铁缺乏后进行

## 关键术语

- 铁蛋白（Ferritin）/血清铁蛋白
- 缺铁性贫血（Iron Deficiency Anemia, IDA）
- 血红素铁/非血红素铁（Heme/Non-heme Iron）
- 运动性贫血（Sports Anemia）
- 溶血（Hemolysis）
- 运铁蛋白饱和度（Transferrin Saturation）
