# 第22章 残疾运动员的特殊需求

## 章节概述

本章讨论残疾运动员的营养管理，包括不同类型残疾（脊髓损伤、脑瘫、截肢等）对能量需求、体成分、营养素需求和营养实践的特殊影响。随着残奥运动的日益发展，这是运动营养学越来越重要的领域。

## 核心知识点

### 1. 不同类型残疾的营养影响

**脊髓损伤（SCI）**：
- 损伤水平以下肌肉麻痹，降低总能量消耗
- 骨骼肌萎缩导致BMR降低
- 体温调节功能受损（损伤水平以下丧失出汗能力）
- 神经源性肠道和膀胱功能障碍
- 骨密度迅速下降（尤其损伤后前几年）
- 压疮风险增加，需关注蛋白质和微量营养素摄入

**脑瘫（CP）**：
- 痉挛增加能量消耗，但同时运动受限
- 由于口腔运动功能障碍导致的进食困难
- 胃食管反流常见
- 一些运动员体成分异常（高体脂和低瘦体重）

**截肢**：
- 行走能耗增加（随截肢水平增加）→能量需求增高
- 残肢和假肢界面的皮肤完整性问题
- 多发性截肢的能耗尤其高昂

### 2. 能量需求

残疾运动员的能量需求高度个体化，基于：
- 残疾类型、水平和程度
- 运动类型、强度和持续时间
- 残存肌肉量和代谢率

关键挑战：准确预测能量需求困难。间接测热法（如果可行）是最佳RMR评估方法。对SCI运动员，常用预测公式会显著高估需求。

### 3. 体成分评估

标准体成分评估方法在残疾运动员中面临挑战：
- BMI参考范围不适用（因肌肉萎缩或肢体缺失）
- 皮褶厚度方程未针对残疾人群验证
- BIA在有金属植入物时禁忌
- DXA扫描受限（设备设计、体位问题）

### 4. 关键营养素关注

- **蛋白质**：压疮预防和治疗时需求增加（1.2-1.5 g/kg/d）
- **维生素D和钙**：SCI运动员骨密度下降，补充可能有益
- **维生素C和锌**：对伤口愈合（如压疮）重要
- **铁**：月经女性SCI运动员仍有铁缺乏风险
- **液体**：SCI运动员出汗功能障碍，体温调节受损，需谨慎管理液体摄入

### 5. 特殊的实践挑战

- 进食独立性问题（自理能力限制）
- 胃肠道功能改变（排便管理）
- 液体摄入限制（因排尿管理挑战而主动减少饮水）
- 旅行和比赛中无障碍食物获取
- 补剂的安全性和兴奋剂风险

### 6. 运动营养支持策略

- 与物理治疗师和职业治疗师协作评估功能能力
- 在比赛环境中评估膳食和液体的可及性
- 针对个体残疾类型和水平的精细能量和营养素处方
- 体重分级项目（如轮椅橄榄球、力量举）的体重管理需格外谨慎

## 关键术语

- 脊髓损伤（Spinal Cord Injury, SCI）
- 神经源性肠道（Neurogenic Bowel）
- 压疮（Pressure Ulcer/Decubitus Ulcer）
- 出汗功能障碍（Anhidrosis）
- 残奥运动（Paralympic Sport）
