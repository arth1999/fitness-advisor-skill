# 临床运动营养学 — 实践模式与决策框架

## 1. 运动员营养管理循环

```
评估 → 制定目标 → 实施干预 → 监控与反馈 → 调整优化 → 评估...
```

这是贯穿全书的运动员营养管理核心范式，各步骤包含：

**评估（Assessment）**：膳食摄入（第2章）、人体测量（第3章）、生化指标、训练和比赛档案
**目标设定（Goal Setting）**：基于运动项目需求、运动员个体特征和训练周期
**干预实施（Intervention）**：膳食调整、营养教育、补剂策略
**监控反馈（Monitoring）**：定期复查膳食摄入、体成分、血液指标和运动能力
**调整优化（Adjustment）**：根据监控数据和运动员反馈调整方案

## 2. 比赛营养三段式策略

```
赛前准备 → 赛中维持 → 赛后恢复
```

这是第12-14章的核心框架，每一项比赛都应整合三者：

- **赛前（Days-Hours）**：CHO负荷、赛前膳食、水合准备
- **赛中（During Event）**：液体和CHO摄入、电解质补充
- **赛后（Hours-Days）**：糖原再合成、蛋白质修复、再水合

## 3. 能量平衡四因素模型

```
E_in = 食物摄入 - 排泄损失
E_out = RMR + TEF + TEA
```

实际应用中需要评估：
- 运动员是否处于能量平衡（体重稳定）
- 能量摄入不足的风险因素（性别、运动项目、训练量、限制饮食史）
- 能量可利用性对健康（月经、骨骼、免疫）和运动能力的影响

## 4. 证据驱动的补剂决策框架

（基于第16章AIS框架）

```
问题识别 → 膳食优化 → 补剂评估 → 风险/收益分析 → 实施决策 → 效果评估
```

**补剂评估四问**：
1. 是否有合法有效证据支持该场景使用？（证据水平A/B/C/D）
2. 是否存在安全风险或兴奋剂污染风险？
3. 该运动员是否已优化膳食和训练基础？
4. 净收益是否大于成本和风险？

## 5. 特殊人群营养适配模式

常见于第17-22章的特殊人群：

```
标准运动员建议 → 识别特殊需求 → 调整基础建议 → 额外监测 → 跨学科协作
```

- **标准建议作为起点**：运动营养的一般原则
- **识别特殊需求**：青少年（生长需求）、老年（合成代谢抵抗）、糖尿病（血糖管理）、素食（微量营养素风险）、残疾（功能限制）
- **调整建议**：基于特殊需求修改营养素剂量、时机和形式
- **额外监测**：增加该人群特有的监测指标（如血糖、B12、骨密度）
- **跨学科协作**：与医生、物理治疗师、心理师协作

## 6. 体重管理决策树

```
体型评价 → 是否需要改变体重/体成分？
  ├── 否 → 维持当前能量平衡策略
  ├── 增重 → ch04策略（能量盈余 + 蛋白质 + 力量训练）
  └── 减重 → 评估紧迫性
        ├── 慢性体重管理（第7章）→ 适中能量限制 + 长期行为改变
        └── 急性降体重（第6章）→ 仅限体重分级项目，需最小化危害
```

## 7. 环境适应营养阶梯

（基于第24章）

```
基础训练营养 → 环境特异性适应 → 环境应激期间的精准营养

热环境：水合优先 → 增加钠和CHO → 热习服完成
高原：铁状态优化 → 抗利尿行动 → 抗氧化保护 → 高原适应完成
```

## 8. 团队膳食四维规划模型

（基于第25章）

四个维度同时满足：
- **营养维度**：满足运动营养科学要求（CHO、蛋白质、微量营养素充足）
- **文化维度**：尊重文化和宗教膳食需求
- **安全维度**：HACCP体系、食物过敏原管理
- **教育维度**：利用供餐环境进行运动员营养教育
