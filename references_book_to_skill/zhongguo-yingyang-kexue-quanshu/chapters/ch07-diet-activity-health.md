# 第七卷 膳食、身体活动与健康

## 卷概述

本卷聚焦膳食模式和身体活动与人类主要慢性非传染性疾病（NCDs）的关系，是全书"从营养素到食物、从食物到膳食模式、从膳食模式到健康结局"逻辑链的最终环节。共10章，覆盖了现代公共健康领域最为关注的膳食相关疾病。

## 各章核心内容

### 第一章 膳食模式与健康
现代营养学从关注单一营养素转向关注整体膳食模式（dietary pattern）：
- **膳食模式分类方法**：先验法（基于膳食指南评分，如HEI、AHEI、DASH评分）和后验法（基于统计学的因子分析和聚类分析）
- **已被证实的健康膳食模式**：
  - 地中海膳食（Mediterranean Diet）：高水果蔬菜、全谷物、豆类、坚果、橄榄油、适量鱼、少量红肉。与心血管疾病、糖尿病和认知功能下降风险降低相关
  - DASH膳食（Dietary Approaches to Stop Hypertension）：高水果蔬菜、低脂乳制品、全谷物、低钠。降压效果明确
  - 健康的植物基膳食
- **中国传统膳食模式**：以植物性食物为主的江南膳食模式研究

### 第二章 饮食行为与健康
饮食习惯不仅取决于营养素知识，还受到心理学、社会学、经济学和环境因素的深刻影响：
- **饮食行为的影响因素**：个人因素（口味偏好、信念、知识）、人际因素（家庭、同伴）、环境因素（食物可获得性、价格、营销）、政策因素（食品标签、税收）
- **早餐与健康**：规律早餐与较低肥胖风险和改善学童认知功能相关
- **进餐频率和规律性**：不规律进餐与代谢紊乱风险增加相关
- **外食行为**：中国城市居民外食频率增加与外食的高能量密度和高钠特征

### 第三章 身体活动与健康
规律身体活动是健康四大基石之一（合理膳食、适量运动、戒烟限酒、心理平衡）：
- **推荐量**：成人每周≥150分钟中等强度或≥75分钟高强度有氧运动+2天力量训练；儿童青少年每天≥60分钟
- **健康益处**：降低全因死亡率和心血管疾病、糖尿病、部分癌症、抑郁症风险；改善骨健康、认知功能和睡眠
- **中国身体活动现状**：居民身体活动水平普遍呈下降趋势

### 第四章到九章 — 膳食和身体活动与主要NCDs
**超重和肥胖（第四章）**：
- 中国超重率约30%（2020数据）、肥胖率约15%，且持续增长
- 能量平衡是根本，但膳食质量同样重要
- 治疗阶梯：生活方式干预 → 药物治疗 → 减重手术

**心血管疾病（第五章）**：
- 膳食钠是高血压的主要膳食危险因素（中国居民钠摄入约10.5 g盐/d，远超<5 g/d推荐）
- 低钾摄入同样是风险因素
- 饱和脂肪和反式脂肪酸替代为不饱和脂肪酸的保护效果
- 增加全谷物、水果蔬菜、鱼类的保护证据

**糖尿病（第六章）**：
- 中国糖尿病患病率约11%-12%
- 膳食干预是糖尿病管理的基石：CHO质量（GI/GL概念）、膳食纤维、MNT
- 身体活动对胰岛素敏感性的改善作用
- 从预防（IGT人群生活方式干预可降低约50%糖尿病进展）到治疗的全链条

**癌症（第七章）**：
- WCRF/AICR持续更新膳食与癌症关系的证据
- 确立的证据：加工肉类→结直肠癌（1类致癌物）；红肉→结直肠癌（2A类）；酒精→多种癌症
- 可能的保护因素：膳食纤维、全谷物、乳制品→结直肠癌；水果蔬菜→多种癌症
- 肥胖本身就是多种癌症的确认风险因素

**骨质疏松症（第八章）**：
- 钙、维生素D和蛋白质的营养关键三角
- 峰值骨量的决定因素（约60%-80%由遗传决定）
- 身体活动尤其是负重运动对骨密度的保护作用
- 中国居民钙摄入普遍不足（平均约366-400 mg/d，远低于800 mg/d的RNI）

**痛风（第九章）**：
- 嘌呤代谢紊乱和/或尿酸排泄障碍→高尿酸血症→痛风
- 传统低嘌呤膳食(<300 mg/d)的挑战：限制过严难以坚持且对心血管风险的管理不利
- 现代管理转向：促进健康膳食模式+体重管理+限制酒精和果糖饮料。红肉和海鲜适量

### 第十章 营养缺乏病
尽管全球范围内蛋白质-能量营养不良（PEM）大幅减少，但微量营养素缺乏（"隐性饥饿"）仍然是全球和中国人群面临的营养挑战：
- **铁缺乏和缺铁性贫血**：全球最常见的微量营养素缺乏。中国育龄妇女和儿童是高危人群
- **碘缺乏**：中国曾是碘缺乏重灾区，普遍食盐加碘政策使得碘缺乏病大幅下降
- **维生素A缺乏**：主要影响儿童，导致夜盲症和感染风险增加
- **维生素D缺乏**：全球普遍存在的问题，中国居民同样面临
- **锌缺乏**：影响生长发育和免疫功能
- **叶酸缺乏**：与神经管缺陷密切相关

## 核心信息

1. 膳食模式（而非单一营养素）是慢性病预防的现代焦点
2. 膳食-慢性病关系有不同水平的证据强度（从确立因果到提示性证据）
3. 中国面临的"双重负担"：营养不足和超重肥胖相关慢性病并存
4. 身体活动与膳食同等重要——二者是健康生活方式的一体两面
5. 公共卫生策略（如碘盐强化）在消除微量营养素缺乏方面具有最高成本效益比
