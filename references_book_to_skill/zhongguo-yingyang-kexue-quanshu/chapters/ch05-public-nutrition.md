# 第五卷 公共营养

## 卷概述

公共营养（public health nutrition）将营养科学的理论和方法应用于群体健康促进，研究人群营养状况、膳食结构、膳食指南、营养政策法规和食品安全等。本卷共9章，涵盖了从营养调查到政策制定的完整公共营养实践链，是营养学社会属性的集中体现。

## 各章核心内容

### 第一章 公共营养概论
公共营养的定义是"应用营养科学和公共卫生的方法改善人群营养健康"。它与临床营养的区别在于关注群体而非个体，侧重预防而非治疗。公共营养的三大核心功能是：评估（Assessment）、政策制定（Policy Development）和保障（Assurance）。

### 第二章 营养调查
营养调查是公共营养的基础信息工作：
- **目的**：了解人群膳食摄入、营养状况和营养相关健康问题
- **内容组成**：膳食调查 + 体格检查 + 生化检测 + 健康问卷
- **中国代表性调查**：中国居民营养与健康状况调查（10年周期），2010-2012年调查首次实现31省全覆盖
- **调查发现**：中国面临营养不足（微量元素缺乏）与超重肥胖的双重挑战

### 第三章 营养监测
营养监测是连续的、长期的数据收集和分析系统，用于追踪人群营养状况的时间变化趋势：
- 监测指标：膳食摄入、人体测量、生化指标、食品供应、健康结局
- 中国营养监测体系的建立和发展：常规监测点网络
- 监测数据用于政策评估：如碘盐推广效果的持续监测

### 第四章 膳食结构与膳食指南
**中国居民膳食结构变迁**：
- 传统模式（以植物性食物为主）→ 转型模式（动物性食物增加）
- 谷物消费量下降，动物性食物和食用油消费量上升
- 与营养相关的慢性非传染性疾病（NCDs）患病率上升

**《中国居民膳食指南》（2016）** — 六条核心推荐：
1. 食物多样，谷类为主
2. 吃动平衡，健康体重
3. 多吃蔬果、奶类、大豆
4. 适量吃鱼、禽、蛋、瘦肉
5. 少盐少油，控糖限酒
6. 杜绝浪费，兴新食尚

**膳食宝塔**：以图形化方式展示每一类食物的每日推荐摄入量范围，是膳食指南的直观教育工具。

### 第五章 食物营养规划与营养改善
营养规划（Nutrition Planning）是在国家层面将营养目标纳入食物生产和供应政策：
- 食物平衡表方法
- 营养改善措施：食物强化（碘盐、铁强化酱油等）、膳食多样化促进、营养补充剂发放
- "三减三健"行动：减盐、减油、减糖；健康口腔、健康体重、健康骨骼

### 第六章 营养教育与健康促进
营养教育是通过信息传播和行为干预，帮助个体和群体掌握营养知识、形成健康饮食行为的系统工程：
- 教育对象：从个体（患者/消费者）到群体（学校、社区）
- 技术与方法：人际传播、大众媒体、社会营销
- 全民营养周（每年5月的第三周）是中国最大规模的营养科普活动
- 行为改变理论：知信行模式（KAP）、健康信念模型、社会认知理论

### 第七章 社区营养
以社区为单位的营养改善活动：
- 社区营养诊断（Community Nutrition Assessment）
- 社区营养干预项目设计和评估
- 社区参与和赋权（Empowerment）
- 典型案例：农村义务教育学生营养改善计划

### 第八章 营养政策法规与标准
营养相关的法律法规体系是公共营养的制度保障：
- 《食品安全法》及其实施条例
- 营养标签相关标准：预包装食品营养标签通则（GB 28050）
- 营养强化相关标准：食品营养强化剂使用标准（GB 14880）
- 特殊膳食用食品标准系列
- 营养师立法和注册制度的推进
- 国际框架：ICN2（第二届国际营养大会，2014）、联合国营养行动十年（2016-2025）

### 第九章 食物保障与食品安全
食物保障（Food Security）的四维定义：食物可获得性（Availability）、可及性（Access）、利用（Utilization）和稳定性（Stability）。
- 中国食物保障的进展：从"吃得饱"到"吃得好、吃得健康"
- 食品安全风险：微生物污染（头号风险）、化学污染（农药、重金属）、非法添加
- 食品可追溯体系
- 食品安全风险监测网络

## 公共营养实践框架

公共营养的工作循环：
```
营养状况评估（调查/监测）→ 营养问题识别 → 
政策/干预设计 → 实施 → 效果评估 → 反馈调整
```

这是中国营养改善工作的基本逻辑路线，从碘缺乏病的消除到营养改善计划的实施均遵循此框架。
