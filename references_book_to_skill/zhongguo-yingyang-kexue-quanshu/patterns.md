# 中国营养科学全书 — 知识框架与实践模式

## 1. 营养科学知识体系的整体结构

```
基础理论 → 专业理论 → 应用技术
```

- **基础理论层**：人体解剖学、生物化学、生理学、微生物学、免疫学等
- **专业理论层**：基础营养、食物营养、人群营养、公共营养、临床营养
- **应用技术层**：DRIs、膳食指南、营养监测评估、营养教育、营养治疗等

这是全书7卷组织结构的理论依据——从基础科学到人群应用到公共政策。

## 2. 营养素必需性推理范式

```
食物谱系分析 → 缺乏症状观察 → 动物模型验证 → 
人类耗竭-补充实验 → 生化功能阐明 → DRIs确定 → 
缺乏病防治 → 人群应用
```

这是营养学最重要的认知路径，从19世纪发现第一个维生素到现代确立42种必需营养素，均遵循此范式。

## 3. 全生命周期营养框架

```
孕前 → 孕期 → 哺乳期 → 婴儿期 → 幼儿期 → 
学龄前 → 学龄期 → 青春期 → 成年期 → 老年期
```

每个阶段有其特定的营养重点：
- 孕前/孕期：叶酸、铁、碘、DHA
- 婴儿期：母乳喂养、辅食添加
- 青春期：钙（峰值骨量）、铁（月经）、蛋白质
- 老年期：蛋白质（抗肌肉衰减）、钙/维生素D（骨健康）、B12

## 4. 从营养素到食物到膳食模式的发展逻辑

```
单一营养素研究 → 食物营养价值 → 膳食模式分析 → 健康结局
```

营养科学正在经历从"以营养素为中心"向"以食物和膳食模式为中心"的范式转变。这一转变体现在：
- 第二卷：食物营养 → 第三卷：研究方法 → 第四卷：人群应用 → 第七卷：膳食模式与慢性病

## 5. 公共营养行动周期

```
营养调查 → 问题识别 → 政策制定 → 实施干预 → 
营养监测 → 效果评估 → 政策调整 → 继续监测...
```

中国公共营养政策遵循此核心周期，从碘缺乏病的消除到营养改善计划的实施。

## 6. 临床营养决策路径

```
入院 → 营养风险筛查(NRS 2002等) → 
├── 无风险 → 定期复查
└── 有风险 → 营养评定(SGA等) →
    ├── 可经口进食 → 膳食处方 + ONS
    ├── 肠内营养(EN) → 管饲途径选择 + EN制剂选择
    └── 肠内不可用 → 肠外营养(PN) → 外周/中心静脉
        → 动态评估 → 调整方案 → 评估结局
```

核心原则：肠内优先（"If the gut works, use it"）

## 7. 膳食-疾病关系证据分级模式

按WCRF/AICR框架：
```
确立因果关系 ←→ 可能增加/降低风险 ←→ 有限证据 ←→ 不太可能有关联
```

类似中国DRIs的证据等级体系：
```
A级(系统评价/RCT) → B级(队列/病例对照) → 
C级(横断面/生态学研究) → D级(专家意见/个案)
```

## 8. 中国营养科学发展的三大里程碑

```
必需营养素发现时代(1900-1950) → 
营养缺乏病防治时代(1950-1980) → 
慢性病预防与营养优化时代(1980-至今)
```

当前中国处于第三个时代，面临营养不足和营养过剩的双重挑战——"双重负担"（Double Burden of Malnutrition）。

## 9. 膳食质量的评价框架

现代营养学评价膳食质量的四个维度：
1. **充足性（Adequacy）**：所有必需营养素的摄入满足DRIs
2. **平衡（Balance）**：各营养素之间比例适当（如蛋白质-能量比、钠-钾比）
3. **能量控制（Calorie Control）**：能量摄入与消耗平衡，维持健康体重
4. **多样性（Variety）**：各类食物的广泛摄入，最大化食物生物活性成分的覆盖

这四个维度构成了中国居民膳食指南(2016)的科学基础。
