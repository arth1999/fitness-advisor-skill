# 中国居民膳食指南（2022）速查表

## 中国居民平衡膳食宝塔（2022）— 成人每日推荐摄入量

> 基于1600\~2400kcal能量需要量水平。适用于18\~65岁轻体力活动健康成年人。

## 八条平衡膳食准则速查

| 准则 | 核心要点 | 关键数字 |
|------|----------|----------|
| 一、食物多样，合理搭配 | 五大类食物，谷类为主 | 每天≥12种，每周≥25种 |
| 二、吃动平衡，健康体重 | 能量平衡，保持运动 | 每周≥150分钟中强度运动，6000步/天 |
| 三、多吃蔬果奶类全谷大豆 | 蔬果奶豆为核心 | 蔬菜≥300g，水果200\~350g，奶≥300ml |
| 四、适量吃鱼禽蛋瘦肉 | 优选鱼禽，控制畜肉 | 总量120\~200g/天，鸡蛋1个/天 |
| 五、少盐少油控糖限酒 | 清淡饮食 | 盐<5g，油25\~30g，糖<25\~50g，酒精≤15g |
| 六、规律进餐足量饮水 | 三餐定时定量 | 水1500\~1700ml，早餐25%\~30%能量 |
| 七、会烹会选会看标签 | 营养标签+健康烹饪 | 高营养素密度食物优先 |
| 八、公筷分餐杜绝浪费 | 饮食卫生+节约 | 生熟分开，公筷公勺 |

## 高盐/高糖/高脂食品识别

| 类别 | 标准阈值 | 常见超标食品 |
|------|----------|------------|
| **高盐** | 钠≥800mg/100g | 咸菜、腊肉、薯片、方便面、酱油 |
| **高糖** | 糖≥11.5g/100ml(饮料) | 碳酸饮料、果味饮料、甜点 |
| **高脂** | 脂肪≥20g/100g | 油炸食品、膨化食品、奶油蛋糕 |
