# 术语表

## 膳食模式类

| 术语 | 英文 | 定义 |
|------|------|------|
| 膳食模式 | Diet Pattern | 膳食中各食物的品种、数量及其比例和消费频率，是长期形成的结果 |
| 平衡膳食模式 | Balanced Diet Pattern | 根据膳食营养素参考摄入量和营养健康状况，推荐的食物种类和比例，最大限度满足不同年龄阶段健康人群的营养需要 |
| 合理膳食 | Adequate Diet | 在平衡膳食基础上，考虑健康状况、地域资源、生活习惯、信仰等调整后的膳食 |
| 健康膳食 | Healthy Diet | 针对健康结局（慢性病发病、预期寿命等）而言，特征常包括少油盐、少加工食品、多蔬果全谷物 |
| 东方健康膳食模式 | Eastern Healthy Diet Pattern | 基于浙江、上海、江苏、福建等地区的膳食特点：食物多样、清淡少油、丰富蔬果、多鱼虾海产品、多奶类和豆类。慢性病发病率和死亡率较低，预期寿命较高 |
| 素食 | Vegetarian Diet | 不包含动物性肉类食物的膳食模式，包含全素食、蛋奶素食等多种亚型 |
| 纯母乳喂养 | Exclusive Breastfeeding | 除维生素/矿物质补充剂、口服补液盐、药物外，不给婴儿任何其他食物或液体，应持续6个月 |

## 食物描述类

| 术语 | 英文 | 定义 |
|------|------|------|
| 全谷物 | Whole Grain | 未经精细加工或经碾磨/粉碎/压片等处理，仍保留胚乳、胚芽、麸皮及天然营养成分的谷物 |
| 添加糖 | Added Sugar | 加工和制备食品时添加到食物或饮料中的糖或糖浆，包括蔗糖、葡萄糖、果糖、各种糖浆 |
| 深色蔬菜 | Dark-colored Vegetables | 深绿色、深黄色、紫色、红色等有色蔬菜，富含维生素、植物化学物和膳食纤维 |
| 营养素密度 | Nutrient Density | 食物中某营养素含量与其能量之比（每1000kcal的能量所含营养素质量单位数） |
| 能量密度 | Energy Density | 100g食物所含能量值（kcal/100g或kJ/100g），与水分和脂肪含量密切相关 |
| 纯能量食物 | Empty Calorie Food | 能量来源单一，除能量外几乎不含其他营养素的食物，如精制糖、淀粉、动植物油脂、酒等 |

## 定量描述类

| 术语 | 含义 | 参考标准 |
|------|------|----------|
| 多吃（喝） | 每天必须吃，或倡导比以前摄入量多 | 该食物在多数人群中摄入不足 |
| 常吃（适量） | 每周摄入频率3\~5次 | 膳食指南推荐量 |
| 少吃（喝） | 特别注意减少食用 | 大部分人群中摄入过高 |
| 控和限吃（喝） | 控制或限制食用 | 如糖、酒 |
| 高能量 | 提供≥400kcal/100g能量的食物 | — |
| 高盐 | 钠含量≥800mg/100g的食品 | 尤指零食和膨化食品 |
| 高糖 | 饮料糖含量≥11.5g/100ml | 固体饮料按冲调后计 |
| 高油/高脂 | 脂肪含量≥20g/100g的食品 | 尤指零食和膨化食品 |
| 低盐 | 钠含量≤120mg/100g(固体)或100ml(液体) | 预包装食品标准 |
| 低糖 | 碳水化合物（糖）≤5g/100g(固体)或100ml(液体) | 预包装食品标准 |
| 瘦肉 | 脂肪含量≤10%的畜肉和禽肉类 | — |

## 人群与生理指标

| 术语 | 英文 | 标准 |
|------|------|------|
| 成人正常BMI | Normal BMI | 18.5\~23.9 kg/m² |
| 成人超重 | Overweight | 24.0\~27.9 kg/m² |
| 成人肥胖 | Obesity | ≥28.0 kg/m² |
| 老年人适宜BMI | Elderly BMI | 20.0\~26.9 kg/m² |
| 中等强度身体活动 | Moderate-intensity Activity | 运动后即刻心率≈170-年龄（次/分）；主观感觉微出汗、可交谈但不能唱歌 |
| 经常参与体育锻炼 | Regular Exercise | 每周≥3次、每次≥30分钟中等及以上强度运动 |

## 营养素相关

| 术语 | 缩写 | 说明 |
|------|------|------|
| 膳食营养素参考摄入量 | DRIs | 中国营养学会2013年发布第8版 |
| 营养素参考值 | NRV | 预包装食品营养标签中用于比较营养素含量的参考值 |
| 血糖生成指数 | GI | 衡量食物引起血糖升高程度的指标 |
| 反式脂肪酸 | TFA | 部分氢化植物油中产生，摄入过多增加心血管疾病死亡风险 |
