# 第七章：特定人群 — 孕妇与乳母膳食指南

## 备孕和孕期妇女膳食指南

### 孕前准备

- 调整孕前体重至正常范围：**BMI 18.5\~23.9 kg/m²**
- 低体重者（BMI<18.5）：适当增加食物量和运动，每天加餐1\~2次，增奶100\~200ml
- 超重/肥胖者：减少高能量高脂肪食物，增加中等强度运动30\~90分钟/天
- 至少孕前3个月开始每天补充**叶酸400μg**
- 孕前每天摄入鱼、禽畜瘦肉和蛋类共计150g

### 孕期核心推荐

| 序号 | 推荐 |
|------|------|
| 1 | 调整孕前体重至正常范围，保证孕期体重适宜增长 |
| 2 | 常吃含铁丰富的食物，选用碘盐，合理补充叶酸和维生素D |
| 3 | 孕吐严重者少量多餐，保证摄入含必需量碳水化合物的食物（≥130g/天） |
| 4 | 孕中晚期适量增加奶、鱼、禽、蛋、瘦肉的摄入 |
| 5 | 经常户外活动，禁烟酒，保持健康生活方式 |
| 6 | 愉快孕育新生命，积极准备母乳喂养 |

### 孕期体重增长推荐

| 孕前BMI | 总增重范围(kg) | 中晚期每周增重(kg) |
|----------|---------------|-------------------|
| 低体重（<18.5） | 11.0\~16.0 | 0.46(0.37\~0.56) |
| 正常体重（18.5\~23.9） | 8.0\~14.0 | 0.37(0.26\~0.48) |
| 超重（24.0\~27.9） | 7.0\~11.0 | 0.30(0.22\~0.37) |
| 肥胖（≥28.0） | 5.0\~9.0 | 0.22(0.15\~0.30) |

### 孕期食物增量

| 时期 | 增加内容 |
|------|----------|
| 孕中期 | 奶类增至500g/天；鱼禽蛋肉增加150\~200g/天 |
| 孕晚期 | 奶类增至500g/天；鱼禽蛋肉增加175\~225g/天 |
| 全孕期 | 每周1\~2次动物血或肝脏（20\~50g），2\~3次海产鱼类 |

### 关键营养素补充

| 营养素 | 推荐量 | 来源 |
|--------|--------|------|
| 叶酸 | 400μg/天 | 补充剂+肝脏、蛋类、绿叶蔬菜 |
| 铁 | 中期24mg/天，晚期29mg/天 | 动物血、肝脏、红肉+维生素C蔬果 |
| 碘 | 碘盐 | 海产品（海带、紫菜等） |
| 维生素D | 400IU/天 | 补充剂+户外阳光照射 |
| 钙 | 1000mg/天 | 奶制品、豆制品 |

### 身体活动

- 健康孕妇每天≥30分钟中等强度身体活动
- 如散步、孕妇瑜伽、游泳等

## 哺乳期妇女膳食指南

### 核心推荐

- 合理安排乳母膳食，保证充足能量和营养素
- 保证充足的优质蛋白质（鱼、禽、蛋、瘦肉）
- 保证充足的钙（奶及奶制品每天400\~500ml）和碘
- 建立母乳喂养信念，促进乳汁分泌
- 合理饮用汤、水和茶（不宜过浓茶和咖啡）
- 产后进行科学的体重管理

### 增加的关键营养素

- **蛋白质**：每天比非哺乳期增加25g
- **钙**：每天1200mg
- **铁**：每天24mg
- **维生素A**：通过动物肝脏（每周1\~2次）补充
- **水分**：保证充足液体摄入，催乳汤水适量

### 产后体重管理

- 产后体重滞留是女性肥胖的重要原因
- 哺乳有助于消耗孕期储备脂肪（每天约消耗300\~500kcal）
- 通过合理膳食和逐渐恢复运动来控制体重
