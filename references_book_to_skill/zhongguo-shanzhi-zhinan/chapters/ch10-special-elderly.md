# 第十章：特定人群 — 老年人膳食指南

## 一、一般老年人膳食指南（65\~79岁）

### 生理特点

- 代谢能力下降，呼吸功能衰退，心脑功能衰退
- 视觉、听觉、味觉、嗅觉等感官反应迟钝
- 肌肉衰减（40岁后每年丢失约8%，70岁后加速至每年约15%）
- 咀嚼和消化能力下降，容易出现营养不良

### 核心推荐

| 序号 | 推荐 |
|------|------|
| 1 | 食物品种丰富，动物性食物充足，常吃大豆制品 |
| 2 | 鼓励共同进餐，保持良好食欲，享受食物美味 |
| 3 | 积极户外活动，延缓肌肉衰减，保持适宜体重 |
| 4 | 定期健康体检，测评营养状况，预防营养缺乏 |

### 关键数值

| 项目 | 推荐值 | 备注 |
|------|--------|------|
| BMI适宜范围 | **20.0\~26.9 kg/m²** | 不同于成人18.5\~23.9 |
| 动物性食物总量 | 120\~150 g/天 | 鱼40\~50g，畜禽肉40\~50g，蛋40\~50g |
| 奶及奶制品 | 300\~400 ml/天 | 或相当量蛋白质的奶制品 |
| 大豆制品 | 相当于15g大豆/天 | 豆腐、豆浆、豆腐干等换着吃 |
| 谷类 | 200\~250 g/天 | 含全谷物和杂豆 |
| 蛋白质 | 1.0\~1.2 g/kg体重/天 | 抗阻训练者≥1.2\~1.5g/kg |
| 优质蛋白质比例 | ≥50% | 来自动物性食物和大豆 |

### 延缓肌肉衰减的关键营养素

1. **蛋白质**：优质蛋白质分散到三餐
2. **n-3多不饱和脂肪酸**：海鱼类（EPA/DHA）、亚麻籽油
3. **维生素D**：户外阳光照射+必要时补充
4. **抗氧化营养素**：维生素C、维生素E、类胡萝卜素、硒
5. **钙**：牛奶300\~400ml/天

### 身体活动建议

- 散步、快走、太极拳、门球等动作缓慢柔和的运动
- 户外活动促进维生素D合成
- 运动靶心率：170 - 年龄（如70岁→100次/分）
- 减少久坐时间，每小时起来活动

### 重要提醒

- 不要求偏胖老年人快速降低体重，维持稳定
- 无主动减重措施下体重明显下降→主动进行营养和医学咨询
- 定期体检，做营养状况测评

## 二、高龄老年人膳食指南（≥80岁）

### 核心推荐

- 多种方式鼓励进食，保证充足食物摄入
- 选择适当加工方法，使食物**细软易消化**
- 经常监测体重，进行营养评估和膳食指导
- 衰弱及其测评——及时干预
- 合理使用营养强化食品和营养素补充剂
- **吞咽障碍**老年人选用及制作易食食品
- 坚持身体活动和益智活动

### 高龄特殊关注点

- **口腔和牙齿健康**：维护咀嚼功能，必要时使用义齿
- **食物加工**：切碎、煮烂、打泥；采用蒸、煮、炖、焖
- **体重监测**：经常测量，防止体重过度下降（衰弱标志）
- **营养补充**：在医师或营养师指导下合理使用营养补充剂
- **吞咽困难**：调整食物质地（糊状、羹状），注意进食姿势

### 老年营养风险筛查要点

- 近期体重下降（3个月内下降>5%）
- 进食量明显减少
- 咀嚼吞咽困难
- 多种慢性病同时用药
- 独居、缺乏社会支持

## 老年饮食与运动的中国特色

- 东方传统饮食文化中，老年人偏好粥、汤等易消化食物——符合生理特点
- 中国社区老年食堂（长者食堂）正在推广，促进老年人共同进餐
- 太极拳、八段锦等传统运动适合老年人——低冲击、改善平衡
