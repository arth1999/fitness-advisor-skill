# 第四章：准则三、四 — 蔬果奶类+全谷大豆 & 鱼禽蛋肉适量

## 准则三：多吃蔬果、奶类、全谷、大豆

### 核心推荐

- 餐餐有蔬菜，每天摄入**≥300g**蔬菜，深色蔬菜应占**1/2**
- 天天吃水果，每天摄入**200\~350g**新鲜水果，果汁不能代替鲜果
- 奶制品摄入量相当于每天**300ml以上**液态奶
- 经常吃全谷物、豆制品，适量吃坚果

### 蔬菜选择与烹饪

**挑选三原则**：
1. **重"鲜"**——优先选择新鲜应季蔬菜，减少营养流失
2. **选"色"**——深色蔬菜（深绿、深黄、紫色、红色）占1/2以上
3. **多"品"**——叶菜、根茎、茄果、菌藻等多品种轮换

**保持营养的烹饪方法**：
1. 先洗后切——减少水溶性维生素流失
2. 开汤下菜——缩短加热时间
3. 急火快炒——减少营养损失
4. 炒好即食——避免长时间保温

### 奶类和大豆摄入

- **奶制品**：鲜奶、酸奶、奶粉、奶酪多样化。乳糖不耐受者可选低乳糖奶或酸奶
- **大豆制品**：豆腐、豆浆、豆腐干、腐竹等换着花样吃。整粒熟大豆蛋白质消化率约65%，豆腐/豆浆可达80%以上
- **全谷物**：全麦粉、糙米、燕麦、荞麦等——每天50\~150g
- **坚果**：每周70g左右（约每天10g），花生、核桃、杏仁等

### 科学证据

- 增加蔬菜摄入降低心血管疾病、食管癌、结肠癌风险
- 增加水果摄入降低心血管疾病、胃癌、食管癌风险
- 增加奶类摄入提高骨密度，降低结直肠癌风险
- 增加大豆制品摄入降低乳腺癌、骨质疏松风险
- 适量坚果摄入降低全因死亡和心血管疾病风险

## 准则四：适量吃鱼、禽、蛋、瘦肉

### 核心推荐

- 成年人每天动物性食物总量：**120\~200g**
- 每周鱼类**≥2次**或300\~500g
- 每周畜禽肉类300\~500g
- 每周蛋类300\~350g（约每天1个鸡蛋）
- 优先选择鱼和禽类（脂肪含量较低）
- 少吃肥肉、烟熏和腌制肉制品

### 各类动物性食物详解

| 类别 | 每天建议(g) | 主要营养特点 |
|------|------------|------------|
| 畜禽肉 | 40\~75 | 优质蛋白质、铁、维生素B族；畜肉脂肪含量较高（选择瘦肉） |
| 水产品 | 40\~75 | 优质蛋白质、不饱和脂肪酸（EPA/DHA）、低脂肪 |
| 蛋类 | 40\~50（约1个） | 全面营养，蛋黄含胆碱、卵磷脂、维生素A、叶黄素、锌 |

### 控制摄入的实用方法

1. **控制总量，分散食用**——分散到每餐，避免集中摄入
2. **小份量，量化有数**——了解常见份量（如50g肉≈手掌心大小）
3. **在外就餐时减少肉类摄入**——主动选择荤素搭配
4. **多蒸煮，少烤炸**——减少高温烹饪产生的有害物质
5. **喝汤更要吃肉**——汤中营养成分远不及肉本身

### 科学证据

- 增加鱼肉摄入可降低成年人全因死亡风险（确信证据A级）
- 增加鱼肉摄入可降低脑卒中和中老年痴呆风险
- 鸡蛋适量摄入（每天1个）不增加健康人心血管疾病风险
- 过多摄入畜肉可增加肥胖和2型糖尿病风险
- 过多摄入烟熏肉制品可增加胃癌和食管癌风险
