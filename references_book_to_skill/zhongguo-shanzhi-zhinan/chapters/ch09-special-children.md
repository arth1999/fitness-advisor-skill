# 第九章：特定人群 — 儿童膳食指南

## 一、学龄前儿童膳食指南（2\~6岁）

### 核心推荐

1. **食物多样，规律进餐，自主进食不挑食**
   - 每天3餐+2次加餐
   - 每天饮奶300\~400ml
   - 培养专注进食和自主进食能力

2. **每天饮奶，足量饮水，合理选择零食**
   - 首选白水，控制含糖饮料
   - 零食选择水果、奶制品、少量坚果
   - 避免高盐高糖高脂零食

3. **合理烹调，少盐少油，从小培养清淡口味**
   - 2\~3岁：盐<2g/天，油15\~20g/天
   - 4\~5岁：盐<3g/天，油20\~25g/天

4. **培养认知食物与喜爱食物**
   - 让儿童参与食物选择和制作
   - 通过游戏认识食物名称、颜色、形状

5. **经常参加户外活动和身体活动**
   - 每天≥180分钟身体活动（包括60分钟中等及以上强度）
   - 限制久坐行为和视屏时间（<1小时/天）

### 学龄前儿童食物推荐量

| 食物类别 | 2\~3岁 | 4\~5岁 |
|----------|--------|--------|
| 谷类 | 85\~100g | 100\~150g |
| 蔬菜 | 200\~250g | 250\~300g |
| 水果 | 100\~150g | 150g |
| 畜禽肉鱼 | 50\~75g | 50\~75g |
| 蛋类 | 50g | 50g |
| 奶类 | 300\~400ml | 300\~400ml |
| 大豆 | 5\~15g | 10\~20g |
| 食用油 | 15\~20g | 20\~25g |

## 二、学龄儿童膳食指南（6\~17岁）

### 核心推荐

1. **提高营养素养**
   - 积极学习营养健康知识
   - 主动参与食物选择和制作——培养生活技能
   - 家庭创造健康食物环境，学校构建健康食物环境

2. **养成健康的饮食行为**
   - **吃好早餐**：含谷薯类、奶蛋类、蔬菜水果至少3类食物
   - **合理选择零食**：选择清洁卫生、营养丰富的食物
   - **在外就餐做到合理搭配**：注意荤素比例

3. **选择健康饮品**
   - 天天喝奶（每天300ml以上）
   - 足量饮水（6岁800\~1000ml，7\~10岁1000\~1300ml，11\~13岁1300\~1500ml，14\~17岁1500\~1700ml）
   - **不喝含糖饮料**，禁止饮酒和含酒精饮料

4. **积极开展身体活动**
   - 每天累计≥60分钟中高强度身体活动
   - 每周≥3天高强度身体活动+抗阻运动
   - 减少视屏时间（<2小时/天）
   - 保证充足睡眠（6\~12岁9\~12小时，13\~17岁8\~10小时）

5. **保持适宜的体重增长**
   - 定期监测身高和体重
   - 正确认识和评估体型
   - 预防和改善营养不良（消瘦、生长迟缓）
   - 预防和控制超重肥胖——不过度节食

|------|-----------|---------|---------|---------|------------|---------|--------|---------|
| 男 | 2000\~2400 | 225\~300 | 400\~500 | 250\~350 | 100\~150 | 50 | 300 | 15\~25 |
| 女 | 1800\~2200 | 200\~250 | 350\~450 | 250\~300 | 75\~125 | 50 | 300 | 15\~20 |

### 关键提示

- 儿童肥胖干预应通过调整膳食结构+增加运动，避免极端节食
- 青春期铁需要量增加（尤其女孩月经来潮后），注意富含铁食物的摄入
- 维生素D缺乏在我国儿童中较普遍，注意户外活动和补充
