# 第六章：准则七、八 — 会烹会选+会看标签 & 公筷分餐+杜绝浪费

## 准则七：会烹会选，会看标签

### 核心推荐

- 认识食物营养特点，挑选**营养素密度高**的食物
- 学会通过**食品营养标签**比较选择健康食品
- 学习烹饪，掌握新工具，传承传统美味
- 在外就餐或外卖时，注意适宜份量和荤素搭配

### 食物挑选技巧

**1. 认识食物营养特点**
- 了解五大类食物的主要营养素贡献
- 谷薯类→碳水化合物；蔬菜水果→维生素/矿物质/膳食纤维；动物性食物→蛋白质/脂肪；奶类/大豆→钙/蛋白质；油脂→脂肪酸

**2. 了解营养素密度**
```
营养素密度 = (食物中某营养素含量 / 该食物能量值) × 1000
```
- 高营养素密度食物：在同等能量下提供更多营养素
- 鼓励"多吃"：蔬菜、水果、全谷物、奶类、大豆、鱼虾、蛋类
- 适当"少吃"：高盐食品、高油食品、高糖食品、加工肉制品

**3. 利用当季、当地食物资源**
- 优先选择本地当季食物，更新鲜、营养更好

### 食品营养标签阅读

**三步法**：
1. **看配料表**——按含量降序排列，排第一的是主要原料
2. **看营养成分表**——能量、蛋白质、脂肪、碳水化合物、钠是强制标示项
3. **看营养素参考值(NRV%)**——某营养素含量占每日推荐摄入量的百分比

**营养声称**：
- "高钙"：钙含量≥30% NRV/100g
- "低脂"：脂肪≤3g/100g(固体)
- "低糖"：糖≤5g/100g(固体)
- "低盐"：钠≤120mg/100g

### 健康烹饪原则

| 推荐方法 | 理由 |
|----------|------|
| 多蒸、煮、炒 | 保留营养，减少用油 |
| 少煎、炸 | 减少高温产生的有害物质 |
| 用天然香料 | 葱姜蒜辣椒醋替代部分盐 |
| 选择新型烹饪工具 | 微波炉、空气炸锅、电蒸锅减少用油 |
| 控制烹调油用量 | 使用定量油壶 |

### 外卖及在外就餐技巧

1. 纳入膳食计划——不把外卖当"例外"
2. 挑选主食不忘全谷物——尽量选糙米饭/全麦面食
3. 挑选菜肴少油炸，注意荤素搭配
4. 不要大份量，适量不浪费
5. 主动提出少油、少盐健康诉求

## 准则八：公筷分餐，杜绝浪费

### 核心推荐

- 选择当地、新鲜卫生的食物，不食用野生动物
- 食物制备**生熟分开**，储存得当，彻底加热
- 使用**公筷公勺**，采用分餐或份餐
- 珍惜食物，按需备餐，不铺张浪费
- 社会餐饮倡导文明用餐，促进食物系统可持续发展

### 食品安全六要点

1. **首选当地当季食物**——更新鲜，减少储运损失
2. **学会辨别食物新鲜程度**——观察色泽、气味、质地
3. **水果蔬菜要洗净**——流水冲洗，必要时浸泡
4. **食物生熟要分开**——生熟砧板、刀具、容器分开
5. **食物加热和煮熟**——彻底加热，中心温度≥70°C
6. **食物储存要得当**——冷藏≤4°C，冷冻≤-18°C；熟食2小时内冷藏

### 公筷分餐的公共卫生意义

- 减少幽门螺杆菌等食源性病原体传播
- 新冠肺炎疫情后更强调饮食卫生的重要性
- 在家吃饭也应鼓励公筷公勺和分餐制

### 杜绝浪费

- **按需选购，合理储存**——减少家庭食物浪费
- **小份量、光盘行动**——每餐适量
- **合理利用剩饭剩菜**——确保安全前提下再加工食用
- **外出就餐按需点餐**——不铺张，剩余打包

### 食物系统可持续发展

- 保障粮食安全
- 保证资源的可持续性（水资源、土地、能源）
- 发展可持续经济和加强人类福祉
- 传承健康文明，树立饮食新风
