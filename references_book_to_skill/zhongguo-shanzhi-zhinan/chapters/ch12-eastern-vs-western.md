# 第十二章：东方健康膳食模式与中西对比

## 东方健康膳食模式

### 定义

《中国居民膳食指南（2022）》**首次**定义和推荐"东方健康膳食模式"（Eastern Healthy Diet Pattern）。该模式以中国**浙江、上海、江苏、福建**等江南及东南沿海地区为代表。

### 核心特征

| 特征 | 描述 |
|------|------|
| 烹调方式 | **清淡少盐**，蒸、煮、炒为主 |
| 食物多样性 | 食物种类丰富多样 |
| 谷物 | 米类为主，搭配杂粮 |
| 蔬菜水果 | 丰富多样，摄入量充足 |
| 水产品 | 经常吃鱼虾等海产品 |
| 豆制品 | 经常食用豆腐、豆浆等 |
| 奶类 | 摄入量相对较高 |

### 健康效果

该地区居民：
- 高血压等慢性病患病率**较低**
- 预期寿命**较高**
- 膳食模式被认为是传统东方膳食向健康膳食模式转变的优秀范例

### 不同地区的膳食模式

| 地区 | 特点 | 优势 | 不足 |
|------|------|------|------|
| 江南/东南沿海 | 清淡少盐、鱼虾多、蔬果丰富 | 慢性病率低，寿命长 | — |
| 北方地区 | 面食为主、口味偏咸 | 面食文化深厚 | 盐摄入偏高 |
| 西北地区 | 牛羊肉多、奶制品多 | 优质蛋白质充足 | 蔬菜水果偏少 |
| 西南地区 | 辣味烹饪、食材多样 | 食材种类丰富 | 油脂用量有时偏高 |

## 中国膳食指南 vs 西方膳食指南对比

### 核心差异

| 维度 | 中国膳食指南(2022) | 美国膳食指南(2020-2025) | 地中海饮食 |
|------|-------------------|------------------------|------------|
| **基础框架** | 膳食宝塔（5层） | MyPlate（餐盘） | 金字塔 |
| **主食定位** | **谷类为主**（占总能量50\~65%） | 谷物约占1/4餐盘 | 全谷物为基础 |
| **蛋白质来源** | 大豆制品地位突出 | 肉类、禽类、蛋类为主 | 鱼类+豆类 |
| **乳制品** | 300ml/天（奶类消费低） | 3杯/天（约720ml） | 适量酸奶和奶酪 |
| **烹调油** | 25\~30g/天（定量限制） | 强调植物油种类（无明确上限量） | 橄榄油为核心 |
| **水产品** | 40\~75g/天，优先选择 | 8盎司/周（约226g） | 高频次大量 |
| **大豆/豆制品** | **独立食物组，核心推荐** | 归入蛋白质食物组 | 归入豆类组 |
| **茶** | 推荐饮茶 | 未提及 | 未提及 |
| **红肉** | 少吃，优选禽肉 | 限制饱和脂肪来源 | 极少红肉 |
| **饮酒** | ≤15g/天酒精 | 男性≤2杯/天，女性≤1杯 | 适量红酒 |

### 独特的"东方元素"

1. **谷类为主+全谷物**：突出碳水化合物基础地位（50\~65%能量），区别于低碳水饮食风潮
2. **大豆和豆制品**：豆腐、豆浆等在中国膳食中有独立且突出的地位，是优质蛋白质重要来源
3. **茶文化**：明确推荐饮茶，且有流行病学研究数据支持
4. **清淡烹饪传统**：蒸、煮、炒为主，更健康
5. **药食同源意识**：食物与健康关系深入人心
6. **深色蔬菜强调**：深色蔬菜应占1/2——中国传统蔬菜种类丰富
7. **分餐制+公筷**：公共卫生考量结合现代文明风尚
8. **东方健康膳食模式**：基于中国本土流行病学证据，而非简单照搬地中海/DASH

### 中国膳食指南的独特优先关切

| 问题 | 中国现状 | 国际对比 |
|------|----------|----------|
| 钠摄入过量 | 极为严重（平均约10.5g/天） | 全球最高水平之一 |
| 全谷物摄入不足 | 精白米面消费为主 | 全球排名倒数 |
| 奶类摄入严重不足 | 仅约发达国家1/10 | 极低水平 |
| 水果摄入不足 | 低于推荐量 | 造成最大死亡负担的饮食因素之一 |
| 畜肉摄入过量 | 以猪肉为主，增长趋势明显 | 红肉占比偏高 |
| 含糖饮料消费上升 | 儿童青少年消费量快速增长 | 与全球趋势一致 |

### 与其他健康膳食模式的共同点

- 强调食物多样性和植物性食物基础
- 限制添加糖、饱和脂肪和钠
- 保持能量平衡和适宜体重
- 鼓励身体活动
- 重视生命周期各阶段的特定营养需求
