# 动作模式关系与发展序列

## 人类动作发展里程碑

所有婴儿（无论文化、气候、家庭状况）在生命前21个月按以下序列发展：

### 姿势发展序列
```
躺着 → 双肘支撑俯卧 → 四肢着地 → 坐立 → 跪着 → 站立 → 单腿站立
```

### 模式发展序列
```
滚动 → 匍匐爬行 → 手爬行 → 攀爬 → 弓步 → 下蹲 → 站立 → 行走 → 跑
```

每一个阶段建立在前一个阶段的基础上，每一个发展动作都为后续更复杂的动作奠定基础。这些是唯一被公认的人类运动指标。

## FMS 7项动作筛查

FMS筛查的7个动作模式来源于发展进程，具有自限性。

| 序号 | 动作模式 | 英文 | 筛查目的 | 发展姿势 |
|------|----------|------|----------|----------|
| 1 | 主动直腿上抬 | Active Straight Leg Raise (ASLR) | 髋关节灵活性、核心稳定性、下肢分离能力 | 有支撑（仰卧） |
| 2 | 肩部灵活性 | Shoulder Mobility | 肩关节活动范围、肩胛骨稳定性、胸椎灵活性 | 有支撑（站立→原始仰卧） |
| 3 | 旋转稳定性 | Rotary Stability | 躯干与骨盆在多平面上的反射性稳定 | 悬空（四肢着地） |
| 4 | 俯卧撑 | Trunk Stability Push-up | 上肢与下半身水平位稳定性、全身张力 | 悬空 |
| 5 | 直线弓步蹲 | In-line Lunge | 下肢对称性、动态平衡、髋/膝/踝协调 | 站立 |
| 6 | 跨栏步 | Hurdle Step | 单腿站立稳定性与迈步控制 | 站立 |
| 7 | 过顶深蹲 | Deep Overhead Squat | 全身对称性、多关节协调、整体动作质量 | 站立 |

### 纠正优先级序列（从简单到复杂）

```
主动直腿上抬 → 肩部灵活性 → 脚踝灵活性 → 旋转稳定性 → 俯卧撑 → 弓步蹲 → 跨栏步 → 深蹲
```

**关键原则：**
- 先解决灵活性瓶颈（ASLR、肩部灵活性、踝关节灵活性）
- 再处理运动控制与稳定性（旋转稳定性、俯卧撑）
- 最后整合至复杂直立模式（弓步蹲、跨栏步、深蹲）

## SFMA 首要层级模式

SFMA在站立位评估7个首要层级动作，按复杂度从简单到高级排列：

```
颈椎 → 肩部 → 多部位屈曲或伸展 → 多部位旋转 → 单腿站立 → 深蹲
```

| 动作 | 英文 | 评估内容 |
|------|------|----------|
| 颈椎运动 | Cervical Spine | 头颈部运动控制与灵活性 |
| 肩部模式 | Shoulder Pattern | 肩带功能 |
| 多部位屈曲/伸展 | Multi-Segmental Flexion/Extension | 脊柱分节运动 |
| 多部位旋转 | Multi-Segmental Rotation | 躯干旋转能力 |
| 单腿站立 | Single Leg Stance | 单侧平衡与稳定 |
| 深蹲 | Overhead Deep Squat | 全身整合功能 |

### SFMA分解逻辑

每个首要层级模式出现DN（功能障碍、无疼痛）时，通过系统分解确定问题来源于：
- **软组织或关节自由度丧失**（灵活性不足）
- **完整性、稳定性或运动控制丧失**（稳定性不足）

> 当评估揭示3个以上DN模式时，先集中处理1-2个最基本的模式。纠正更复杂的模式（如多部位屈曲）之前，先确认其组成部分（如颈椎、肩部）的功能完整性。

## 姿势稳定性阶梯

| 姿势层级 | 英文 | 静态稳定性 | 动态稳定性 | 对应筛查 |
|----------|------|------------|------------|----------|
| 有支撑 | Supported | 伸展（手臂或腿） | 滚动 | FMS: ASLR, 肩部灵活性 / SFMA: 滚动, 头颈 |
| 悬空 | Hovering | 四肢着地, 平板支撑 | 爬行 | FMS: 俯卧撑, 旋转稳定性 / SFMA: 四足摇摆 |
| 堆叠 | Stacked | 坐着, 半跪, 双膝跪地 | 坐→跪过渡 | SFMA: 长坐躯干旋转, 窄距单膝跪 |
| 站立 | Standing | 站立, 单腿站立 | 行走/迈步/弓步/下蹲 | FMS: 蹲/弓步/跨栏 / SFMA: 蹲/单腿站立/屈伸/旋转 |

> 从基础（地面/有支撑）开始确认稳定性，逐步进阶到站立动态控制。如果在某一姿势下呼吸出现急促或变浅，说明已触及控制能力的边界。

## 逐个关节理论模式图

```
足弓(稳定) → 踝关节(灵活) → 膝关节(稳定) → 髋关节(灵活)
    ↕
腰椎(稳定) → 胸椎(灵活) → 肩胛骨(稳定) → 盂肱关节(灵活)
```

### 灵活性优先处理区域
1. **踝关节** — 一切动作从地面向上传导，踝关节背屈不足是预测损伤的最强指标之一
2. **髋关节** — 灵活性受限最常见的区域之一
3. **胸椎** — 缺乏自由活动的另一关键区域

### 功能障碍传导链
- 踝关节受限 → 膝关节代偿 → 髋关节/下背部疼痛
- 胸椎僵硬 → 腰椎过度活动 → 肩关节功能障碍

## 动作模式关系检验法（筛查→假设→验证）

以过顶深蹲功能障碍为例的系统推理路径：

```
深蹲功能障碍
├── 测试ASLR：正常？→ 不太可能是髋关节问题
├── 测试肩部灵活性：受限？→ 可能是肩部/胸椎问题
│   └── 改善肩部灵活性后重测深蹲 → 改善？→ 肩部是驱动因素
├── 测试弓步蹲：不对称？→ 可能是脚踝问题
│   └── 脚踝活动度测试 → 确认后处理
└── 系统收集：不跳过任何模式，因为功能障碍可能影响多种模式
```

## FCS 能力发展序列

```
动作控制 → 姿势控制 → 爆发力控制 → 冲击力控制
(运动控制筛查) (提物走筛查) (跳远) (三连跳)
```

FCS在FMS确认功能性动作胜任后，进一步量化：
- 能否在多个平面上展示平衡和控制能力？
- 能否在负荷和疲劳下保持姿势和对位？
- 能否通过身体有效产生力量？
- 能否有效吸收和重新使用力量？
