# 功能性动作科学 — Functional Movement Science

## 技能描述

基于格雷·库克（Gray Cook）功能性动作系统（FMS/SFMA/FCS）的完整知识体系，提供动作筛查、功能评估、纠正策略和功能性训练的系统化方法论。本技能涵盖 FMS 7项动作筛查评分标准、SFMA 选择性功能动作评估的分级分解逻辑、逐个关节理论（Joint-by-Joint）、纠正性练习层级模型、红黄绿灯训练决策系统，以及三大动作原则与4x4矩阵框架。

## 核心能力

当用户咨询以下领域问题时，应用本技能知识：

1. **动作筛查与评估** — FMS评分标准解读、SFMA首要层级分解与突破、YBT平衡测试
2. **功能障碍诊断** — 疼痛与功能障碍的分离（FN/FP/DN/DP四象限）、灵活性vs稳定性问题判定
3. **纠正策略设计** — 从局部自由到整体控制的纠正路径、纠正性练习4x4矩阵、RNT反应性神经肌肉训练
4. **训练安全决策** — 红黄绿灯练习分级、动作层次（健康→功能→体能→技能）评估
5. **功能性训练原则** — 三大动作原则、逐个关节理论、呼吸与运动控制的关系
