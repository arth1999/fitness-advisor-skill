# 功能性动作科学 — 快速参考卡片

## FMS 7项动作筛查速查

| # | 动作 | 0分 | 1分 | 2分 | 3分 |
|---|------|-----|-----|-----|-----|
| 1 | 主动直腿上抬 | 疼痛 | 踝骨垂线在大腿中点以下 | 踝骨垂线在大腿中点与髌骨之间 | 踝骨垂线在大腿中点以上（与对侧比较） |
| 2 | 肩部灵活性 | 疼痛 | 双拳间距>1.5手长 | 双拳间距<1.5手长 | 双拳间距<1手长 |
| 3 | 旋转稳定性 | 疼痛 | 无法完成同侧动作 | 可完成同侧动作 | 可完成对侧动作（同侧肘触对侧膝） |
| 4 | 俯卧撑 | 疼痛 | 男性拇指在头顶以上/女性拇指在下巴以上无法撑起 | 男性拇指与下巴齐平/女性拇指与锁骨齐平撑起 | 男性拇指与头顶齐平/女性拇指与下巴齐平撑起 |
| 5 | 直线弓步蹲 | 疼痛 | 无法保持平衡或膝盖/足部偏离 | 在板内完成但膝盖/足部偏离 | 保持躯干垂直，膝盖轻触板面，足部在矢状面 |
| 6 | 跨栏步 | 疼痛 | 髋/膝/踝在矢状面外 | 在矢状面内但有代偿 | 在矢状面内无代偿 |
| 7 | 过顶深蹲 | 疼痛 | 大腿高于水平面 | 大腿低于水平面但有代偿 | 大腿低于水平面，上半身与胫骨平行或垂直 |

> 注：以上为简化描述。完整评分需结合测试杆位置、身体对位、代偿模式综合判断。具体评分细则见 `chapters/01-fms-scoring.md`。

## 评分优先级决策算法

```
筛查得分处理优先级：
1. 识别0分（疼痛）→ 保护，转诊医疗评估
2. 识别1分（功能障碍）→ 纠正
3. 识别不对称模式（1/2, 2/3）→ 恢复对称性
4. 识别2分 → 通过发展继续纠正（功能性训练）
5. 3分 → 继续当前训练，进入FCS评估
```

## SFMA 首要层级分解算法

```
进行SFMA首要层级评估
    ↓
标记每个模式：FN / FP / DN / DP
    ↓
FP or DP？→ 需要医疗评估疼痛来源
    ↓
DN模式 > 3个？→ 仅处理1-2个最基本的模式
    ↓
选择最初级DN模式（颈椎→肩部→多部位屈伸→多部位旋转→单腿站立→深蹲）
    ↓
分解：软组织/关节自由度丧失？or 完整性/稳定性/运动控制丧失？
    ↓
应用干预 → 重新测试 → 确认变化 → 进阶到下一个DN模式
```

## 纠正性练习3步流程

```
步骤1：重置 (Reset)
├── 被动灵活性干预（手法/软组织/泡沫轴）
├── 主动辅助灵活性（外部帮助/弹力带辅助）
├── 主动灵活性（动态拉伸/PNF）
└── 呼吸重置（3:1呼-吸比 → 1:1）
    ↓ 重新测试，确认积极变化
步骤2：强化 (Reinforce)
├── 保护（-）：移除加重功能障碍的活动/练习
├── 纠正（+）：纠正性练习4x4矩阵
│   ├── 辅助动作（被动/主动辅助）
│   ├── 独立动作（主动）
│   ├── 反应性模式（RNT）
│   └── 独立抗阻动作（静态/动态加载）
└── 发展（=）：逐步增加负荷与控制要求
    ↓
步骤3：重建 (Reload)
└── 回归正常训练/活动，持续监控
```

## 红黄绿灯练习速查

| FMS模式 | 红灯（避免） | 黄灯（谨慎） | 绿灯（安全） |
|----------|-------------|-------------|-------------|
| 主动直腿上抬 | 跑步、冲刺、硬拉、高翻、壶铃摇摆 | 深蹲、蹬台阶 | 单膝跪地下劈/上拉、上肢训练、核心训练 |
| 肩部灵活性 | 过头推举、引体向上、倒立、抓举 | 划船、水平推 | 硬拉、高翻、摇摆、下半身训练 |
| 跨栏步 | 跑步、冲刺、单腿练习 | 对称负重深蹲、硬拉 | 单膝跪地下劈/上拉、手提箱硬拉、上肢训练 |
| 深蹲 | 负重深蹲、高翻、抓举 | 单腿练习、分腿蹲、弓步蹲 | 硬拉、土耳其起立、单膝跪地下劈/上拉 |

## 功能性训练三大原则速记

| 原则 | 一句话 | 实践应用 |
|------|--------|----------|
| 原则一：首先动作好 | 质量先于数量 | 先筛查FMS，确认达到最低标准后再进行体能训练 |
| 原则二：保护→纠正→发展 | 不伤害→消除功能障碍→增强能力 | 0分保护、1分纠正、2分发展、3分加强 |
| 原则三：创建系统 | 标准化操作程序 | 测试→行动→重测 循环，4x4矩阵指导决策 |

## 4x4矩阵速查

|  | 意识(?) | 保护(-) | 纠正(+) | 发展(=) |
|--|---------|---------|---------|---------|
| **生产力** | 技能/运动表现特定测试 | 移除压力大的活动 | + | = |
| **身体强健** | FCS/1RM/VO₂max | 移除造成伤害的练习 | + | = |
| **全面健康** | FMS/风险因素 | 移除动作风险因素 | + | = |
| **身体健康** | SFMA/生命体征 | 移除健康风险因素 | + | = |

## 沟通脚本模板

### 筛查前
> "我将让你做一些基本动作。在做每个动作之前我会告诉你，如果你在动作过程中感到任何疼痛，请立即告诉我。"

### 筛查后（不先解释，先问客户）
> "你认为你做得怎么样？"
> "你认为自己在哪个方面做得最差？"
> "你有感觉到任何不适或不平衡吗？"

### 创造体验后
> "你注意到前后的变化了吗？你感觉哪里不一样？"

### 设定预期
> "在你感觉到变好的两三天前，你的动作会做得更好。如果你发现动作做得更好，但在4天内没有感觉更好，来找我复查。"

## 关键数字记忆

| 数值 | 含义 |
|------|------|
| 21分 | FMS满分（7项×3分） |
| 14分 | FMS风险阈值（低于此分受伤风险增加） |
| ~20% | 接受FMS筛查人群中至少一种模式出现疼痛的比例 |
| 4-6周 | 新活动开始后运动表现进步最大的窗口期（神经适应） |
| 70% | 单纯呼吸练习后观察到动作积极变化的概率 |
| 85% | 系统性收集数据后，首次干预即可测量到积极变化的概率 |
| 3:1 | 初始呼吸训练的呼气与吸气比率 |
| 5+ | 风险因素数量超过5个时，损伤概率显著上升 |
| 3 | 存在3个以上SFMA的DN模式时，应优先处理1-2个最基本模式 |
