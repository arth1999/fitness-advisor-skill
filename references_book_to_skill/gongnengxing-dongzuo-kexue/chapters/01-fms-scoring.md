# 第1章 — FMS 7项动作筛查评分标准

## FMS概述

功能性动作筛查（Functional Movement Screen, FMS）由7个动作模式组成，用于快速识别动作功能障碍和疼痛。FMS并非诊断工具，而是筛查工具——它告诉我们是否需要更深入的调查，而非告诉我们"问题是什么"。

### 评分系统核心逻辑

| 分数 | 含义 | 本质上意味着 |
|------|------|-------------|
| **0分** | 动作时出现疼痛 | 有机体问题——动作存在物理障碍。需要医疗评估。 |
| **1分** | 无法完成动作 | 环境问题——面对环境要求时身体/行为反应不足。功能障碍。 |
| **2分** | 通过代偿完成动作 | 可通过训练计划、生活方式或环境的简单调整进行补救。 |
| **3分** | 无须代偿完成动作 | 展示了执行功能模式所需的灵活性和控制力。 |

**关键认知：**
- 分数争议的重点应该在0分和1分、1分和2分之间，而非2分和3分之间。
- 总分14分是早期研究的风险阈值（7项×3分=21分满分），NFL平均得分14-15分。
- 不对称（如1分/2分、2分/3分）比总分更具临床指导价值。
- 21分满分不代表"完美"——过度灵活（hypermobile）的人可能获得高分，但在极端环境中仍有失败风险。
- 疼痛报告率：约20%接受FMS筛查的人至少在一种模式中出现疼痛。

## 七项动作详解

### 1. 主动直腿上抬（Active Straight Leg Raise, ASLR）

**筛查目的：** 髋关节灵活性、被动伸展性、核心稳定性、下肢分离能力。

**发展意义：** 在发育序列中对应婴儿仰卧时四肢的分化运动。这是最基础的模式之一——如果无法在无负重仰卧位完成此动作，站立状态下更不可能完成。

**评分参考（简化）：**

| 分数 | 标准（以踝骨垂线与对侧大腿的关系为准） |
|------|------------------------------------------|
| 3分 | 踝骨垂线在大腿中点和髂前上棘（ASIS）之间；双侧对称 |
| 2分 | 踝骨垂线在大腿中点和膝盖骨（髌骨）之间 |
| 1分 | 踝骨垂线在膝盖骨以下；或双侧显著不对称 |
| 0分 | 动作过程中出现疼痛 |

**关键代偿信号：** 对侧腿外旋、抬腿时骨盆后倾、屈膝代偿。

**双侧不对称的临床意义：** ASLR的不对称直接揭示髋关节功能不均衡，是下肢损伤的重要风险因素。

---

### 2. 肩部灵活性（Shoulder Mobility）

**筛查目的：** 双侧肩关节活动范围——一侧的伸展、内收加内旋与另一侧的屈曲、外展加外旋的组合。评估肩胛骨稳定性与胸椎灵活性。

**发展意义：** 肩部灵活性模式对应婴儿早期上肢的独立运动能力。

**评分参考（简化）：**

| 分数 | 标准（双拳在背部靠近的距离以"手长"计） |
|------|----------------------------------------|
| 3分 | 双拳间距在一只手长以内 |
| 2分 | 双拳间距在一只手半以内 |
| 1分 | 双拳间距超过一手半 |
| 0分 | 动作过程中出现疼痛 |

**附加排除测试：** 肩部撞击测试——筛查对象将测试侧手放在对侧肩上并尝试抬肘。出现疼痛则记录为阳性并标记0分。

---

### 3. 旋转稳定性（Rotary Stability）

**筛查目的：** 躯干与骨盆在多平面上的反射性稳定控制。反映爬行发展阶段的核心控制能力。

**发展意义：** 对应婴儿四肢着地阶段的躯干反射性稳定模式。

**评分参考（简化）：**

| 分数 | 标准 |
|------|------|
| 3分 | 能完成对侧动作（同侧肘触对侧膝，然后伸展） |
| 2分 | 只能完成同侧动作（同侧肘触同侧膝，然后伸展） |
| 1分 | 无法完成同侧动作 |
| 0分 | 动作过程中出现疼痛 |

**脊柱屈曲排除测试：** 四肢着地后坐回脚后跟，如果出现疼痛记0分。

---

### 4. 俯卧撑（Trunk Stability Push-up）

**筛查目的：** 在封闭动力链中，上肢与下半身在矢状面内的对称稳定性。评估全身张力（"在站起来之前把全身皮肤绷紧"的能力）。

**发展意义：** 对应婴儿悬空姿势（四肢着地）阶段的核心与上肢综合控制。与抬腿、肩部灵活性、旋转稳定性不同，俯卧撑是高阈值运动，不将身体切成左右两半来比较。

**评分参考（简化）：**

| 分数 | 标准（拇指起始位置） |
|------|---------------------|
| 3分 | 男性拇指与头顶齐平；女性拇指与下巴齐平 |
| 2分 | 男性拇指与下巴齐平；女性拇指与锁骨齐平 |
| 1分 | 无法在指定的手位完成 |
| 0分 | 动作过程中出现疼痛 |

**脊柱伸展排除测试：** 俯卧位推起上半身（腰部伸展），出现疼痛记0分。

---

### 5. 直线弓步蹲（In-line Lunge）

**筛查目的：** 下肢对称性、动态平衡、髋/膝/踝在矢状面内的协调能力。

**发展意义：** 对应婴儿开始站立行走后的下肢协调分化。

**评分参考（简化）：**

| 分数 | 标准 |
|------|------|
| 3分 | 在测试板内保持躯干垂直，膝盖轻触板面，足部在矢状面内 |
| 2分 | 能在板内完成，但出现膝盖偏离、足旋前或躯干倾斜 |
| 1分 | 无法保持平衡或膝盖/足部明显偏离，无法完成 |
| 0分 | 动作过程中出现疼痛 |

---

### 6. 跨栏步（Hurdle Step）

**筛查目的：** 单腿站立稳定性与迈步控制（双任务挑战：站立腿的稳定+迈步腿的灵活）。

**发展意义：** 对应行走阶段的单腿支撑与迈步分化。

**评分参考（简化）：**

| 分数 | 标准 |
|------|------|
| 3分 | 在矢状面内无代偿完成跨栏步 |
| 2分 | 在矢状面内但有代偿（骨盆倾斜、躯干侧移） |
| 1分 | 髋/膝/踝在矢状面外，或触碰跨栏线 |
| 0分 | 动作过程中出现疼痛 |

---

### 7. 过顶深蹲（Deep Overhead Squat）

**筛查目的：** 全身对称性、多关节协调、整体动作质量的综合评估。这是最复杂的模式——它集成了其他6种模式的能力。

**早期经验教训：** 格雷·库克承认最初错误地认为所有动作模式都可以在深蹲中体现，"我们假设深蹲是基本的起点...我们错了。" 深蹲依赖于更原始、更简单、更基本模式的完整性。

**评分参考（简化）：**

| 分数 | 标准 |
|------|------|
| 3分 | 大腿低于水平面，上半身与胫骨平行或胫骨垂直，测试杆在足上方 |
| 2分 | 大腿低于水平面但有代偿（脚跟抬起、上半身前倾过多、膝盖内扣等） |
| 1分 | 大腿高于水平面，或无法完成 |
| 0分 | 动作过程中出现疼痛 |

## 筛查应用的核心原则

### 1. 完整筛查不可拆分
筛查不是自助餐——不能挑选某些测试而跳过其他测试。7项测试之间存在关系，形成一个探索同一主题的难度阶梯。像视力表一样，需要确认被筛查者能否辨认第一行，而不是任意选一行测试。

### 2. 双侧测试先测"好"的一侧
在双侧测试中（ASLR、肩部灵活性、旋转稳定性、弓步蹲、跨栏步），先筛查客户更具功能性的一侧。这让他们更可能感知到两侧的不同，增强自我意识。

### 3. 筛查用于检测失败，而非预测成功
FMS检测的是失败的可能性，不是成功的保证。高分数不意味不会受伤；低分数意味失败/受伤的风险更大。NFL中每个运动员都有100%的受伤可能——筛查识别的是那些未达到环境最低要求的人。

### 4. 结果传达策略
完成筛查后，不要立即解释分数。先问："你认为你做得怎么样？"让客户用自己的语言描述体验，然后将他们的感受与客观测量连接起来。

### 5. 不对称比总分更重要
一个18分但有一个0分模式的人，受伤风险可能比14分但所有项目都是2分的人更大。不对称（1分/2分、2分/3分）是需要优先解决的临床目标。

## 实施建议

**教练应用场景：**
- 伤病后恢复训练/运动的现有客户
- 度假或休息后回归训练的客户
- 开始新训练阶段前（赛季期/休赛期/体能期/比赛期）

**临床医生应用场景：**
- 重新评估期间使用FMS
- 用于有慢性损伤或复发性损伤的患者
- 用于准备出院的患者

**逐步实施策略：**
1. 每周做一定数量的筛查——每天1-2次，20天20次（先在非付费人群中练习）
2. 编写你的筛查脚本，精简语言
3. 拍摄记录筛查过程以便回顾
4. 在筛查机制上先追求一致性→有效性→效率
5. 让筛查对象尝试自我筛查和自我评分——他们能描述结果吗？
