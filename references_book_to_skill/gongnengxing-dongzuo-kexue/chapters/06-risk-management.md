# 第6章 — 损伤风险管理与红黄绿灯系统

## 风险公式

> **风险 = 威胁 × 脆弱性**

- 威胁：外部环境中不可控或难以预测的因素（运动强度、比赛密度、环境条件等）
- 脆弱性：个体的内部状态（功能障碍、不对称、不良生活方式、既往伤病史等）

可以通过控制威胁（减少暴露）或减小脆弱性（改善内部状态）来降低风险。脆弱性是更容易被主动管理的变量。

## 已确定的动作风险因素

### 不可改变的风险因素

| 因素 | 说明 |
|------|------|
| 年龄 > 26岁 | 随年龄增长，组织恢复能力下降 |
| 既往受伤史 | 曾经受伤及因该损伤损失的时间 |

### 可改变的风险因素

#### 身体健康与全面健康层

| 风险因素 | 监测方式 |
|----------|----------|
| 运动时出现疼痛 | FMS 0分模式 |
| 自我感知的恢复程度 | 生活方式问卷 |
| 体重指数偏高（BMI） | 体测 |
| 握力不对称或低于年龄标准 | 握力测试 |
| 体力活动减少 | 活动量记录 |
| Y平衡测试不对称 | YBT |
| Y平衡测试综合得分低于标准 | YBT |
| FMS中多项获得1分 | FMS |
| FMS中多项不对称 | FMS |
| 踝关节背屈不对称 | 踝关节活动度测试 |

#### 身体强健层

| 风险因素 | 监测方式 |
|----------|----------|
| 肌肉力量不对称 | 双侧力量测试 |
| 心血管健康状况不佳 | 两英里跑/最大摄氧量测试 |

## 风险因素的累积效应

> "这些风险因素都可用于预测损伤，但起作用的风险因素的数量更为关键。

| 风险因素数量 | 优势比（Odds Ratio） | 相对风险 |
|-------------|---------------------|----------|
| > 5个 | 1.9 ~ 9.6 | 1.5 ~ 2.2 |
| 3-4个 | 1.7 | 1.2 ~ 1.4 |
| 0-2个 | 0.6 ~ 1.2 | 0.7 ~ 1.1 |

**关键信息：**
- 大多数人平时就携带4-5个危险因素（即使没有受伤），这意味着许多人正在向危险靠拢
- 单个风险因素不足以显著预测损伤——累积效应是关键
- 消除任何一个可改变的风险因素，都能降低整体损伤概率

## 红黄绿灯练习决策系统

### 系统概述

> "你的锻炼和活动决策以及患者和客户的决策可以通过一个系统来过滤筛查结果，从而完成智能指导，这个系统类似于交通信号灯系统。"

### 三级分类

| 灯色 | 含义 | 对应FMS分数 | 行动 |
|------|------|------------|------|
| 红灯 | 必须避免 | 0分、1分、不对称1分/2分 | 直接挑战疼痛或功能障碍模式的动作，禁止进行 |
| 黄灯 | 谨慎使用 | 2分、不对称2分/3分 | 不直接挑战功能障碍模式，但需在训练后重新筛查验证 |
| 绿灯 | 可以安全使用 | 3分 | 不会对功能障碍模式产生负面影响 |

---

## 红黄绿灯练习详细分类

### 主动直腿上抬（ASLR）

| 灯色 | 条件 | 练习示例 |
|------|------|----------|
| 红灯 | 0分/1分/不对称1-2分 | 跑步、冲刺、硬拉、高翻、壶铃摇摆 |
| 黄灯 | 2分/不对称2-3分 | 深蹲、蹬台阶 |
| 绿灯 | 3分 | 单膝跪地下劈或上拉、上肢训练、核心训练 |

### 肩部灵活性

| 灯色 | 条件 | 练习示例 |
|------|------|----------|
| 红灯 | 0分/1分/不对称1-2分 | 过头推举、引体向上、倒立、抓举 |
| 黄灯 | 2分/不对称2-3分 | 划船、水平推 |
| 绿灯 | 3分 | 硬拉、高翻、摇摆、下半身训练 |

### 跨栏步

| 灯色 | 条件 | 练习示例 |
|------|------|----------|
| 红灯 | 0分/1分/不对称1-2分 | 跑步、冲刺、单腿练习 |
| 黄灯 | 2分/不对称2-3分 | 对称负重深蹲、硬拉 |
| 绿灯 | 3分 | 单膝跪地下劈或上拉、手提箱硬拉、上肢训练 |

### 过顶深蹲

| 灯色 | 条件 | 练习示例 |
|------|------|----------|
| 红灯 | 0分/1分/不对称1-2分 | 负重深蹲、高翻、抓举 |
| 黄灯 | 2分/不对称2-3分 | 单腿练习、分腿蹲、弓步蹲 |
| 绿灯 | 3分 | 硬拉、土耳其起立、单膝跪地下劈或上拉 |

> 注：1分/2分和2分/3分是指在身体两侧进行的测试中表示不对称运动的分数（抬腿、肩部灵活性、旋转稳定性、弓步和跨步）。

## 黄灯练习的验证方法

黄灯练习需要特别谨慎——它们不会直接挑战功能障碍模式，但可能产生消极影响。

**验证流程：**
```
1. 纠正存在功能障碍的模式
2. 让客户进行黄灯练习/活动
3. 重新测试该模式
    ├─ 功能障碍再次出现？→ 这是一个教学时刻 + 完善纠正策略的机会
    └─ 功能保持？→ 该黄灯练习是安全的
```

## 不损害完整性的原则

> "变形的动作不应该被重复，并且绝对不应该为了某个体能训练的效果而重复做变形的动作。在训练中，我们需要始终努力保持动作的完整性。在功能障碍之上增加练习次数、技能训练或行为，意味着你可能会锁定一个你永远无法摆脱的模式。"

### 训练中的三个容忍度层次

| 场景 | 动作变形 | 是否可接受 |
|------|----------|-----------|
| 比赛中/测试日（最大努力） | 可预料 | 可接受（一次性） |
| 训练日（积累训练量） | 应避免 | 不可接受 |
| 纠正性训练 | 绝对避免 | 完全不可接受 |

### 留有余地的策略

- 训练时间、训练量和训练强度不要同时最大化
- 以质量为优先——"留点余地"
- 不将体能训练的效果建立在变形的动作之上

## 管理生活方式风险因素

### 忽略生活方式的代价

> "世界上最伟大的训练或康复计划无法对抗不良饮食、睡眠不足或一天13个小时的久坐。"

| 生活方式因素 | 对动作的影响 |
|-------------|-------------|
| 睡眠不足 | 损害灵活性、平衡能力、力量和耐力 |
| 脱水 | 降低组织弹性、影响运动控制 |
| 营养不良 | 影响恢复、能量供应和激素水平 |
| 慢性压力 | 触发交感神经优势，破坏动作的自然表达 |

### 交感神经优势的影响

> "在错误的准备状态下进行正确的练习可能只会让某人比以前稍微好一点。消除压力的外围触发因素或学习如何更好地管理这些无益的行为，可以发掘出令人难以置信的动作根基。"

当客户处于交感神经主导状态时：
- 看似简单的纠正练习会让客户感到非常疲劳
- 单脚站立等基本动作可能出现挣扎
- 神经系统不能有效接收训练输入

## 风险管理实施清单

### 评估阶段
- [ ] 收集完整的损伤/手术史
- [ ] 询问自我感知的恢复程度
- [ ] 记录睡眠时长和质量
- [ ] 评估水合作用
- [ ] 进行完整的FMS（不跳过任何模式）
- [ ] 如发现疼痛，记录0分模式位置
- [ ] 识别所有不对称

### 保护阶段（立即行动）
- [ ] 标示所有红灯练习，告知客户暂时避免
- [ ] 标示所有黄灯练习，纳入验证流程
- [ ] 针对可改变的生活方式风险因素给出明确建议
- [ ] 设定睡眠/水合/营养的最低标准

### 纠正阶段（持续监控）
- [ ] 在训练课后重新筛查功能障碍模式
- [ ] 每周追踪生活方式的变化
- [ ] 积累至少2周无负面变化的模式，再升级到发展
- [ ] 重新评估风险因素数量是否减少

### 教育阶段
- [ ] 教导客户如何自我筛查和识别自己的"红灯"
- [ ] 让客户理解不训练某个模式 ≠ 不训练
- [ ] 解释功能障碍模式与疼痛之间的关系
- [ ] 让客户认识到他们的风险因素数量

## 沟通高风险信息的策略

> "告诉人们应该停止训练或锻炼，需要他们暂时搁置一些事情...可能是一个敏感的话题。"

### "不放弃，只替代"的沟通策略

不要说："你不能做深蹲了。"
而是说："我们可以暂时把深蹲换成硬拉和土耳其起立，同时我们一起解决你深蹲中的灵活性问题。一旦你能够无代偿地完成深蹲，我们就可以安全地重新加入训练计划。你认为需要多长时间能看到变化？"

### 客户承诺请求
> "你能承诺暂时不健身锻炼，而是和我一起做其他可能产生更大影响的事情吗？我能否先将你的健身目标放一边，制定一个一到两周的纠正策略，然后再重新评估？"

## 预防不良康复

> "如果我们不能防止受伤，让我们问问自己：我们能防止不良的康复吗？我们能防止不佳的运动前体检吗？我们能阻止人们加入高失败率的活动吗？我们能设计出更好的训练计划并提供更好的策略来帮助客户独立识别和管理其薄弱环节吗？"

答案是：我们能。
