# 第5章 — 三大动作原则与功能性训练理论

## 三大动作原则总览

格雷·库克经过30年实践，将功能性动作系统的哲学基础提炼为三个原则：

| 原则 | 类别 | 一句话表述 |
|------|------|----------|
| 原则一 | 自然原则 | **首先动作好，然后常运动** |
| 原则二 | 道德原则 | **保护、纠正，然后发展** |
| 原则三 | 实践/文化原则 | **创建系统来支持理念** |

> "如果你相信原则一，请用原则二来尊重它。要按照原则二采取行动，请实施原则三。"

---

## 原则一：首先动作好，然后常运动（自然原则）

### 为什么这个原则起作用？

自然原则源于对人类发育过程的观察。无论气候、文化或家庭状况如何，婴儿在生命前21个月都遵循相同的发育路径：

```
头部控制 → 翻滚 → 爬行 → 攀爬 → 跪下 → 蹲下 → 站立 → 迈步 → 行走 → 奔跑
```

每一个动作的发展建立在前一个动作的基础上，为后续身体功能奠定基础。这些是"预装的操作系统"——人类动作的基本软件。

### 与常见健身理念的根本冲突

> "将数量放在质量之前的锻炼模式颠倒了第一原则，这通常等于在功能障碍的基础上实现身体强健——人们经常运动，并期待着好的动作会随之而来。但这种情况很少出现。"

**错误的假设链：**
```
更多锻炼 → 更强壮/更快 → 身体更好
```

**正确的顺序：**
```
先确认基础动作质量达标 → 再增加运动量/强度/复杂度
```

### 三大支柱概念

1. **我们无法像自然一样更好地发展自己或他人**
   - 自然环境提供丰富多样的动作刺激
   - 最健康的人通常不需要"锻炼器材和教练"

2. **我们可以比自然更安全、更快速地发展自己和他人**
   - 通过保护和策略性干预，在不制造伤害的前提下加速适应
   - "正确的发展是在到达下一个发展水平之前，在当前所处发展水平上表现出良好的行为"

3. **适应来自对强加需求的特定适应（SAID原则）**
   - 但该原则仅在系统有足够的基础能力时才能安全应用

### 定义"动作好"

世界卫生组织（WHO）为不同年龄制定了心率和血压等健康指标的正常范围，但对于"什么是健康的人类运动"的协议仅适用至21个月大的婴儿。FMS的7项动作模式正是基于这一发展进程设计的。

> "动作好"意味着能够对接收的环境信号做出适当的反应，然后进行渐进的动作学习和成长。动作能力（动作好）优先于动作体能（常运动）。

## 原则二：保护、纠正，然后发展（道德原则）

### 道德原则的内涵

源于希波克拉底誓言的核心——"首先，不要造成伤害"。应用于动作领域：

```
保护 (Protect)  →  纠正 (Correct)  →  发展 (Develop)
     ↓                  ↓                  ↓
消除负面影响      恢复功能基线      增强与维持适应
```

### 保护：优先消除负面因素

> "在大多数情况下，答案来自环境和行为。消除负面影响，然后用积极的活动或环境填补空白，这是促进和发展健康的唯一真正解决方案。"

**可以消除的负面因素：**
- 睡眠不足/质量差
- 脱水（每天少于足够的水摄入量）
- 加工食品和不良饮食习惯
- 引发疼痛或功能障碍的训练动作
- 慢性压力
- 久坐不动的生活方式

### 纠正：从功能障碍恢复到最低标准

纠正不是在功能障碍之上添加更多练习，而是：
1. 通过暴露问题来增加体验
2. 创造一种让客户自己"发现"问题的情境
3. 用可衡量的变化来验证纠正的有效性

> "我试图传达通过行动来创造意识的过程...我们不仅要揭露一个存在功能障碍的动作，然后对此进行训练，我们还要参与其中，与被纠正者共同创造一种学习体验。"

### 发展：在功能基础上构建体能

- 只有功能层达标，才能安全地追求体能和技能
- 发展意味着给动作模式增加负荷、速度、复杂度
- 持续的监控确保发展不损害基础完整性

### 三大不可协商事项

在原则一和原则二之下，三个根本命题构成了所有哲学、程序或系统的基石：

1. 我们无法像自然一样更好地发展自己或他人
2. 我们可以比自然更安全、更快速地发展自己和他人
3. 正确的发展是在到达下一个发展水平之前，在当前所处发展水平上表现出良好的行为

## 原则三：创建系统来支持理念（实践/文化原则）

### 为什么需要系统？

> "良好的流程和系统所形成的建筑结构恰到好处，可以提高自由度。它们的目标首先是通过设置护栏和检查点为你的工作提供保护，其次是确认你行动的有效性。"

### 系统的功能

1. **自我检查** — 如果操作得当，系统会进行自我检查（像视力表和血压计一样，无论在什么环境下使用，功能几乎都相同）
2. **防止偏见** — 算法思维帮助我们构建系统来收集可靠、客观的数据，消除证实偏见
3. **提供反馈回路** — 测试→行动→重新测试，创造客观的反馈循环
4. **保护创造力** — 系统不是限制性的教条，而是提供了安全操作的基础，允许在边界内创新

### 标准操作程序（SOP）在动作中的应用

对照飞行员的航前检查清单——标准操作程序旨在：
- 通过清单对照法降低风险
- 提高运动表现
- 最大限度地减少差异和误解
- 确保问题尽早被发现
- 在遇到麻烦时可以回溯线索

### 测试→行动→重测模型

```
┌───────────────────────────────────────────────────────┐
│  1. 执行筛查或测量                                      │
│     ↓                                                 │
│  2. 采取所需的行动（干预）                               │
│     ↓                                                 │
│  3. 重新筛查或重新测量                                   │
│     ↓                                                 │
│  4. 看到变化了吗？                                      │
│     ├─ 是 → 如果变化积极，继续；如果变化负面，换方案      │
│     └─ 否 → 我测量的对吗？提供的战术正确吗？需要更长时间？  │
└───────────────────────────────────────────────────────┘
```

## 整体动作模式：策略→战术

### 策略优先于战术

> "在健康和健身的世界里，我们花了过多的时间来讨论战术，却没有先理解或明确策略。我们争论哪些测试更敏感或更具体，哪些手动治疗的效果最好...一直以来，我们都假设所有能知道的都已经知道了。"

| 概念 | 定义 | 动作领域的例子 |
|------|------|--------------|
| **策略** | 获得预期结果的行动计划 | "先确保髋关节灵活性，再训练深蹲深度" |
| **战术** | 用于支持策略的每个具体行动 | "泡沫轴放松髋屈肌+PNF拉伸+高脚杯深蹲" |
| **原则** | 指导策略的不可动摇的真理 | "首先动作好，然后常运动" |

### 流水线厨师 vs 主厨

- **流水线厨师**：知道怎么做，按食谱操作，技能有限但可重复
- **主厨**：了解烹饪原则，知道为什么和怎么做，能根据环境变化调整策略

> "烹饪的艺术并非来自随意选择食材或方法，而是来自理解为什么和明白怎么做，然后根据在烹饪参数范围内的观察、品尝和体验做出决定。"

## 功能性动作系统的整体框架

### 动作与健康的四层关系

```
                    ← 复杂度递增 →
                    ┌─────────────┐
生产力 (技能)        │ FCS + 专项  │  ← 为特定目标精确移动
                    ├─────────────┤
身体强健 (体能)      │ FCS         │  ← 能力与环境匹配
                    ├─────────────┤
全面健康 (功能)      │ FMS         │  ← 动作能力/风险因素
                    ├─────────────┤
身体健康 (结构/生理) │ SFMA        │  ← 无疼痛/结构完整
                    └─────────────┘
                    ← 基础性递增 ↓
```

每一层都支持和强化下一层。任何一层的缺陷都会限制更高层的发展。

### 功能作为切入点

> "功能适用于每一个负责掌管身体潜能的医疗保健、健身和运动表现领域的专业人士...如果使用功能作为切入点，就需要一个测试系统来建立基线——每一层的最低接受水平。"

FMS填补了健身（身体强健）和健康（身体健康）之间的空白——它关注的是功能问题，提供了一个环境让客户在过早进入健身锻炼或医疗保健系统之前解决问题。

## 彻底透明 + 算法思维

这两个原则来自雷·戴利奥（Ray Dalio），是功能性动作系统的方法论核心：

| 原则 | 含义 | 在FMS中的应用 |
|------|------|-------------|
| **彻底透明** | 寻求可衡量的真相、沟通和问责 | 用客观评分标准消除主观判断；不跳过任何筛查模式 |
| **算法思维** | 构建系统收集可靠数据，指导决策模型 | 优先级序列、分解逻辑、红黄绿灯系统 |

> "房间里最客观的人是那些知道自己不客观并创建反馈系统来管理自己专业知识的人。"

---

## 功能性训练的最终目标

功能性动作系统的真正目标不是揭示动作问题然后分配练习来纠正它们，而是：

> "让专业人士和客户都认识到基本动作在维持生活的各个方面的重要性，以及生活的各个方面会如何影响动作，然后让他们通过体验式学习的方式来逐步学会欣赏，获得整合，而不是用嘴去谈论它。"

**最终追求的成果：**
1. 自我意识——知道自己的动作能力边界
2. 独立性——能够自我监控和管理身体状态
3. 适应性——在面对各种环境压力时保持功能完整性
4. 可持续性——不依赖专业人士就能维持动作健康
