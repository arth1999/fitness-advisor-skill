# 第13章 — 高强度间歇训练（HIIT）的基本方法与计划设计

## 核心框架

HIIT是短时间内交替进行高强度运动和低强度恢复或休息的训练方法。本章阐述HIIT的科学原理、各种形式及对不同人群的应用建议。

## 一、HIIT的基本概念

### 定义
高强度间歇训练是指在达到或超过无氧阈/最大摄氧量的强度下，进行反复短时间的运动，期间穿插低强度恢复或完全休息。

### HIIT的核心优势
- **时效性**：短时间（≤30min）即可获得与持续性训练相似甚至更好的有氧和代谢适应
- **多样性**：可适用于跑步、自行车、划船、自重训练、负重训练等多种运动形式
- **双重适应**：同时刺激有氧系统和无氧系统

### 生理效应
- VO₂max ↑（与持续性训练效果相当）
- 线粒体密度和氧化酶活性 ↑（有氧适应）
- 糖酵解酶活性 ↑（无氧适应）
- 胰岛素敏感性 ↑（代谢健康效益）
- 脂肪氧化能力 ↑（24h静息代谢升高）

## 二、HIIT的形式与方法

### 按强度分类

**1. 有氧HIIT（Aerobic HIIT）**
强度：90%-100% VO₂max（或85%-95% HRmax）
工作:恢复 ≈ 1:1（如4min快+4min慢 × 4组）
总时长：20-40min
适用：有氧基础较好的运动员

**2. 无氧HIIT（Anaerobic HIIT/SIT）**
强度：>100% VO₂max（全力/近乎全力）
工作:恢复 = 1:3至1:6（如30s全力+3-4min恢复 × 4-6组）
总时长：10-20min
适用：发展无氧能力和爆发力

**3. 短间歇HIIT（Short Interval HIIT）**
工作期≤30s，恢复期≤30s（1:0.5至1:1）
如：Tabata（20s全力+10s休息 × 8循环 = 4min）
适用：在极短时间内获得高代谢刺激

### 按运动形式分类
- **跑步HIIT**：跑道/跑步机间歇
- **自行车HIIT**：功率自行车/动感单车（可精确控功率）
- **划船机HIIT**：全身参与，低冲击
- **自重HIIT**：波比跳、登山者、高抬腿、蹲跳等组合
- **负重HIIT**：壶铃甩摆、哑铃抓举等组合（力量+有氧融合）

## 三、HIIT的计划设计

### 设计要素
| 要素 | 选择 |
|------|------|
| 运动形式 | 根据目标（专项性vs一般体能）+可得器材 |
| 工作期强度 | 80%-100% HRmax 或 RPE 8-10 |
| 工作期时间 | 10s-5min |
| 恢复期强度 | 被动（完全休息）或主动（30%-50% HRmax） |
| 恢复期时间 | 视目标：有氧HIIT 1:1；无氧HIIT 1:3-6 |
| 循环次数 | 4-12（根据强度和时间调整） |
| 频率 | 1-3次/周（与其他训练配合） |

### 训练周期中的HIIT整合
- **基础准备期**：以有氧HIIT为主，强度相对较低，建立代谢基础
- **专项准备期**：加入无氧HIIT，缩短工作-恢复比例
- **竞赛期**：减少HIIT频率，降低量但保持强度
- 注意：HIIT不应与高强度力量/爆发力训练同日进行（至少间隔6h）

## 四、不同人群的HIIT建议

**运动员**
- 2-3次/周，作为持续训练和专项训练的补充
- 根据项目特征选择HIIT形式（球类项目常用短间歇HIIT模拟比赛节奏）

**普通健身人群**
- 1-2次/周即可获得显著健康收益
- 初始选择较低强度、较长工作/较宽恢复比（如1min快+2min慢）
- 循序渐进——切勿在第一次训练就尝试极量HIIT

**超重/肥胖人群**
- 选择低冲击形式：自行车、椭圆机、划船机、水中运动
- 减短工作时长、增加恢复时长(1:2-1:3)
- HIIT结合饮食管理的减脂效果显著

**老年人**
- 必须有足够的健康筛查
- 选择低冲击形式，工作期RPE 6-8（非极量），恢复期RPE 3-4
- 频率≤2次/周
- 更多侧重健康效益（代谢健康、心血管保护）而非竞技表现

**中国HIIT应用现状**
HIIT在中国大众健身领域近年来迅速普及——"Tabata"、"HIIT燃脂"成为热门词汇。但在竞技体育领域，HIIT的系统化应用仍在发展中，常作为传统耐力训练的补充而非主训手段。
