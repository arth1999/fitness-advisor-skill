# 第12章 — 耐力训练的基本方法与计划设计

## 核心框架

耐力是长时间维持特定运动强度的能力。本章涵盖有氧耐力和无氧耐力的训练原理、方法和计划设计。

## 一、耐力的分类与基本原理

### 耐力分类矩阵
|  | 有氧耐力 | 无氧耐力 |
|------|---------|---------|
| 能量系统 | 有氧氧化 | 糖酵解(+磷酸原) |
| 强度区间 | <乳酸阈 | >乳酸阈 |
| 持续时间 | >3min | 30s-3min |
| 代表项目 | 长跑/自行车/赛艇 | 400m/800m/100m游泳 |

### 有氧耐力的决定因素
- **最大摄氧量（VO₂max）**：有氧能力的上限（中枢因素+外周因素）
- **乳酸阈（无氧阈）**：可持续强度的分界线——乳酸阈处的%VO₂max越高，有氧表现越好
- **运动经济性**：给定速度下的能耗——氧耗越低，经济性越高
- **底物利用**：脂肪氧化比例越高，糖原节省化越好
- **肌纤维类型**：I型纤维比例

### 无氧耐力的决定因素
- 糖酵解能力和速率
- 缓冲能力（H⁺耐受与清除）
- 磷酸原系统恢复速度（组间/间歇恢复）

## 二、耐力训练的基本方法

### （一）持续性训练法
无间歇地长时间持续运动。

- **LSD（长距离慢速训练）**：≤70% VO₂max或60%-70% HRmax，持续≥30min，目标：提高脂肪氧化能力+毛细血管密度
- **中等强度持续训练**：70%-85% VO₂max或70%-85% HRmax，20-60min，目标：提高VO₂max和乳酸阈
- **高强度持续训练**：85%-95% VO₂max或85%-95% HRmax，15-30min，目标：乳酸耐受和清除（接近阈值训练）

### （二）间歇训练法
在训练中穿插间歇（恢复期），使运动员能在较高强度下完成更大的总训练量。

- **有氧间歇**：工作期2-5min，强度85%-100% VO₂max；恢复期=工作期（1:1），主动恢复
- **无氧间歇**：工作期30s-2min，强度>100% VO₂max；恢复期=1:3至1:6，被动或主动恢复
- **短间歇**：工作期≤30s，恢复期15-30s（1:0.5至1:1）——提高重复冲刺能力

### （三）重复训练法
以竞赛速度或更高速度进行短段训练，恢复充分（心率<120bpm或≥5min）。

### （四）法特莱克（变速训练）
在自然地形中进行自发性变速跑——结合持续训练和间歇训练的优点。

## 三、耐力训练计划设计

### 训练变量
| 变量 | 有氧耐力 | 无氧耐力 |
|------|---------|---------|
| 训练频率 | 3-6次/周 | 1-3次/周 |
| 每次训练时间 | 30-120min | 15-45min |
| 强度 | 60%-95% HRmax | >85% HRmax |
| 周总量 | 视项目需求 | 视项目需求 |

### 周期安排（以耐力项目为例）
- **准备期（基础）**：LSD+中等强度持续跑为主，建立有氧基础
- **准备期（专项）**：阈值训练+有氧间歇，提高乳酸阈和VO₂max
- **竞赛前期**：高强度间歇+重复训练+竞赛配速训练
- **竞赛期**：维持为主，减少训练量+保持强度
- **过度期**：积极恢复（LSD+交叉训练）

### 极化训练模型（Polarized Training）
约80%训练时间为低强度（<乳酸阈），约20%训练时间为高强度（>>乳酸阈），极少中等强度训练。这一模式在多国耐力运动员中显示出优越效果。

### 金字塔训练模型
低强度占大部分→中等强度占中间→高强度占顶部。传统耐力训练的常用模式。

## 四、能量系统训练的目标心率区
- **恢复/热身**：<60% HRmax
- **LSD/基础耐力**：60%-70% HRmax
- **中等强度/阈值训练**：70%-85% HRmax
- **高强度**：85%-95% HRmax
- **最大强度**：>95% HRmax

心率储备（HRR）= HRmax - 安静心率。靶心率 = %HRR + 安静心率。

## 五、中国耐力训练特点

中国传统耐力训练受苏联"大运动量"理念影响深远——偏好**高训练量**（每周跑量很大）+**高原训练**（昆明/多巴等高原基地是重要训练据点）。近年来逐渐融合西方"两极化训练"理念，在保持量的同时更加注重强度分布的合理性。中国竞走/中长跑项目在奥运舞台的成绩变化可反映这一演化过程。
