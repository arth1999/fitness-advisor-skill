# 第二章第一节: 运动系统概述

## 核心思想
运动系统由**骨(杠杆)、骨连结/关节(枢纽)、骨骼肌(动力)** 三者组成，约占成人体重60%-70%。三者互相制约、互相依存、缺一不可。全身206块骨，600余块骨骼肌，四肢肌占全身骨骼肌80%(下肢占50%)。

## 骨的框架
### 骨的分类
- **长骨**: 长管状(两端膨大骨端+中央骨体)，分布于四肢，运动中以杠杆作用为主
- **短骨**: 立方形成群分布，分布于手腕/足踝部，分散压力、灵活运动
- **扁骨**: 板状面积大，分布于颅骨/胸骨，保护脏器、提供肌肉附着面
- **不规则骨**: 形状不规则如椎骨；含气骨如上颌骨
- **籽骨**: 肌腱内钙化小骨(如髌骨)，保护肌腱、改变肌牵引方向、增大力臂

### 骨的结构
- **骨膜**: 外膜(外层致密坚韧+内层有成骨/破骨细胞与修复有关)；内膜(衬骨髓腔内面)
- **骨质**: 骨松质(骨小梁交织，排列与压力/张力方向一致，可适应性改变)；骨密质(哈佛氏系统/骨单位是最坚固的支持结构)
- **骨髓**: 红骨髓有造血功能

### 影响骨生长的因素(运动相关)
1. 机械力因素: 骨形态和骨量随受力情况不断自我调整——经常受肌牵拉的部位(如胫骨粗隆)明显突出
2. 激素、营养(蛋白质+维生素C/D)、遗传
3. **关键应用**: 儿童青少年(17-20岁前骨骺未闭合)进行适宜运动可促进骺软骨增殖分裂，有利于身高增长

## 关节的框架
### 关节三要素(基本结构)
- **关节面**: 一凹一凸，覆有关节软骨(2-7mm)，减少摩擦、缓冲震动
- **关节囊**: 外层纤维层(致密结缔组织，厚薄与负重有关——下肢厚而紧、上肢薄而松)；内层滑膜层(分泌滑液)
- **关节腔**: 密闭腔隙，内含少量滑液，呈负压状态维持关节稳定

### 关节分类(按运动轴数目)
| 类型 | 关节举例 | 运动轴数 |
|------|----------|----------|
| 单轴关节: 滑车关节 | 指间关节 | 1(冠状轴，屈伸) |
| 单轴关节: 圆柱/车轴关节 | 桡尺近/远侧关节 | 1(垂直轴，旋转) |
| 双轴关节: 椭圆关节 | 桡腕关节 | 2(屈伸+内收外展) |
| 双轴关节: 鞍状关节 | 拇指腕掌关节 | 2(屈伸+内收外展+环转) |
| 多轴关节: 球窝关节 | 肩关节/髋关节 | 3(屈伸+外展内收+旋转) |
| 多轴关节: 平面关节 | 腕骨间关节 | 3(微动) |

### 关节五种运动形式
1. **屈与伸**: 绕冠状轴，矢状面内运动。通常向前为屈、向后为伸。特例: 小腿/足向前为伸(背屈)，向后为屈(跖屈)
2. **外展与内收**: 绕矢状轴，冠状面内运动。远离正中面为外展，靠近为内收
3. **旋内与旋外**: 绕垂直轴，水平面内运动。由前向内侧旋转为旋内(旋前)，向外侧为旋外(旋后)
4. **水平屈与水平伸**: 上臂/大腿外展90°后在水平面内的运动
5. **环转**: 绕冠状轴、矢状轴及中间轴连续圆周运动。具有冠状轴和矢状轴的关节均可

### 影响关节运动幅度的6个因素
1. 关节面面积差(差越大→幅度越大，如肩关节)
2. 关节囊厚薄和松紧度
3. 韧带多少与强弱(多而强→稳固性好→幅度小)
4. **关节周围骨骼肌的伸展性和弹性**(核心训练目标)
5. 关节周围骨结构(骨性突起小→幅度大)
6. 年龄/性别/运动训练(儿童>成人，女性>男性，训练者>非训练者)

## 骨骼肌的框架
### 训练相关核心概念
1. **肌拉力线**: 起点中心到止点中心的连线。从关节冠状轴前方通过→屈；后方通过→伸。从矢状轴外侧通过→外展；内侧通过→内收。
2. **近固定与远固定**: 近固定=起点相对固定(如哑铃弯举)；远固定=止点相对固定(如引体向上)。这是设计训练动作时判断哪个关节在运动的关键。
3. **生理横断面**: 所有肌纤维横断面之和，是决定肌力最重要的解剖学因素。绝对力=生理横断面×比肌力(男9.2kg/cm²，女7.1kg/cm²)
4. **肌初长度**: 生理范围内初长度越长→收缩加速度越大→力量越大。预拉伸可增力(如足背屈60°时小腿三头肌力量从384kg增至598kg)
5. **肌纤维类型**: 红肌(慢缩/Ⅰ型)→有氧代谢，耐力强，不易疲劳；白肌(快缩/Ⅱ型)→无氧酵解，力量大速度快，易疲劳。短跑运动员白肌70-90%，马拉松运动员红肌约90%

### 骨骼肌的物理特性
- **伸展性与弹性**: 可被动拉长及恢复原长。适度提高→增大动作幅度、增强柔韧性
- **黏滞性**: 肌肉内部阻力。温度越低黏滞性越大。**热身运动(准备活动)可升温减少黏滞性，降低损伤风险**。冬季运动特别需要重视准备活动

### 发展力量素质的6条训练原则
1. **针对性训练原则**: 动作结构(近/远固定)、肌拉力方向、工作类型须与专项一致
2. **抗阻力训练原则**: 极限负荷和超量负荷是发展肌力的基本原则——可动员更多肌纤维
3. **匀速训练原则**: 控制运动速度，降低阻力的速度比提升阻力的速度更重要
4. **全幅度训练原则**: 从全屈到全伸，使各部分肌纤维都参与工作
5. **全面训练原则**: 各部肌肉均衡发展，避免肢体畸形
6. **核心力量训练原则**: 肩关节以下、髋关节以上的中间区域，负担稳定重心、传导力量

### 发展柔韧素质的训练原则
- **准备活动原则**: 温度升高→黏滞性降低→拉伸效果好且不易受伤
- **循序渐进原则**: 长期阶段性训练，先易后难，方法柔和，切忌快速用力
- **静力拉伸为主、动静结合**: 达到"酸、胀、痛"程度停留一段时间(约30秒)
- **柔韧与力量相结合**: 单纯力量会降柔韧，单纯柔韧影响稳固性
- **利用多关节肌"被动不足"**: 在一个关节已拉长后，利用其他关节运动进一步拉伸

## 心智模型
1. **杠杆-枢纽-动力模型**: 分析任何动作时，先问三个问题: 哪个骨做杠杆? 哪个关节是枢纽? 哪些肌肉提供动力? 三者缺一则运动不可能发生。
2. **力线-轴线关系模型**: 要判断某块肌肉使环节做什么运动，只需看肌拉力线从关节轴的哪一侧通过。

## 关键启示
1. 儿童青少年是柔韧素质训练的"黄金时期"——骨有机物多、可塑性大、肌纤维伸展性好
2. 热身不是可有可无——黏滞性的温度依赖性是解剖学铁律
3. 比肌力是常数→增加绝对力的唯一途径是增大生理横断面(肌肉肥大)
